When a drive fails, the "Failure" field is set to "1" on the day it fails. The next day, thedrive isremoved from the list and is no longer counted, reducing the overall number of drives. Onthe other hand,new drives are added on a regular basis increasing the overall number of drives. In otherwords, count thenumber of drives each day.

当一个驱动器失败时，＆＃34;失败＆＃34;字段设置为＆＃34; 1＆＃34;在它失败的那一天。第二天，从列表中举行的媒体，不再计算，减少了整个驱动器的数量。另一方面，在定期增加新的驱动器，增加了驱动器的总数。换句话说，每天都有驱动器的数量。

Each day a drive is listed in a daily snapshot file it counts as one drive day. For example,if there are35,000 drives listed in a daily snapshot file that equals 35,000 drive days.

我第一份开发人员工作的超现实体验

2021-08-05 21:01:44

大约十年前，我获得了数学和计算学位，对从事数学和编程的职业抱有浓厚的兴趣，但不知道如何去做。首先也是最重要的是，我决定在大学毕业后留在曼彻斯特，而不是在我搬回家后冒着被困在父母那里的风险。所以我和一个大学的朋友租了一套公寓，开始找工作。我花了很多时间在大学学习 Web 开发——不是从课程中学习，而是自学，因为课程远远落后。我很高兴使用 LAMP 堆栈制作网站，我觉得自己很有能力，而且我知道曼彻斯特地区有很多这样的工作——所以我开始申请“Web 开发人员”和“PHP 开发人员”以及各种类似的职位的本地公司。我的第一次面试是在斯托克波特的一家小公司做一般开发人员的工作——这份工作需要 Microsoft Access 经验，出于某种原因，这仍然是我认为自己拥有的东西，并且愿意向我所拥有的其他人承诺，这让我感到困惑我现在。我最终在面试中与公司老板进行了非常愉快的交谈。他问了一些关于数据分析的很棒的面试问题，我们花了大约 90 分钟的时间交谈——他对我在大学学习的东西很着迷——特别是数据可视化和动态系统与混沌。我确实得到了这份工作，但他为时已晚——我已经进行了第二次面试，他们要求我立即开始。我的第二次采访是在索尔福德一个老式工业区的办公楼里的一家名为 App Start 的公司（如果你把媒体城想象成索尔福德，索尔福德的其余部分会让人震惊）。由于低估了从火车站步行的距离，我一直在拖延时间，而我因匆忙赶到并试图找到这个地方而有点慌张。迎接我的是一个穿着沙滩裤和人字拖的男人。我穿着全套西装、衬衫、领带和漂亮的鞋子，我告诉他“我是来采访克里斯的”——他的回答“哦，克里斯不再在这里工作了”应该引起警报钟声，但是嘿，我年轻而天真。穿着人字拖的亚历克斯现在似乎掌权了。他问我申请了哪个职位——我说“网络开发人员”，他邀请我加入。大约有 15 到 20 人在台式 PC 上工作。有一个大沙发。地板是裸露的混凝土。在写有密码的白板之间，你可以看到墙壁上的涂鸦。在“面试”中，他试图在 Chris 的收件箱中找到我的简历，我向他展示了我制作的一些网站。他问我是否可以第二天来，做一天的工作作为试验——我照做了。第二天是星期五——我似乎记得建立一个社交网站的原型——虽然我不知道为什么。无论如何——他似乎认为我足够好所以他给了我这份工作——从周一开始——这对我来说很棒——我马上就找到了一份全职工作，这意味着我可以支付房租并开始我的职位- 在曼彻斯特的大学生活。但我一接受，他就问：“你明天有空吗？” ——我犹豫了一下，但还是回答了这个问题——“是的”。 “你能进来吗？”因为我不知道还能说什么，所以我说是的。同样，这应该是一个标志。业主是一位名叫安德鲁的年轻企业家，我听说他通过他的房地产开发和管理公司 Fresh Start Living 赚了数百万美元。他决定进入“应用程序”市场（请记住，这是 2011 年——iPhone 应用程序非常庞大），因此他决定给他的 IT 人员 (Alex) 一笔预算，聘请一些开发人员来制作大量应用程序。其中一个肯定会取得巨大的成功，他会赚到数百万美元。星期六我又回去了，发现办公室里没有任何人。我等了一会儿，但不知道该怎么办。我没有亚历克斯的联系电话。过了一会儿，他出现了。他一直在大楼的Fresh Start Living一侧。我们在办公室独自完成了一天的工作。他送我回家，我原定在星期一开始真正的工作。

我周一到达，并被告知我将从事的项目。简短的版本是我被告知我们必须在任何人回家之前完成它。我们一直待到晚上 9 点。那是我成为全职员工的第一天。所以在我的第一天，亚历克斯告诉我我将从事的项目叫做技能购买，并继续解释这个概念。这个想法类似于当时流行的拍卖网站，您可以对产品出价，但每次出价都需要付费——因此，如果您看到中标的 iPhone 为 1 便士，您可以出价 2 便士，但如果您出价高，你就失去了 2 便士——最终有人会以便宜的价格赢得它，而牺牲在他们之前出价的每个人的利益，公司就会发财。它们被称为便士拍卖，您可以在本文中了解它们：在线便士拍卖是一种赌博形式吗？无论如何——我被告知的想法是相似的，但你不是竞标，而是玩游戏（由我合作的游戏开发者制作的 Flash 或 Unity 游戏）——它就像一个街机游戏——获胜者得到奖品。您会购买积分，将积分用于玩游戏，如果您在比赛倒计时结束时成为最佳射手，那么您将赢得奖品。实际上，您会以 99% 的折扣赢得奖品，因为他们似乎认为这绕过了赌博法。在没有 Web 开发人员的情况下，实现这个想法非常棘手，Alex 一直在使用他自己的万事通技能将一些东西组合在一起。他有一些管理电子商务网站的经验（后来我发现他经营着自己的在线情趣用品店……），因此他使用 PHP 电子商务框架 Magento 组装了一个 Skill Buy 原型。他会在网站的管理区域添加产品（奖品），并修改模板以显示倒数计时器和东西。不知何故，亚历克斯承诺该网站将准备好在第二天向安德鲁展示 - 下午的某个时候，他开始感到压力，并对整个房间说“在完成之前没有人回家”。其中一位游戏开发者在 Unity 中制作了一个目标射击练习游戏，我们必须弄清楚如何将其嵌入网站并让游戏将玩家的分数提交给网站的 PHP 后端。我们不知道从哪里开始，但我们整个下午都在努力，一直到晚上，我们让它按照亚历克斯要求的任何标准工作，最后回家了。在接下来的几周里，我在这个网站上工作，我厌倦了被困在这个电子商务框架中——不得不使用它的超规范化 MySQL 数据库（EAV 模型），而且我正在构建的产品是这不是一家真正的商店——所以我问亚历克斯他是否介意我写一些定制的东西。我让自己的自定义 PHP/MySQL 站点快速启动并运行，不久我们就启动了该站点。我很高兴编写蹩脚的 PHP，实施可疑的安全保护（阻止人们发送虚假分数的微弱尝试），并学习诸如如何集成 Facebook 登录、PayPal 支付、使用 AJAX 以及与游戏开发人员合作以集成评分系统。我真的很喜欢在那里工作一段时间！这是一种解放，知道我正在按照自己的条件使用自学的网络开发技能 - 并因此获得报酬。这家应用程序公司由亚历克斯和一个叫利亚姆的 19 岁孩子经营，他的叔叔参与了房地产业务。不知何故，利亚姆获得了一项实验性的副业业务，以帮助他在大学的“实习年”。可能让您感到奇怪的一件事是奇异的图形。设计师为计划中的 iPhone 游戏制作了所有这些牛在场的图形 - 但他们搁置了它并告诉她在网站上重新使用它们。她讨厌他们被这样使用——这对其他人来说也没有意义。亚历克斯经常问你周六是否有空。前几次我说是的，只是多工作了一天——我想我认为我会自动获得加班费——但我没有。最终我学会了说不。

我加入后不久，一个叫 Bob 的人出现并担任了某种产品经理的工作，类似于 Alex 和 Liam。和利亚姆一样，他看起来还不错，但有点格格不入。他是亚历克斯的朋友。他曾经整天盯着谷歌分析，然后过来告诉我关于退回时间和推荐的事情。他过去常常在午饭后离开，无论出于何种原因，他赢得了“半职鲍勃”的绰号。不过他并没有坚持多久。一天，他正在与安德鲁会面，他显然已经看穿了他并且受够了。我开始发现，当安德鲁决定他不喜欢某人，而不是解雇他们时，他会尝试一下，让他们感到非常生气，他们想要离开。鲍勃走进门，把门砰地关在身后，大声自言自语“他不会那样跟我说话”，然后向房间宣布他“不再听安德鲁的狗屎了”，然后冲了出去。我们推出了这个网站，他们开始投放 Facebook 广告来宣传它。亚历克斯甚至订购了大量印刷材料和标志。我记得有一次，在送我回家的电梯时，他停下来下车，在一组交通信号灯的栏杆上用缆绳把一条技能购买横幅绑在了栏杆上。我们有人玩——注册后你会得到一些免费积分，有些人实际上也在购买积分。更便宜的噱头奖品的时间很短，我们有获奖者领取奖品——利亚姆把奖品寄出去了。我们在那里也有一些大奖，比如劳力士手表、一年的免费住宿和房子。这个想法是，如果网站获得足够的收入，安德鲁可以放弃使用 Fresh Start 的一些出租物业甚至房子。住宿和房子的时间为 3 个月，这肯定足以建立网站用户群。过了一会儿，安德鲁决定做出改变。您无需购买积分，而是购买游戏门票，只有当足够多的人购买门票时，游戏才会开放，以保证收入足以支付奖品成本。这基本上停止了网站上的所有活动。人们没有玩游戏和争夺奖品，他们只是在等待，几乎没有人有兴趣买票。这意味着该网站不再在奖品上花钱，因此它没有任何支出，但没有任何迹象表明它获得了所需的用户数量。在我的收件箱中搜索与 App Start 相关的任何内容时，我发现了这封来自 Andrew 的电子邮件，由 Alex 转发给我： Hi Ben – 这些是 Andrew 要求的技能购买更改。开始转发消息：为房产奖励制作一个技能购买部分，仅用于替换任何当前的房产部分 显示 3 个选项，讨论基于拥有现金的收益 1 栋联排别墅 1 套公寓 @ bispham 3 学生舱蒙哥马利 呼叫所有这些价值 100 k 加 A入场门票成本为 100 英镑 每场比赛只有 1000 张门票 1 位获胜者是最高得分者 1000 张门票售出 72 小时后 主要卖点 您有千分之一的机会获得获胜技能 在星期一之前设置好将开始通过电话销售售票 使其突出 易于导航和易于支付（找到一种方法），最重要的是显示迄今为止游戏中的人数 有一个部分，他们可以记下销售代理的名称，以便我们可以付款通讯和明显获取客户详细信息等 Splash 网站 - 每周可能达到 60000 人 电话销售 - 很快将达到每周 100,000 人每周 160,000 人都看到或听到在 additi 购买投资物业的消息他们应该看到 1000 英镑有 1000 人有机会赢得改变人生的投资房产 每周 160,000 人 @ 0.25% 转换 – 每周 40,000 英镑 转换的关键是使网站变得流畅和吸引人 易于填写 易于支付玩等，如果你使它光滑简单的销售和吸引力如果它有效，那么在 3 周内你会赠送一个房子公关会做剩下的。 ！今天做明天不存在！使用 Orange 的 BlackBerry® 发送 同时，团队的其他成员正在开发 iPhone 游戏。那里有一些非常有才华的设计师和开发人员——他们制作了一些他们引以为豪的伟大游戏。他们制作了一款名为 Splat（有点像 whack-a-mole）的游戏，这是一种令人上瘾的游戏，并且他们设法让当地的广播电台采访了他们。他们还制作了一款名为 Pigs Might Fly 的游戏——他们全力以赴为这款游戏进行营销。亚历克斯打印了一些大猪标志并将它们贴在氦气球上，然后让它们在曼城中心周围飞来飞去。他们让每个人都给广播电台发短信，说“我刚刚在曼彻斯特看到一只飞猪从我的车旁飘过”，然后他们把其中的一些读出来，问是否有人知道这是关于什么的。然后有人打电话给广播节目并开始播放该应用程序。我刚刚发现公司的 Twitter、Flickr 和 YouTube 帐户仍然有效，而且一切都在那里：争论的焦点是安德鲁希望他们淘汰尽可能多的应用程序，然后等待它们中的任何一个是否受到关注。但是他们想选择一个想法并花几个月的时间来研究它，以完成一份体面的工作。

在我完成将 Skill Buy 转换为新格式后，它不需要任何正在进行的工作，直到它拥有所需的用户，因此我被要求加入团队构建应用程序的其余部分。我们被分配到三人一组：每组两名开发人员和一名设计师。我们必须每周为每个小组制作 10 个应用程序。因此，每个开发人员都必须每天制作一个应用程序，设计师创建我们需要的任何资产。我们刚刚列出了一些你可以毫不费力地做出的糟糕想法。它们大多是选择题测验，或者是一个做了一些傻事的按钮。我使用 PhoneGap 来引导应用程序、创建图标和元数据，并仅在技能购买服务器上托管的 PHP 脚本中编写应用程序逻辑。这些应用程序看起来很糟糕，它们毫无意义，也没有潜力。但我们正在达到我们的目标。 Andrew 还向我们提供了他自己的想法，例如“咨询专家”系列，我们将构建一个您可以向其提交问题的应用程序，然后“专家”会做出回应。他似乎认为自己认识一些专家。大约在这个时候，发生在鲍勃身上的同样的事情也发生在亚历克斯身上——他只是进来大喊“我完了，Cya！”走了出去。我们知道他一定是安德鲁日常工作的接班人。我自己和安德鲁也有类似的经历。有一天，房地产行业的 Web 开发人员生病了，我被要求为他做准备，因为 Andrew 已被承诺在当天结束时将准备好一些新功能。房地产业务的 Steph 向我介绍了情况，我与开发商进行了 Google 聊天。我按要求做了，斯蒂芬对此很满意。我被叫去把它展示给 Andrew，他似乎相信这不是他要求的——所以他嘲笑这项工作，称之为浪费时间，告诉我“回到应用程序”然后继续在我离开时对斯蒂芬大喊大叫。所以我们继续了一会儿。我们制作了蹩脚的应用程序。我们达到了目标。我们告诉我们 19 岁的老板，我们不能再这样下去了，这些应用程序都不会成功。我决定是时候继续前进了。我一直等到月底，确保我收到了报酬，然后礼貌地告诉利亚姆我要离开了。他们从来没有给我合同，所以我没有通知期或任何事情。我刚刚离开并开始寻找新的东西。大约一个月后，我遇到了一位设计师，他说有一天 Andrew 关闭了他们，他们都失业了。很遗憾我的第一份真正的工作只持续了大约 4 个月，但这是一次疯狂的旅程和疯狂的经历。直到现在我还没有真正考虑过那段时间。回想起来，感觉很奇怪，故事看起来很超现实，所以我想写下来。安德鲁很有个性，但与加兹相比，他算不了什么。我认为 Gaz 是 Andrew 最年长也是最好的朋友。当安德鲁赚到钱时，他聘请了加兹作为他的司机/保镖。盖兹看起来像个保镖。他是那些你一见钟情的人之一——但他真的很令人愉快，很乐意与之交谈。一天，我们听到一辆汽车在停车场周围行驶，发出很大的噪音。安德鲁刚买了一辆玛莎拉蒂作为送给 Gaz 的“礼物”——开车送他进去，他们正在工业区转了几圈。自然，这辆车被称为 Gazerati。在网上搜索这篇文章时，我看到了一篇题为房地产开发商被判缓刑的新闻文章，它的开头是：

安德鲁·约翰·卡米勒里 (Andrew John Camilleri) 是一名曾与破产的房地产公司 Fresh Start Living 有关联的商人，他被曼彻斯特刑事法院的陪审团一致判定在个人自愿安排提案中作出虚假陈述。 placenorthwest.co.uk 它还指出，在 2011 年初，安德鲁向债权人作出虚假陈述，试图消除 900 万英镑的债务。因此，他不是百万富翁，而是相反——一个消极的百万富翁。似乎像 App Start Ltd 一样的 Fresh Start Living 已不复存在。

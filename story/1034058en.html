JuiceSync：云存储的并行化rsync

2020-11-10 00:10:21

Juicesync is a tool to move your data in object storage between any clouds or regions, also support local file, sftp and HDFS.

JuiceSync是一款可以在任意云或地域之间移动对象存储中的数据的工具，也支持本地文件、sftp和HDFS。

Juicesync will scan all the keys from two object stores, and comparing them in ascending order to find out missing or outdated keys, then download them from the source and upload them to the destination in parallel.

JuiceSync会扫描两个对象存储中的所有密钥，并按升序进行比对，找出丢失或过期的密钥，然后从源下载并并行上传到目的地。

We use go mod to manage modules, if not sure how to use this, refer to The official document.

我们使用Go mod来管理模块，如果不确定如何使用，请参考官方文档。

Auto discover endpoint for bucket of S3, OSS, COS, OBS, BOS, SRC and DST can use format NAME://[ACCESS_KEY:SECRET_KEY@]BUCKET[/PREFIX] . ACCESS_KEY and SECRET_KEY can be provided by corresponding environment variables (see below).

S3、oss、cos、obs、bos、src、dst存储桶的自动发现端点可以使用NAME://[ACCESS_KEY:SECRET_KEY@]BUCKET[/PREFIX]格式。ACCESS_KEY和SECRET_KEY可以由相应的环境变量提供(见下文)。

S3: The access key and secret key for S3 could be provided by AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY, or IAM role.

S3：S3的访问密钥和秘密密钥可以由AWS_ACCESS_KEY_ID和AWS_SECRET_ACCESS_KEY或IAM角色提供。

OSS: The credential can be provided by environment variable ALICLOUD_ACCESS_KEY_ID and ALICLOUD_ACCESS_KEY_SECRET , RAM role, EMR MetaService.

操作系统：凭据可以由环境变量ALICLOUD_ACCESS_KEY_ID和ALICLOUD_ACCESS_KEY_SECRET、RAM角色、EMR MetaService提供。

Qiniu:The S3 endpoint should be used for Qiniu, for example, abc.cn-north-1-s3.qiniu.com.If there are keys starting with "/", the domain should be provided as QINIU_DOMAIN.

七牛：七牛应该使用S3端点，例如abc.cn-North-1-s3.qiniu.com。如果有以"；/"；开头的密钥，则域名应提供为七牛域名。

sftp: if your target machine uses SSH certificates instead of password, you should pass the path to your private key file to the environment variable SSH_PRIVATE_KEY_PATH, like SSH_PRIVATE_KEY_PATH=/home/someuser/.ssh/id_rsa juicesync [src] [dst].

Sftp：如果您的目标机器使用SSH证书而不是密码，您应该将私钥文件的路径传递给环境变量SSH_PRIVATE_KEY_PATH，如SSH_PRIVATE_KEY_PATH=/home/someuser/.ssh/id_rsa JuiceSync[src][dst]。

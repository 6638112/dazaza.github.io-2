微软正在提高Xbox Live Gold订阅的价格，三个月会员资格的月费上涨1美元，至10.99美元，月费上涨5美元，至29.99美元。

2021-01-23 02:58:59

微软即将提高其Xbox Live Gold订阅的价格。微软已经开始通知现有的Xbox Live Gold会员某些市场的变化，它将看到价格上涨1美元，在美国达到每月10.99美元，为期3个月的成员上涨5美元。

12个月和6个月的价格也在上涨，但涨幅不会影响此处的现有订户。现在，三个月的价格为29.99美元，六个月的价格为59.99美元。微软还允许Xbox Live Gold会员将其剩余的Gold时间转换为Xbox Game Pass Ultimate（最长36个月）。转换意味着，如果您的帐户上还有11个月的Xbox Live Gold，则可以升级到Xbox Game Pass Ultimate，并将其转换为11个月的Ultimate，而无需额外费用。

除非您订阅Xbox Live Gold，否则确实意味着该服务的价格将从每年60美元提高到120美元。微软去年悄无声息地取消了它的12个月选项，该选项以前的价格为60美元。索尼仍以60美元的价格提供12个月的PlayStation Plus，目前尚不清楚微软会提供什么价格来证明突然的价格上涨。无论哪种方式，此举都旨在将Xbox Live订户推向Xbox Game Pass Ultimate，后者提供Xbox Live Gold访问权限，xCloud流媒体以及对100多种游戏的访问权限。

许多人一直希望微软能够完全取消其Xbox Live费用，或者至少允许免费游戏免费访问Xbox控制台上的多人游戏功能。现在显然不是事实，对于许多而言，这标志着Xbox Live Gold多年来的首次价格上涨。 Xbox Live Gold团队说：“在许多市场中，Xbox Live Gold的价格多年来没有变化，在某些市场中，它已经超过10年没有变化。”

这些更改要等到Xbox Live订户收到Microsoft通知他们提价的消息至少45天后才会生效。一些地区已经开始接收通知，但大多数地区还没有。微软表示：“如果您正在调整价格的地区，您将在下个月收到一封电子邮件和一个消息中心通知，让您知道会员的新价格是多少。”

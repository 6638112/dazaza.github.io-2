三星发布6.7“Galaxy Note 20和6.9”Note 20 Ultra，搭载Android 10、骁龙865+，以及更多，起价分别为999美元和1,299美元

2020-08-05 23:39:09

三星的Galaxy Note阵容多年来一直在演变，但一直是超级用户的系列产品。今天，三星发布了Galaxy Note 20系列，拥有更强大的规格，更新的设计，S Pen改进，等等。

Galaxy Note 20系列从旗舰产品--Galaxy Note 20 Ultra开始。理论上，它与Galaxy S20 Ultra非常相似。它拥有相同的108MP摄像头传感器，类似的6.9英寸QHD+SuperAMOLED显示屏，在规格表上也有更多的共同之处，如下所示。

显示屏还进行了大幅升级，以赶上Galaxy S系列，刷新率为120 Hz。3088×1440面板还实现了496ppi的像素密度，并提供HDR10+支持。居中的打孔机也装有1000万像素的自拍相机。

当然，Galaxy Note 20 Ultra之所以脱颖而出，是因为它内置了S Pen。有了Note 20 Ultra-而不是更小、更便宜的Note 20-三星已经设法将输入延迟降低到只有9ms，以实现更“逼真的精确度”。此外，S Pen还获得了一些用于滚动、打开应用程序和系统导航的新手势。

与同名的Galaxy S20相比，Galaxy Note 20Ultra搭载了类似的相机系统。它有一个108MP的主后置摄像头和两个1200万像素的传感器作为后盾-一个是长焦的，另一个是超宽的。该系统可提供高达50倍的混合变焦效果。为了解决困扰Galaxy S20 Ultra的自动对焦问题，三星在Note 20 Ultra上安装了激光自动对焦系统。

与Ultra相比，标准的Galaxy Note 20在几个规格上有所下降。这两款设备都使用相同的骁龙865+处理器，但普通的Note 20只有6.7英寸、60 Hz的1080p显示屏、1200万像素的主摄像头、8 GB的RAM，没有microSD卡插槽。

三星还在其Galaxy Note 20系列上加入了一些新功能，首先是三星Notes。内置的S钢笔优化笔记应用程序正在增强您的手机、三星平板电脑和Windows PC之间的同步。这款应用程序还增加了自动保存和改进的文件夹管理。最重要的是，笔记所有者现在可以在录制音频反馈的同时对PDF进行注释和高亮显示，音频反馈与S钢笔绘制的内容同步。Dex还对Galaxy Note 20/Ultra进行了重大升级，可以无线连接到任何支持Miracast的电视。

在相机部门，三星增加了对24fps和21：9宽高比的8K录制的支持。还有一个带有焦点、音频、曝光、变焦速度和其他控制的“专业视频”模式。三星正在添加一项聪明的新功能，允许手机从连接的一对蓝牙耳塞捕获音频。

三星还在Note 20 Ultra中加入了UWB无线电，可以使用同样的无线电与附近的设备通信。从技术上讲，这可以在未来用于智能家居设备或其他技术，但目前，它仅限于谷歌的附近共享功能，使用UWB，可以根据你面对的人对共享目标列表进行排序。

三星Galaxy Note 20的售价为999美元，而Note 20 Ultra的起价为1,299美元。这两款设备都将从Samsung.com以及零售和运营商合作伙伴处购买。直接从三星获得价值高达650美元的折价产品，三星货币用户将获得额外的7.5%的折扣。

Note 20将以神秘青铜、灰色和绿色三种颜色发布。另一方面，Note 20 Ultra将有神秘的青铜色、白色和黑色三种款式。

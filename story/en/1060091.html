One of the key features of LindowsOS, is its ability to install software with literally one-click of the user's mouse. Rather than the traditional installation hassles (downloading, uncompressing, installing, etc.), with Click-N-Run TM everything is performed automatically and seamlessly. LindowsOS users can simply visit the Click-N-Run "Warehouse" and browse through dozens of software titles. For each title, the user can find screenshots, descriptions, specifications, and comparisons. When they find an application they are interested in, they simply select the Click-N-Run TM button... ...and the software is automatically downloaded and installed on the user's LindowsOS computer. Additionally, the selected software title will be saved in the user's My.Lindows.com account, so that they can Click-N-Run™ it on other machines they may own or new machines they may have later on. Can I use Click-N-Run™ to install software from the Click-N-Run Warehouse on machines not running LindowsOS? Because Click-N-Run™ relies on technology built into the actual operating system, it requires LindowsOS and its native web browser to operate. Where do the software titles found in the Click-N-Run Warehouse come from? Lindows.com simply provides the marketplace to house applications that others develop. "Publishers" place the software in our warehouse, decide how to describe the software program, what price to charge, and so on.

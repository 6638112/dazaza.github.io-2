IodéOS= LineageOS 17.1以及MicroG和adblocker

2020-12-13 19:26:17

Introduction iodéOS is a privacy-focused operating system powered by LineageOS and based on the Android 10 mobile platform. iodéOS aims at protecting the user's privacy with a built-in adblocker and by freeing the smartphone from snitches. The objectives in the conception of this ROM are threefold: To keep the stability and security level of LineageOS, by minimizing the modifications made to the system. Apart the system modifications required by the adblocker, we mainly only added a few useful options commonly found in other custom ROMs, made some cosmetic changes, modified a few default settings to prevent data leaks to Google servers.

简介iodéOS是由LineageOS提供支持并基于Android 10移动平台的注重隐私的操作系统。 iodéOS旨在通过内置的adblocker来保护用户的隐私，并通过将智能手机从告白中解放出来。此ROM概念的目的有三点：通过最小化对系统的修改，保持LineageOS的稳定性和安全性。除了adblocker所需的系统修改之外，我们主要仅添加了一些其他自定义ROM中常见的有用选项，进行了一些外观更改，修改了一些默认设置以防止数据泄漏到Google服务器。

To ease a quick adoption of this ROM by new users. We especially target users that are concerned by the protection of their privacy, but are not reluctant to still use inquisitive apps like Google ones. We thus included MicroG as well as a coherent set of default apps (all open source, with one exception), and simplified the initial setup of the system. Particularly, an initialization of MicroG has been made with GCM notifications allowed by default, a privacy-friendly network location provider (DéjàVu) pre-selected, as well as Nominatim Geocoder.

为便于新用户快速使用此ROM。我们特别针对那些关注其隐私保护但又不愿意仍使用诸如Google之类的查询性应用程序的用户。因此，我们包含了MicroG以及一组连贯的默认应用程序（所有开源程序，一个例外），并简化了系统的初始设置。特别是，MicroG的初始化是使用默认允许的GCM通知，预先选择的隐私友好的网络位置提供商（DéjàVu）以及Nominatim Geocoder进行的。

To provide a new and powerful way of blocking ads, malwares, data leaks of all kinds to many intrusive servers. We are developing an analyzer, tightly integrated into the system, that captures all DNS requests and network traffic, as well as a user interface (the iodé app). Compared to some other well-known adblockers, this has the advantages of: Avoiding to lock the VPN for that use. You can even use another adblocker that uses VPN technology alongside our blocker.

提供一种新颖而强大的方法来阻止广告，恶意软件，各种数据泄漏到许多侵入式服务器。我们正在开发一种紧密集成到系统中的分析器，它可以捕获所有DNS请求和网络流量以及用户界面（iodé应用程序）。与其他一些知名的adblocker相比，它具有以下优点：避免针对该用途锁定VPN。您甚至可以在我们的阻止程序旁边使用另一个使用VPN技术的adblocker。

Being independent of the kind of DNS server used by the system or set by an independent app: classical DNS on UDP port 53 or any other one, DNS over TLS (DoT), DNS over HTTPS (DoH), ..., as we capture the DNS requests before they are transmitted to the system function that emits the DNS request. What we do not support, is DoH when it is natively built into applications, i.e. when an app communicates directly with a DoH server, without asking name resolution to the system. It would require to decrypt HTTPS packets between such an app and the DoH server, which may create a big security hole.

不受系统使用或由独立应用程序设置的DNS服务器类型的影响：UDP端口53或任何其他端口上的传统DNS，基于TLS的DNS（DoT），基于HTTPS的DNS（DoH），...在将DNS请求传输到发出DNS请求的系统功能之前捕获它们。我们不支持在本机内置于应用程序中的DoH，即当应用程序直接与DoH服务器通信而不向系统询问名称解析时。它将需要解密此类应用程序与DoH服务器之间的HTTPS数据包，这可能会造成很大的安全漏洞。

Precisely mapping DNS requests and network packets to the Android apps that emitted (or received) them.

精确地将DNS请求和网络数据包映射到发出（或接收）它们的Android应用程序。

Deciding which apps have a filtered network usage (by default, all apps), and which ones can communicate with blacklisted servers.

确定哪些应用程序具有过滤的网络使用情况（默认情况下，所有应用程序），以及哪些可以与列入黑名单的服务器通信。

The iodé blocker is already perfectly usable, although still in its infancy. Many features are lacking, like the possibility of clearing statistics (for specific apps or all), forbidding the collection of statistics for some apps, personalizing the blacklist, etc. We are actively developing it, and new functionalities will be regularly added.

尽管仍处于起步阶段，但碘阻断剂已经可以完美使用。缺少许多功能，例如清除统计信息（针对特定应用程序或所有应用程序），禁止收集某些应用程序的统计信息，个性化黑名单等的可能性。我们正在积极开发它，并将定期添加新功能。

Features Changes in LineageOS to prevent data leaks: Pre-installed apps: Browser: our own fork of Firefox (with Qwant as default search engine and telemetry disabled) instead of Lineage’s default browser Jelly.

在LineageOS中进行更改以防止数据泄漏：预先安装的应用程序：浏览器：我们自己的Firefox分支（具有Qwant作为默认搜索引擎，并且禁用了遥测功能），而不是Lineage的默认浏览器Jelly。

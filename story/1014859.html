SiFive Core IP 20G1

2020-07-29 13:45:11

我们很高兴地宣布，随着SiFive 20G1的发布，SiFive的RISC-V IP产品组合进行了全面更新。此版本为SiFive Core IP(业界最广泛的RISC-V IP产品组合)带来了重要的增强功能和新功能，范围从功能超强的SiFive U7系列到极为流行的SiFive E2系列，提供高达2.8倍的性能(1)、高达25%的低功耗(2)和高达11%的小面积(3)。要深入了解这一激动人心的版本的细节，请注册参加我的SiFive Connect网络研讨会。

SiFive提供基于自由开放的RISC-V ISA的处理器核心--毕竟，SiFive是由RISC-V的发明者创立的。SiFive的使命是支持领域专用硅的开发，市场反馈结合SiFive客户经验告诉我们，处理器内核必须包括为CPU内核(SiFive Shield和SiFive Insight)集成、保护和开发软件所需的所有组件。

SiFive Insight是业界首个针对RISC-V处理器内核的预集成调试和跟踪IP，可实现更快的芯片调试、软件/硬件集成和应用程序开发。

独一无二的是，SiFive Insight具有原生的ARM®CoreSight™兼容性。通过与ARM®CoreSight™无缝集成，开发人员可以将基于SiFive RISC-V的内核集成到混合ISA设计中，并维护他们现有的开发环境。SiFive Insight获得了诸如Green Hills Software、IAR Systems、Lauterbach、Segger等领先软件公司的广泛行业支持，同时还提供SiFive Freedom Studio免费工具。

SiFive 20G1版本对SiFive Core IP进行了几项关键增强，以提高功耗、性能和面积(PPA)。高性能U7系列应用处理器的增强功能降低了超过25%的功耗(2)，同时将流数据应用(如AI加速应用中的应用)的负载带宽提高到2.8倍(1)。

SiFive E3和E7系列现已推出RISC-V“嵌入式”扩展RV32E，与RV32I相比，面积减少高达11%(3)，进一步提高了这些内核的PPA。根据客户反馈，对于需要面积受限的高性能嵌入式CPU的产品设计人员来说，RV32E将是一个非常受欢迎的选择。SiFive 3系列、5系列和7系列内核更新了新的、增强的实时功能，为SiFive的获奖产品组合带来了高可靠性设计的确定性性能。

SiFive先进的平台工程专业知识使我们的路线图开发能够在FPGA上快速集成和验证，以快速测试我们产品线的改进和新功能。SiFive在客户产品包中包含FPGA位流，以实现高性能软件开发平台，目前包括基于Xilinx UltraScale+系列FPGA的VCU118，为包括大型多核配置在内的整个SiFive Core IP产品组合提供软件开发平台。

屡获殊荣的SiFive Core Designer基于云的开发应用程序用于配置您自己的处理器核心，将SiFive 20G1版本捆绑在一起，使SiFive Core IP简单易用。在SiFive20G1版本中，我们采用了一个新的版本方案，从“年/月”命名为“年/发布”。SiFive 20G1版本是2020年的第一个通用版本，它包含了SiFive 9个月的辛勤工作和进展，我们正在向由路线图驱动并与客户工程周期保持一致的新版本进度表迈进。SiFive Core Designer允许芯片设计者微调SiFive处理器以满足他们的工作负载要求，选择所需的特性和功能。

SiFive Shield是一种开放、可扩展的平台架构，旨在实现RISC-V的整个SoC安全，已预先集成到SiFive Core IP中，并通过SiFive Core Designer进行配置。SiFive 20G1版本中新增了对SiFive Shield硬件加密加速器(HCA)IP附加选项的支持。SiFive HCA IP提供AES、SHA和TRNG功能的可配置组合，以实现安全信任根和硬件加速加密功能。SiFive Shield HCA使用SiFive Core Designer进行配置，并与SiFive Core IP交付件一起预先集成和验证，包括随附的文档、软件和驱动程序。

在软件讨论中，我们首先被问到的问题之一是关于FreeRTOS™实时操作系统。对RISC-V的FreeRTOS支持已于2019年初集成到官方FreeRTOS版本中，SiFive Freedom E SDK现在包括FreeRTOS v.10，并将源代码和构建脚本集成到SDK中。除了FreeRTOS内核，SiFive Freedom E SDK还提供了几个基于FreeRTOS的示例，包括使用物理内存保护(PMP)来实现安全的运行时操作。

SiFive Freedom E SDK主板支持包(BSP)已经升级，包括系统视图描述(SVD)文件，从而更紧密地集成到调试环境中。

SiFive Freedom Studio通过增强对SiFive Insight高级跟踪和调试的支持，实现更快的新产品开发，包括新的跟踪流程视图，以加快调试速度。

SiFive 20G1版本提供了令人印象深刻的PPA改进，并结合了新的预集成SiFive Insight跟踪和调试功能，通过SiFive对ARM®CoreSight™的独特支持来最大限度地减少工作流程中断，以及广泛的工具生态系统支持，从而节省时间和精力。使用SiFive Shield安全功能和附加选项为您的下一应用或实时SoC定义和采用高性能、高效率、安全的RISC-V处理器，SiFive从未像现在这样吸引您。

我们迫不及待地想与客户接触，他们将使用比以往更快、更高效、更强大的SiFive Core IP产品组合的实质性新功能来打造令人难以置信的新设备。

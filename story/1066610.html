Visual Studio 2022预览1现在可用

2021-06-18 07:25:02

我们很高兴宣布，Visual Studio 2022的第一个预览版本已准备好安装！这是第一个64位Visual Studio的版本，我们很乐意下载它，尝试一下，并加入我们通过反馈塑造Visual Studio的下一个主要版本。

我们使用此预览的关键目标是测试和调整新的64位平台的可扩展性！使用新的64位平台Visual Studio现在能够扩展以利用所有系统资源，您必须在使用复杂的解决方案或长时间使用Visual Studio时，您必须提高Visual Studio的可靠性。 64位转换工作会影响Visual Studio的每个部分，因此范围比我们通常的预览更大。

我们知道您希望Visual Studio的质量，稳定性和可扩展性。并完全透明，我们可以获得最快捷的方式，当时像你这样的真实开发人员告诉我们我们可以改进的地方。我们真的注意每个错误报告，建议和升空！您可以通过报告问题从Visual Studio中提交。

Visual Studio 2022预览可以与早期版本的Visual Studio并排安装，可在所有三个版本（社区，Pro和Enterprise）中获得，并且可以自由使用。

我们鼓励您尝试预览，就像Visual Studio 2019一样使用它。我们感谢您提供反馈和完成内容调查的时间，这在使Visual Studio 2022为您提供最佳开发人员环境中是非常宝贵的。

我们特别喜欢了解您在Visual Studio 2022中使用非常大而复杂的解决方案的经验。在64位升级之前，具有这种解决方案的客户有时会在Visual Studio中遇到问题，因为它耗尽内存以便使用主要的32位过程。在Visual Studio 2022的早期测试期间，同一客户能够在几天运行IDE，即使包含包含700（或更多）的项目的解决方案。

由于大多数预览1升级与64位支持有关，我们将释放令人兴奋的新功能和绩效改进的SLATE，从预览中开始。您可以阅读Visual Studio RoadMap上即将到来的即将到来的功能。您可以立即尝试的一个新功能是IntelliciCode的更新 - 您可以一次自动完成代码，一次最多一线。

将Visual Studio移动到64位仍然存在一些工作，并且Visual Studio 2019中的少数功能不包括在Visual Studio 2022预览中。您可以在发行说明中找到即将到来的功能列表。

在Visual Studio 2022预览期间，我们构建您使用和爱的扩展的合作伙伴将正在努力更新他们的扩展。当他们这样做时，他们的扩展将立即在Visual Studio 2022中提供。

Visual Studio 2022 for Mac的第一个预览将即将推出，让您首次查看Visual Studio的新现代麦斯科斯UI。在我们觉得它准备好开发人员的反馈之前，我们还有一些工作要做，我们会在Visual Studio博客上让您在此处更新其进度。

如果您拨出扩展，我们希望帮助您将扩展程序纳入Visual Studio 2022.为了让您入门，查看我们的指南以将扩展迁移到预览1。

还！今年6月18日，您可以在Visual Studio团队上的一名高级PM上观看Mads Kristensen，演示如何更新Visual Studio 2022的扩展。Livestream在下午3点开始。太平洋时间：https://youtu.be/-pkiptw6km0

我们将继续与我们的推广作者和维护者共享详细信息，因为我们更接近Visual Studio 2022的官方产品版本。如果您对如何更好地使这一进程有任何想法，您可以在这次短期调查中分享您的思考。

通过安装它，使用它，参加调查，并在开发人员社区分享您的思考来参与塑造Visual Studio 2022。

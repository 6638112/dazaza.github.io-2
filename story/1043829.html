Linux Mint 20.1长期支持版本已经发布

2021-01-10 11:55:35

Linux Mint 20.1现在可用； Linux Mint在20.1中的第一个稳定版本以Cinnamon，MATE和Xfce三种版本提供。 Linux发行版的新版本基于Ubuntu 20.04 LTS和Linux内核5.4。

由于它是一个长期支持版本，因此将在2025年之前支持Linux Mint 20.1。根据开发团队的说法，Linux发行版附带了更新的软件，改进的功能和新功能。

Linux Mint 20.1带有一个统一的文件系统，该文件系统可以看到某些目录与其在/ usr中的对应目录合并在一起，例如/ bin与/ usr / bin合并，/ lib与/ usr / lib合并，以实现兼容性。

Linux Mint 20.1包括几个新功能。开发人员添加了一个选项，可以将网站转换为新版本的桌面应用程序。启动新的Web应用管理器以打开任何网站，例如将gHacks，Twitter或YouTube集成到网络应用中。

Web应用程序的行为在很大程度上类似于桌面程序。它们在自己的窗口中启动并使用自定义图标，使用它们时，您可以在Alt-Tab界面中找到它们。可以固定Web应用程序，并在创建后在应用程序菜单中找到它们。

Linux Mint中的另一个新应用程序是IPTV播放器Hypnotix。该应用程序支持直播电视和播放列表，如果有VOD部分可用，则支持电视服务和电影。该应用程序默认支持免费IPTV提供商Free-IPTV，而该IP-TV只允许访问免费直播电视。

通过删除对Linux Mint 20中引入的ippusbxd的支持并出于可靠性目的恢复Linux Mint 19.3打印机和扫描仪的支持，从而改善了打印和扫描。

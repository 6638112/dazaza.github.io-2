AWS Kinesis现在支持最长一年的数据保留期

2020-11-06 07:08:32

You can now store streaming data for up to one year in Amazon Kinesis Data Streams. Long term retention of streaming data enables you to use the same platform for both real-time and older data retained in Amazon Kinesis Data Streams. For example, you can train machine learning algorithms for financial trading, marketing personalization, and recommendation models without moving the data into a different data store or writing a new application. You can also satisfy certain data retention regulations, including under HIPAA and FedRAMP, using long term retention. You only pay for data stored and retrieved without provisioning additional storage or compute resources.

现在，您可以在Amazon Kinesis数据流中存储最长一年的流数据。通过长期保留流数据，您可以对Amazon Kinesis数据流中保留的实时数据和较旧数据使用相同的平台。例如，您可以针对金融交易、营销个性化和推荐模型训练机器学习算法，而无需将数据移动到不同的数据存储或编写新的应用程序。您还可以使用长期保留来满足某些数据保留规定，包括HIPAA和FedRAMP下的规定。您只需为存储和检索的数据付费，而无需调配额外的存储或计算资源。

Amazon Kinesis Data Streams is a massively scalable and durable real-time data streaming service. It can continuously capture gigabytes of data per second from hundreds of thousands of sources like website clickstreams, IoT data, database event streams, financial transactions, social media feeds, IT logs, and location-tracking events. The data collected is available in milliseconds for real-time analytics use cases like dashboards, anomaly detection, dynamic pricing, and more.

Amazon Kinesis数据流是一种可大规模扩展且耐用的实时数据流服务。它可以持续每秒从数十万个来源(如网站点击流、物联网数据、数据库事件流、金融交易、社交媒体馈送、IT日志和位置跟踪事件)捕获千兆字节的数据。收集的数据在毫秒内即可用于仪表盘、异常检测、动态定价等实时分析用例。

For more information, refer to Changing the Data Retention Period in the Amazon Kinesis Data Streams Developer Guide and visit our pricing page. Long term retention of data is available in all AWS regions where Amazon Kinesis Data Streams is available.

有关更多信息，请参阅Amazon Kinesis数据流开发人员指南中的更改数据保留期，并访问我们的定价页面。所有提供Amazon Kinesis数据流的AWS地区都提供长期数据保留服务。

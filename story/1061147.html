泡沫轧制对性能和恢复影响的荟萃分析

2021-05-07 04:47:22

泡沫滚动被认为提高肌肉性能和灵活性以及减轻肌肉疲劳和疼痛。出于这个原因，泡沫轧制已成为各种运动环境的流行干预，用于提高培训或竞争准备效率以及速度恢复。该荟萃分析的目的是比较泡沫轧制在（预轧制为热升高活动）之前应用的泡沫轧制的影响（作为回收策略后滚动后）在冲刺，跳跃和强度表现上进行运动与柔韧性和肌肉疼痛结果一样，并识别与泡沫辊还是滚子按摩器的自料更有效。使用PubMed，Google Scholar，Pedro和Cochrane Library搜索引擎进行全面和结构化文献搜索。符合纳入标准的二十一项研究。二十项研究使用预轧，而七项研究使用后滚动。预滚动导致Sprint性能的细微改善（+ 0.7％，G = 0.28）和柔韧性（+ 4.0％，G = 0.34），而对跳跃的影响（-1.9％，g = 0.09）和强度性能（ + 1.8％，g = 0.12）可忽略不计。轧制略微减弱的运动诱导的冲刺（+ 3.1％，G = 0.34）和强度性能（+ 3.9％，G = 0.21）。它还降低了肌肉疼痛感知（+ 6.0％，G = 0.47），而其对跳跃性能的影响（-0.2％，g = 0.06）是微不足道的。在二十一项研究中，14个使用的泡沫辊，而另一个七件二手滚筒按摩条/棍子。发现泡沫辊的趋势，以提供比滚轮样机的强度性能（+ 5.6％，G = 0.27与-0.1％，G = -0.01）的恢复更大的影响。泡沫滚动装置在预轧制方面效果的差异似乎似乎没有实际相关性（整体性能：+ 2.7％，G = 0.11与+ 0.4％，G = 0.21;灵活性：+ 5.0％， g = 0.32与+ 1.6％，g = 0.39）。总体而言，确定泡沫轧制对性能和恢复的影响是不成熟的，并且部分可以忽略不计，但在某些情况下可以是相关的（例如，增加冲刺性能和灵活性或减少肌肉疼痛感觉）。证据似乎是合理地，泡沫滚动的广泛使用作为预热活动而不是恢复工具。

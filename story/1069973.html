Element 筹集了 3000 万美元以推动 Matrix

2021-07-27 15:54:55

今天的大新闻：由创建 Matrix 的团队创立的初创公司 Element 刚刚筹集了 3000 万美元的 B 轮资金，以进一步加速 Matrix 的开发并改进 Element，旗舰 Matrix 应用程序。该轮融资由我们在 Protocol Labs 和 Metaplanet 的朋友领投，Metaplanet 基金由 Jaan Tallinn（Skype 和 Kazaa 的联合创始人）创立。 Protocol Labs 和 Metaplanet 都出色地参与了我们的去中心化通信任务，您真的无法要求更好的资金来源来帮助将 Matrix 提升到一个新的水平。感谢您相信 Matrix 并领导 Element 的最新资金！您可以在 Element 博客上从 Element 的角度阅读有关它的所有内容，但可以说这对于整个 Matrix 生态系统来说是一个巨大的新闻。除了改造 Element 应用程序，在 Matrix 方面，这意味着现在有具体的资金保证：完成 P2P Matrix 的构建并使其上线（包括完成 Dendrite，因为我们的 P2P 工作建立在 Dendrite 上！）所有正常的日常工作都在以下方面进行：许多姗姗来迟的有趣点点滴滴（是的，自定义表情符号，我们正在看着你）。如果您想知道 Protocol Labs 的投资是否意味着我们将看到 IPFS 和 Matrix 之间更多的重叠，那么是的 - 在技术上这样做有意义的地方，我们希望更紧密地合作；例如在我们的 P2P 工作中与 libp2p 团队合作（我们仍然需要正确地尝试 gossipsub！），或者给 MSC2706 一些关注。但是，短期内没有计划在 Matrix 或 Element 中使用加密货币激励措施。所以，激动人心的时刻即将到来！我们非常感谢多年来一直支持 Matrix 的每个人——尤其是我们的 Patreon 支持者，他们的捐款为所有 matrix.org 基础设施付费，同时鼓励其他人打开支票簿； Element 现有投资者（尤其是 Notion 和 Automattic，本轮再次进入）；所有大规模的 Matrix 部署都有效地将 Matrix 变成了一个行业（你好 Gematik！） - 以及所有曾经运行过 Matrix 服务器、贡献代码、使用规范制作他们自己的 Matrix 驱动的创作，或者只是在 Matrix 上聊天的人。

不用说，如果没有你，Matrix 就不会存在：如果不是所有支持它的人，协议和网络早就失效了（matrix.org 服务器现在可以在网络上看到超过 3550 万可寻址用户！）-和同时社区和核心团队不断增加的能量相结合，使协议以前所未有的速度向前推进。我们将尽一切努力成功创建期待已久的开放网络安全通信层，我们期待 Element 的大量新资金直接用于核心 Matrixdevelopment :)

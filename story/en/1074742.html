This small guide will help you setup your board to get started with the jupyter notebook

这个小指南将帮助你设置你的电路板，开始使用jupyter笔记本

1. iotSDR-7010 or 7020 board 2. Computer with compatible browser (Google Chrome) 3. Ethernet cable (100M or 1G) 4. Micro USB data cable OR Power Adapter of 5V for Power Jack 5. Micro-SD card(class 10) with preloaded image (16GB recommended)

1.iotSDR-7010或7020板2。配备兼容浏览器的电脑（谷歌Chrome）3。以太网电缆（100M或1G）4。电源插座5的微型USB数据线或5V电源适配器。带有预加载图像的Micro SD卡（10级）（建议16GB）

following the Etcher instructions Write the image to a blank Micro SD card (16GB recommended)

按照蚀刻机说明将图像写入空白Micro SD卡（建议16GB）

Set the ** Boot** jumper to the SD position. (This sets the board to boot Linux from the Micro-SD card)

将**引导**跳线设置到SD位置。（这会将主板设置为从Micro SD卡引导Linux）

Insert the Micro SD card with respective pre-loaded iotSDR image file into the Micro SD card slotConnect the Ethernet cable, one end with iotSDR and other with PCConnect the USB cable to your PC/Laptop, and to the PWR + UART MicroUSB porton the board (next to power jack) iotSDR Board should now power up and get into boot sequence

将带有各自预加载的iotSDR映像文件的Micro SD卡插入Micro SD卡插槽连接以太网电缆，一端带有iotSDR，另一端带有PC将USB电缆连接到PC/笔记本电脑，并连接到板上的PWR+UART MicroUSB端口（电源插座旁边）。iotSDR板现在应通电并进入引导顺序

After setting the SD card header position, inserting the SD card with pre-loaded image file, and the USB cable is connected, the green LED will come on immediately to confirm that the board has power. Following to that, after few seconds, the Blue / Done LED should light up to show that bit file is loaded in the Zynq® device. After some time The Linux OS will be booted and ready for use.

设置SD卡头位置，插入带有预加载图像文件的SD卡，并连接USB电缆后，绿色LED将立即点亮，以确认电路板已通电。接下来，几秒钟后，蓝色/完成LED应亮起，以显示位文件已加载到Zynq®设备中。一段时间后，Linux操作系统将启动并准备好使用。

Once board setup up is complete, you can connect your browser to it to start using Jupyter notebook.

电路板安装完成后，您可以将浏览器连接到它，开始使用Jupyter笔记本电脑。

Follow the below procedure to configure ethernet connection between Host PC and iotSDR before running Jupyter notebook on browser

在浏览器上运行Jupyter notebook之前，请按照以下步骤配置主机PC和iotSDR之间的以太网连接

1. Assign your computer a static IP address from the pool of 192.168.2.xx (except .99 and .1) 2. Connect the board to your computer’s Ethernet port 3. ping to 192.168.2.99 to confirm that you are connected 4. Now open chrome browser and write http://192.168.2.99:9090/lab in the search bar 5. Enter password window will appear The password is: xilinx

1.从192.168.2的池中为计算机分配一个静态IP地址。xx（除.99和.1外）2。将电路板连接到计算机的以太网端口3。ping至192.168.2.99以确认您已连接4。现在打开chrome浏览器并写入http://192.168.2.99:9090/lab在搜索栏5中。输入密码窗口将出现密码为：xilinx

After the password tab window, two folder of the notebooks on the left of webpage will appear and the files can be accessed and user can run the demo projects.

在“密码”选项卡窗口之后，网页左侧会出现两个笔记本文件夹，可以访问这些文件，用户可以运行演示项目。

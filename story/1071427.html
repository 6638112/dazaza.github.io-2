微软暂停其新 Windows 365 服务的免费试用，该服务允许用户在“大量需求”后通过网络浏览器流式传输 Windows 10 或 11

2021-08-05 21:25:52

本周早些时候，微软推出了新的云 PC Windows 365 服务，由于需求，该公司已经不得不暂停免费试用。 Windows 365 可让您租用云 PC（配备各种 CPU、RAM 和存储选项），然后通过 Web 浏览器流式传输 Windows 10 或 Windows 11。该服务仅在注册一天后就达到了最大容量。 Microsoft 365 Twitter 帐户的声明中写道：“随着大量需求，我们已经达到了 Windows 365 试用的容量。” “我们看到了对 Windows 365 的令人难以置信的反响，需要在我们提供额外容量的同时暂停我们的免费试用计划，”Windows 365 计划管理总监 Scott Manchester 解释说。 Windows 365 目前仅适用于企业，如果该公司在免费试用一天后就遇到了容量问题，那么其需求显然超出了微软的预期。微软周一推出了这项服务，如果您已经拥有 Windows 10 许可证，则每个用户的定价为一个 vCPU、2GB 内存、64GB 存储和 12GB 带宽的每月 20 美元起。 Windows 365 有多种配置，包括高端配置，包括 8 个 vCPU、32GB 内存和 512GB 存储空间，每月 158 美元。 Windows365.com 上提供了所有不同的价格选项，但您现在需要注册免费试用，因为 Microsoft 已达到容量上限。

Mark Gurman：Mac 上的 Face ID 可能会在几年内推出，所有 iPhone 和 iPad 也将在那个时间段内过渡到 Face ID

2021-07-25 16:13:57

受人尊敬的彭博社记者马克·古尔曼今天在最新一期的 Power On 通讯中表示，苹果计划在未来“几年”内​​将 Face ID 引入 Mac。在时事通讯中，古尔曼表示，他相信苹果的最终目标是将其所有产品都转移到 Face ID，包括 iPhone SE 和 iPad Air 等低端 iPhone，它们都具有 Touch ID。 Gurman 表示，“Touch ID”仍然是 Apple 产品阵容的重要组成部分，尤其是对于低端机型，因为它是 Face ID 的“更便宜的替代品”，同时继续为用户提供安全保障。但我希望这最终会改变。今年不会发生，但我敢打赌 Mac 上的 Face ID 会在几年内推出。我希望所有 iPhone 和 iPad 也能在这段时间内过渡到 Face ID。最终，嵌入屏幕的摄像头将通过消除顶部的缺口来帮助区分苹果价格更高的设备。面部识别传感器为 Apple 提供了两个核心功能：安全性和增强现实。 Touch ID，方便与否，只提供前者。 Gurman 此前曾报道称，由于苹果计划最近推出重新设计的 24 英寸 iMac，它最初计划包含 Face ID，但 Face ID 的实施已推迟到即将到来的 iMac 重新设计。 Gurman 指出，与 iPhone 和 iPad 不同，Mac 笔记本电脑的屏幕明显更薄，因此更难安装面部识别所需的深度传感器。更进一步，Gurman 表示，苹果最终将把 Face ID 嵌入到屏幕中，而不再需要 iPhone 上的刘海。苹果分析师郭明錤认为，这种“iPhone”最早可能会在 2023 年问世。 至于 Mac 台式机，苹果正在继续开发 27 英寸 iMac 的替代品，它可能具有更大的屏幕和更强大的“M2X”芯片.今年早些时候，苹果公司暂停了更大的‌iMac‌的工作，专注于今年四月发布的24英寸‌iMac‌。不明白为什么这需要苹果这么长时间。 Mac 可以说是 FaceID 最强大的用例。我觉得古尔曼在这些时事通讯中比实际提供新信息更具预测性。

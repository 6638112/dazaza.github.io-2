Radeon RX 6800和6800 XT性能标志着AMD重返高端显卡

2020-11-21 12:50:58

在过去的几周中，PC游戏玩家一直热切期望推出AMD的Radeon RX 6000系列，即“ Big Navi”。在过去的几个月中，NVIDIA凭借其基于Ampere的GeForce RTX 30系列显卡撼动了GPU市场，该显卡提供的性能和价值远高于其前代产品。但是，由于GeForce 30系列卡的有限供应，使一大批爱好者陷入困境，一堆现金烧掉了他们的口袋里的一个小孔，渴望升级。因此，当AMD的Radeon RX 6000系列于10月下旬正式亮相时，据透露，Radeon RX 6800，RX 6800 XT和旗舰Radeon RX 6900 XT将直接与NVIDIA所提供的最好的NVIDIA产品一起完成，许多这样的爱好者在游戏GPU市场的高端市场上实现了一种可行的选择，花这些钱可能很快就到了。 AMD在其活动中显示的性能数字无疑描绘了一个有趣的图画。 AMD声称Radeon RX 6800将击败GeForce RTX 2080 Ti（并扩展为GeForce RTX 3070）。 Radeon RX 6800 XT可以与GeForce RTX 3080作战。强大的Radeon RX 6900 XT可以与功能强大的GeForce RTX 3090并驾齐驱。考虑到GeForce RTX 30系列显卡在市场上的主导地位，AMD的至少可以说有趣。今天，我们开始了解Radeon RX 6800系列的性能如何独自发挥作用。我们已经准备好了Radeon RX 6800和Radeon RX 6800 XT，并在接下来的页面上与您共享大量数据（Radeon RX 6900 XT将在几周后到达）。不过，在深入探讨之前，让我们快速浏览一下这些卡并找出它们使它们打勾的原因。Radeon RX 6800和RX 6800 XT围绕相同的基于Navi 21 RDNA 2的GPU构建，在台积电的7nm工艺节点上制造。 GPU由大约268亿个晶体管组成，管芯尺寸为519mm2。与为当前Radeon RX 5700 XT（具有251平方毫米管芯的103亿个晶体管）供电的芯片相比，该芯片要大得多。它不仅仅是附加的晶体管。据AMD称，其RDNA 2架构的总体目标是帮助推动性能领先地位，并推动AMD在GPU市场高端领域重新回到竞争地位。考虑到AMD尚无法通过多代单GPU在这一领域进行真正的竞争，因此这是一个崇高的目标。尽管Polaris，Vega和第一代Navi都有其优点，但多年来，AMD一直没有一个可以与NVIDIA最好的产品并列的GPU。为了实现这一壮举，RDNA 2的计算单元将Radeon RX 5700 XT的CU的性能（或更高）提高了一倍，并且每瓦性能提升了约50％。此外，RDNA 2提供了领先的功能集，其中包括对DirectX 12 Ultimate及其附带的所有功能的支持，包括可变速率着色，网格着色器，采样器反馈和射线跟踪。还将支持Microsoft的DirectStorage API，以加快从SSD到GPU内存的直接游戏加载速度，尽管PC游戏需要一段时间才能获得对DirectStorage的支持。每个RDNA 2 CU都具有集成的光线跟踪加速器，每个周期能够进行4个Box或1个Triangle相交，与纯软件，非加速实现相比，其相交性能提高了10倍。不过，AMD从未在其驱动程序中为当前Radeons启用DXR支持，因此我们无法轻松地在那里进行直接比较。边界体积层次（BVH）的遍历和射线结果的阴影由在CU上运行的着色器代码处理，并且新的“无限缓存”（稍后将进行介绍）足够大，可以容纳大量BVH靠近CU，以降低延迟。我们还应该提到的是，AMD正在开发一种新的超级采样和缩放模式，以在将来的游戏中启用光线跟踪时提高性能，这有点类似于NVIDIA的DLSS，但目前仍处于开发的早期阶段，还没准备好黄金时间。 AMD还为RDNA 2配备了更高带宽，更节能的数据路径，并在整个GPU中集成了细粒度时钟门控。 RDNA 2也具有新的缓存层次结构，它具有一个128MB的大型高速缓存池，称为“无限缓存”。根据AMD的说法，当将Infinity Cache与256位GDDR6内存接口一起使用时，交付给GPU的带宽增加了一倍以上。 Infinity缓存最终可以提高绝对性能和每瓦带宽，因为它可以以更低的延迟和更高的缓存命中率将数据馈送到图形管道，从而进一步提高了整体效率。尽管Navi 21上的256位内存接口以及16Gbps的GDDR6内存可提供高达512GB / s的本机带宽，但是当充分利用无限缓存时，有效带宽将跃升至1,664GB / s。结合新架构Infinity Cache的优势，以及AMD对用于构建Big Navi的7nm工艺节点的优化库的使用，以及第一批基于RDNA 2的GPU中的计算单元，最终增加了约30％比上一代产品更节能。但是，新的架构和经过调整的过程也使AMD的频率也提高了约30％。最终效果是每瓦性能提高54％-65％，具体取决于所测试的特定型号GPU，并且与上一代产品相比，性能（主动或被动）提高约2倍。 Radeon RX 6000系列还支持许多新功能，即“ AMD Smart Access Memory”和“ Rage Mode”。 AMD称AMD智能访问内存（ASAM）为PCIe规范中称为Resizable BAR的一部分的功能的实现。当在兼容的主板和芯片组上使用SAM时，AMD Ryzen 5000系列处理器可以通过其PCIe 4接口访问连接到Radeon RX 6000系列GPU的整个内存池。处理器一直能够窥探GPU内存的某些部分（仅256MB的传统限制），但是Smart Access Memory授予Ryzen 5000处理器完全访问权限，这可以在某些工作负载下提高性能。目前，已在少数基于500系列芯片组的主板上启用了支持，但将来会越来越多。 Rage Mode品牌标志着AMD提供了一种新的一键式功率增强模式，该模式将在其Radeon图形驱动程序中提供，该模式可以追溯到几年前ATI的Rage图形芯片。愤怒模式实质上会提高卡的功率限制，从而导致频率和电压曲线更加激进，从而有效地提高性能。它与上一代Radeons提供的“高性能”配置文件相同。 AMD展示了Smart Access Memory和Rage模式的综合优势，将Radeon RX 6800 XT在各种游戏中的性能提高了2％-13％-我们将很快对其进行测试。 Navi 21 GPU的完整配置具有80个计算单元-这就是Radeon RX 6900 XT出现在几周后的结果。 Radeon RX 6800和Radeon RX 6800 XT的核心相同，但按比例缩小以适应不同的价格。 Radeon RX 6800具有60个CU，总共3,840个流处理器，96个ROP和128MB的无限缓存。它的游戏时钟和加速时钟分别为1,815MHz和2,105MHz，并且通过256位内存接口将16MB的标准GDDR6连接到GPU，从而提供总计512GB / s的峰值带宽。电路板总功率为250瓦。单精度性能仅是16 TLFOPS的一半，而半精度则是32 TFLOPS的两倍。 AMD将579美元的Radeon RX 6800与GeForce RTX 2080 Ti以及最近推出的GeForce RTX 3070相对。向上移动，我们有了Radeon RX 6800 XT。 Radeon RX 6800 XT还通过256位接口连接了128MB的Infinity缓存和16GB的GDDR6内存，但是CU数量增加到72个，总共有4,608个流处理器，并且具有128个ROP。 6800 XT的游戏时钟也更高，达到2,015MHz，而Boost时钟则高达2,250MHz。额外的CU和更高的时钟将电路板功率提升到300W，但也带来了超过20.7 TFLOPS的单精度计算性能（41.47 TFLOPS半精度）的出色性能提升。像素填充率也明显提高。 Radeon RX 6800 XT的入门价格为649美元，厂商建议零售价比NVIDIA GeForce RTX 3080低约50美元。RadeonRX 6000系列中的所有显卡均采用类似的设计，并配有带三轴流风扇的程式化散热器。在铝制外壳的内部。压铸铝框架和背板增加了一些额外的刚度，还有助于冷却。那个护罩位于一个巨大的蒸气室顶部，蒸气室贯穿PCB的整个长度。卡上的散热器之间唯一的区别是它们的高度– Radeon RX 6800的宽度为两个插槽，而Radeon RX 6800 XT（和6900 XT）为两个半插槽。高端卡上的散热片和导流罩只是更高，以增加一些表面积。所有卡的长度和高度均相同（267mm x 120mm）。 Radeon RX 6000系列卡建立在14层PCB上，具有4个2oz的铜层和15个功率级相位–其中两个专用于内存。该卡具有双标准8针PCI Express电源连接器和内置RGB照明控件。 Radeon RX 6000系列虽然可能不如GeForce RTX 30系列Founders Edition卡的冷却解决方案那么优雅，但它无疑是AMD迄今为止生产的最优质的产品。 Radeon RX 6000系列的端口包括一个带固定速率链接（FRL）支持的HDMI 2.1输出，两个全尺寸DisplayPort（1.4）和一个USB C型连接器。 FRL是HDMI

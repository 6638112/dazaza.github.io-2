Nintendo DS-TV-Out Restoration Project
Nintendo DS-TV-Out恢复项目

2021-02-24 21:18:09

During late 2020, we discovered that the Nintendo DS Lite had a leftover feature in its SoC allowing it to easily have cheap hardware video output. With a little circuitry and some software hacks, we were able to restore it and make it usable for anyone. No FPGA’s, no bulky or cumbersome hardware. This mod is specially useful to revive consoles with only the lower screen, being able to watch the upper screen on your TV. Or to create a GBA Macro with additional TV Output.

在2020年代末，我们发现Nintendo DS Lite在其SOC中有一个剩余的功能，使其能够轻松获得廉价的硬件视频输出。通过一点电路和一些软件黑客，我们能够恢复它并使其可供任何人使用。没有FPGA，没有笨重或繁琐的硬件。此MOD对于仅具有下屏幕的恢复游戏，能够在电视上观看上部屏幕。或者使用额外的电视输出创建GBA宏。

First iteration of the TV-OUT board in action If you are just interested in installation, this is the current method while we work on simpler methods and more features you have requested:

首先迭代电视输出板在操作中，如果您对安装感兴趣，这是当前方法，同时我们正上工作，您要求的更多功能：

Donwload the “NDS TV OUT ENABLE.nds” homebrew from the NDS TV OUT repo

从NDS TV Out Repo中卸载“NDS TV Out Enable.nds”rewlow

Copy both the NDS TV OUT ENABLE and Twilight Menu .nds files to a flashcart.

将NDS TV Out Enable和Twilight菜单复制到FlashCart。

Use flashme to autoboot into the flashcart. You can do this by pressing A + B + Start + Select while booting. Run Twilight menu, and from there, run the enabler homebrew.

使用flashme到autoboot进入flashcart。您可以通过按下启动时按A + B + START + SELECT执行此操作。运行暮光菜单，从那里运行启动器Homebrew。

The console will return to Twilight Menu. Now you can use the buttons on the board to swap between the different screen modes (Upper Screen to TV, Bottom Screen to TV, etc) and launch your games.

控制台将返回暮光菜单。现在，您可以使用电路板上的按钮在不同的屏幕模式（上屏幕到电视，底部屏幕到电视等）之间交换，并启动您的游戏。

The retail firmware of the Nintendo DS Lite disables this specific feature early in the boot process. To reenable it, we use a custom firmware like flashme, which is very easy to install and is required only once, plus a homebrew. Despite that, we are working on an even simpler solution to make it available to as many people as possible, our own custom firmware which integrates patches to enable this feature directly on boot. Additionally, we are currently working with homebrew developers to integrate control of this new feature into existing software for the DS Lite.

Nintendo DS Lite的零售固件在启动过程中早期禁用此特定功能。要重新启用它，我们使用像FlashMe这样的自定义固件，这非常易于安装，只需要一次，加上一个家庭用品。尽管如此，我们正在致力于更简单的解决方案来使其可以使用尽可能多的人，我们自己的自定义固件将修补程序集成到启动时直接启用此功能。此外，我们目前正在使用Homebrew开发人员将对DS Lite的现有软件集成到现有软件中。

This feature is only found on the Nintendo DS Lite. Nintendo DS Phat does not contain this feature nor does the Nintendo DSi. It is important to remark that this is not the same hardware found on Devkits or other special units. This hardware feature is present in virtually every single Nintendo DS Lite out there. The reason why it was left there is unknown, but as said before, it is not related to development units, those use a different video capture hardware. Perhaps Nintendo imagined the Nintendo Switch as early as 2006?

此功能仅在Nintendo DS Lite上找到。 Nintendo DS Phat不包含此功能，也不包含任天堂DSI。重要的是要备注，这不是在Devkits或其他特殊单位上找到的硬件。此硬件功能几乎存在于每个单个Nintendo DS Lite中。它被遗弃的原因有未知，但如前所述，它与开发单位无关，那些使用不同的视频捕获硬件。也许任天堂早在2006年就会想象任天堂交换机？

We only need a few extra hardware components to make this video signal usable. You will be able to download the schematics and gerber files for our open hardware circuit board from the repository. The latest version is revision 1.2 which fixes some minor issues with a component in the board.

我们只需要一些额外的硬件组件来使该视频信号可用。您将能够从存储库下载开放式硬件电路板的原理图和格柏文件。最新版本是修订1.2，它修复了电路板中的组件的一些小问题。

First prototype and tests before designing a proper board The final, production-ready board contains a DAC (Digital to Analogue Converter) which turns the 10 bits digital signal at 16.7 MHz provided by the DS Lite into a proper analogue signal. This signal then goes through an operational amplifier and it’s ready to be delivered to your nearest TV trough composite video.

第一个原型和测试在设计适当的板之前，最终生产的板包含一个DAC（数字到模拟转换器），其将DS Lite提供的16.7 MHz的10位数字信号变为适当的模拟信号。然后，该信号通过运算放大器，并准备好送到最近的电视槽复合视频。

We are currently considering creating an additional PCB revision which would allow to install the mod on consoles without lossing a working upper screen.

我们目前正在考虑创建额外的PCB修订版，允许在不损失工作Upper屏幕上安装MOD。

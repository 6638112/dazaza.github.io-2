新笔记本电脑：Thinkpad P14s

2021-08-06 22:17:17

大约 4.5 年前，我购买了戴尔 XPS 15。它是一款与该尺寸/价格不相匹配的动力马。更换电池后，电池膨胀（并且被迫以这种方式飞行并不有趣），它再次发生。斯洛文尼亚只有一家公司提供保修服务，并在法律规定的 40 个工作日内完成工作（例如更换电池）。我决定重新使用 Thinkpad，因为它们运行 Linux 的整体体验和质量是无与伦比的。受到 Ben Gamari 帖子的启发，我发现我对新笔记本电脑有以下要求：14" 或 15" 屏幕，我还没有下定决心，但这两个对我来说都很好。重量轻（小于 2 公斤），以便我可以轻装旅行（如果我必须移动几米，可以用一只手握住笔记本电脑）。

32 GB 的 RAM，因此我不必担心达到限制（这些天我什至不配置交换）。没有独立的 GPU，这使得玩视频游戏变得不那么方便，并且电池耗电量更少。安静的粉丝，我以前没有为此烦恼过。 XPS 真的很吵，而且我变老了。电池续航时间长（理想情况下为 8 小时或更长时间），这样我就可以完成一天的工作而不必担心插头。快速的维修周转时间，我在斯洛文尼亚的联想公司大约花了一天时间（如果他们有备件）。处理器：AMD Ryzen 7 PRO 5850U Octa Core（8 核/16 线程）1.90 - 4.40 GHz，4 MB L2 和 16 MB L3 缓存内存：32 GB DDR4-3200 MHz（16GB 焊接）（无插槽空闲，最大 48GB）硬磁盘：1 TB SSD M.2 PCIe 3.0 x4 NVMe OPAL2Display：14"UHD (3840x2160) IPS, Dolby Vision500 nits, 16: 9, 1500: 1 对比度, 170°, anti-glare100% DCI-P3 显卡: AMD RGraphon RX Vega 8Network / 通信：10/100/1000 Mbit RJ45 EthernetRealtek WLAN RTL8852AE 11ax，2x2Bluetooth 5.2IR 网络摄像头 720p 与 ThinkShutter 为 Windows HelloWWAN / LTE-A 准备，模块可选接口：2x USB 3.2 供电）2x1（ 3.2 Gen 2 (DisplayPort 1.4a, Power Delivery 3.0)1x HDMI 2.01x SideDocking Port (CS18)1x 3.5mm Mic-In/Audio-Outcombined1x MicroSD 读卡器1x SmartCard ReaderSecurity:- Match-On Touch FingerPrint Reader- TPM 2.0 Security Chip - Kensington 端口输入设备：6 行精密键盘德语，带背光UltraNav / UltraNav Touch 3 + 3 Button DesignBattery：3 芯锂聚合物电池特里 (50Wh) 内部。运行时间长达 10 小时。快速充电：65W USB-C 适配器在 1 小时内充电 80%。尺寸和重量：329 x 227 x 17.9 毫米，1.47 公斤其他：65W USB-C 电源保修：3 年制造商保修一年电池 CPU 基准测试声称我的 CPU 分数从 6945 增加到 19504（相比之下，Apple M1 有 15139）。相当令人印象深刻！

总的来说，除了电池之外，它满足了我的所有要求（虽然我还没有测试过，但我希望它能持续 8 小时以下）。拥有德国键盘是我为当前芯片短缺付出的代价，因为很难获得带有美国/英国键盘（也是英国脱欧）的 P14s 型号。是否有某种钥匙更换服务？由于 NixOS 尚不支持安全启动，您需要做的第一件事是启动到 BIOS 并禁用它。我已经为 Thinkad P14s 编写了构建可启动 ISO 映像教程和 nixos 硬件支持，以便能够使用支持 wifi 卡的 ISO 映像启动。 NixOS 的安装几乎与官方说明相同，只是加密了根分区。 Wifi 工作（虽然速度有问题），蓝牙工作，亮度控制工作，网络摄像头工作，触控板工作，暂停工作。

英特尔为消费者淘汰了所有仅适用于Optane的固态硬盘，未计划更换

2021-01-17 06:35:14

出乎意料的是，英特尔宣布将停止面向消费市场的所有仅适用于Optane的SSD。令人惊讶的是，该公司表示，这些驱动器将不会出现仅适用于Optane的替换，显然标志着其发烧友支持的台式PC的Optane SSD的终结。

英特尔停止生产Optane Memory M10、800P，900P和905P SSD，这代表了其用于台式机的仅Optane系列产品的全部。英特尔的900P和905P停产通知书指出：

＆＃34;英特尔不会为客户细分市场提供新的大容量Optane内存SSD作为过渡产品。英特尔将专注于面向客户细分市场的带有固态存储的新型Optane Memory H20。

M10和800P系列的停产通知也将Intel的客户指向Optane Memory H20驱动器。 H20驱动器在同一设备上配备了Optane内存和QLC闪存。这些缓存驱动器的速度几乎与仅Optane的驱动器（即使在同一个联盟中也没有）一样快，并且主要是为笔记本电脑和OEM市场设计的。

英特尔的停产期也非常短，从通知发布之日起，或者在我们看到的800P系列产品的前几天，不再提供所有停产的驱动器：

所有型号的最新订单将于2021年2月26日发货。英特尔所有已停产的消费类驱动器均配备了第一代Optane内存（3D XPoint），但该公司最近推出了用于服务器的新型Optane SSD P5800X。这款新驱动器首次将第二代Optane推向市场，这表明英特尔仍致力于将奇异的内存用于企业市场。目前，第二代Optane似乎不会出现在台式机上。

英特尔正处于将其NAND业务出售给SK hynix的多年旅程的开始，但它将保留其Optane内存业务和IP。考虑到奇特的底层3D XPoint技术是由英特尔和美光公司共同设计的，并且不为其他内存生产商所采用，这使英特尔成为面向高利润企业市场的差异化产品，这不足为奇。

显然，英特尔决定降低其在NAND存储市场的份额是伴随着对Optane品牌企业市场的新关注。英特尔目前不批量生产Optane内存，它只是从美光公司购买内存。

面向消费市场的英特尔Optane内存驱动器是世界上最快的SSD，在几乎所有可以想到的指标（成本和容量）中，都给竞争的基于NAND的SSD造成浪费。正如我们经常看到的那样，便宜和足够好往往会在市场上取胜，并且各种花园TLC SSD的价格和容量优势，以及对大多数消费者而言足够好的性能，使英特尔的Optane驱动器成为最坚硬的发烧友或专业人士的利基市场。工作站市场。

考虑到以消费者为中心的Optane驱动器的消亡，看来英特尔将依靠其企业级Optane驱动器来解决包括工作站市场在内的所有专业领域。

Sila 提供 1300 万美元用于提供单一 API 来开发金融产品和服务

2021-07-26 21:54:38

Revolution Ventures 领投了这轮融资，现有投资者 Madrona Venture Group、俄勒冈风险基金和 Mucker Capital 以及 Wise 联合创始人 Taavet Hinrikus 跟投。这笔资金使总部位于俄勒冈州波特兰的 Sila 迄今为止的总投资达到 2000 万美元。该公司由 Shamir Karkal、Angela Angelovska、Isaac Hines 和 Alex Lipton 于 2018 年创立，旨在以合规方式简化数字支付和存储，并以区块链技术为基础。首席执行官 Karkal 在金融科技领域有着悠久的历史，他于 2009 年共同创立了 Simple，一款将各种账户统一到一张可访问的银行卡中的应用程序。 它于 2014 年被 BBVA 以 1.17 亿美元收购，并于今年早些时候关闭。 Karkal 告诉 TechCrunch，Sila 的想法源于在创办另一家银行时的挫败感。他看到了对金融应用程序开发的需求，但受到“仍停留在 20 世纪”的银行系统的阻碍。他认为消费者期望不同水平的服务，这就是许多人涌向金融科技公司的原因。然而，每当一家企业试图连接现有的银行系统时，金融科技和加密货币创新者，随着它的建立和扩展，总会遇到技术和合规问题，Karkal 说。 “与银行合作的问题在于，你必须弄清楚如何与他们的大型机集成，”他补充道。 “在这个过程中，你最终也必须成为合规专家才能做到这一点。” Karkal 花了三年时间为其他公司建立银行流程，而 Sila 则用了 18 个月。其银行 API 使开发人员能够创建自己的数字钱包，从而无需与传统金融机构集成。 Sila 还与包括 Plaid、Alloy、Lithic 和 Arcus 在内的金融科技平台建立了合作伙伴关系以转移资金，并得到 Evolve Bank 和 Trust 的支持。 Sila 现在可以在六到八周内让客户开始运作。 Karkal 说，与几乎完全专注于电子商务的竞争对手不同，Sila 的大多数客户都在金融科技、保险科技、商业房地产和加密货币领域进行受监管的支付，而这些领域的合规性基础往往更加复杂。

自公司推出平台以来，业务稳步发展，并于 2020 年下半年起飞。该公司在今年早些时候筹集了 770 万美元的种子轮。在过去 12 个月中，Sila 的收入增长了 10 倍，客户的最终用户在过去七个月中增长了 500% 以上。 Sila 将利用新资金增加员工人数、瞄准更多合作伙伴并扩展产品功能，包括其 Ethereum MainNet 稳定币发行以及 FedWire 与 Nacha 自动清算所网络之间的互操作性。 “美国出现了大量金融科技浪潮，而我们几乎没有触及皮毛，”卡卡尔说。 “像印度、非洲和拉丁美洲这样的地方可能会同时加速，因为它们主要是从零开始。我们在这里‘武装反叛者’，帮助这些创新者构建应用程序，为所有最终用户提供更好的财务体验。”作为投资的一部分，Revolution Ventures 的合伙人 Clara Sieg 将加入该公司的董事会。她告诉 TechCrunch，她通过波特兰生态系统认识了该公司的联合创始人。革命倾向于从消费者的角度看待金融科技初创公司。她说，认识到建设基础设施的问题意味着与银行打交道，该公司开始着手寻找一家建造管道的公司来解决这个问题。在金融科技领域，她认为 Dwolla 是 Sila 的竞争对手。上周，该公司筹集了 2100 万美元用于继续开发其 API，该 API 使公司能够构建和促进快速支付，特别是专注于 ACH。然而，这归结为实际签约客户，而且竞争格局非常薄弱，Sieg 补充道。 “Sila 正在为人们建立一种简单的方式来规划资金并以监管的眼光看待事物，”Sieg 说。 “当 Shamir 构建 Simple 时，他可以看到现有企业提供开发人员嵌入金融服务所需的工具是多么具有挑战性，这就是我们对他获胜的能力充满信心的原因。”

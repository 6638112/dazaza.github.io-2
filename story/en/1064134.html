Apple details upcoming updates to Safari, with better tab management and Tab Groups, extensions on iOS and iPadOS, and a more compact design on macOS
Apple详细信息即将到达Safari的更新，具有更好的标签管理和标签组，IOS和iPado的扩展以及宏的更紧凑设计

2021-06-08 03:14:10

Apple just announced at WWDC 2021 that it is bringing Safari web browser extensions to its mobile devices. With iOS 15 and iPadOS 15, iPhone and iPad will have access to the same web extensions available in Safari for Mac.

Apple刚刚在WWDC 2021宣布，它将Safari Web浏览器扩展带入其移动设备。使用iOS 15和iPados 15，iPhone和iPad将访问Safari for Mac中的相同Web扩展。

Web browser extensions are used to add more features to a browser, with things like ad blockers, VPNs, password managers, and much more. Previously restricted to Safari on the Mac, web browser extensions are now coming to Safari on the iPhone and iPad with iOS 15.

Web浏览器扩展用于向浏览器添加更多功能，以及广告拦截器，VPN，密码管理器等内容。以前仅限于Mac上的Safari，Web浏览器扩展现在正在iPhone和iPad上的Safari与iOS 15。

Developers will now be able to create universal extensions that work on Mac, iPhone, and iPad with the new software available later this year.

开发人员现在将能够在今年晚些时候使用新软件创建在Mac，iPhone和iPad上工作的通用扩展。

Interestingly, this comes after Apple teamed up with Google, Mozilla, and Microsoft to improve universal browser extensions for Safari, Chrome, Firefox, and Edge. In June last year, Apple added support for  WebExtension API in the macOS version of Safari. This API is a cross-platform technology for offering the same extension in different web browsers, including Google Chrome and Microsoft Edge.

有趣的是，在Apple与Google，Mozilla和Microsoft合作，以改善Safari，Chrome，Firefox和Edge的通用浏览器扩展。去年6月，Apple在麦克斯版Safari中添加了对WebExtension API的支持。此API是一个跨平台技术，用于在不同的Web浏览器中提供相同的扩展，包括Google Chrome和Microsoft Edge。

iOS 15, iPadOS 15, and macOS 12 will be available to the public this fall.

IOS 15，IPADOS 15和MacOS 12将于公众提供这一秋季。

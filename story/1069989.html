Element 是一个建立在去中心化协议 Matrix 之上的消息传递应用程序，从 Protocol Labs 和其他公司筹集了 3000 万美元的 B 轮融资，使总融资额达到 4800 万美元

2021-07-27 18:18:23

Element 是由 Matrix 协议支持的端到端加密团队消息平台背后的公司，已在 B 轮融资中从众多知名支持者那里筹集了 3000 万美元，其中包括 Skype 联合创始人 Jaan Tallinn 的 Metaplanet，WordPress.com 的母公司公司 Automattic、Protocol Labs 和 Notion。在 Salesforce 正式收购 Slack 的一周内，这笔 277 亿美元的巨额收购及时提醒人们，团队聊天和协作工具在迅速接受远程工作的世界中发挥着重要作用。与此同时，开源软件几乎渗透到商业世界的每一个角落和缝隙，这一组合使四年的 Element 处于有利地位。总部位于伦敦的 Element 在 Matrix 之上开发了一个开源消息传递客户端，Matrix 是 Matthew Hodgson 和 Amandine Le Pape 于 2014 年在 Amdocs 内部创建的基于去中心化开放标准的通信协议。二人于 2017 年离开 Amdocs，完全专注于 Matrix 的发展和商业化，首先是通过一家名为 New Vector 的公司，该公司开发了 Matrix 托管服务和名为 Riot 的跨平台 Slack 替代方案。 2018 年，Matrix.org 基金会成立，以巩固 Matrix 作为一个中立的非营利实体的发展，而去年 Hodgson 和 Le Pape 将 New Vector 和 Riot 应用程序更名为 Element。 Matrix 可能最好与电话网络或电子邮件之类的东西进行比较，因为它是一种互操作的通信系统，不会将人们锁定在封闭的生态系统中。使用电子邮件，您可以向其他人发送消息，无论他们使用什么服务提供商或客户端（应用程序）——雅虎、Gmail、Outlook 和 ProtonMail 用户都可以很好地相互发送消息。 Matrix 协议正在努力实现类似的目标，但在其十字准线中基于现代互联网的消息传递——在 Matrix 世界中，WhatsApp 用户可以轻松地与 Slack 和 Skype 用户通信。由于 Element 建立在 Matrix 之上，因此它本质上是更广泛的 Matrix 网络增长的催化剂。任何使用 Element 的人都在参与一个由数千万用户组成的开放式全球网络，这些用户分布在来自不同组织的数千个部署中。这也意味着使用 Element 的人不会被锁定在 Element 中——他们可以切换到任何其他由 Matrix 驱动的客户端，而不会丢失任何数据。另外，一种称为“桥接”的方法开放了对不是基于 Matrix 协议构建的第三方应用程序的支持，包括 Telegram、Slack、WhatsApp 和竞争对手的开源消息传递工具，例如 Mattermost。今年早些时候，一家名为 Beeper 的新创业公司加入了竞争，由 Eric Migicovsky 提供，他于 2016 年将他之前的 Pebble 智能手表业务卖给了 Fitbit。 Beeper 是一个基于 Matrix 的通用聊天应用程序，它使用桥接在更多人之间传递消息十多种聊天应用程序，包括 iMessage、WhatsApp、Slack、Twitter、Hangouts 和 Facebook Messenger。

上周还有消息称，德国负责医疗保健系统数字化的国家机构 Gematik 正在转向 Matrix。在此之前几年的孤立数字化转型导致各个医疗机构无法有效地相互沟通，同时也引发了对他们选择传输机密医疗数据的系统的安全性和隐私性的质疑。大约 150,000 个独立的组织构成了德国的医疗保健系统，涵盖医院、当地医生、诊所、保险公司等。切换到 Matrix 为不同方在他们使用的特定应用程序方面提供了一些灵活性。它们可能有不同的用例，但通过遵守共同的标准，所有应用程序仍然可以相互通信。 “这是一项艰巨的任务，到目前为止有 15 家供应商参与其中——包括 Element，”Element 首席执行官兼首席技术官 Hodgson 告诉 VentureBeat。 “将会有来自不同供应商的与系统兼容的广泛应用程序。”其他几家商业公司之前已经在 Matrix 之上构建了产品，包括爱立信的 Contextual Communication Cloud，这是一项托管服务，可让企业将高级通信和协作服务集成到其应用程序中。法国巨头泰雷兹为企业推出了一项名为 Citadel.Team 的即时消息服务，与 Element 并没有太大的不同。 Element 最近还续签了与法国政府的合同，通过该合同，它的消息传递和协作工具集面向法国各地超过 550 万名公务员，而去年它与 Dataport 签署了新合同，向德国教育系统出售商业许可证。总而言之，Matrix 的势头似乎不止一点点，在过去 12 个月中增长了 190%，在 75,000 多个部署中拥有超过 3500 万“可寻址用户”。作为推动 Matrix 协议的主要商业实体，Element 处于有利地位，可以利用这一激增继续关注数据隐私和所有权。数据主权是让组织选择托管数据的位置和方式的概念，从而选择适用于其治理的法律——这对于高度监管的行业和民族国家来说是一笔巨额交易。在一个越来越多地转向云和私有基础设施的世界中，许多情况导致组织对将所有数据移交给一家大公司感到不安。

Element 的新投资者（和 Skype 联合创始人）塔林在一份声明中说：“当通信集中化时，它就会成为一个非常有吸引力的滥用目标——无论是通过宣传、监视、审查还是更糟糕的方式。” “消费者需要从监视资本主义中解脱出来，组织需要一种安全的中立沟通方式。 Matrix 是提供缺失的通信层的最先进平台。” Salesforce 上周斥资数十亿美元收购 Slack，这很好地提醒人们注意什么是利害关系。属于全押在 Slack 上的企业的数据现在归 Salesforce 所有，无论他们喜欢与否。 Slack 最初决定将其出售给 Salesforce 的原因之一是由于微软积极捆绑 Teams 作为其更广泛的 Office 软件套件的一部分，这可能很快会在欧洲面临反垄断审查。关键是，这整集都聚焦于“大科技”可以运用的力量和控制力，而像 Matrix 这样的去中心化“开放”协议有助于解决这一问题。 “Salesforce 收购 Slack 对 Matrix 来说是个好消息，”Hodgson 说。 “这清楚地表明，当您使用 Slack 或 Teams 时，您的数据正在快速传输——仅在传输和休息时加密（即不是端到端）——进入 Salesforce 或 Microsoft 的大型技术云。各种形式和规模的组织都在疯狂地寻找替代方案，让他们对自己的数据拥有完全的主权，而不是盲目地相信这些科技巨头会充当良好的保管人。”同样的原则也适用于无数其他遵循开源精神的企业团队沟通工具，例如 Slack-alternative Mattermost；总部位于巴西的 Rocket.Chat，在其核心开源聊天平台之上提供各种托管和自托管层，并在几个月前筹集了 1900 万美元；和 Kandra Labs，它在开源聊天平台 Zulip 之上开发了一系列商业产品。与此同时，Element 根据客户的需求提供各种定价计划。例如，该公司提供 Element Matrix Services (EMS)，它承诺本地部署的所有数据主权优势，但作为托管服务除外。但这究竟是什么意思？ “它是托管的，因为您可以获得一个专门为您服务的实例，由我们在您选择的地理位置运行，您可以将 DNS 指向该实例，”Hodgson 解释说。 “结合端到端加密，这意味着你拥有对对话的完全所有权和控制权——你实际上只是将服务器的管理外包给了最初编写它的人。” Element 之前筹集了大约 1800 万美元，通过最新的现金注入，该公司现在不仅可以支持自己的产品，而且可以支持基础 Matrix 协议，该协议现在为许多开源和商业服务提供支持。在这个过程中，它可能只是在某种程度上打破了沟通孤岛。

“Element 的资金意味着对 Matrix 协议的持续大量投资，这极大地有利于整个 Matrix 生态系统，”Hodgson 说。 “Matrix 将用于通信，就像网络用于信息共享一样。就像网络一样，它是一个开放的标准，去中心化和通用。” VentureBeat 的使命是成为技术决策者的数字城镇广场，以获取有关变革性技术和交易的知识。我们的网站提供有关数据技术和策略的重要信息，以指导您领导您的组织。我们邀请您成为我们社区的一员，以访问：封闭的思想领袖内容和我们珍贵活动的折扣访问权限，例如 Transform 2021：了解更多 成为会员

Cordless Project Closes Down
无线项目关闭

2020-11-23 22:34:26

Hey, so I know this is somewhat of a bummer, but I got banned because of ToS violation today. As that's basically a conformation for what we've believed would never be enforced, I decided to not work on the cordless project anymore. I'll be taking down cordless in package managers in hope that no new users will install it anymore, especially less illiterate ones. I believe that if you manage to build it yourself, you've probably read the README and are aware of the risks.I'll keep the repository up for now and see what I'll do with it. Might archive it, might delete it, I am unsure. Either way, thanks for sticking around everyone :)

嘿，所以我知道这有点不知所措，但是由于今天违反了ToS规定，我被禁止了。因为这基本上是我们一直认为不会执行的标准，所以我决定不再从事无绳项目。我将在软件包管理器中删除无绳电话，希望不再有新用户安装它，尤其是文盲较少的用户。我相信，如果您设法自己构建它，您可能已经阅读了README并意识到了风险。我将暂时保留该存储库，看看我将如何处理它。我不确定可以将其存档，也可以将其删除。无论哪种方式，感谢大家的支持:)

The application only uses the official Discord API and doesn't send data tothird parties. However, this application is not an official product byDiscord Inc.

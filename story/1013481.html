向您的Mac添加外部GPU可能比购买新GPU更好

2020-07-22 22:07:56

苹果在几年前增加了对eGPU的支持，这是通过在Mac上增加Thunderbolt3端口来实现的。它们具有非常高的吞吐量，使得内部机箱中的GPU能够提供几乎与内部连接的GPU相同的图形处理能力。但是，尽管苹果已经直接销售了几款eGPU，并在本地支持AMD显卡，而不需要任何特殊的驾驶体操，但这仍然是一个利基类别。但是，对于任何希望将现有Mac的寿命再延长几年的人来说，等待苹果硅谷转型的结果是一个很好的选择，来自苹果和主要软件合作伙伴的更新是一个很好的选择。

这里有几个迅雷3eGPU外壳选项，供那些考虑这一升级途径的人选择，以及每个选项的相对优点。请记住，每一款的价格都是针对机箱的--您必须添加自己的eGPU才能使其正常工作，但好消息是，随着图形技术的改进，您可以不断升级和更换这些显卡，从而给Mac带来更大的提升。

Razer Core X Chroma是Razer的顶级GPU机箱，它支持宽达三个插槽的全尺寸PCIe显卡，最大功率为500瓦。集成电源提供700W的功率，可以为任何连接的笔记本电脑充电提供100W的输出，在eGPU的背面，你会发现四个额外的高速USB端口，以及一个用于联网的千兆位以太网端口。Chroma版本还配有可调的LED照明，可为用户提供额外的定制选项。Razer为我提供了Core X Chrome、AMD RADEON RX 5700XT和NVIDIA GeForce RTX 2080Ti，以便在Mac和PC系统上进行测试。

这不是最小的机箱，但这在一定程度上是因为它支持3插槽卡，这在许多竞争对手中是无与伦比的。它也相对较短和较长，这使得它成为一个很好的选择，藏在桌子下面，甚至可能放在桌子下的支架上(有足够的间隙让风扇排气正常工作)。它在操作中很安静，只有当里面的GPU实际上正在为兼容的软件工作时，才会真正发出任何可听得见的噪音。

我的大部分测试都集中在Mac上使用Razer Core X Chroma，要做到这一点，你需要坚持使用AMD的GPU，因为苹果本身并不支持MacOS中的Nvidia显卡。然而，AMD RADEON RX 5700 XT是一只野兽，它提供了足够的马力来改善照片和视频编辑等活动，并为您提供了额外的显示输出选项，通常只是为系统提供额外的资源以供利用。

多亏了Adobe在其Lightroom、Photoshop和Premiere产品中添加eGPU支持的工作，你可以在所有这些应用程序的整体渲染和输出方面获得很大改进，特别是如果你的Mac电脑只有一个集成的GPU。苹果自己的应用程序也是如此，包括Final Cut Pro X。

根据我的经验，使用eGPU极大地改进了Adobe和Apple的PRO视频编辑软件的导出功能，将导出时间缩短了至少一半。一般来说，在Lightroom中工作要快得多，响应速度也更快，缩略图和预览的渲染时间大大减少，这在我2018年的Mac mini上通常需要相当长的时间。

苹果还使用eGPU来加速任何使用Metal、OpenGL和OpenCL的应用程序的性能，这就是为什么当你插入一个时，你可能会注意到系统性能的微妙普遍改善。很难量化这种影响，但总体系统性能感觉不那么迟缓，反应更灵敏，特别是在同时运行大量应用程序的情况下。

Razer Core X Chrome额外的扩展槽、安静的运行和最大功率传输，都使它成为您寻找机箱来处理所需一切的首选，而且它可以为Mac和Windows PC提供巨大的颠簸-如果您碰巧同时使用这两个平台，则两者可以互换。

如果你想花更少的钱，并得到一个更基本的外壳，但仍然提供出色的性能，看看Akitio节点泰坦。机箱制造商Akitio被OWC收购，OWC是一家广受欢迎的Mac外围设备制造商和销售商，几十年来一直提供第三方RAM、坞站、驱动器等。Node Titan是他们的高端eGPU机箱。

Node Titan的外壳比Razer Core X略小，并且采用了类似太空灰色的颜色，与苹果的Mac笔记本更匹配。较小尺寸的折衷是它只支持2插槽显卡，但它还具有集成的弹出手柄，与更轻、更紧凑的设计相结合，使其更加方便，便于随身携带。

Akitio的Node Titan搭载了一个650W的电源，这对高消耗显卡来说是件好事，但这也意味着，与Core X Chrome相比，这种情况下的另一个折衷方案是，Titan只向任何连接的笔记本电脑提供85W的输出。这低于最新的16英寸MacBook Pro全速充电所需的96W，但仍足以让你的笔记本保持供电，并为苹果Mac笔记本系列的其他产品提供全速充电。

Node Titan还在机箱本身上只提供一个端口-用于连接到您的计算机的迅雷输出。但是，与其配合使用的图形卡将提供自己的显示连接，用于连接外部显示器。

在性能方面，Akitio Node Titan与适用于Mac的AMD RADEON RX 5700 XT(以及适用于PC的AMD和NVIDIA卡)在连接时提供相同的潜在收益，因为GPU规格是使用盘柜时最重要的。它的运行噪音更大一些，特别是在发出安静但仍然可以检测到的持续嗡嗡声方面，即使在GPU没有征税的情况下也是如此。

然而，Node Titan仍然是一个很好的选择，对于那些想要更便携和更实惠的人来说，这可能是一个更好的选择，但代价是牺牲笔记本的最大功率输出和大量方便的端口扩展。

回到更多的Mac电脑可以选择用户可扩展RAM的时候，这是一个很好的方式，可以从外部机器中挤出一点更多的生命，让速度更慢的机器感觉更快。现在，在苹果的产品阵容中，只有几款Mac让升级内存变得很容易，甚至有可能。添加eGPU可能会有类似的效果，特别是如果你花大量时间在创意编辑应用程序上，包括Adobe的套件、苹果的Pro应用程序或包括DaVinci Resolve在内的各种其他第三方应用程序。

EGPU设置的总价格(包括卡)可以接近甚至与新Mac的价格相当，但即使是更便宜的卡也能带来显著的好处，您可以随时根据需要更换。值得一提的是，尽管苹果已经表示会支持迅雷，但苹果硅Mac对eGPU的支持前景并不确定。尽管如此，eGPU可以推迟数年的升级需求，使人们更容易等待和观察，看看这一过程过渡对Mac用户真正意味着什么。

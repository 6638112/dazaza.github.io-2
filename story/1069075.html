快速移动和破坏事物使我们失去了我们的隐私和安全

2021-07-23 02:20:02

Denis Mars 是 Proxy 的首席执行官兼联合创始人，该公司设计和构建隐私优先、以人为本的身份技术。在 Y Combinator 担任招生主管期间，我看到了数百个创业项目。许多人都有一个共同的特点：他们走的是快速增长的用户之路，并将从用户那里提取的数据货币化。随着时间的推移，我开始看到我们的技术正在创造的全貌：一个“少数派报告”世界，我们的一举一动都被跟踪并货币化。一些公司，如 Facebook，奉行“快速行动，打破常规”的格言。他们不仅破坏了事物，而且通过传播虚假信息和宣传使我们失望，最终使一些人丧生。发生这种情况是因为不惜一切代价增长的心态。 21 世纪一些最大的面向消费者的硅谷公司通过使用数据销售广告而很少或根本不考虑用户隐私或安全，从而蓬勃发展。我们拥有一些技术领域最聪明的人才；如果我们真的愿意，我们可以改变一些事情，至少，人们不必担心隐私和信息安全。我们可以转向一种模式，让人们可以更好地控制自己的数据，硅谷探索隐私和数据安全方面的创新。虽然有多种长期方法和潜在的新商业模式可供探索，但在短期内还是有办法实现隐私至上的心态。这里有几种方法可以开始迈向人们可以控制他们的数据的未来。我们需要通过有意识地设计一个技术以安全和合乎道德的方式为人类、企业和社会服务的未来来处理技术。在不了解或不考虑后果的情况下接近技术增长已经侵蚀了硅谷的信任。我们必须做得更好——我们可以从工作场所开始，通过自我主权身份更好地保护个人数据，这种方法让人们可以控制和拥有自己的数字身份。

将工作场所作为提高人们数字身份隐私和安全性的起点是有道理的，因为许多已被广泛采用的技术——比如个人电脑、互联网、手机和电子邮件——在成为家庭技术之前就开始于工作场所从而继承了基本原则。随着即将回归办公室生活，现在是重新审视我们如何在工作场所采用新做法的最佳时机。那么雇主将如何做到这一点呢？首先，他们可以将重返办公室作为非接触式访问和数字 ID 的推动力，以防止物理和数字数据泄露，后者正变得越来越普遍。员工可以通过安全存储在手机上的数字 ID 或标记化 ID 进入办公室。他们将不再需要使用印有个人信息和照片的塑料卡片，这些卡片很容易伪造或复制，从而提高了雇主和雇员的安全性。如今，非接触式访问也不是一个大的飞跃。大流行使我们为数字识别做好了准备——因为 COVID 加速了非接触式支付的使用，对许多人来说，非接触式 ID 的更改将是无缝的。令牌化识别将权力交到用户手中。这不仅对于工作场所访问和身份至关重要，而且对于许多其他更重要的原因也很重要。令牌化的数字身份证经过加密，只能使用一次，因此如果系统遭到破坏，任何人几乎不可能查看数字身份证中包含的数据。它就像 Signal，但适用于您的数字 ID。随着更复杂的技术的推出，将产生更多的个人数据（这意味着更多的数据容易受到攻击）。我们必须担心的不仅仅是我们的驾照、信用卡或社会安全号码。我们的生物识别技术和个人健康相关数据（例如我们的医疗记录）越来越多地在线访问并用于验证目的。由于黑客攻击和身份盗窃的盛行，加密的数字 ID 非常重要。如果没有代币化的数字 ID，我们都很容易受到攻击。我们看到了最近发生的 Colonial Pipeline 勒索软件攻击。它使美国管道系统的大部分瘫痪数周，表明我们基础设施的关键部分极易受到破坏。

最终，我们需要考虑制造为人类服务的技术，而不是相反。我们还需要问问自己，我们创造的技术是否不仅对用户有益，而且对整个社会有益。构建更好地为人类服务的技术的一种方法是确保它保护用户及其价值。随着其他技术的出现，自我主权身份将成为我们未来的关键。除此之外，我们将看到我们的数字钱包不仅仅包含信用卡，这使得对安全数字身份证的需求变得更加重要。最重要的是，个人和公司只需要控制自己的数据。鉴于近年来对隐私和安全的普遍认识，雇主必须认真对待个人数据漏洞的威胁，并在自主身份方面走在前列。通过在工作场所使用非接触式访问和数字 ID 的第一步，我们可以开始朝着更安全的未来迈进，至少在我们自己的数据和身份方面。

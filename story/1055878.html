Windows 95 - 今天它如何看？

2021-04-03 07:48:29

Windows 95是Microsoft的“下一代”操作系统：重新设计的UI，长文件名支持，32位应用程序和许多其他更改。一些Windows 95组件今天仍在使用中。它看起来怎么样？让我们测试它并搞清楚。

我不会对Windows 95进行审查，这是在很多年前这么多的盟友。但是一旦我看到了视频，就像“青少年对Windows 95反应”一样，很明显，已经不仅有青少年，而且全新一代20-30岁的成年人在他们的生命中使用过Windows 95太年轻，实际上记得它。无论如何，Windows 95是计算机的伟大里程碑，历史上的伟大里程碑，Windows 3.1到一个新系统的巨大改进，我们现在正在使用的部分。

将Windows 95安装到现代PC可能很棘手，但实际上不需要 - Windows 95可以在虚拟机上运行。 Windows 3.1在DOSBOX中运行完美，这是一个漂亮而轻便的模拟器，甚至不需要安装，可以从任何文件夹执行。但是Dosbox正式​​不支持Windows，理论上，它可能是可能的，但它更容易使用完全成熟的虚拟机。我已经使用Oracle VirtualBox，这是免费的，可以安装在任何现代PC上。

首次将许多UI元素添加到Windows 95中，它们仍然存在于今天： 开始菜单。 微软试图重新设计，但现在仍在那里，25年后。 顺便说一下，第一键盘没有特殊的“赢”键，但快捷方式Ctrl + Esc今天仍在Windows 10中工作。 系统托盘与屏幕右下角的时钟和图标。 回收站，底部的任务栏和其他元素，如“命令提示符”，等等。 一些组件，如“设备管理器”，字面上没有在25年内更改（右侧的Windows 10和Windows 95）： 有趣的是，“关闭”窗口允许在MS-DOS模式下启动PC： 此功能不可用，但在Linux中，在控制台模式下启动的可能性仍然存在。

通常，Windows 95用户界面仍然熟悉，即使我们考虑过多年过去了的话，它很迷人。

Internet是第90次最新技术的最新技术，Windows 95可以支持拨号和以太网连接。

最受欢迎的是使用电话线的拨号连接。 “调制解调器”是一个特殊的“魔术盒”，它是将计算机连接到互联网提供商。在连接开始时，2个调制解调器正在交换有关线路质量和连接速度的信息，此时扬声器处于活动状态，并且可以听到连接声音。它们根据所选速度而不同，经验丰富的用户甚至能够通过听到声音来识别连接质量。典型的连接速度为每秒大约3-4千字节，因此只需要几分钟即可在Web浏览器中打开本文。最后但并非最不重要的是，电话线在用户在线时忙碌。

Windows 95中的Web浏览器的图标仅称为“Internet”，但如果我们打开它，我们可以看到“Microsoft Internet Explorer”名称，即使是今天熟悉。默认主页http://home.microsoft.com无法使用：

此浏览器显然无法支持HTTPS，因此大多数现代网站无法打开。可以打开www.google.com网页，但它是正确显示的：

至少，HTTP连接有效，我能够使用Python -M HTTP.Server 80命令在我的PC上运行本地HTTP服务器，并在Windows 95浏览器中查看和下载文件。在不制定共享文件夹的情况下交换文件是相对简单的方式。

1996年ICQ - 首先所谓的“即时信使”，已发布。现在它是标准的总是在线，并在Slack或Whatsapp中有不同的聊天，但在1996年它是一种新的想法。很多年前，我有一个ICQ号码，但经过这么多年，不可能记住它。我试图进行新的注册，但它也明显失败了，服务器现在关闭：

顺便说一下，这个问题今天更重要。现在，大多数服务都在网上和云中，我可以猜测所有“智能”设备，我们今天正在使用，将无法在25年后开始。

通常，有趣的是，实际上，Windows 95可以连接到因特网。但这种联系实际上是无用的 - 网络标准在过去25年中迄今为止，99％的网站无法打开。我可以平网站，这主要是我能做的最大值：

Windows 95是一个32位操作系统，而软件可以在理论上是在现代操作系统中运行（具有一些限制）。但相反的不是真实的 - 由于不同的API版本和缺少的库，现代软件在Windows 95上不起作用。因此，从一侧，Windows 95看起来像窗户;）但是从另一边，它太老了，无法在它上运行任何现代软件。至少我们可以测试在此期间创建的软件。

新的32位Microsoft Office于1995年发布，它包括多个应用程序，如Word，Excel或PowerPoint。即使在今天，所有这些应用程序也在市场上。在屏幕上，在屏幕右侧看到特殊的“发射器”也有趣 - “标准”快速启动面板仅在Windows 98中发布。

Adobe Photoshop 5.0于1998年发布，它是第一个为Windows 95制作的版本，前一个是Windows 3.1所做的。我想，它的功能对于大多数用户来说也可以足够：

Photoshop 5.0无法从数码相机打开原始文件，但是大多数其他照片编辑功能（层，曲线，过滤器等）可用。看来Photoshop设计通常在过去25年内没有改变也很有趣（想要看到1.0版，1990年为Mac发布的那些，可以阅读另一篇文章）。

流行的媒体播放器Winamp 0.99于1997年发布，顺便说一下，应用规模仅为146千克。如今即使是现代媒体播放器的飞溅屏幕可能更大......

现在是几乎所有人在硬盘驱动器上收集MP3文件的时间，Winamp是最可下载的Windows应用程序之一。不需要的在线流媒体服务尚未发布。

微软发布基于Chromium的Edge Preview for Linux

2020-10-21 08:46:22

Last month, Microsoft officials said they'd release a preview of the new Chromium-based Edge browser for Linux some time in October. On October 20, Microsoft made good on the promise, making available the Edge Dev Channel build for Linux. The new release supports Ubuntu, Debian, Fedora and openSUSE Linux distributions. Microsoft is planning to release weekly builds, like it does with the Dev Channel builds for other platforms. To get started, users can download and install a .deb or .rpm package directly from the Edge Insider site, which will configure a system to get future automatic updates. Or users can install Edge from Microsoft's Linux Software Repository. More detailed instructions are available on Microsoft's Chredge-on-Linux blog post.

上个月，微软官员表示，他们将在10月份的某个时候发布Linux版基于Chromium的新Edge浏览器的预览版。10月20日，微软兑现了承诺，推出了适用于Linux的Edge Dev Channel版本。新版本支持Ubuntu、Debian、Fedora和openSUSE Linux发行版。微软正计划每周发布一次版本，就像它为其他平台发布的Dev Channel版本一样。要开始，用户可以直接从Edge Insider站点下载并安装.deb或.rpm软件包，这将配置系统以获取未来的自动更新。或者，用户可以从微软的Linux软件存储库安装Edge。更详细的说明可以在微软的Chredge-on-Linux博客上找到。

Officials announced the planned Chromium-based Edge (aka Chredge) on Linux release date on Sept. 22, the opening day of its virtual  Ignite 2020 IT Pro conference.

官方宣布计划在Linux上发布基于Chromium的Edge(又名Chredge)，发布日期为9月1日。22日，其虚拟的Ignite 2020 IT Pro大会开幕当天。

Microsoft already supports the new Edge browser on Windows 7, 8.1, and Windows 10, as well as on macOS. Microsoft is including the new Edge browser as part of its Windows 10 20H2 feature update, which began rolling out to mainstream users today, October 20. (It also rebranded its browsers on iOS and Android so they share the same logo as Chredge.)

微软已经在Windows 7、8.1和Windows 10以及MacOS上支持新的Edge浏览器。微软将把新的Edge浏览器作为其Windows1020H2功能更新的一部分，从10月20日开始向主流用户推出。(它还重新命名了iOS和Android浏览器的品牌，以便它们与Chredge共享相同的标志。)。

As it did when introducing the new Edge on macOS, Microsoft is positioning Edge on Linux as more of an offering for IT pros and developers who want to test web sites than as a browser for "normal" users on those platforms. However, any user on any supported platform can use the new Edge.

正如微软在MacOS上推出新的Edge时所做的那样，微软将Linux上的Edge定位为面向IT专业人士和开发人员的产品，而不是面向这些平台上的普通用户的浏览器。但是，任何受支持平台上的任何用户都可以使用新的Edge。

Twitter表示，它“调查目前影响所有Twitter API V2端点的可能的系统不规则”，这导致某些用户的问题

2021-04-18 18:32:12

它看起来像Twitter上的问题，星期五晚上持续到周六早上。美国东海岸的人报告说，虽然页面加载，但他们无法发送推文和DMS。截至11月11日星期六，该网站仍然遇到了许多用户的问题。

没有官方的词语，上断如何普遍普遍，但Twitter表示周六，它与其服务器“修复问题”。

Twitter支持在周五晚上发布了一条简短的信息，它正在调查此事，并于周六早上更新。

我们知道Twitter的部分仍然不适用于你们中的一些人。我们正在与我们的服务器修复问题，以便尽快将其恢复正常。谢谢你与我们粘在一起。

- Twitter支持（@twittersupport）4月17日，2021年4月17日

没有官方的话语如何普遍普遍存在。监测现场中断的下降员在周五晚上8:25左右举行了巨大的飙升。事情似乎在一小时内恢复正常，但周六早上左右左右，它再次显示报告的问题。通常这些中断是亚马逊Web服务等中间服务问题的结果，但它的仪表板显示了这种写作的所有绿色。

Twitter的API状态网站发布了一条消息，即其数据团队正在调查“目前影响所有Twitter API V2端点的可能的系统不规则，从2021-04-17 12:00 UTC开始。目前尚未确定任何开发人员影响的存在和范围，但我们将立即提供更新。“

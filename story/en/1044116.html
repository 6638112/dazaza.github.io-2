2021 Cloud Report
2021年云报告

2021-01-15 20:12:01

The 2021 Cloud Report is the only cloud performance report to compare AWS, Azure, and GCP on benchmarks that reflect critical applications and workloads.

《 2021年云报告》是唯一可在反映关键应用程序和工作负载的基准上比较AWS，Azure和GCP的云性能报告。

The 2021 Cloud Report runs over 1,000 microbenchmark tests to evaluate CPU, network, storage, and TPC-C performance.

《 2021年云端报告》运行了1,000多个微基准测试，以评估CPU，网络，存储和TPC-C性能。

We want to help others understand the performance and cost tradeoffs of each cloud and its machines.

我们想帮助其他人了解每个云及其机器的性能和成本权衡。

For three years running, AWS has provided the lowest network latencies of any of the major clouds.

在过去的三年中，AWS提供了所有主要云中最低的网络延迟。

New to the report were machines running Amazon's ARM-based Graviton2 processors, which excelled on multi-core CPU benchmarks.

该报告的新成员是运行基于Amazon ARM的Graviton2处理器的计算机，该处理器在多核CPU基准测试中表现出色。

Azure's ultra disk machines were high performers overall and proved themselves worth the higher cost for many workloads.

Azure的超磁盘计算机总体上具有较高的性能，并证明了自己值得承受许多工作负载的较高成本。

This project is open-source and you are welcome to run the benchmarks yourself. You may just end up saving costs, increasing performance, and learning something new. If you have any questions about the report, we encourage you to get in touch with us via our public Slack channel or learn more via our YouTube channel.

该项目是开源的，欢迎您自己运行基准测试。您可能最终会节省成本，提高性能并学习新知识。如果您对报告有任何疑问，建议您通过我们的公共Slack频道与我们联系，或者通过我们的YouTube频道了解更多信息。

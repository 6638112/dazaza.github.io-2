美国唱片工业协会获得YouTube-DL启动GitHub

2020-10-24 09:35:09

Jump to navigation Jump to search Microsoft GitHub has removed all traces of the very useful youtube-dl utility for downloading videos from YouTube and other websites, including this one, following a questionable DMCA request from the Recording Industry Association of America.

跳转到导航跳转到搜索Microsoft GitHub删除了从YouTube和其他网站(包括本网站)下载视频的非常有用的youtube-dl实用程序的所有痕迹，此前美国唱片业协会(Record Industry Association Of America)提出了一项有问题的DMCA请求。

The popular video player mpv playing a YouTube-hosted Creative-Commons licensed video from this years NextCLoud conference using youtube-dl to download pieces of the video on-the-fly.

流行的视频播放器MPV播放YouTube托管的Creative-Commons授权视频，该视频来自今年的NextCLoud会议，使用youtube-dl动态下载视频片段。

youtube-dl is a simple command-line utility that lets you easily download audio adn videos from just about any website with a file file embedded in it. It works on sites like this one. A lot of software, including the popular video player mpv, can use it to download video fragments on the fly so videos embedded in web pages can be opened and played as if they were local files.

Youtube-dl是一个简单的命令行实用程序，可以让您轻松地从几乎任何嵌入了文件文件的网站下载音频和视频。它适用于像这样的网站。许多软件，包括流行的视频播放器MPV，都可以使用它动态下载视频片段，这样嵌入到网页中的视频就可以像本地文件一样打开和播放。

The Recording Industry Association of America submitted a DMCA request to Microsoft GitHub demanding that youtube-dl gets removed from the Internet on October 23rd, 2020. The complaint contains this rather misleading claim:

美国唱片业协会于2020年10月23日向微软GitHub提交了DMCA请求，要求将YouTube-dl从互联网上移除。起诉书中包含这样一种颇具误导性的说法：

"The clear purpose of this source code is to (i) circumvent the technological protection measures used by authorized streaming services such as YouTube, and (ii) reproduce and distribute music videos and sound recordings owned by our member companies without authorization for such use. We note that the source code is described on GitHub as “a command-line program to download videos from YouTube.com and a few more sites. "

此源代码的明确目的是：(I)规避授权流媒体服务(如YouTube)使用的技术保护措施，以及(Ii)未经授权复制和分发我们的成员公司拥有的音乐视频和录音制品以供此类使用。(I)绕过授权流媒体服务(如YouTube)使用的技术保护措施；以及(Ii)未经授权复制和分发我们的成员公司拥有的音乐视频和录音。我们注意到该源代码在GitHub上被描述为“一个从YouTube.com和其他几个网站下载视频的命令行程序”。"；

"We also note that the provision or trafficking of the source code violates 17 USC §§1201(a)(2) and 1201(b)(1). The source code is a technology primarily designed or produced for the purpose of, and marketed for, circumventing a technological measure that effectively controls access to copyrighted sound recordings on YouTube, including copyrighted sound recordings owned by our members. "

我们还注意到，源代码的提供或贩运违反了USC§17§1201(A)(2)和1201(B)(1)。源代码是一种主要为规避有效控制访问YouTube上受版权保护的录音(包括我们的会员拥有的受版权保护的录音)的技术措施而设计或生产并销售的技术。"；

RIAA's claim that youtube-dl is primarily designed for and used to download their extremely untalented members pathetic attempts at making music seems ridiculous. The Recording Industry Association of America does not represent anyone who's even remotely talented or worth listening to. They do not represent any south-Korean groups like ATEEZ, BlackPink, BTS, Lovelyz, Mamamoo, WJSN or anyone else who's videos are regularly removed from YouTube's "Trending" list in order to hide how popular they are in comparison to RIAA largely talentless members.

RIAA声称YouTube-dl主要是为下载他们极无天赋的会员而设计的，并用来下载他们可悲的音乐尝试，这似乎是荒谬的。美国唱片业协会并不代表任何有一丁点才华或值得一听的人。他们并不代表任何韩国团体，如ATEEZ、BlackPink、BTS、Lovelyz、Mamamoo、WJSN或其他任何经常从YouTube的热门视频列表中删除的人，目的是掩盖与RIAA基本上没有才华的成员相比，这些视频有多受欢迎。

"These are the people who previously claimed damages from Napster that were higher than all money existing in the world at the time, so for them it's just medium levels of stupidity. "

这些人之前向纳普斯特索赔的赔偿金比当时世界上所有的钱都要高，所以对他们来说，这只是中等水平的愚蠢。"；

Microsoft GitHub has gone along with RIAA's DMCA request. They have now removed youtube-dl and all forks of it from GitHub. The youtube-dl repository and all the forks are currently showing this statement:

微软GitHub已经同意了RIAA'；DMCA的请求。他们现在已经将youtube-dl及其所有分支从gihub上删除。Youtube-dl存储库和所有分支当前显示以下语句：

"This repository is currently disabled due to a DMCA takedown notice. We have disabled public access to the repository. "

"；由于DMCA关闭通知，此存储库当前已禁用。我们已禁用对存储库的公共访问。"；

All the pages on the youtube-dl website at youtube-dl.org, except the front page, were in reality hosted by Microsoft GitHub, not the German server hosting the front page of that site. The tar.bz2 source code download link on that site is a simple redirect to Microsoft GitHub. Clicking the latest tarball download link listed on that page results in a ERROR 451: Unavailable for Legal Reasons. error from GitHub's servers.

Youtube-dl网站youtube-dl.org上的所有页面，除了首页，实际上都是由microsoft gihub托管的，而不是托管该网站首页的德国服务器。该站点上tar.bz2源代码下载链接是指向Microsoft GitHub的简单重定向。单击该页面上列出的最新tarball下载链接会导致错误451：由于法律原因不可用。来自GitHub'；服务器的错误。

The latest youtube-dl-2020.09.20.tar.gz youtube-dl source code archive can still be acquired from any of the Gentoo Linux distfiles mirrors such as mirror.yandex.ru/gentoo-distfiles/ ( youtube-dl-2020.09.20.tar.gz).

最新的youtube-dl-2020.09.20.tar.gz youtube-dl源代码存档仍然可以从任何Gentoo Linux发行版文件镜像(如mirror.yandex.ru/gentoo-Distfiles/(youtube-dl-2020.09.20.tar.gz))获得。

The youtube-dl developers can follow the DMCA legal process and appeal GitHub's decision or move on to another hosting provider who aren't as willing to remove free software source code if someone asks them to.

YouTube-dl开发者可以遵循DMCA的法律程序，对GitHub的决定提出上诉，或者转向另一个主机提供商，如果有人要求他们删除自由软件源代码，他们也不愿这样做。

使用 Amnesty 的工具检查您的手机是否有 Pegasus 间谍软件

2021-07-22 21:36:45

国际特赦组织是帮助打破记者和国家元首成为 NSO 政府级间谍软件 Pegasus 目标消息的组织的一部分，它发布了一个工具来检查您的手机是否受到影响。该工具旁边是一组很棒的说明，它们应该可以帮助您完成一些技术检查过程。使用该工具涉及将手机备份到单独的计算机并对该备份运行检查。如果您在新闻发布后一直盯着手机看，并且正在寻找使用 Amnesty 工具的指南，请继续阅读。首先要注意的是该工具是基于命令行或终端的，因此运行需要一定的技术技能或耐心。我们试图涵盖很多您在此处启动和运行所需的知识，但在开始之前需要了解一些内容。第二个注意事项是 Amnesty 正在运行的分析似乎最适合 iOS 设备。 Amnesty 在其文档中表示，其工具可以在 Android 手机备份上运行的分析是有限的，但该工具仍然可以检查潜在的恶意 SMS 消息和 APK。同样，我们建议遵循其说明。要检查您的 iPhone，最简单的开始方法是使用 iTunes 或 Mac 或 PC 上的 Finder 进行加密备份。然后，您需要找到该备份，Apple 提供了相关说明。 Linux 用户可以按照 Amnesty 的说明使用 libimobiledevice 命令行工具创建备份。备份手机后，您需要下载并安装 Amnesty 的 mvt 程序，Amnesty 还提供了相关说明。如果您使用 Mac 运行检查，则首先需要安装 Xcode（可从 App Store 下载）和 Python3，然后才能安装和运行 mvt。获取 Python3 的最简单方法是使用名为 Homebrew 的程序，该程序可以从终端安装和运行。安装这些之后，您就可以开始运行 Amnesty 的 iOS 说明了。你需要确保你的 iPhone 的备份是用密码加密的

如果您在尝试解密备份时遇到问题，您并不孤单。当我试图将它指向我的备份时，该工具给了我错误，它位于默认文件夹中。为了解决这个问题，我将备份文件夹从该默认位置复制到我桌面上的一个文件夹中，并将 mvt 指向它。我的命令最终看起来像这样：（仅用于说明目的。请使用 Amnesty 说明中的命令，因为程序可能已更新。）运行实际扫描时，您需要指向妥协指标文件，国际特赦组织以名为 pegasus.stix2 的文件形式提供。那些刚开始使用终端的人可能会对如何实际指向一个文件感到困惑，但只要你知道文件在哪里，它就相对简单。对于初学者，我建议将 stix2 文件下载到 Mac 的下载文件夹。然后，当您到达实际运行 check-backup 命令的步骤时，添加到选项部分。作为参考，我的命令最终看起来像这样。 （同样，这仅用于说明目的。尝试复制这些命令并运行它们将导致错误）：（作为参考， ~/ 或多或少充当用户文件夹的快捷方式，因此您不要必须添加诸如 /Users/mitchell 之类的内容。）同样，我建议您遵循 Amnesty 的说明并使用其命令，因为该工具总是有可能已更新。 Twitter 上的安全研究员 @RayRedacted 也有一个很好的主题，介绍了您在运行该工具时可能遇到的一些问题以及如何处理这些问题。调查没有发现美国手机被 Pegasus 入侵的证据

最后，Amnesty 仅提供在 macOS 和 Linux 系统上安装该工具的说明。对于那些希望在 Windows 上运行它的人，The Verge 已经确认可以通过安装和使用 Windows Subsystem for Linux (WSL) 并遵循 Amnesty 的 Linux 说明来使用该工具。使用 WSL 需要下载和安装 Linux 发行版，比如 Ubuntu，这需要一些时间。但是，它可以在您等待手机备份时完成。运行 mvt 后，您将看到一个警告列表，其中列出了可疑文件或行为。值得注意的是，警告并不一定意味着您已被感染。对我来说，一些完全光明正大的重定向出现在它检查我的 Safari 历史记录的部分（sheets.google.com 重定向到 docs.google.com，reut.rs 重定向到 reuters.com 等）。同样，我也遇到了一些错误，但这只是因为该程序正在检查我手机上未安装的应用程序。围绕 Pegasus 的故事可能让我们中的许多人对我们的手机产生了比平时更多的怀疑，无论我们是否可能成为民族国家的目标。虽然运行该工具可以（希望）有助于缓解一些担忧，但对于许多美国人来说，这可能不是必要的预防措施。据《华盛顿邮报》报道，NSO Group 曾表示，其软件不能用于带有美国号码的手机，而且调查没有发现任何证据表明 Pegasus 成功入侵了美国手机。虽然很高兴看到 Amnesty 以可靠的文档提供了这个工具，但它确实有助于解决围绕 Pegasus 的隐私问题。正如我们最近看到的，政府不需要针对你手机的麦克风和摄像头来获取私人信息——即使你的手机没有 Pegasus，数据经纪人行业也可能会出售你的位置历史记录。

新型M是美国制造的键盘，可为您的打字增添弹性

2021-02-18 16:54:40

1984年是计算领域具有里程碑意义的一年。当然，这是Macintosh的首次亮相，但它也催生了另一款永恒的计算机硬件：IBM Model M键盘，Matt Neuburg认为这是有史以来最伟大的键盘（请参阅“有史以来最伟大的计算机键盘”）。 ？”（2009年2月27日）。

M型代表了为满足打字机用户而设计的悠久的键盘时代，其特征在于令人满意的可点击的翘曲弹簧键。 1992年，IBM将大部分键盘制造业务转移给利盟，后者继续为Big Blue生产Model M键盘。最终，该行业转向了像橡胶穹顶这样的更便宜，更笨拙的机制，而利盟和IBM在1990年代后期退出了M型业务。但是，一群利盟员工购买了位于肯塔基州列克星敦的工厂，并组建了Unicomp。

Clicky键盘曾经是一种小众产品，近年来又重新流行起来。现在它们如此流行，主流公司提供了机械“游戏”键盘，您可以在许多实体店甚至沃尔玛那里购买。同时，自从苹果公司明显倒闭以来，Unicomp就一直在生产那些经典的Model M clicky键盘。

Unicomp甚至生产Mac专用的型号，Spacesaver M，但我不愿意推荐它。这是因为，多年来，旧的Lexmark工具已经破旧，导致出现了有关可靠性问题以及装配和加工不良的广泛报道。最近，Unicomp投资购买了全新的模具，以在25年中制造出第一个全新的Model M，简称为New ModelM。

我最近发现自己需要新的键盘。虽然Model M并不是我的第一个键盘，但它却是我在高中打字课上使用的键盘，从那以后，我就一直渴望那种那种轻快，弹力十足的打字快感。

由于COVID-19大流行以及随后的停工对小型企业造成了灾难性的影响，我将新年做为一项新的决议，将为支持本地小型企业提供更多支持。 Unicomp工厂位于我家北部3个小时的地方，因此我认为，即使我对键盘不满意，但我至少会尽自己的一份力量来支持仍在该公司生产产品的少数计算机外围设备制造商之一。美国。

我差点订购了带有Windows键和PC键盘的所有存货的New ModelM。毕竟，我并不太在乎键盘，因此，如果我的Command键上印有Saint Hannes十字勋章，Windows徽标或Linux企鹅，我也不会在意。无论如何，您都可以换下键帽，Unicomp出售各种键帽。

我的挂断是沿着顶部行的Mac专用键，可让我控制亮度，音量，任务控制和其他功能。加上基本型号缺少Apple键盘上的功能键。当然，我可以重新编程以代替它，但是我决定使用Mac布局的键盘可以大大简化我的生活。

我在12月中旬联系了Unicomp支持人员，以查看他们是否愿意为我建造一个与Spacesaver M布局相同的New ModelM。我知道Unicomp受到COVID-19的严重打击，从而延迟了即将推出的Mini M键盘原定于2020年3月，所以我没有寄予希望。但是，他们很快做出回应，告诉我他们可以在1月中旬制作一个自定义键盘，然后再次与他们联系。

快闪到1月中旬，我再次与Unicomp支持联系，以了解他们是否可以构建我的自定义New ModelM。他们说可以，并提供了有关如何订购它的说明：

结帐过程非常笨拙，定制和运输后的总成本为145.87美元。这很昂贵，但仅比目前市场上的许多高端机械键盘高。

有人告诉我，组装和运输键盘可能需要最多两周的时间，但由于运输距离很近，它仅在三天内就交付了，并在第二天就到达了。

我现在输入的键盘可能非常独特：Mac布局的New ModelM。

New Model M的实用工业设计不会赢得任何时尚大奖。它具有简单的黑色底座以及白色和灰色按键组合。 Mac的变种非常裸露。它包括您在全尺寸Mac键盘中期望的所有键-命令，选项，功能，亮度等，但不要在键帽上看到Susan Kare设计的图标。诸如Command和Exposé之类的特殊键具有简单的文本标签，而亮度，音量和媒体键则具有通用的象形文字。

就尺寸而言，New Model M是野兽。它宽17.9英寸（45.5厘米），深7.5英寸（19厘米），高1.96（5厘米）英寸，重3磅11盎司（约1.7千克）。数字键盘使键盘特别宽，因此，如果您将键盘和定位设备并排放置，可能会造成妨碍。我更喜欢将键盘放在膝盖上，所以这对我来说不是问题。

New Model M可以与我的iMac一起使用。音量，亮度和媒体键效果很好，Exposé也是如此。我必须对Dashboard键进行编程，以在“系统偏好设置”＆gt;中触发启动板。键盘＆gt;快捷方式（有趣的是，仪表板键注册为F12）。

奇怪的是，Unicomp用Eject替换了Insert键，直到我尝试在ThinkPad上键入内容并按DVD并弹出DVD托盘时，我才注意到这一点。尽管这是全尺寸Mac键盘的标准配置，但另一个奇怪之处是数字键盘上没有Num Lock键。而是由清除键代替。另外，我对手指枪的小象形文字在“删除”键上杀死字母“ a”感到很满意。

如果您是在标准布局之间争论，还是在Mac布局上多花了20美元，我认为如果您要使用键盘的主计算机是Mac，这笔额外的钱是值得的。

让我们谈谈真正重要的是什么：在New Model M上打字感觉如何？很难在文本中传达这种经验，但我会尽力而为。

请务必注意，M型不是机械键盘！取而代之的是，它使用了一种弯曲弹簧机制，该机制比传统的机械键盘机制要古老得多。两者之间的主要区别在于弹簧的大小。在屈曲的弹簧键盘中，您按下一个键帽，该键帽将按下一个直接位于促动按键的水平面上的弹簧。在机械键盘中，一点点塑料就能操纵按键，而弹簧只需将按键返回到其原始位置即可。

在屈曲弹簧机构中，弹簧是秀场上的明星。如果拔下其中一个键帽，您会看到弹簧固定在孔中。

功能上的区别在于，翘曲键的弹簧具有某种“弹力”，而机械键则没有。很难传达，但是您键入时按键会发出轻微的“刺耳”的声音。虽然不是绝对安静，但声音不是特别大。我不担心孩子们在隔壁房间里醒来时醒来，它似乎比亚当·恩格斯特（Adam Engst）可靠的老式Das键盘安静，当我们在Slack通话时，这听起来像是在踩马。我录制了一段在New Model M上打字的视频，因此您可以感觉到声音。

屈曲弹簧因需要大量的力来致动而闻名。我之前很担心，因为反复按下按钮时前臂肌肉会变得很紧张，这就是为什么我使用立式鼠标并且很少再在计算机上玩游戏的原因（请参阅“ Anker的立式鼠标可提供廉价的RSI缓解功能”，7 2018年12月）。但是，我对新模型M所需的启动力丝毫没有感到困扰。

实际上，我发现按键非常轻巧。弹簧具有足够的抵抗力，因此您不会仅通过触摸键帽就可以意外激活键，但是一旦您决定按下该键，它就会在如此令人满意的咔嗒声之前移动。我估计实际的激活行程与Apple的Magic Keyboard大致相同。

单击和按键触底之间的额外行程有助于缓冲手指。当我用Apple键盘打字时，有时指尖会不断地撞到键盘上，这使我感到痛苦。我对New Model M没有这种痛苦。

当我在键盘之间时，我使用了超级便宜的沃尔玛键盘，它是如此糊状，就像在土豆泥上打字一样。这对我的前臂造成了压力，因为感觉就像我的手指在泥泞中工作。我不能说New Model M消除了我的前臂疼痛，但在我尝试过的所有键盘中，这种疼痛加重幅度最小。

我喜欢用此键盘打字，但没有完美的选择。首先，虽然美学是主观的，但我发现新模型丑陋。但是，比起整体美观而言，更令人反感的是为Caps Lock和功能指示灯使用了明亮的蓝色LED。也许只是我的眼睛很敏感，但我发现蓝色LED太亮了，这些天它们在计算机组件中已完全被过度使用。

延迟已久的Mini M（如果曾经发售）将具有我希望New Model M具有的一项功能：可拆卸的USB电缆。以我的经验，像这样的长期使用物品，首先要使用的是电缆（例如，我的2007年索尼MDR-V6耳机上的电缆），因此能够轻松更换它是一个很好的选择。

关于新模型M的一些事情要比我在高中使用的旧模型M逊色。当我挤压键盘的两侧时，会有轻微的吱吱声。最初的M型女士坚如磐石，大概是因为它们是用石头雕刻而成的。但是，这在键入时没有任何区别。另一个降级是使用单件式键帽。较旧的Model M型号在每个按键上都有一个外壳，可以轻松打开和关闭，因此您可以更改按键的外观，而不必卸下整个按键，也不会弄乱弹簧机构。在新型号M上，您必须删除整个密钥。同样，这是一个无所作为。

从长远来看，我可能遇到的最后一个小问题是Command键上的尖锐边缘。由于我的左手拇指的一侧经常碰到钥匙的角，因此边缘会刺激我的拇指。最好在Control，Option和Command键上倒角。

如果我不喜欢New Model M，那么我至少可以安慰自己，因为我已经向一些小型的本地公司投入了一些钱。但是我喜欢它！虽然它不会赢得选美比赛或视频游戏比赛的胜利，但打字却是一种乐趣。我每天都在打字。

有一种称为林迪效应的理论，该理论指出，不易腐烂的事物持续的时间越长，持续的时间就越长。例如，如果某本书已出版50年，则可以合理地假设它会再出版50年。通过这种测量，拥有37年历史的Model M确实是“ lindy”。这是永恒的设计，我很高兴Unicomp继承了这一传统。

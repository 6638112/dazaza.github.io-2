Microsoft now lets Word for web and OneNote users embed Pinterest Pins into their documents as interactive content
微软现在允许Word for Web和OneNote用户将Pinterest Pins作为交互式内容嵌入到他们的文档中

2021-02-25 02:16:10

Microsoft is now allowing Word for web and OneNote users to embed Pinterest Pins into their documents. The new feature lets Pinterest users paste a URL of any Pin into a OneNote page or Word document to render it as interactive content. This adds a new way to create pages, sections, or notebooks full of Pinterest content in OneNote.

Microsoft现在允许Word for Web和OneNote用户将Pinterest Pins嵌入到他们的文档中。新功能使Pinterest用户可以将任何Pin的URL粘贴到OneNote页面或Word文档中，以将其呈现为交互式内容。这增加了一种在OneNote中创建充满Pinterest内容的页面，部分或笔记本的新方法。

The embeds also work in Word, but only on the web version right now. You can embed Pinterest content into a Word document online, and it will be displayed when you open the document on the mobile or web versions of Word.

嵌入内容也可以在Word中使用，但目前仅在Web版本上可用。您可以将Pinterest内容在线嵌入到Word文档中，并且在Word的移动版本或Web版本上打开该文档时将显示该内容。

Microsoft is targeting this new feature at schools and students, particularly the OneNote embeds for Pinterest. “Over the past few years, we have heard many requests for a Pinterest integration with OneNote,” explains Mike Tholfsen, a product manager on Microsoft’s education team.

微软正针对学校和学生推出这项新功能，特别是为Pinterest嵌入的OneNote。 “在过去的几年中，我们听到了许多与OneNote集成Pinterest的请求，”微软教育团队的产品经理Mike Tholfsen解释说。

The Pinterest support for Word and OneNote follows similar integration in Microsoft’s Edge browser. Microsoft built Pinterest support directly into its new Collections feature inside Edge last year, allowing Pinterest users to export these collections of links, images, and text directly to Pinterest.

对Word和OneNote的Pinterest支持在Microsoft Edge浏览器中进行了类似的集成。微软去年在Edge内部的新“收藏”功能中直接内置了Pinterest支持，允许Pinterest用户将这些链接，图像和文本的收藏直接导出到Pinterest。

Microsoft has also reportedly been trying to acquire Pinterest recently. The Financial Times reported earlier this month that the software maker had approached Pinterest in recent months about a deal but that talks aren’t currently active.

据报道，微软最近还试图收购Pinterest。英国《金融时报》本月早些时候报道说，这家软件制造商最近几个月曾就一项交易与Pinterest进行过接触，但目前谈判并不活跃。

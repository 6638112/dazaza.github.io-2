在Runcode上解决编码挑战。忍者在线比赛，11月6-9日

2020-10-30 20:06:44

Annual programming competition Runcode.ninja is back again in its fourth year, beginning Friday, Nov. 6. RunCode is a nonprofit organization staffed by volunteers working in their spare time and focused on providing educational opportunities for coders and infosec folks. The online event allows programmers of all experience levels to tackle a wide array of challenges, using any of 14 supported programming languages.

一年一度的编程大赛Runcode.Ninja再次进入第四个年头，从11月6日(星期五)开始。RunCode是一个非营利性组织，由业余时间工作的志愿者组成，专注于为程序员和信息安全人员提供教育机会。在线活动允许所有经验水平的程序员使用14种受支持的编程语言中的任何一种来应对广泛的挑战。

This year, the competition theme is "all things web"—which means that most challenges will have something to do with websites; although the "something" can vary pretty drastically, from user interaction to server log analysis. The event will have more than 30 available challenges, grouped into easy, intermediate, and hard, for competitors to find and upload solutions for.

今年的比赛主题是万物互联，这意味着大多数挑战都与网站有关，尽管从用户互动到服务器日志分析，这些挑战可能会有很大的不同。该活动将有30多个可用挑战，分为简单挑战、中级挑战和困难挑战，供参赛者查找和上传解决方案。

For each challenge, competitors will be given a problem description, a sample data set, and an expected output to make the desired order and formatting clear. Competitors are expected to generate more test data of their own and thoroughly verify the correctness of their code against all the corner cases they can think of; solutions tested against only the sample data provided will likely fail the challenge. Uploaded code is run in a sandboxed Docker container and its output tested for correctness.

对于每项挑战，将向参赛者提供问题描述、样本数据集和预期输出，以使所需的顺序和格式清晰。预计竞争对手将生成更多自己的测试数据，并针对他们能想到的所有角落情况彻底验证其代码的正确性；仅针对提供的样本数据测试的解决方案很可能无法通过挑战。上传的代码在沙盒Docker容器中运行，并测试其输出的正确性。

Amazon gift cards ($100, $200, $300, $400, and $500) will be awarded to this year's top 5 finishers. The event begins Friday, Nov. 6, at 4pm EST and runs through Monday, Nov. 9, at 8am EST. Interested coders and would-be coders can sign up at https://2020.runcode.ninja/quick_reg—which is an entirely different registration from the one on the main site, which allows you to view and try to solve challenges whenever you'd like.

亚马逊礼品卡(100美元、200美元、300美元、400美元和500美元)将颁发给今年排名前5位的获奖者。该活动于美国东部时间11月6日(星期五)下午4点开始，一直持续到11月9日(星期一)美国东部时间上午8点。感兴趣的程序员和潜在的程序员可以在https://2020.runcode.ninja/quick_reg上注册-这是一个与主网站上的注册完全不同的注册，允许您随时查看并尝试解决挑战。

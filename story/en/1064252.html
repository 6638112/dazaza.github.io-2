Briar 1.3 released – image attachments, profile images and disappearing messages
BRIAR 1.3发布 - 图像附件，档案图像和消失的消息

2021-06-08 12:58:58

The Briar Project released version 1.3 of its Android app today.Thanks to support from eQualit.ie, this release adds several new features that have been requested by many users over the years.

The Briar Project今天发布了其Android应用程序的1.3版。谢谢来支持Companit.ie，此版本增加了多年来许多用户要求的几个新功能。

With today’s release, users can upload profile pictures that will be visible only to their contacts.

通过当今的发布，用户可以上传只能可见的配置文件图片。

Lots of people have asked for a way to send images via Briar. We listened! This release adds the ability to send images in private conversations.Images are still heavily compressed, so high resolution images might show pixel artifacts.

很多人都要求通过Briar发送图像。我们听了！此版本增加了在私有对话中发送图像的能力.Images仍然严重压缩，因此高分辨率图像可能显示像素伪影。

Also, disappearing messages are now available in private conversations.Users can set a timer on their messages, and they will be automatically deleted when the timer expires.We hope this feature will go a long way towards improving users’ privacy, especially for people communicating in high-risk areas.

此外，现在可以在私人对话中消失消息。用户可以在其邮件中设置计时器，当计时器到期时，它们将自动删除。我们希望此功能能够实现沟通人的隐私，特别是对于人们沟通在高风险区域。

To improve the user experience, the settings screen has been re-organized and split into sub-screens. The redesign creates space for the user’s profile picture, and the improved categorization into sub-screens makes it easier to find specific settings.

为了提高用户体验，可以重新组织和分成子屏幕的设置屏幕。重新设计为用户的个人资料图片创建了空间，并且将改进的分类转换为子屏幕使其更容易找到特定设置。

Work on the next big release is already underway. Among other things, the team is working on a way to exchange messages over storage media (SD card or USB stick) as well as sharing the app offline.

在下一个大发行版上已经进行了工作。除此之外，该团队正在努力在存储媒体（SD卡或USB棒）上交换消息以及离线共享应用程序。

Briar is a messaging app designed for activists, journalists,and anyone else who needs a safe, easy and robust way to communicate.Unlike traditional messaging tools such as email, Twitter or Telegram,Briar doesn’t rely on a central server -messages are synchronized directly between the users’ devices.If the internet’s down,Briar can sync via Bluetooth or Wi-Fi, keeping the information flowing in a crisis.If the internet’s up, Briar can sync via the Tor network, protecting usersand their relationships from surveillance.

Briar是一个用于活动家，记者和其他任何人需要一种安全，简单且强大的方式来沟通的消息传递应用程序。诸如电子邮件，推特或电报等安全，轻松和强大的方式，如电子邮件，推特或电报，Briar不依赖于中央服务器-Messages在用户的设备之间直接同步。如果互联网的下降，Briar可以通过蓝牙或Wi-Fi同步，使信息在危机中流动。如果互联网上，Briar可以通过TOR网络同步，保护用户和他们的关系从监视中保护用户的关系。

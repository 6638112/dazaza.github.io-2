ClearURLs – automatically remove tracking elements from URLs
clearurls - 自动从URL删除跟踪元素

2021-05-05 20:54:11

ClearURLs is an add-on based on the new WebExtensions technology and is optimized for Firefox and Chrome based browsers.

Clearurls是基于新的WebExtsions技术的加载项，并针对Firefox和基于Chrome的浏览器进行了优化。

This extension will automatically remove tracking elements from URLs to help protect your privacy when browse through the Internet,which is regularly updated by us and can be found here.

此扩展将自动从URL中删除跟踪元素，以帮助保护您的隐私，当浏览互联网时，该互联网经常更新，可以在此处找到。

Many websites use tracking elements in the URL (e.g. https://example.com?utm_source=newsletter1&utm_medium=email&utm_campaign=sale) to mark your online activity.All that tracking code is not necessary for a website to be displayed or work correctly and can therefore be removed—that is exactly what ClearURLs does.

许多网站在URL中使用跟踪元素（例如https://example.com?utm_source=newsletter1& um_medium=email & um_campaign=sale）来标记您的在线活动。所有跟踪代码不需要显示或正常工作，因此可以删除 - 这正是Clearurls所做的。

You can choose between two options to contribute. You can create a merge request, or you can use the POEditor to translate ClearURLs.

您可以选择两个选项以贡献。您可以创建合并请求，或者您可以使用Poeditor翻译Clearurls。

Hint: The description field in the translation files are only an information for what the translation is used.It is not necessary to translate the description field; in the most cases it is empty.

提示：翻译文件中的描述字段只是介绍了所使用的翻译的信息。不需要翻译描述字段;在大多数情况下，它是空的。

If you want to create a merge request, you must open the path _locales/en/messages.json in the ClearURLs repoand translate the english terms into terms of your language. Once you have translated all the terms, you make a pull request of your translation.Please push your translation into the folder _locales/{language code}/messages.json.

如果要创建合并请求，则必须打开Clearurls中的路径_locales / en / messages.json repoand将英文条款转换为您的语言。一旦翻译所有条款，您会提取您的翻译请求。请将翻译推入文件夹_locales / {language代码} /messages.json。

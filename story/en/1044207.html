即将发布的Windows 10X最终版与Chrome操作系统非常相似，“开始”按钮位于任务栏的中央，缺少Live Tiles，更多

2021-01-15 20:39:37

After years of waiting, it looks like Microsoft now has a true answer to Chrome OS. A new and near-final version of Windows 10X has leaked, and it offers a first look at the changes Microsoft has made to the upcoming operating system to get it ready for laptops. Windows 10X first started off life as a variant of Windows 10 designed for dual-screen devices. It was supposed to launch alongside Microsoft’s Surface Neo, a tablet-like device with two separate nine-inch displays that fold out to a full 13-inch workspace.

经过多年的等待，看起来微软现在对Chrome OS有了真正的答案。 Windows 10X的一个接近最终版本的新版本已经泄漏，它首次展示了Microsoft对即将到来的操作系统进行的更改，以使其可用于笔记本电脑。 Windows 10X最初是为双屏设备设计的Windows 10变体，从此开始。它应该与微软的Surface Neo一同发布，Surface Neo是一款平板电脑式设备，具有两个独立的9英寸显示屏，可以折叠成一个完整的13英寸工作区。

Microsoft revealed last year that Windows 10X is now being reworked for “single-screen” devices like laptops, and Surface Neo has been delayed. While the company has spent years differentiating Windows 10X for foldable and dual-screen hardware, it now looks and feels more like Chrome OS than ever before.

微软去年透露，Windows 10X现在正在针对笔记本电脑等“单屏”设备进行重新设计，而Surface Neo已被推迟。该公司花了多年的时间为可折叠和双屏硬件区分Windows 10X，但现在看起来和感觉都比以往任何时候都更像Chrome操作系统。

The latest version that has leaked is a near-final form of what will ship on Chromebook-like devices later this year. Windows 10X isn’t an operating system you will upgrade to, or an update that will appear for existing machines. It will only ship on new hardware, and it’s a lightweight version of Windows designed for those who might be tempted to buy a Chromebook instead.

泄漏的最新版本是今年晚些时候将在类似Chromebook的设备上发布的最终版本。 Windows 10X不是您要升级到的操作系统，也不是现有计算机上将出现的更新。它仅在新硬件上提供，它是Windows的轻量级版本，专为可能打算购买Chromebook的用户设计。

Windows 10X already looked a little Chrome OS-like on dual-screen devices, but it’s even more apparent now that it’s designed for single-screen laptops instead. Microsoft has simplified almost every area of Windows inside Windows 10X, to the point where it feels like a portal to the web rather than a portal to Windows apps.

Windows 10X在双屏设备上看起来已经有点像Chrome操作系统，但是现在它更是为单屏笔记本电脑而设计。微软已经简化了Windows 10X中Windows的几乎每个区域，以至于感觉就像是Web的门户，而不是Windows应用程序的门户。

The Start button is now centered on the taskbar, with a Start menu that acts like a launcher. It doesn’t have the Live Tiles found on Windows 10; instead, it’s a list of apps and recent documents. You can also search for apps, documents, or even content on the web through Microsoft’s Bing search engine.

现在，“开始”按钮位于任务栏的中央，带有类似于启动程序的“开始”菜单。它没有在Windows 10上找到的Live Tiles；相反，它是应用程序和最新文档的列表。您还可以通过Microsoft的Bing搜索引擎在网络上搜索应用程序，文档甚至内容。

The comparisons to Chrome OS are immediately obvious here, with a launcher and the focus on web apps. You can even install Progressive Web Apps (PWAs) from the built-in Chromium Edge browser and pin them to the taskbar.

与Chrome OS的比较在这里很明显，它具有启动器并专注于Web应用程序。您甚至可以从内置的Chromium Edge浏览器安装渐进式Web应用程序（PWA），然后将其固定到任务栏。

This Windows 10X taskbar is also an area of the OS that has been massively simplified. You might be used to seeing lots of system tray icons from apps, but these are nonexistent in Windows 10X. Instead, there’s a simple Action Center that displays the time and pops out to reveal quick settings like shutting a machine down, enabling a VPN, connecting displays, or managing the volume. This also includes access to notifications and an updated widget to control media playback.

Windows 10X任务栏也是操作系统的一个已大大简化的区域。您可能习惯于从应用程序中看到很多系统任务栏图标，但是Windows 10X中不存在这些图标。取而代之的是，有一个简单的操作中心，可以显示时间并弹出以显示快速设置，例如关闭计算机，启用VPN，连接显示器或管理音量。这还包括对通知的访问以及用于控制媒体播放的更新的小部件。

Windowing in Windows 10X has also been simplified compared to Windows 10. You can drag and drop apps to run them side by side, but there’s no way to allow apps to freely resize. As a result, the middle button you’re used to seeing in the top right-hand side of apps (which switches between full-screen and windowed modes) has been removed in 10X. You can simply minimize apps or close them here. Laptops that ship with Windows 10X will also have gesture trackpad support for multitasking apps.

与Windows 10相比，Windows 10X中的窗口化也得到了简化。您可以拖放应用以使其并排运行，但是无法允许应用自由调整大小。因此，您以前在应用程序右上角看到的中间按钮（在全屏模式和窗口模式之间切换）已被删除10倍。您可以简单地最小化应用程序或在此处关闭它们。随Windows 10X一起提供的笔记本电脑还将为多任务应用程序提供手势触控板支持。

Microsoft has also been working on a modern version of File Explorer in Windows 10X. It’s primarily designed as a way to access OneDrive cloud files, but you can also manage downloaded files here or ones from USB storage. There’s no way to access local files in this new File Explorer, as 10X is primarily designed for cloud storage.

微软还一直在研究Windows 10X中现代版本的文件资源管理器。它的主要目的是访问OneDrive云文件，但您也可以在此处管理下载的文件或USB存储设备中的文件。由于10X主要用于云存储，因此无法在此新的文件资源管理器中访问本地文件。

It appears that Microsoft might not be shipping desktop app support in Windows 10X, at least initially. Microsoft has built a container technology to run traditional desktop apps in a lightweight virtual machine. This was originally designed to be used to ensure legacy apps worked properly with dual-screen devices and didn’t hit their battery life or interfere with how apps are displayed on multiple screens.

看起来，Microsoft可能至少在最初没有提供Windows 10X中的桌面应用程序支持。 Microsoft已经建立了一种容器技术，可以在轻量级虚拟机中运行传统的桌面应用程序。最初设计该功能是为了确保旧版应用程序可以在双屏设备上正常使用，并且不会耗尽电池寿命或干扰应用程序在多屏上的显示方式。

Microsoft is still working on this container technology, and it appears as a developer-only mode in this latest version of 10X. That could mean end users won’t be able to install regular desktop apps on Windows 10X when it ships. Either way, it’s clear Microsoft is focusing on a combination of apps from the Microsoft Store here as well as web apps.

Microsoft仍在研究此容器技术，在最新版本的10X中，它仅作为开发人员模式使用。这可能意味着最终用户将无法在Windows 10X出厂时在其上安装常规的桌面应用程序。无论哪种方式，显然微软都将重点放在此处Microsoft Store应用程序和网络应用程序的组合上。

In some ways Windows 10X feels like a preview of some of the UI and UX changes we’ll likely see in regular Windows 10 later this year. Microsoft is planning a “ sweeping visual rejuvenation of Windows,” which will include a modernized Start menu, File Explorer, and built-in apps that make the overall Windows 10 UI feel more consistent.

在某些方面，Windows 10X就像是一些UI和UX更改的预览，我们可能会在今年晚些时候的常规Windows 10中看到。微软正在计划“使Windows焕然一新”，其中将包括现代化的“开始”菜单，“文件资源管理器”和内置应用程序，这些应用程序将使整个Windows 10 UI更加一致。

Microsoft still hasn’t officially confirmed when Windows 10X will ship, or which laptop manufacturers will be launching devices with this OS variant. It’s clear this is a more direct attempt to challenge Chromebooks, after Microsoft has tried for years to awkwardly position Windows 10S as an alternative. That project failed and turned into a simple S Mode. In contrast, Windows 10X feels like a much bigger effort to recreate the simplicity of Chrome OS with the added benefit of Microsoft services and apps.

微软尚未正式确认Windows 10X何时发布，或哪些笔记本电脑制造商将启动具有此操作系统变体的设备。显然，这是微软在尝试Windows 10S替代品的尴尬多年之后，更直接地尝试挑战Chromebook。该项目失败并变成了简单的S模式。相比之下，Windows 10X则需要付出更多的努力来重新创建Chrome操作系统的简便性，同时还要增加Microsoft服务和应用程序的优势。

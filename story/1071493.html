苹果开始销售配备 Touch ID 的妙控键盘和新的 Mac Pro GPU

2021-08-05 21:58:59

Apple 正处于主要产品发布之间的缓慢时期，但周二标志着某些 Mac 用户可能会感兴趣的一些外围设备和附加组件的发布。这些产品包括配备 Touch ID 的妙控键盘（以前仅与 24 英寸 iMac 一起包装）和三个用于 Mac Pro 台式机的新显卡模块。您还可以在 Apple 商店找到 Magic Mouse（79 美元）和 Magic Trackpad（129 美元）的更新版本，但它们并没有太大的不同。例如，妙控板的形状略有不同，以更好地匹配变化更大的妙控键盘。每个更新的配件都配有 USB-C-to-Lightning 充电线。键盘具有全新的形状和附加功能，包括 Touch ID 指纹识别器。 Touch ID 是一种生物识别身份验证功能，以前仅在 Apple 的笔记本电脑和移动设备中可见。全新妙控键盘设计是今年春季推出的配备 M1 的 24 英寸 iMac 的专属设计。由于一体式桌面有多种颜色，因此键盘有相同的色调。但似乎独立的妙控键盘只是采用了许多 Apple 配件中常见的标准银色和白色图案。带 Touch ID 的妙控键盘售价 149 美元，不带 Touch ID 的键盘售价 99 美元。还有配备数字小键盘的非 Touch ID 和 Touch ID 键盘版本（分别为 129 美元和 179 美元）。 Mac Pro 远非大众市场的消费产品。它主要用于专业视频编辑托架等。但是，如果您是少数拥有它的人之一（或者如果您为拥有 IT 的团队管理 IT），那么您现在有一些新的升级选项。 Mac Pro 具有可升级的显卡，但它使用预配置的模块而不是现成的消费级 GPU。从今天开始，可以为 Mac Pro 购买三个新的图形模块。我们将不得不等待一些基准测试的出现才能获得有关性能的可靠见解，但每个模块都包括 4 个 Thunderbolt 3 端口和一个 HDMI 2.0 端口。 Mac Pro 可以容纳两个模块，因此如果您使用上面列出的 Duo 模块，您可以拥有多达 4 个 GPU。

随着这些新模块的推出，Apple 将不再提供 AMD Radeon Pro Vega II 或 AMD Radeon Pro Vega II Duo 模块作为新 Mac Pro 在购买时的配置选项。但是，它们仍将作为 Apple 现有客户的独立选项提供。注意：Ars Technica 可能会通过附属计划从该帖子上的链接中获得销售补偿。

SolidRun 1U 2节点Arm服务器

2021-03-01 22:41:15

本周，SolidRun在其基于NXP Layerscape LX2160A的系统中有了新的发展。新的SolidRun HoneyComb LX2服务器将他们熟悉的两个HoneyComb LX2K板包装到一个机箱中。其中包括电源和2.5英寸或3.5英寸SATA存储的容纳空间。尽管该系统核心板上的电路板不是新部件，但这确实标志着Arm系统在没有太多其他关注的层上的可部署性的里程碑。

退一步看一下该系统的着陆点，Arm生态系统目前在价格和性能范围的顶部和底部有很多活动，而在中间没有很多选择。这就是SolidRun新型双节点服务器旨在竞争的地方。

价格在5到100美元之间的Raspberry Pi系统已出货超过3000万台。这样的开发人员套件在学习和DIY空间中获得了一定程度的普及，因为它们使在编程或“物理计算”中尝试新概念变得非常容易。虽然Raspberry Pi基金会已经演示了在生产中使用Raspberry Pi 4的情况，包括运行Raspberry Pi 4s为其网站发布Pi 4提供服务，但这并不是他们设计的目的。在该示例中，基金会在性能更高的负载平衡器和数据库计算机之间包括了自己的硬件，以展示其功能强大的新设备，而不是像他们可能已替换的Xeon一样。

另一方面，距离几个数量级，Ampere Altra正在为某些客户永久替换其中一些至强处理器提供理由。 Ampere的性能价格比令人印象深刻，例如2个插槽服务器的160个内核和每个插槽128个PCIe Gen4通道的数字也令人印象深刻。这突显了这是一个专为云提供商和超大规模客户设计的平台。您可以查看STH的Ampere Altra Wiwynn山。 Jade Server评论以了解更多信息。

顶级Arm服务器引起了足够的兴趣，开发人员想知道是否没有更好的工作站级/入门级/中端服务器系统。潜在的Arm开发人员希望开发板100美元到每个插槽2U的价格在4000美元之间。 （对Ampere Altra系列另一端的800美元32核1.7GHz SKU表示歉意。）

SolidRun在这一领域已定位多年。其LX2产品继续积极开发。 HoneyComb的成熟度与SolidRun在两个方面所做的工作息息相关。它正在为自己的市场提供服务并推动特定的开发（例如，他们的高带宽网络），但返回他们的开发工作是为了使主线Linux易于采用。

HoneyComb LX2K是COM Express Type 7载体，用于NXP Layerscape LX2160A片上系统，具有16x Arm A72内核，运行频率高达2GHz。 COM Express Type 7模块在SODIMM中具有双通道DDR4，总容量高达64GB。载板具有4个SATA，一个M.2和一个Micro SD插槽用于存储，四个SFP +（直接来自SoC）和一个1Gb RJ45（用于联网），两个USB 3.0端口以及一个开放式PCIe x8插槽，这些插槽用于网络连接。使用标准ATX电源的标准迷你ITX外形尺寸。甚至还有ATX标准电源指示灯和重置按钮标头。目标似乎是创建一个易于集成到现有机箱和系统中的平台。

在运行高度可部署的服务器以替换Xeon的情况下，这可能与硬件成熟度水平不完全相同。另一方面，拥有一个可以部署在机架中并进行Arm开发的双节点1U平台是一大进步。我们不仅获得了更大的CPU，而且还具有使用许多标准组件的能力，这也将有助于增强生态系统。添加带有散热和电源的机箱似乎没什么大不了的，但是市场的很大一部分都不希望搜索或拼凑这些解决方案。相反，如果节点已经在1U机架中，则可以减少摩擦和进入开发的障碍。这仍然不是400-500美元的PC或Atom服务器。尽管如此，这仍是朝正确方向迈出的重要一步。

SolidRun的HoneyComb LX2K是一个可访问的平台，能够在64位Arm架构上作为工作站进行开发。对于具有1U机箱的设备型功耗敏感边缘应用程序，它也变得更加易于部署。对在生产中使用HoneyComb板感兴趣的开发人员面临着尝试将ITX板机架安装的挑战。英国的Bamboo Systems为准备部署8节点1U的开发人员提供了一个解决方案，但这提高了Ampere Altra的规模。小型服务器节点的挑战在于，随着规模的扩大，拥有更大的节点往往会更加高效。 AWS Gravition2是该行业的重要产品，但它也与AWS生态系统联系在一起。可以在1U中部署两个节点的新可用性针对的是那些有兴趣在野外狂奔Arm的人们的可访问性和能力的交叉点。

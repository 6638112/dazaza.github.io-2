EASTL：替代 C++ 标准库

2021-08-05 21:09:42

EASTL 代表电子艺界标准模板库。它是一个包含容器、算法和迭代器的 C++ 模板库，可用于跨多个平台的运行时和工具开发。它是这样一个库的一个相当广泛和健壮的实现，并且强调高性能高于所有其他考虑因素。如果您熟悉 C++ STL 或使用过其他模板化容器/算法库，您可能不需要阅读本文。如果您根本不熟悉 C++ 模板，那么您可能需要比本文档更多的内容来加快速度。在这种情况下，您需要了解模板如果使用得当，是轻松创建优化 C++ 代码的强大工具。 C++ 模板的描述超出了本文档的范围，但 Internet 上有大量此类文档。 EASTL 适用于 EASTL 功能有用的任何工具和运输应用程序。现代编译器能够使用模板生成良好的代码，并且许多人在从嵌入式系统到服务器和大型机的多个平台上的当代和下一代应用程序中使用它们。 Conan 中的 EASTL 包由 Conan 团队成员和社区贡献者保持最新。如果版本已过时，请在 Conan Center Index 存储库上创建问题或拉取请求。 vcpkg 中的 EASTL 端口由 Microsoft 团队成员和社区贡献者保持最新。如果版本已过期，请在 vcpkg 存储库上创建问题或拉取请求。 EASTL 由 Paul Pedriana 创建，他维护了该项目大约 10 年。 EASTL 随后由 Roberto Parolin 维护了 8 年多。他是 EASTL 开源的推动者和支持者。Rob 是团队所有成员的导师，并教会了我们我们想知道的关于 C++ 诡异的一切。

Max Winkler 是 EA 中当前的 EASTL 所有者和主要维护者，负责开源存储库。

Xbox All Access似乎是游戏界最划算的交易之一

2020-09-10 04:43:49

今天，微软公布了下一代游戏机的价格：从11月10日开始，Xbox系列S和Xbox系列X的价格分别为299美元和499美元。但微软也在讨论一项智能手机风格的计划，作为名为Xbox All Access的订阅计划的一部分，玩家可以免费获得这两种系统中的任何一种。

我们以前从未见过像这样的Xbox硬件首付计划--自2018年以来，Xbox One系列就已经推出了类似的All Access计划。但我们正在利用这个高调的场合提醒你一个关于Xbox All Access鲜为人知的事实：从长远来看，它可以为新的游戏机购买者节省资金，也可以预付费用。

首先，让我们计算一下数字。使用Xbox All Access，您承诺两年内支付24.99美元/月(对于Xbox系列S)或34.99美元/月(对于Xbox系列X)。作为对这一承诺的交换，你可以预先获得相关的硬件，并保留两年的Xbox Game Pass旗舰版订阅。

游戏通行证旗舰版通常每月收费14.99美元，所以你的所有访问权限月度付款最终会稍微高一点，以弥补之前免费的硬件。但是，除了不必一次性花费数百美元之外，所有接入用户实际上都可以在两年后脱颖而出。

因此，与单独购买游戏机和游戏通行证的玩家相比，所有接入订户都节省了一大笔钱，从长远来看，也节省了一些钱。从各方面考虑，这笔交易还不错。

在你跳上全通道列车之前，有几件事要牢记在心。

首先，只有当你对微软的游戏通行证终极版订阅感兴趣时，所有的访问显然才是值得的。然而，订阅已经是相当划算的了，它提供了Xbox历史上数百款游戏的可下载和XCloud流媒体访问，以及200多款供PC播放器使用的Windows游戏。游戏通行证订阅还包括微软游戏工作室发布当天的每一款游戏，现在还附带了整个EA Play库作为奖金。

不过，如果你更喜欢单独购买游戏，而不是受困于微软的订阅选择，那么你最好提前购买你的游戏机。

即使你认为Game Pass看起来很吸引人，但请记住，所有访问权限也会将您锁定在该订阅的整整两年内。如果您试用了Game Pass，发现您想在6个月、12个月或23个月后取消订阅，您仍将在整个两年的月度付款剩余时间内承担责任。如果您希望更灵活地对订阅计划进行采样，则并非所有访问权限都适合您。

同样重要的是要记住，游戏通行证终极版订阅经常在14.99美元/月的价目表价格上提供大幅折扣。与Xbox All Access计划相比，利用这些交易可以为精明的游戏机买家节省一点钱。

最后要记住的是，Xbox All Access并不是每个Xbox购买者都可以使用的。微软已经宣布了11个国家，参与其中的零售商将在这个假日季提供订阅服务，明年还计划提供更多：

即使在这些国家/地区，也不是每个客户都能使用All Access订阅。参与需要公民一号银行的信用检查和信用额度的批准，该银行正在与微软合作开展该项目。这一信贷额度不收取任何费用或利息，可以提前还清，不会受到处罚。尽管如此，过去有过信用问题的客户可能会发现，他们一开始就不能获得所有访问权限的批准。

一段时间以来，微软已经表示，它不再单纯以Xbox硬件的游戏机市场份额来衡量成功。Xbox首席执行官菲尔·斯宾塞(Phil Spencer)在2015年表示：我们的动机不是击败索尼，我们的动机是尽可能多地赢得客户。最近，斯宾塞表示，他认为亚马逊(Amazon)和谷歌(Google)等流媒体巨头是未来的主要竞争对手，而不是任天堂(Nintendo)和索尼(Sony)这样的公司。

所有的访问在很大程度上反映了这种新的关注点。微软没有要求预付款来抵消其硬件成本，而是通过将所有Access客户捆绑在一个为期两年的订阅计划中来进行长期博弈。对于微软来说，这两年的前期收入和整体收入的损失(与标准定价相比)对于让更多玩家习惯订阅游戏通行证的想法来说，是一笔很小的成本。在这两年里，微软的前期收入和整体收入的损失(与标准定价相比)是让更多玩家习惯于订阅游戏通行证的一小部分成本。

微软并不只是希望那些所有的接入用户在两年后忘记取消他们昂贵的月度订阅。一位微软代表告诉Ars Technica，一旦Xbox All Access附带的24个月Xbox Game Pass旗舰版订阅到期，您将需要购买新的Xbox Game Pass旗舰版会员或购买独立的Xbox Live Gold和Xbox Game Pass会员资格，才能继续享受Xbox Live Gold和Xbox Game Pass福利。

换句话说：在你记得告诉他们停止之前，微软不会强迫你实施自动续费计划，扣掉你的信用卡。24个月后，续签还是违约由你决定。

微软似乎相信，在体验了这项服务两年后，一定比例的All Access客户将会续签。然而，即使是那些不喜欢的人，也将被锁定在微软的游戏机生态系统中，而不是索尼的游戏机生态系统中，而不需要预先承诺数百美元。

与Game Pass相结合，Xbox All Access是我们在很长一段时间内考虑购买新一代游戏机硬件的方式中最有趣的变化之一。我们将饶有兴趣地关注这一免费的计划是否会在这个假日季吸引大量寻找新游戏机的消费者。

引入开放式Web文档

2021-01-28 21:54:58

Web平台技术的高质量文档是我们共享的开放式数字基础架构的至关重要的组成部分。今天，我们很高兴公开推出Open Web Docs，这是一个集体项目，旨在为技术作家群体提供有关战略创建和长期维护Web平台技术文档的支持，该文档对所有人开放。

创建Open Web Docs的目的是确保MDN Web Docs等事实上的标准资源上的Web平台文档的长期运行状况，而与任何单个供应商或组织无关。通过全职员工，社区管理以及我​​们的合作伙伴组织网络，我们使这些资源能够更好地维护和维护核心Web平台技术的文档。 Open Web Docs致力于通过我们的贡献来改善现有平台，而不是创建新的文档站点。

Florian Scholz于2020年11月作为我们的内容负责人加入该项目，与利益相关者一起确定初始工作流程。我们2021年的优先事项包括与Mozilla的MDN编写者和工程师合作，以支持最近的基础架构过渡，并确定优先顺序并推进关键文档工作，围绕核心Web技术文档，浏览器兼容性数据开发贡献者社区，并改善JavaScript文档。即将宣布更多员工，有兴趣者可以通过opencollective.com/open-web-docs、https://github.com/openwebdocs和@OpenWebDocs关注我们的更新。

Open Web Docs的工作人员得到了创始赞助商Coil，Google和Microsoft的支持，并获得了Igalia和Open Source Collective慷慨支持的额外资金支持。 Mozilla，三星和W3C提供额外的支持和参与。参与的组织通过每周的编辑和OWD指导委员会会议来合作进行内容工作，并且随着我们对工作的深入研究，有计划创建共享的流程。

微软在黑暗模式下回溯；默认情况下，Windows 11 将以浅色模式发货

2021-07-24 02:42:33

几天前，我们分享了一些让一些人感到不安的 Windows 11 新闻。在 Microsoft Inspire 上，宣布 Windows 11 将默认启用暗模式。这不是谣言或猜测，而是微软员工在微软官方活动中发布的公告。但现在发现关于默认暗模式的说法是不正确的。微软已与 BetaNews 取得联系以澄清。目前尚不清楚通信和消息传递出了什么问题，但 Microsoft Inspire 出了点问题。当 Windows 企业营销总监 Melissa Grant 说“我们将在 IT 最爱中发布所有 Windows 11 商业 SKU - 默认情况下漂亮的黑暗模式”时，她是错误的。我们知道，最近分享了有关 Windows 11 在暗模式下默认发送给所有商业 SKU 的不准确信息，并为造成的混乱道歉。澄清一下，默认情况下，Microsoft 将在浅色模式下提供 Windows 11 SKU。但是，原始设备制造商可以选择在暗模式下运送他们的设备，客户可以选择在“设置”中轻松自定义他们的体验到暗模式或亮模式。与我们所有的产品一样，我们将继续倾听客户的反馈，以确保 Windows 11 满足客户的需求，无论他们处于计算之旅的哪个阶段。正如在 Inspire 上明确表示的那样，用户仍然可以在明暗模式之间进行选择，但默认情况下每个人都会看到明暗版本。

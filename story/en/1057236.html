GCC 10.3 has been released
GCC 10.3已被释放

2021-04-09 16:45:08

The GNU project and the GCCdevelopers are pleased to announce the release of GCC 10.3.

GNU项目和GCCDevelopers很高兴地宣布发布GCC 10.3。

This release is a bug-fix release, containing fixes for regressions inGCC 10.2 relative to previous releases of GCC.

此版本是错误修复版本，其中包含相对于GCC的先前版本的回归10.2的修复。

GCC used to stand for the GNU C Compiler, but since the compilersupports several other languages aside from C, it now stands for theGNU Compiler Collection.

GCC用于代表GNU C编译器，但由于编译除了C的几种其他语言之外，它现在代表了Gnu编译器集合。

The GCC developers would like to thank the numerous people that havecontributed new features, improvements, bug fixes, and other changes aswell as test results to GCC.This amazinggroup of volunteers is what makes GCC successful.

GCC开发人员要感谢众多人，以为新功能，改进，错误修复以及其他更改，以及作为测试结果的测试结果。这是志愿者的惊人组成的是使GCC成功的原因。

For additional information about GCC please refer to the GCC project web site or contact the GCC development mailing list.

有关GCC的其他信息，请参阅GCC项目网站或联系GCC开发邮件列表。

For questions related to the use of GCC,please consult these web pages and the GCC manuals.

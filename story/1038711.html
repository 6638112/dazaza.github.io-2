Xbox云游戏服务将于2021年春季登陆iOS和Windows PC

2020-12-10 02:57:48

微软在今天的博客文章中概述了从即将推出的游戏到Xbox Series X / S计划的所有内容，微软宣布Xbox云游戏将在2021年春季登陆iOS移动设备和Windows PC。

在Windows PC上，游戏将通过Xbox应用程序或Web浏览器流式传输，而该服务将仅限于iOS设备上的移动Web浏览器。

微软的游戏流功能需要Xbox Game Pass Ultimate订阅，该订阅还包括一个Xbox平台和Windows PC的按需下载游戏的库，EA Play可下载游戏库以及Xbox Live Gold，Microsoft 39; s的在线多人游戏服务。 Xbox Game Pass Ultimate在美国的价格为每月14.95美元，但微软目前为新用户提供1美元的3个月折扣试用期。

此外，微软表示云游戏和Xbox Game Pass Ultimate将在春季进入新市场，包括澳大利亚，巴西，日本和墨西哥。

Xbox游戏流式传输已在Android设备上使用了几个月，并且有望最终在Windows PC上实现。但是它在iOS上的存在还远远不能确定。

苹果公司的App Store指南最初禁止使用任何类型的游戏流媒体服务，理由是对质量和非淫秽行为的策划是App Store运行方式的重要原则。苹果公司辩称，其内部审核员无法评估各种流媒体服务上的单个游戏，因此无法在App Store上提供诸如Microsoft或Google的竞争对手Stadia服务之类的服务。

微软和其他公司指出，苹果公司对待游戏的态度似乎与其他形式的媒体有所不同。库比蒂诺在其App Store上基于订阅的类似应用程序中并未出现电视节目，书籍，音乐，杂志或其他媒体的出现。它仅将这些限制应用于游戏。

经过激烈的公关斗争和新闻界的辩论后，苹果略微放松了限制，称诸如Xbox云流或Stadia之类的游戏订阅服务可以在iOS App Store上使用，前提是该服务中提供的每个游戏都列为单独的应用。 App Store，因此需要经过Apple的审查程序。 Google的Stadia宣布了完全放弃这一麻烦的计划，而是通过基于浏览器的体验来提供其游戏，而现在（如先前所传闻），微软已经走了同样的道路。

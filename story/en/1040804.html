level 2 So crafting a lot can be a major source of bloat. There was discussion of this over at the CB77 modding discord. It looks like crafting an item permanently increases the size of your save file in order to store the new item's info. However, selling or disassembling the item does not revert the size. It will still have the new chunk of space, but it will go unused by anything else.

2级因此，大量制作可能是造成膨胀的主要来源。在CB77改装不和谐中对此进行了讨论。制作商品似乎会永久增加保存文件的大小，以便存储新商品的信息。但是，出售或拆卸产品不会恢复尺寸。它将仍然具有新的空间，但是其他任何地方都不会使用。

While janky, this is good news because it means that borked saves could be repaired by a save game editor (WIP) in the future. Save editors have to decompress the save (LZ4 compression like The Witcher 3 used) parse the info into different categories, allow you to change a piece, then pack it all up again. There are multiple parts in the save file that have a header that contains a length key for the following section, so each of those need to get updated prior to cooking the modified save back into a normal save.dat. So, a lot of work left to do, but fixing a save bloated by crafting should be the easier of things to do.

虽然简陋，但这是个好消息，因为这意味着将来可能会由保存游戏编辑器（WIP）修复无用的保存内容。保存编辑器必须解压缩保存文件（像The Witcher 3一样使用LZ4压缩）将信息解析为不同的类别，允许您更改片段，然后重新打包。保存文件中有多个部分，这些部分的标题包含下一部分的长度键，因此在将修改后的保存内容重新制作为普通的save.dat之前，每个部分都需要进行更新。因此，还有很多工作要做，但是解决因手工制作而save肿的保存应该是一件容易的事。

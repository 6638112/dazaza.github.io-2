Valetudo is a cloud-free web interface for robot vacuum cleaners
Valetudo是用于机器人吸尘器的无云Web界面

2021-01-22 10:43:14

In my review of Kyvol Cybovac S31 LDS smart robot vacuum cleaner, I noted that app permissions meant both the map of your house and GPS coordinates may be sent to the cloud. But it was pointed out to me that earlier LDS robot vacuum cleaners from Roborock/Xiaomi were supported by Valetudo project that removes the need to connect to the cloud, and implements a mobile-friendly web interface as well as MQTT support for integration with Home Assistant or Node-RED.

在我对Kyvol Cybovac S31 LDS智能机器人吸尘器的评论中，我注意到应用程序权限意味着您的房屋地图和GPS坐标都可以发送到云中。但是向我指出的是，Valorudo项目为Roborock / Xiaomi的早期LDS机器人真空吸尘器提供了支持，该项目消除了连接到云的需求，并实现了移动友好的Web界面以及与Home Assistant集成的MQTT支持。或Node-RED。

The project is not a custom firmware for the robots, and instead, the stock firmware is patched with Valetudo which developers describe as an “alternative App implementation + mock cloud which runs on the robot itself.”

该项目不是针对机器人的自定义固件，而是使用Valetudo修补了库存固件，开发人员将其描述为“替代的App实现+在机器人本身上运行的模拟云”。

The good news is that you don’t necessarily need to teardown your vacuum cleaner to root it and install Valetudo, but it will depend on the model, and manufacturing date/firmware installed. The less good news is that the process can still be complicated, and only works on a few models including:

好消息是，您不一定需要拆除真空吸尘器才能生根并安装Valetudo，但这取决于型号和安装的生产日期/固件。不太好的消息是，该过程仍然很复杂，并且仅适用于以下几种模型：

Roborock V1 (Xiaomi Mi SDJQR02RR) with pre-March 2020  and post-March 2020 models requiring different installation methods.

具有2020年3月之前和2020年3月之后型号的Roborock V1（Xiaomi Mi SDJQR02RR）需要不同的安装方法。

If you’re lucky you can update the firmware over OTA (WiFi) using some commands in a Linux terminal, or a web tool called Dustbuilder.  In other cases, getting root access via the USB port, and for some models… well a full teardown is in order:

如果幸运的话，您可以使用Linux终端中的某些命令或称为Dustbuilder的网络工具通过OTA（WiFi）更新固件。在其他情况下，可以通过USB端口（对于某些型号）获得root用户访问权限，因此需要进行完整的拆除：

Once you’ve done the update the Xiaomi app will not work anymore, and you’d only access the robot vacuum cleaner via its web interface which, in most cases, comes with the same features as the mobile app minus cloud connectivity. However, if you change your mind, you can simply factory reset the device to remove Valetudo and continue with the Xiaomi app, at least on Roborock models.

完成更新后，小米应用程序将无法再使用，您只能通过其网络界面访问机器人吸尘器，在大多数情况下，该界面具有与移动应用程序相同的功能，但会减少云连接。但是，如果您改变主意，则只需恢复出厂设置即可删除Valetudo，然后至少在Roborock型号上继续使用小米应用程序。

Dark Mode Coming to GitHub After 7 Years
7年后，暗模式将进入GitHub

2020-11-29 10:22:28

This issue has been opened for seven years. You're a developer focused platform and a majority of developers use dark themes in nearly all their applications, including IDEs and text editors. I don't understand the rationale for not having a simple option when other websites like YouTube, Twitter, Reddit, etc have given people the option, even if opt-in. While some like Discord started out with light themes and made the dark themes default when they realized that that's actually what a majority of users wanted and used despite being opt-in. This seems like such a simple yet obvious feature request that would benefit people. Half the people in this comment thread are pointing out browser-specific plugins and workarounds to get what they want which will inevitably break if you make changes and then you'll just be hit with support requests for when things are not working because they used third party website-altering plugins. So if not for any other reason than to save yourself some support ticket costs, add what is a couple line CSS toggle for us.

Maddy: Composable all-in-one mail server in Go
Maddy：Go中可组合的多合一邮件服务器

2020-11-22 10:06:46

Maddy Mail Server implements all functionality required to run a e-mailserver. It can send messages via SMTP (works as MTA), accept messages via SMTP(works as MX) and store messages while providing access to them via IMAP.In addition to that it implements auxiliary protocols that are mandatoryto keep email reasonably secure (DKIM, SPF, DMARC, DANE, MTA-STS).

Maddy Mail Server实现了运行电子邮件服务器所需的所有功能。它可以通过SMTP发送邮件（作为MTA），可以通过SMTP接收邮件（作为MX），并在通过IMAP提供对邮件的访问权限的同时存储消息。此外，它还实施了辅助协议，这些协议对于保持电子邮件的合理安全性（DKIM， SPF，DMARC，DANE，MTA-STS）。

It replaces Postfix, Dovecot, OpenDKIM, OpenSPF, OpenDMARC and more with onedaemon with uniform configuration and minimal maintenance cost.

它用onedaemon替换了Postfix，Dovecot，OpenDKIM，OpenSPF，OpenDMARC等，具有统一的配置和最低的维护成本。

Note: IMAP storage is "beta". If you are looking for stable andfeature-packed implementation you may want to use Dovecot instead. maddy stillcan handle message delivery business.

注意：IMAP存储是“ beta”。如果您正在寻找稳定且功能丰富的实现，则可能要使用Dovecot。 maddy stillcan处理消息传递业务。

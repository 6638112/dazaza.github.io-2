微软将收购ZeniMax Media及其游戏发行商Bethesda Softworks

2020-09-21 21:32:29

标志性的游戏组合、出版专业知识和世界级人才加速了微软游戏业务的增长。

华盛顿州雷德蒙德。2020年9月21日-地球上超过30亿人为了好玩、逃避和人际联系而玩游戏。与任何其他媒体不同，游戏赋予人们创造力、战略思维和团队合作的能力，使他们沉浸在由一些世界上最令人惊叹的创作者创造的互动故事和世界中。游戏的文化现象使其成为世界上规模最大、增长最快的娱乐形式--该行业2021年的年收入预计将超过2000亿美元。

随着游戏行业从以设备为中心的时代转变为以玩家为中心的时代，新技术提供了在任何设备上与朋友一起玩游戏的自由，微软(纳斯达克市场代码：MSFT)周一宣布计划收购Bethesda Softworks的母公司ZeniMax Media，后者是世界上最大的私人持股游戏开发商和发行商之一。贝塞斯达是广受好评和畅销的游戏系列的创造者，其中包括长老卷轴和Fallout等。贝塞斯达为Xbox带来了令人印象深刻的游戏、技术、人才组合，以及轰动一时的商业成功记录。根据协议条款，微软将以75亿美元现金收购ZeniMax Media。

由于在内容、社区和云方面的独特投资，微软的游戏战略与其他公司不同，它使人们能够在他们想要的任何地方，与他们想要的人一起玩他们想玩的游戏。游戏是游戏的主要增长引擎，游戏正在推动Xbox Game Pass等新的云游戏服务，Xbox Game Pass的订户已超过1500万，达到了一个新的里程碑。随着贝塞斯达的加入，微软的创意工作室团队将从15个增加到23个，并将把贝塞斯达的标志性特许经营权添加到Xbox Game Pass中。这包括微软打算将贝塞斯达未来的游戏带入Xbox Game Pass，同一天这些游戏在Xbox或个人电脑上推出，比如贝塞斯达游戏工作室目前正在开发的备受期待的新太空史诗电影“星际迷航”(Starfield)。

微软首席执行官萨蒂亚·纳德拉(Satya Nadella)表示：“游戏是娱乐业中最广泛的类别，因为世界各地的人们都会转向游戏来与朋友联系、社交和玩耍。”“高质量的差异化内容是Xbox游戏PASS增长和价值背后的引擎--从”我的世界“到”飞行模拟器“。作为一家久经考验的游戏开发商和发行商，贝塞斯达公司在每一类游戏上都取得了成功，我们将共同推进我们的雄心壮志，为全球30多亿游戏玩家提供支持。“。

“对于Xbox迷来说，这是一个令人敬畏的时代。仅在过去的10天里，我们就公布了两款新游戏机的详细信息，这两款游戏机明天开始预购，在Xbox Game Pass旗舰版中推出云游戏，现在我们又在我们战略中最关键的部分：游戏上进行了另一项投资，“微软游戏执行副总裁菲尔·斯宾塞(Phil Spencer)说。一代又一代的游戏玩家已经被贝塞斯达产品组合中的著名特许经营权迷住了，而且在未来的几年里，作为Xbox的一部分，这种情况还将继续下去。“。

计划中的收购包括遍布全球的出版办公室和开发工作室，拥有超过2300名员工，其中包括Bethesda Softworks、Bethesda Game Studios、id Software、ZeniMax Online Studios、Arkane、MachineGames、Tango Gameworks、Alpha Dog和Roundhouse Studios。贝塞斯达广受好评的畅销系列包括“长老卷轴”、“堕落”、“末日”、“地震”、“沃尔芬斯坦”和“迪肖诺德”等。

贝塞斯达母公司ZeniMax Media成立于1999年，由董事长兼首席执行官罗伯特·A·阿尔特曼创建；贝塞斯达的结构和领导力将继续存在。

“对于这家公司、我们的员工和我们的粉丝来说，这是激动人心的一天。几十年来，我们与微软一直保持着密切的合作伙伴关系，这笔交易是这些年合作的自然发展，“Altman说。“今天最大的赢家是我们的球迷。我们正在继续开发我们的AAA级游戏，但现在有了微软的规模和整个游戏栈，我们的游戏只会变得更好。“。

该交易受惯例成交条件和完成监管审查的制约，微软预计收购将在2021财年下半年完成，对2021和2022财年非GAAP营业收入的影响微乎其微。非GAAP不包括采购会计调整的预期影响，以及整合和交易相关费用。

关于ZeniMax媒体公司。ZeniMax Media是一家总部位于华盛顿特区郊外的私营媒体机构，在全球设有国际出版办事处。通过其子公司，ZeniMax Media为控制台、个人电脑和手持/无线设备创建和发布原创互动娱乐内容。ZeniMax媒体部门包括Bethesda Softworks、Bethesda Game Studios、id Software、Arkane Studios、Tango Gameworks、MachineGames、ZeniMax Online Studios、Alpha Dog Games、Roundhouse Studios、ZeniMax欧洲有限公司、ZeniMax Asia K.K.、ZeniMax亚太有限公司和ZeniMax Australia Pty Ltd。

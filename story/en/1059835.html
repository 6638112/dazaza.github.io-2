LuaRT, a comprehensive framework for Windows to develop in Lua
Luart，在Lua开发的窗户综合框架

2021-04-23 04:30:55

LuaRT 0.9.2 beta 1 has just been released, and can be downloaded here. Go to Lua rtCommunity to see the changelog. Documentation for the net module has been published too. Lua rt is a multipurpose programming framework for the Windows operating system, using the Lua programming language. Lua is known to be fast, light and simple for newcomers. Lua rt provides a single-click installer with a specific runtime library and binaries for Windows operating systems (including files, sockets, compression, gui...) to develop quickly. Lua rt main goal is to provide a simple and fast way to develop for Windows operating systems in Lua without external dependencies, out of the box, and without any C compilation hassle.

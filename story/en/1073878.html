Microsoft releases a new Windows 11 Preview build with a customizable Start menu, five new touch gestures, Live Captions, Task Manager redesign, and more
微软发布了一个新的Windows 11预览版，带有可定制的开始菜单、五种新的触摸手势、实时字幕、任务管理器重新设计等等

2022-02-17 17:21:08

Microsoft is bringing new touch gestures, Start menu folders, a redesigned Task Manager, and lots more improvements to Windows 11. A new build of Windows 11 is available to testers today, and it’s full of new features that look like they will significantly improve the usability of Windows 11 overall.

微软正在为Windows 11带来新的触摸手势、开始菜单文件夹、重新设计的任务管理器，以及更多的改进。今天，测试人员可以使用Windows 11的新版本，其中充满了新功能，看起来它们将显著提高Windows 11的整体可用性。

The new Start menu folders will allow Windows 11 users to customize the pinned section of the Start menu into folders. You drag an app on top of another to create a folder, and you can rearrange apps within folders and remove them. The ability to rename folders is coming in future test builds, but it’s good to see more customization on the Start menu, given Windows 11 launched without many ways to tweak the menu design.

新的“开始菜单”文件夹将允许Windows 11用户将“开始菜单”的固定部分自定义为文件夹。你可以将一个应用拖动到另一个应用上，创建一个文件夹，然后可以在文件夹中重新排列应用并将其删除。重命名文件夹的功能将在未来的测试版本中出现，但考虑到Windows 11在启动时没有很多方法来调整菜单设计，在“开始”菜单上看到更多的定制是件好事。

If you’ve always been a fan of the gestures in Windows 8, Microsoft is bringing five new touch gestures to Windows 11 soon that will make using tablets a little easier. The first is the ability to swipe up on the taskbar to bring up the Start menu and swipe down to dismiss it again. You’ll also be able to swipe between pinned, all apps, and recommended / more on the Start menu.

如果你一直喜欢Windows 8中的手势，那么微软很快就会在Windows 11中推出五种新的触摸手势，这将使平板电脑的使用变得更简单。第一个功能是在任务栏上向上滑动以打开“开始”菜单，向下滑动以再次关闭它。您还可以在“开始”菜单上的“锁定”、“所有应用”和“推荐/更多”之间滑动。

The third new touch gesture lets you swipe from the bottom right of the taskbar to show the Quick Settings menu — useful if you need to toggle airplane mode quickly or access Bluetooth settings. The animations for bringing up the Notification Center have also been improved, and there’s a new full-screen gripper in touch-oriented apps that will stop you accidentally swiping from the edges of a tablet screen. Microsoft also says rotating your device between portrait and landscape orientation is getting “snapper” and “more responsive.”

第三个新的触摸手势可以让你从任务栏右下角滑动以显示快速设置菜单——如果你需要快速切换飞行模式或访问蓝牙设置，这很有用。启动通知中心的动画也得到了改进，面向触摸的应用程序中有一个新的全屏抓取器，可以阻止你从平板电脑屏幕边缘意外滑动。微软还表示，在纵向和横向之间旋转设备会变得“更快”和“更灵敏”

Microsoft’s new Snap Layouts feature in Windows 11 is also getting improved, allowing you to drag an app to the top of a window and reveal all the layouts you can use. Microsoft is also improving its Focus feature to include a Do Not Disturb mode. It makes it a lot easier to quickly silence notifications, so you only see priority ones or ones for alarms.

微软在Windows 11中的新快照布局功能也得到了改进，允许你将应用程序拖动到窗口顶部，并显示所有可以使用的布局。微软也在改进其聚焦功能，加入了一个“请勿打扰”模式。它使快速关闭通知变得容易得多，因此您只能看到优先级通知或警报通知。

Live Captions will soon be built into Windows 11, accessible from a Winkey + CTRL + L shortcut. Live Captions will greatly benefit people who are deaf or hard of hearing, but they’re also available for anyone to use and will be automatically generated on-device from any content that includes audio. Captions can float in a window or be displayed at the top or bottom of a screen. This looks like a great accessibility improvement to Windows 11.

实时字幕将很快嵌入Windows 11，可以通过Winkey+CTRL+L快捷方式访问。现场字幕将使聋哑人或听力障碍者受益匪浅，但也可供任何人使用，并将在设备上根据包含音频的任何内容自动生成。字幕可以在窗口中浮动，也可以显示在屏幕的顶部或底部。这看起来对Windows 11的可访问性有很大的改进。

Other improvements on the way include the ability to drag and drop on the Windows 11 taskbar. This was a big missing part of the initial launch, so Windows 11 users will be happy to see this return. Microsoft is also making it easier to see your OneDrive storage within File Explorer and improving the Quick Access feature to let you pin files as well as folders.

其他改进包括在Windows 11任务栏上拖放的功能。这是首次发布时缺失的一大部分，因此Windows 11用户将很高兴看到这一回归。微软还在文件资源管理器中更容易查看OneDrive存储，并改进了快速访问功能，使您可以锁定文件和文件夹。

In a push to reduce energy consumption on PCs, Microsoft is also changing the default values for sleep and screen off in the power settings in Windows 11. You’ll only see these new settings if you clean install Windows 11, but new PCs in the future will presumably ship with more aggressive sleep / screen off options.

为了降低个人电脑的能耗，微软还在Windows 11的电源设置中更改睡眠和关机的默认值。只有当你清洁安装Windows 11时，你才能看到这些新设置，但未来的新PC可能会附带更积极的睡眠/屏幕关闭选项。

Last but not least, the Task Manager redesign in Windows 11 is now official. It includes a new command bar and a dark theme, alongside an efficiency mode that lets you limit apps from consuming system resources.

最后但并非最不重要的一点是，Windows 11中任务管理器的重新设计现在是正式的。它包括一个新的命令栏和一个黑色主题，以及一个可以限制应用程序消耗系统资源的效率模式。

As always, these new Windows 11 features are only in testing right now, which means some of them will likely show up in monthly updates throughout 2022. Microsoft just delivered Windows 11’s first big update, and the company revealed it won’t wait to bring new features just once a year.

和往常一样，这些新的Windows 11功能目前只在测试中，这意味着其中一些功能可能会在2022年的每月更新中出现。微软刚刚发布了Windows 11的第一次大更新，该公司透露，它不会等待每年只推出一次新功能。

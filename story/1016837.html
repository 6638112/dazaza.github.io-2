苹果应用商店政策意味着不允许Xbox游戏流媒体

2020-08-07 11:44:47

今年9月，微软计划在视频游戏行业推出一项重大举措：推出世界上首个内置Netflix风格库的游戏流媒体服务。

每月支付15美元，你就可以在智能手机和平板电脑上流媒体播放100多款游戏，但在苹果无处不在的iPhone和iPad上是无法使用的。

苹果公司的一位发言人周四表示，原因是苹果不能审查通过Game Pass提供的每一款游戏。

苹果公司的一位发言人告诉“商业内幕”，App Store的创建初衷是为客户提供一个安全、值得信赖的地方来发现和下载应用程序，同时也是所有开发者的绝佳商机。在应用程序上架我们的商店之前，所有的应用程序都会按照同一套准则进行审查，这些准则旨在保护客户，并为开发者提供公平和公平的竞争环境。

由于微软没有将其流媒体服务上的每款游戏都提交给苹果的审查程序，因此允许访问这些游戏的应用程序被阻止发布。

苹果在给“商业内幕”(Business Insider)的一份声明中表示：我们的客户可以享受数百万开发者提供的优秀应用和游戏，游戏服务完全可以在App Store上推出，只要他们遵循适用于所有开发者的同一套指导方针，包括单独提交游戏供审查，并出现在图表和搜索中。苹果在给“商业内幕”(Business Insider)的一份声明中表示。除了App Store之外，开发者还可以选择通过App Store上的Safari和其他浏览器通过网络联系到所有iPhone和iPad用户。"；

谷歌提供的一项类似的服务，名为Google Stadia，也在苹果的App Store指导方针下遇到了障碍。它可以在Android手机和平板电脑上使用，但不能在苹果设备上使用。

既然苹果允许Netflix和Spotify这样的服务，而不需要审查每一条内容，为什么不允许游戏使用类似的服务呢？根据苹果公司的说法，区别归根结底在于媒介：游戏是互动的，不像音乐和电影，而且App Store中有与游戏相关的消费者期望。

例如，不能通过苹果的内置服务进行应用内支付，也不能对App Store进行评级，等等。

微软广受欢迎的订阅服务Xbox Game Pass一开始就可以将游戏从大型库中直接下载到您的游戏机和PC上。

今年9月，微软将Xbox Game Pass与其流媒体服务Project XCloud结合在一起--这是几年来酝酿的结果。不仅可以将游戏从Game Pass库下载到您的游戏机和PC上，它们还可以流传输到智能手机和平板电脑上。

微软表示，今年9月推出Xbox Game Pass旗舰游戏后，每月只需支付15美元，就可以将100多款游戏直接流媒体到你的智能手机或平板电脑上。然后，您可以在Xbox游戏机上从您停止的位置继续玩这些游戏。

此外，微软发布的每一款主要Xbox游戏，从“光环”到“战争机器”，再到“武力赛车”，都会在发布时发布到该服务上，此外还有一些第三方游戏，如“巫师3：狂野猎杀”和“侠盗猎车手5”。它是持久化库，可以对这些游戏进行流媒体播放。

这是一项已经很成功的服务的下一次演变：Xbox Game Pass目前拥有超过1000万付费用户。这是微软计划永远结束Xbox和PlayStation之间的直接竞争的关键下一步。

但如果没有对苹果手机和平板电脑的支持，微软雄心勃勃的计划将面临重大障碍。该公司的一份声明承诺将继续努力寻求解决方案。

声明称，我们的雄心是通过Xbox Game Pass扩展云游戏，Xbox Game Pass可以在所有设备上使用，但目前我们没有更多关于iOS的信息可供分享。

维生素D和新冠肺炎的首次随机对照试验

2020-09-04 08:14:16

钙化二醇治疗和最佳可用治疗与最佳可用治疗对因新冠肺炎住院的患者重症监护病房入院和死亡率的影响：一项先导性随机临床研究"；

维生素D内分泌系统可能对新冠肺炎进展过程中涉及的细胞和组织有多种作用。

对住院的新冠肺炎患者应用钙化非二醇或25-羟基维生素D可显著降低其重症监护联合入院的需求。

维生素D内分泌系统可能对参与新冠肺炎病情进展的细胞和组织有多种作用，特别是通过减轻急性呼吸窘迫综合征。钙化二醇可迅速升高血清25OHD浓度。因此，我们评估了钙化二醇治疗对因新冠肺炎住院的西班牙患者重症监护病房入院和死亡率的影响。

连续76例新冠肺炎感染住院患者，临床表现为急性呼吸道感染，经病毒性肺炎X线分型和CURB65PCR阳性(总分及Gt；1推荐入院)证实。

所有住院患者均接受相同的标准治疗(按医院方案)：羟氯喹(第一天每12 小时4 0 0 mg，随后5天每12 小时2 0 0 mg)、阿奇霉素(5 0 0 mg口服，共5天)。符合条件的患者在入院当天通过电子随机分配按2：1无钙二醇比例服用骨化二醇(0.532 mg)，或不服用骨化二醇。钙化二醇治疗组的患者在第3天和第7天继续口服钙化二醇(0.266 mg)，然后每周服用一次，直到出院或入院。疗效结果包括ICU入院率和死亡率。

在接受钙化二醇治疗的50名患者中，1名需要进入重症监护病房(2%)，而在26名未接受治疗的患者中，13名需要入院(50%)，x2费舍尔检验p <； 0.001。接受钙化二醇治疗的患者与未接受钙化二醇治疗的患者在重症监护病房中的单变量风险估计优势比：0.0 2(95%CI 0.002-0.17)。接受钙化二醇治疗的患者与未接受钙化二醇治疗的患者在重症监护病房中的多变量风险估计优势比(经高血压和T2 DM调整)：0.03(95%CI：0.003-0.25)。接受钙化非二醇治疗的患者中，无一人死亡，均已出院，无并发症。13名没有接受钙化二醇治疗的患者，也就是没有住进ICU的患者，已经出院了。在入住ICU的13名患者中，2名死亡，其余11名已出院。

我们的初步研究表明，服用大剂量的钙化二醇或25-羟基维生素D(维生素D内分泌系统的主要代谢物)，可以显著减少因确诊为新冠肺炎而需要住院的患者对重症监护病房的治疗需求。钙化非二醇似乎能够降低疾病的严重程度，但需要进行更大规模的试验，并进行适当的分组匹配，才能给出明确的答案。

Gmail Will Now Let You Edit Office Documents Directly From Email Attachments
Gmail现在允许您直接从电子邮件附件中编辑Office文档

2020-12-11 09:31:58

Google is making it even easier to work with Microsoft Office files, with the company now allowing users to directly edit attached Office files in Gmail, much like it already allows with Google Docs or Sheets files.

Google使得使用Microsoft Office文件变得更加容易，该公司现在允许用户直接在Gmail中编辑附加的Office文件，就像它已经允许使用Google Docs或Sheets文件一样。

Google Workspace (the recently rebranded G Suite, which encompasses Google Docs, Sheets, Slides, Meet, and Calendar) has offered compatibility with Office files from Word, PowerPoint, and Excel for some time. But previously, documents had to be imported into Google Drive in order to edit or modify them — while emailed attachments were only able to be viewed.

Google Workspace（最近重新命名为G Suite，包括Google Docs，表格，幻灯片，Meet和Calendar）在一段时间内已经提供了与Word，PowerPoint和Excel中Office文件的兼容性。但是以前，必须将文档导入Google云端硬盘才能进行编辑或修改，而只能通过电子邮件查看附件。

The new workflow, however, streamlines that process considerably. Now, you can directly open and edit an Office file using the Google Docs editor just by clicking on it — just like you would a native Google Doc. But the new editing function doesn’t convert Office files into Google Docs, instead preserving the original file format.

但是，新的工作流程大大简化了该过程。现在，您可以通过单击Google Docs编辑器直接打开和编辑Office文件，就像本地Google Doc一样。但是新的编辑功能不会将Office文件转换为Google文档，而是保留原始文件格式。

Gmail will allow users to respond to the original email and include the now-updated file (still in an Office file format) without first requiring that they download and then re-attach the updated file.

Gmail将允许用户回复原始电子邮件，并包括现在更新的文件（仍为Office文件格式），而无需先要求他们下载然后重新附加更新的文件。

Google is also working to help ensure that Office files work more smoothly in Google Docs, with the company launching a new Macro Converter add-on for Google Workspace that’s designed to help users and organizations import their macros from Excel to Sheets more easily.

Google还在努力确保Office文件在Google Docs中更流畅地运行，该公司针对Google Workspace推出了一个新的Macro Converter加载项，旨在帮助用户和组织更轻松地将其宏从Excel导入到表格中。

Similarly, Google is working on adding better document orientation and image support to Google Docs, allowing for documents with both horizontally and vertically oriented pages, along with images placed behind text and watermarks (although the new image features won’t be available until next year.)

同样，Google正在努力为Google文档添加更好的文档方向和图像支持，允许同时具有水平和垂直页面的文档以及放置在文本和水印后面的图像（尽管新图像功能要到明年才能使用） ）

Correction December 10th, 1:30pm: Google’s suite of products is called Google Workspace, not Google Workplace. We regret the error.

更正12月10日，下午1：30：Google的产品套件称为Google Workspace，而不是Google Workplace。对于错误，我们深表歉意。

万维网今天已经 30 岁了

2021-08-06 10:50:20

Berners-Lee 是计算机科学家的儿子，1955 年（与史蒂夫·乔布斯和比尔·盖茨同年）出生于伦敦，并在牛津大学学习物理学。在 1980 年代受雇于 CERN 期间，Berners-Lee 观察到跟踪该组织遍布全球的数千名研究人员的项目和计算机系统是多么困难。正如他后来所说：“在那个时代，不同的计算机上有不同的信息，但你必须登录不同的计算机才能获得它。此外，有时您必须在每台计算机上学习不同的程序。” 1989 年 3 月，Berners-Lee 向 CERN 的管理人员提出了一个信息管理系统的建议，该系统使用超文本链接连接到 Internet 的不同计算机上的文档。 （超文本是 1963 年创造的一个术语，允许一个人通过单击编码的单词或短语来获取文档或一段内容。）被他的老板标记为“模糊但令人兴奋”的提议最初未被接受。 Berners-Lee 与 CERN 的比利时工程师 Robert Cailliau 合作完善了该提案，1990 年，这位英国人的老板给了他时间参与该项目。在最初将项目称为信息管理后，伯纳斯-李尝试了诸如信息矿井和信息网格之类的名称，然后才决定使用万维网。到 1990 年底，Berners-Lee 使用史蒂夫乔布斯设计的 NeXT 计算机开发了作为 Web 基石的关键技术，包括用于创建网页的超文本标记语言 (HTML)；超文本传输​​协议 (HTTP)，一组用于通过 Web 传输数据的规则；和统一资源定位符 (URL)，或用于查找文档或页面的 Web 地址。他还设计了一个基本的浏览器和 Web 服务器软件。 1991 年 8 月 6 日，伯纳斯·李 (Berners-Lee) 发布了有史以来第一个网站，Web 开始作为 Internet 上的一项公开可用服务出现。恰如其分地，该站点是关于万维网项目的，描述了 Web 以及如何使用它。在伯纳斯-李的 NeXT 计算机上托管在 CERN，该站点的 URL 是 http://info.cern.ch。 Berners-Lee 并没有试图从他的发明中获利，并拒绝了 CERN 要求为他的 Web 技术申请专利的呼吁。他希望网络是开放和免费的，以便它可以尽可能快地扩展和发展。正如他后来说的，“如果这项技术是专有的，并且在我的完全控制之下，它可能不会起飞。你不能建议某物成为一个通用空间，同时保持对它的控制。” 1993 年，伊利诺伊大学国家超级计算应用中心的一个团队发布了 Mosaic，这是第一个受到公众欢迎的 Web 浏览器。在接下来的几年里，雅虎 (1994)、亚马逊 (1995)、eBay (1995) 和谷歌 (1998) 等网站相继推出。根据 Internet Live Stats 的数据，到 2004 年 Facebook 首次亮相时，已有超过 5100 万个网站。与此同时，1994 年，伯纳斯-李离开 CERN 前往麻省理工学院，在那里他创立了万维网联盟 (W3C)，这是一个维护 Web 标准的组织。这位低调的有远见的人后来被《时代》杂志评为 20 世纪 100 位最重要人物之一，并于 2004 年被英国女王伊丽莎白二世封为爵士。 2009 年，伯纳斯-李创立了万维网基金会，该组织致力于确保网络造福人类。在 2012 年伦敦夏季奥运会开幕式上，他因发明了网络而感到荣幸，并在推特上写道：“这是给所有人的。”

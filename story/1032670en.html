基于TensorFlow和KERS的手写几何形状检测器

2020-11-02 04:52:50

When I was a kid I had a magnetic drawing board. It was a very simple but interesting tool for me. I used to draw everything on it. The most satisfying part was the erasing mechanism of the magnets.

当我还是个孩子的时候，我有一块磁力绘图板。对我来说，这是一个非常简单但有趣的工具。我以前什么都画在上面。最令人满意的部分是磁铁的擦除机制。

I don't know how popular it is now these days to the kids. And due to the smart phones era I think kids are more into these gadgets than anything else. But I think we should come up with ways so that kids still find drawing a fun thing to do.

我不知道现在它对孩子们有多受欢迎。由于智能手机时代的到来，我认为孩子们对这些小玩意的兴趣超过了其他任何东西。但是我认为我们应该想办法让孩子们仍然觉得画画是一件有趣的事情。

As a computer engineer I thought why not try to build something like that. And so I tried to recreate a digital version of the magnetic drawing board using Machine Learning.

作为一名计算机工程师，我想为什么不试着建造这样的东西呢？所以我试着用机器学习技术重建一个数字版的磁性绘图板。

I created a dataset of 2,000 images of these four classes each with 500 images of size 28x28 pixels. This is the first version. I am still working on creating more robust dataset. Here's a screenshot of the dataset.

我创建了一个包含这四个类的2000个图像的数据集，每个类都有500个大小为28x28像素的图像。这是第一个版本。我仍在致力于创建更健壮的数据集。这里是数据集的屏幕截图。

The model consists of three convolution blocks with a max pool layer in each of them. There's a fully connected layer with 128 units on top of it that is activated by a relu activation function. A very simple model to understand.

该模型由三个卷积块组成，每个卷积块中有一个最大池层。有一个完全连接的层，上面有128个单元，由RELU激活功能激活。这是一个非常简单的模型，很容易理解。

I used Python Tkinter for the front end of the project. Since Tkinter is very simple and easy to use and almost run on every machine I figured it's the easiest and quickest choice.

我使用Python Tkinter作为项目的前端。由于Tkinter非常简单易用，几乎可以在每台机器上运行，我认为它是最简单、最快捷的选择。

I saved the image and then resized and then preprocessed using Keras api. After that predicted the shape of the drawn picture with the model. The accuracy is 94.04% so far on avg. I'm working on using a more sophisticated model and increase the dataset. Let me know in the comment section if you have any suggestions. I will publish the code on Github soon after cleaning and making the code more readable.

我保存了图像，然后调整大小，然后使用Kera API进行预处理。在此基础上，利用模型对所绘制的图片进行了形状预测。到目前为止，AVG的准确率为94.04%。我正致力于使用更复杂的模型并增加数据集。如果你有什么建议，请在评论区告诉我。我将在清理后不久在Github上发布代码，使代码更具可读性。

Let me know what you think of this project. Any sorts of help and constructive criticism is very much appreciated.

让我知道你对这个项目的看法。我们非常感谢任何形式的帮助和建设性的批评。

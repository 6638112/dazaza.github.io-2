消息来源：美国选举日模拟黑客和破坏投票等攻击的演习显示，美国官员将努力打击网络错误信息

2020-09-24 00:09:35

华盛顿(路透社)-据五名知情人士透露，美国选举日的演习模拟了从黑客到炭疽热再到扰乱投票的各种攻击，显示州和地方官员将难以迅速反击充斥着社交媒体的谎言。

这些评估发表之际，美国情报官员和安全分析人士预计，围绕11月3日大选的数字错误信息将接踵而至。上周，美国国家安全局(National Security Agency)局长中曾根康弘(Paul Nakasone)将军指出，虚假信息是选举面临的最大威胁。

模拟中的情景包括：测试选举官员对网络供电停电的反应的演习，选票填充的虚假说法，匿名来电者对投票站发出的虚假炸弹威胁，以及选举日炭疽疫情在结果接近的特定县爆发的虚假说法。

这些测试至关重要，因为如果社交媒体上的虚假信息传播虚假信息，并开始误导选民，管理选举的州和地方官员往往会成为第一反应者。

科罗拉多州国务卿杰娜·格里斯沃尔德(Jena Griswold)表示，这些官员也将是首批向Facebook和Twitter等社交媒体公司报告这些例子，要求删除内容的官员之一。格里斯沃尔德拥有打击虚假信息的工作人员。

但是，即使内容被删除，让选民相信信息是不正确的仍然是困难的。

“你的回应会影响到同样受到虚假信息影响的观众吗？如果虚假信息成功，你还能真正联系到他们吗？是否存在否认本身放大虚假信息的风险，“约翰·霍普金斯大学的虚假信息专家托马斯·里德(Thomas Rid)说。

自2017年以来，已有超过25个州进行了自己的模拟演习，也被称为“桌面演习”。据国土安全部(Homeland Security Department)的一名发言人称，网络安全和基础设施安全局(CyberSecurity And Infrastructure Security Agency)-国土安全部的一个部门-已经组织了总共55次演习。

总部位于波士顿的网络安全公司Cybereason的一名发言人表示，该公司在过去两年里组织了八次活动，州和地方选举官员以及联邦机构都参加了活动，所有活动都是无偿进行的。

关于模拟演习的大部分细节都是保密的，不鼓励来宾向媒体谈论这些细节。

新墨西哥州国务卿玛吉·图卢兹·奥利弗(Maggie Toulouse Oliver)表示，虽然选举官员总是寻求成为投票信息的主要来源，但在紧急情况下很难做到这一点。

奥利弗补充说：“总的来说，试图让人们了解和理解他们的地方选举官员是谁，以及他们如何在危机时刻直接从他们那里获得信息，这是一个持续存在的差距和挑战，我们需要弄清楚。”

美国联邦调查局(FBI)和国土安全部(Homeland Security Department)周四在一份公共服务公告中表示，选民应该总是“从州和地方选举官员等值得信赖的来源寻找信息”，因为来自“外国行为者和网络罪犯”的虚假信息威胁。

据路透社(Reuters)此前报道，州选举官员已经在数字广告活动中投入了数百万美元，以接触选民，并通过增加自己的社交媒体追随者来迅速解决虚假信息，但进展缓慢。

全国各州的主要选举官员通常只有不到几千名社交媒体粉丝，他们可以立即通过在线帖子联系到他们。在总统特朗普和拜登的关键战场佛罗里达州，国务卿只有约2,100名推特粉丝，没有Facebook账户。在加利福尼亚州，国务卿的“CA SOS投票”推特账户只有1.3万名追随者。

这些情景考验了选举官员与联邦合作伙伴在紧急情况下如何安抚公众，分享准确信息，并保持对局势的控制，以便选举能够正常进行。

他们并不是为了测试特定的州或机构是否会失败，但几名参与模拟的人士告诉路透社，经验证明，传播故意不正确的信息以影响某些地区的选民投票率，是选举官员需要克服的最大挑战之一。

丹佛市前选举主任、演习的早期支持者安珀·麦克雷诺兹(Amber McReynolds)说，“这让你大开眼界，它让你思考可能会发生什么，让人们来计划。”“这通常是关于沟通问题，将信息公之于众，并调整计划。”

她说：“这很有趣，因为很多时候，州或县内的通讯人员并没有提前参加选举的业务对话，他们实际上是一个关键的合作伙伴。”

围绕选举的虚假信息已经成为一个问题。8月下旬，数量不详的密歇根州选民收到了一个机器人电话，错误地警告底特律居民，邮寄投票可能会让他们接受追债和强制接种疫苗。密歇根州总检察长表示正在调查这起事件。

Raspberry Pi在2021年的替代方案：Hackboard，Zimaboard，RockPi，Odroid等

2020-12-30 02:09:41

需要一个项目的SBC吗？ 在下面查看我对2021年Raspberry Pi替代产品的建议。 ZimaBoard小巧但强大的双网卡x86 SBC专为用作家庭服务器而设计。 Rock Pi4一种功能强大的基于Hex Core ARM的SBC，与Raspberry Pi Model B具有相同的物理尺寸 NanoPi R4S带有双NIC和最高4GB RAM的微型Hex Core ARM SBC。 ZeroPi具有超低价格的超廉价四核ARM SBC和千兆位NIC。 Hackboard 2是一种新的众筹中型SBC，比Raspberry Pi 4 B型（120x80mm）大。 电路板设计紧凑，但具有电源按钮，LED电源状态/系统LED和小型RTC电池的功能。 Windows是现成的，可以轻松安装任何x86兼容的Linux发行版。 请密切注意我即将进行的评论！

hackboard 2的起价为$ 99（随Ubuntu一起提供）或$ 140（随Windows 10 Pro一起提供）。与Raspberry Pi 4 Model B相比，这似乎很昂贵，但是Windows 10版本比Windows 10 Pro许可证便宜，几乎所有Linux发行版或x86 OS都兼容。

在SBC上运行家庭服务器并不是什么新鲜事，自2012年发布Raspberry Pi Model B以来，我个人一直在这样做。到现在为止！ ZimaBoard是一种SBC，旨在用于服务器应用程序，并提供两种不同的配置：

ZimaBoard体积小，无风扇且功耗低，其6w Apollo Lake CPU使其非常适合作为轻巧的家用服务器。尽管ZimaBoard尚未获得资金，但预计将在2021年某个时候发布。在这里查看我在ZimaBoard上的新闻稿。

两种型号都将很快上市，价格分别为69.90美元和129.90美元。

Rock Pi 4是一款小型的SBC，其使用与Raspberry Pi Model B相同的外形尺寸。它具有六角核心Rockchip RK3399，在外形尺寸和价格方面都非常强大。

RockPi 4的1GB型号起价为49美元，2GB型号起价为59美元，4GB型号起价为75美元。

ODROID-h2 +是采用2.5Ghz Intel J4115 CPU的四核x86板。尽管由于可移动内存DIMM的原因，这并不是严格意义上的SBC，但它仍然适用于此列表，尽管缺少GPIO可能使其不适用于某些项目。

由于Dual Nics，这将成为出色的软件路由器或家庭服务器。

英特尔四核处理器J4115（14nm）具有4MiB高速缓存，最高2.5Ghz（单线程）或2.3Ghz（多线程）

ODROID-h2 +的价格为$ 119.00，但请记住，您将需要单独购买DDR4-PC19200 SODIMM RAM。

ODROID-h2 +可以在英国或其他地方购买。

NanoPi R4S是一款微型SBC，旨在用作具有Hex Core RK3399 CPU和双千兆以太网的防火墙或路由器。友好的ARM产品系列对我来说并不陌生，他们的支持非常出色。看看我以前对他们的NanoPi Fire 3的评论。

NanoPi R4s的起价为1GB型号45美元和4GB型号69美元。

ZeroPi是Friendly Arm的另一种产品，起价仅为9.99美元，非常低。具有运行在1.2Ghz的四核Allwinner H3 CPU。尽管没有GPIO，对于某些想要低功耗，非常便宜的Linux SBC的人来说，它在某些项目中毫无用处，但这还是便宜的！

ZeroPi现已上市，价格为9.99美元，或者额外支付3美元，您可以拥有一个漂亮的金属外壳。

NVIDIA Jetson Nano是NVIDIA SBC开发套件系列中的最新产品，具有四核Arm A57 @ 1.43GHz和128核心Maxwell GPU。

NVIDIA Jetson Nano在英国亚马逊的售价为64英镑，在世界其他地区的售价约为90美元。

ASUS Tinker Board是ASUS的第一个商业RBC，是Raspberry Pi Model B的直接竞争对手。尽管该板使用了老化技术，但在华硕提供的文档和支持方面，它仍然是可行的选择。

华硕Tinker Board在英国的售价约为40英镑，可以在亚马逊上购买。

关于Sipeed XuanTie C906的消息鲜为人知，因为它尚未发布，但它似乎是第一个使用RISC-V架构的预算SBC，这使其显得有些特殊。

该委员会可用的唯一详细信息已于2020年11月在一条推文中发布，但我已经进行了仔细调查，并确认了以下内容：

您好，我们将于2月份发布它，请继续在Twitter上关注我们，我们将在Twitter上及时更新。非常感谢您对我们的支持和关注。最好的问候，Orgmar www.sipeed.com

好消息：我们获得了第一款基于XuanTie C906（RV64GCV）的芯片，它具有丰富的接口（HDMI / RGB / DVP / MIPI / GMAC /…），并且可以运行Debian系统。最后也是最重要的一点，基本的开发板价格为12.5美元起（HiFive Unleashed的1％）。pic.twitter.com/EJbXTJ5eMb

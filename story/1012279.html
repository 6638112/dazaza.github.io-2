微软表示，将于9月份向其Xbox Game Pass Ultimate订户免费提供名为Project XCloud的游戏流媒体服务

2020-07-16 22:07:11

微软正计划在9月份推出其游戏流媒体服务，目前名为Project XCloud，向其付费Xbox Game Pass Ultimate订户免费提供。XCloud服务将允许Xbox玩家在移动设备上玩游戏，甚至可以在他们的游戏机上开始游戏，然后在手机或平板电脑上恢复游戏。Xbox Game Pass旗舰版将Xbox Live Access、Xbox Game Pass订阅以及从9月份开始的XCloud游戏流媒体整合为每月14.99美元的订阅。

微软承诺，当流媒体服务推出时，将有100多个Xbox游戏通行证游戏可以在手机或平板电脑上播放。不过，微软目前还没有详细说明推出时将支持哪些国家。该公司一直在美国和欧洲部分地区建立Azure数据中心，使用Xbox One S Blade通过XCloud流媒体播放Xbox游戏。微软将在2021年将这些服务器升级到Xbox系列X硬件。

Xbox Game Pass旗舰版将是在发布时访问XCloud流媒体的唯一方式，但它并不总是局限于微软的顶级订阅层。

微软Xbox首席执行官菲尔·斯宾塞(Phil Spencer)在接受The Verge采访时表示：“随着时间的推移，我们将继续扩大将流媒体作为平台的一部分，并玩你拥有的不属于订阅的游戏的方式。”他说：“推出时，我们免费将其安装在旗舰版中。我们认为一开始对我们来说，这是一个很好的观众，而且这是一群玩很多游戏的观众。“。

虽然所有在9月份访问XCloud游戏流媒体的人都将通过手机或平板电脑访问，但微软目前还没有详细说明将支持哪些设备。这家软件制造商一直在与苹果就App Store政策进行谈判，这些政策阻止微软测试具有与Android版本相同功能的XCloud。微软今年早些时候开始在iOS设备上测试XCloud，但测试仅限于单一的Halo游戏。

“我们希望最终将XCloud带到人们可以流媒体游戏的每一个屏幕上。斯宾塞说：“现在我们只是说手机。”“目前正在进行讨论，我们正在努力解决问题。我们将在8月份和9月份的发布会上更具体地讨论哪些移动设备。“。

听起来我们将在8月份全面了解XCloud发布的更多细节。我还问了斯宾塞关于承诺的PS4控制器支持和将XCloud游戏流式传输到PC的问题，他承诺下个月公司的计划会更清晰。

Project XCloud甚至不会是微软游戏流媒体服务的最终名称。“这不是官方的名字，但你应该把它想成游戏通行证流媒体或Xbox流媒体之类的东西，”斯宾塞解释说。“我们实际上现在还没有最终的名字，但不会是Project XCloud.”

除了发布计划，微软在XCloud方面还有更大的雄心，不仅仅是流媒体到移动设备上。斯宾塞说：“XCloud上的浏览和购买功能非常方便，我觉得这在今天是非常有价值的。”“很多时候，我第一次玩游戏实际上会在XCloud上，所以我可以去把它作为我的零食试用体验。”

微软正计划将XCloud作为一种工具，让人们能够快速试用Xbox游戏。例如，这可能需要你登录Xbox游戏机，看到朋友在玩游戏，并在完整下载完成之前快速加入，或者在未来只需点击Facebook Gaming的链接即可。

斯宾塞说：“我们希望试验能像今天在音乐和视频中一样简单，我今天可以在Spotify上给你发送一首曲目，你可以立即进行流媒体播放。”随着时间的推移，无论我在哪里看到一场比赛，我都应该能够试一试。我们的想法是，任何时候你在任何设备或网络浏览器上看到一款游戏，你都应该能够在未来开始通过XCloud流媒体播放它。

其中一些XCloud的雄心壮志听起来与谷歌对Stadia的承诺类似。谷歌的大计划是利用YouTube让人们直接进入游戏。这还没有实现，但是访问游戏的简便性让人感觉它将是未来XCloud和Stadia的重要组成部分。

一些竞争对手的流媒体服务，如NVIDIA的GeForce Now，在未经出版商许可的情况下提供他们的游戏流媒体，从而遇到了麻烦。微软在出版商方面还没有遇到任何类似的问题。“届时将有100多场比赛，”斯宾塞说。“我们已经与这么多出版商建立了现有的关系，因此对我们来说，对话的重点是如何将我们共同建立的伟大业务作为一个平台，作为一个工作室或出版商，并将其扩大。”

这并不能保证每一款Xbox游戏通行证游戏都一定会在XCloud游戏流媒体上提供，但听起来肯定会有绝大多数会出现在XCloud游戏流媒体上。由于Xbox Game Pass，许多游戏开发商从销售增加中受益，特别是独立开发商，他们看到了接触1000多万订户的立竿见影的好处。

微软现在正计划在下周的Xbox Series X游戏发布会上展示更多的XCloud游戏，我们应该会在8月份得到更多关于这项服务的具体发布细节。

自2013年微软首次展示在Windows Phone上运行Halo4以来，微软已经为XCloud的发布进行了长达十年的大部分时间的努力。在对XCloud进行了近一年的公开测试后，9月份将标志着微软朝着覆盖全球数十亿游戏玩家的雄心勃勃的计划迈出了重要的一步。

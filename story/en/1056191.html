Skip to content Website add-on products purporting to improve accessibility go back to the late 1990s with products like Readspeaker and Browsealoud. Both of which added text-to-speech capabilities to the website(s) on which they were installed.

跳至内容网站附加产品，声称提高可访问性，返回20世纪90年代后期，产品如ReadSpeaker和BrowseLealoud。这两者都将文本到语音功能添加到安装的网站上。

Later, similar products came to market that added additional tools to their software that allow user-based control of things like font-sizes and changes to the web pages colors so that contrast is improved. Products like Userway, EqualWeb, AudioEye, User1st, MaxAccess, FACIL'iti, and accessiBe fall into this category. These products are sometimes also white labelled under additional names.

后来，类似的产品来到市场，为他们的软件添加了额外的工具，允许基于用户的对字体大小和Web页面颜色的更改，以便改善了对比度。产品如Usway，ComplyWeb，AudioEye，User1，MaxAccess，Facil＆＃39; ITI，Actia和Accessibe属于此类别。这些产品有时也有白色标记为额外的名称。

As stated above, some overlay products contain widgets which present a series of controls that modify the presentation of the page they're on. Depending on the product, those changes may do things like change the page contrast, enlarge the size of the page's text, or perform other changes to the page that are intended to improve the experience for users with disabilities.

如上所述，一些叠加产品包含窗口小部件，其呈现了一系列控件，该控件修改了它们的页面的演示;重新启动。根据产品，这些变化可能会更改页面对比度，扩大页面的大小，或者对旨在提高残疾用户体验的页面的其他更改。

To laypersons, these features may seem beneficial, but their practical value is largely overstated because the end users that these features claim to serve will already have the necessary features on their computer, either as a built-in feature or as an additional piece of software that the user needs to access not only the Web but all software.

对于PATPERSONS来说，这些功能可能看起来有益，但它们的实际价值很大程度上被夸大了，因为这些功能所声称的最终用户已经在其计算机上拥有必要的功能，作为内置功能或作为额外的软件用户不仅需要访问Web但所有软件。

On this latter point, it is a mistake to believe that the features provided by the overlay widget will be of much use by end users because if those features were necessary to use the website, they'd be needed for all websites that the user interacts with. Instead, the widget is  as—at best—redundant functionality with what the user already has.

在后一点上，认为覆盖小部件提供的功能是最终用户使用的，因为如果这些功能是使用网站所必需的，则为所有网站都需要＆＃39;用户与之交互。相反，窗口小部件与用户已具有的最佳冗余功能。

Some overlay products have capabilities aimed at providing accessibility repairs to the underlying page on which the overlay is added. These repairs are applied when the page loads in the user's browser.

一些叠加产品具有旨在为添加覆盖层的底层页面提供可访问性维修的功能。当用户＆＃39; s浏览器中的页面加载时，应用这些维修。

While it is true that a non-trivial array of accessibility problems can be repaired in this manner, the nature, extent, and accuracy of such repair are limited by a number of important factors:

虽然可以以这种方式修复非琐碎的可访问性问题，但是这种修理的性质，程度和准确性受到许多重要因素的限制：

Automated repair of field labels, error management, error handling, and focus control on forms is not reliable

现场标签的自动修复，错误管理，错误处理和表单上的对焦控制不可靠

Modern, component-based user interfaces, such as those using ReactJS, Angular, or Vue may change the state of all or some of the underlying page independently of the overlay, rendering it unable to fix those JavaScript-driven changes to content.

基于组件的用户界面，例如使用RESTJS，ANGULAR或VUE的用户界面可以独立于覆盖层改变所有或一些底层页面的状态，渲染它无法将这些JavaScript驱动的改变更改为内容。

Repairs to the page can either slow down page load times or cause unexpected page changes for assistive technology users.

修复页面可以减慢页面加载时间或导致辅助技术用户的意外页面更改。

In addition to the above, overlays do not repair content in Flash, Java, Silverlight, PDF, HTML5 Canvas, SVG, or media files.

除了上述外，覆盖层还不会在Flash，Java，Silverlight，PDF，HTML5 Canvas，SVG或媒体文件中修复内容。

An additional class of product exists, which only perform automated repairs and are marketed as a temporary solution. These include Amaze by Deque Systems, Alchemy by Level Access, and Sentinel by Tenon. For purposes of this document, these products aren't considered to be in the same class of product as the overlays that provide widgets. The most notable difference, beyond the lack of a “widget” is that Amaze, Alchemy, and Sentinel are understood by their manufacturers as being intended for use as an interim solution.

存在另外一类产品，只能执行自动维修，并作为临时解决方案销售。这些包括通过Deque Systems，Alchemy通过级别访问和宾夕法尼克州的Sentinel am。出于本文档的目的，这些产品在与提供小部件提供小部件的覆盖层时，这些产品被认为是在同一类产品中。最值得的差异，超出缺乏“小部件”的是，其制造商被理解为amaze，alchemy和sentinel作为旨在用作临时解决方案。

Among the many claims made by overlay vendors is the claim that the use of their product will being the site into compliance with accessibility standards such as WCAG 2.x, related and derivative standards, and laws that mandate compliance with those standards.

在覆盖供应商提出的许多主张中，索取其产品的使用将符合WCAG 2.x，相关和衍生标准等可访问性标准，以及授权遵守这些标准的法律。

Conformance to a standard means that you meet or satisfy the ‘requirements’ of the standard. In WCAG 2.0 the ‘requirements’ are the Success Criteria. To conform to WCAG 2.0, you need to satisfy the Success Criteria, that is, there is no content which violates the Success Criteria. Understanding WCAG 2.0: Understanding Conformance

符合标准意味着您满足或满足标准的“要求”。在WCAG 2.0中，“要求”是成功标准。要符合WCAG 2.0，您需要满足成功标准，即没有内容，违反了成功标准。了解WCAG 2.0：了解一致性

Given that conformance is defined as meeting all requirements of the standard, these products' documented inability to repair all possible issues means that they cannot bring a website into compliance. Products marketed with such claims should be viewed with significant scepticism.

鉴于该一致性被定义为满足标准的所有要求，这些产品＆＃39;记录无法修复所有可能的问题意味着他们无法将网站带入合规性。应以显着的怀疑态度观察与此类索赔销售的产品。

Accessibility on the Web is a big challenge, both for owners of websites and for the users of those websites. The invention of novel approaches to resolving this challenge is to be commended.

Web上的可访问性是对网站的所有者以及这些网站的用户的重要挑战。提出了解决这一挑战的新方法的发明。

However, in the case of overlays—especially those which attempt to add widgets that present assistive features—the challenge is not being met. Even more problematic are the deceptive marketing provided by some overlay vendors who promise that implementing their product will give their customer's sites immediate compliance with laws and standards.

然而，在叠加的情况下 - 特别是那些试图添加呈小部件的小部件，这些辅助功能 - 挑战不会满足。甚至更有问题是由一些覆盖的供应商提供的欺骗性营销，他们承诺实施其产品将为客户提供立即遵守法律和标准的顾客＆＃39;

首家进入 YC 的赞比亚初创公司正在开发非洲首个发卡 API

2021-08-06 15:31:22

Union54（54 是对非洲国家数量的一种认可）是由 Perseus Mlambo 和 Alessandra Martini 创立的金融科技公司。这家初创公司声称是非洲第一个发卡 API，今年才刚刚推出。但要描绘这幅画，Union54 并非凭空而来；这是这对夫妇早期创业公司 Zazu 的一个项目。 Zazu 于 2015 年作为赞比亚的挑战者银行成立。与非洲大陆的任何金融科技公司一样，Zazu 必须创建自己的借记卡，用户可以将其连接到钱包。大多数情况下，Zazu 必须等待该国的合作银行数月才能发行这些卡。 Mlambo 告诉我，有一次他们不得不等待 18 个月。与此同时，创始人开始与该地区的银行合作，开始自己发卡。但银行在他们的方法上无精打采。 “我们刚刚意识到，处理器或银行不一定有能力回答我们的问题或能够为我们提供我们正在寻找的产品，”Mlambo 在接受 TechCrunch 采访时说。这家初创公司决定寻找靶心并与万事达卡会面。我的意思是，当您可以将发行这些卡的人放在首位时，为什么要等待银行，对吗？它声称，最终，该公司获得了万事达卡原则会员资格，这是非洲第一家金融科技公司。作为主要成员，Zazu 被授权充当“发卡行”。换句话说，他们可以提供借记卡并作为“收单方”，这意味着他们可以提供交易处理服务。在此过程中，创始人意识到要真正推动非洲金融科技的发展，必须让任何非洲国家的金融科技更容易发行虚拟或实体借记卡。所以团队从 Zazu 分拆出 Union54。该平台现在有多个 API，使任何金融科技都能轻松发行可编程借记卡。 “我们现在已经利用我们的会员资格来帮助其他公司，任何想要发行自己的卡的非洲金融科技公司。他们可以来找我们，插入我们的 API 并迅速采取行动，而无需花费很长时间进行谈判，”Mlambo 在谈到为其他非洲金融科技公司提供服务时说。

首席执行官补充说，该公司的目标是不想花费数十万美元的设置费来获得虚拟或实体卡的金融科技公司。 Union54 声称可以在几周内通过 API 发行卡，该 API 具有 BIN 赞助、项目管理和结算等功能。能够做到这一点使 Union54 成为非洲第一个发卡 API 的吹嘘权。金融科技公司很少看到这个机会。大多数都专注于从支付网关到钱包的其他领域。这是一个值得注意的有趣点，因为不知何故，这些细分市场中的所有大玩家最终都试图为他们的客户创建虚拟和实体卡，并面临这样做的复杂性。这就是 Union54 想要填补的空白，尽管目前处于测试阶段，但该公司拥有令人印象深刻的未透露姓名的客户，他们已在其候补名单上注册并正在使用该平台。 “这些公司的迷人之处在于它们不是 B 级或 C 级玩家。他们是非洲金融科技的前 5%。对我来说，我总是告诉人们，我们现在处于非洲金融科技的黄金一代。因此，对于发卡产品来说，能够与所有这些在其领域中被视为领导者的人合作确实是最佳时机。这意味着我们确实拥有人们每天都想使用的东西，”首席执行官补充道。在该公司的网站上，其 API 有八个用例：基于分类帐、收单方/网关、立即购买、稍后付款、信用合作社、送货公司、数字银行、信用卡管理和公司卡。还允许使用 Union54 的金融科技公司设计卡片并设置他们希望使用卡片收费的货币，并设置一个广泛的目录，包括谁将使用它们、它们将用于什么、何时使用以及如何使用。将会被使用 。 Union54 对每次 API 调用都按即用即付的方式向金融科技公司收费。如果一家金融科技公司想要创建一张实体卡，他们需要支付 7 到 9 美元的固定费用，以及在交易时未公开的固定费用。 Mlambo 表示，进入 YC 2021 的夏季批次使该公司能够注册第一批客户，因为他们中的大多数都来自 YC 的网络。他称 YC 是一个“从一开始就值得”的项目。

“Union54 成为首个被 Y Combinator 接受的赞比亚金融科技公司，我感到非常兴奋和自豪。第二个在南部非洲。如您所知，当全球投资者关注非洲时，他们通常是从西非的角度来看的，我们进入 Y Combinator 验证了我们更广泛假设的一小部分：有可能从赞比亚等友好司法管辖区为非洲提供服务。”

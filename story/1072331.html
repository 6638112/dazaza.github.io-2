微软的 Windows 11 推广工作进展不顺利

2021-08-09 23:27:12

在 7 月 21 日的直播中，微软项目经理 Aria Carley 回答了微软技术社区用户关于升级到 Windows 11 的最终硬件要求的问题。虽然硬件要求——包括但不限于 TPM 2.0 支持——并不适用于 Windows 11 alpha 图像现在可用，Carley 确认“硬件地板”对于最终版本是真实的。 “所以我们讨论了这个新的硬件平台，哪些设备符合条件，哪些不符合条件，”Carley 说，并补充道，“我们知道有些设备不符合 Windows 11 的条件，这很糟糕。”她继续表示，微软正在实施不受欢迎的硬件层，“以保持设备的生产力、更好的体验，最重要的是拥有比以前更好的安全性，以便他们在这支新员工队伍中得到保护。”尽管承认这种情况对受影响的用户来说“糟糕透顶”，但 Carley 在回答后来的问题时加倍强调硬件层的不灵活，他说：“组策略不会让你绕过 Windows 11 的硬件强制执行。我们是仍然会阻止您升级您的设备......以确保您的设备保持受支持和安全。”不出所料，这些答案似乎并没有被观众接受——根据 Windows Central 的说法，该视频的最高评论写道，“这些答案中的很多都是超级聋子......看起来 Windows 11 将是另一个视窗 8。”其他评论（同样，根据 Windows Central 的说法）推测，看似不必要的硬件要求是推动新计算机销售的隐秘策略，相应地促进了 Windows 许可证销售。不幸的是，我们不得不接受各种博客对 Microsoft 技术社区用户不得不说的话，因为 Microsoft 只是禁用了对视频的评论 - 删除所有现有评论 - 以回应消极情绪。虽然评论消失了，但投票没有——截至今天下午，有 270 人不喜欢，只有 146 人喜欢。在我们看来，微软急于提出新的硬件要求过于激进且处理不当，不管任何人对其未指明的安全优势的合法性有何看法。一个更温和的“为 Windows 11 制造”活动——要求 OEM 硬件供应商在新的、OEM 安装的 Windows 系统上满足这些要求——可能足以在大致相同的时间范围内实现相同的目标。

Windows 10 现在默认阻止可能不需要的应用程序

2021-08-05 21:03:55

微软已经开始在公司的 Windows 10 操作系统中开启 Windows Defender 的潜在有害应用程序 (PUA) 保护。微软表示，潜在不需要的应用程序“可能导致您的设备运行缓慢、显示意外广告，或者在最坏的情况下，安装可能更有害或更烦人的其他软件”。 Microsoft 指出，这些程序不是恶意软件，而是通常不需要或不需要的软件。该公司在 2018 年推出了对在 Windows 10 上阻止 PUA 的支持，但用户必须使用 PowerShell 启用该功能； 2020 年 5 月的 Windows 10 更新中引入了启用或配置 Windows 安全保护下的保护选项。当时仍处于禁用状态，用户现在可以在 Windows 10 设备上的 Windows 安全保护下配置保护。潜在有害应用程序保护将从 2021 年 8 月起默认开启，但仅适用于使用 Windows Defender 而不是第三方安全解决方案的 Windows 10 用户。如果您使用的是 Windows Defender，您可以通过以下方式检查和配置 PUA 保护：如果已打开，请选择是要阻止应用程序和下载，还是两者之一。同一页面链接到保护历史记录。您可以使用它来查看保护操作，包括被 Windows 的 PUA 保护阻止的应用程序或下载。

Configure Defender 等第三方程序也可用于管理保护和其他 Windows Defender 功能。 Windows Defender 不是唯一可以防止有害程序的程序。微软前段时间在其新的 Edge 浏览器中添加了 PUA 保护。潜在有害应用程序保护会阻止下载或安装某些程序。用户仍可取消阻止程序以完成下载或安装。误报很常见，类似于防病毒解决方案中常见的误报，尤其是在涉及新程序时。有经验的用户可能不会从 PUA 保护中获得多少收益，但对于没有经验的用户来说则不然。广告

微软的 Panos Panay 通过“即将推出”的 Focus Sessions 功能挑逗 Spotify 与 Windows 11 的集成

2021-08-06 20:19:39

微软的首席产品官 Panos Panay 最近一直有一些泄密事件，他戏弄 Windows 11 中的新 Spotify 集成以及其他新的应用程序重点。随着 Windows 团队为今年秋季晚些时候推出的 Windows 11 做准备，Panay 过去一周在 Twitter 上取笑了 Windows 工程师在内置应用程序中所做的工作以及与操作系统的日益集成。在 Windows 团队向 Dev 和 Beta 频道参与者发布了 Windows 11 Insider build 22000.120 后不久，其中包含一些 UI 更改、额外的小部件和大量修复，Panay 在 Twitter 上挑逗 Windows 11 的新“焦点会议”。 Panay 将 GIF 发布在他的 Twitter 提要上，简要展示了 Spotify 将如何集成到闹钟中，这似乎使用户能够选择在时钟应用程序中创建的策划时间范围内播放的播放列表。团队的另一个第一眼......即将推出#Windows11 上的#FocusSessions。这对我来说已经改变了游戏规则，尤其是与 @Spotify 集成 #Productivity #Creativity #WindowsInsiders pic.twitter.com/HfJh4niDiS Panay 没有提供太多其他细节，而且功能看起来非常简单，因为它融合了焦点辅助和 Spotify 播放列表。用户可以为他们的“焦点会议”以及他们想要在“焦点会议”期间播放的音乐流设置特定的时间长度与其他 Panay 戏弄一样，集成是微妙的，并且在大多数情况下，已经是Windows 10 的现实。但是，无需跳转到完整应用程序即可选择 Spotify 播放列表的便利性是对 Windows 10 当前可访问性和功能的独特体现。

今天的挑逗也遵循了突出的 Spotify 和 Microsoft 集成工作的历史，其中包括与 Xbox 服务、Xbox 应用程序、Cortana (RIP) 集成、Skype 插件以及 Groove 的首选替代服务的捆绑订阅。由于 Spotify/Clock 的新集成是 Panay 来自共享应用程序图标面板的第二个挑逗，一些人猜测未来公司的首席产品官可能会继续挑逗新的应用程序设计或功能。

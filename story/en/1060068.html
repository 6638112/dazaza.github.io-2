Click Studios, maker of the enterprise password manager Passwordstate, notified its 29K customers about a supply chain attack deployed via an update of the app
点击企业密码管理器MailyState的制造商，通知其29k客户了解通过应用更新部署的29k客户

2021-04-24 05:30:26

A mysterious threat actor has compromised the update mechanism of enterprise password manager application  Passwordstate and deployed malware on its users’ devices, most of which are enterprise customers.

一个神秘的威胁演员损失了企业密码管理器应用程序PasswordState的更新机制，并在其用户的设备上部署恶意软件，其中大部分是企业客户。

Click Studios, the Australian software firm behind Passwordstate, has notified its 29,000 customers earlier today via email.

点击PasswordState背后的澳大利亚软件公司的Studios，今天早些时候推出了29,000名客户。

According to a copy of the company’s communications, obtained by Polish tech news site  Niebezpiecznik, the malware-laced update was live for 28 hours between  April 20, 20:33 UTC and  April 22, 00:30 UTC.

根据公司通信的副本，由波兰科技新闻网站NieBezPiecznik获得，恶意软件更新在4月20日，20：33 UTC和4月22日00:30 UTC之间生活28小时。

Danish security firm CSIS, which dealt with the aftermath of this supply chain attack, published today an  analysis of the attacker’s malware.

丹麦安全公司CSIS处理此供应链攻击的后果，今天发表了对攻击者的恶意软件的分析。

Juan Andres Guerrero-Saade, Principal Threat Researcher at SentinelOne, said the attackers appear to have gained access to the Click Studios servers and added 4KB of code to an older version of the Passwordstate app.

Huan Andres Guantres Guerrero-Saade，Sentinelone的主要威胁研究员表示，攻击者似乎已经获得了点击工作台服务器的访问权限，并将4KB代码添加到较旧版本的PasswordState应用程序。

When installed, the app forced the victim’s password manager to download an additional ZIP file named “ Passwordstate_upgrade.zip” that contained a DLL file named “ moserware.secretsplitter.dll.” After installation, this DLL file would ping a remote command and control server, from where it would request new commands and retrieve additional payloads.

安装后，该应用程序迫使受害者的密码管理器下载包含名为“moseerware.seCretsplitter.dll”的DLL文件的“passwordstate_upgrade.zip”的其他zip文件。安装后，此DLL文件将ping远程命令和控制服务器，从它所请求新命令和检索其他有效载荷。

Unfortunately, once the intrusion was discovered, the attackers immediately took down their C&C server, which has prevented investigators from discovering what additional payloads and other actions the attackers performed.

不幸的是，一旦发现入侵，攻击者立即取下他们的C＆amp; C服务器，阻止调查人员发现攻击者所执行的额外有效载荷和其他操作。

Click Studios released a hotfix package [ ZIP] that would help customers remove the attacker’s malware, which the company named  Moserware. [ instructions are in the image above]

单击工作室发布了一个修补程序包[zip]，可以帮助客户删除攻击者的恶意软件，该公司命名为Moseerware。 [指令在上面的图像中]

But while it hasn’t been verified beyond a doubt what the attackers wanted from infected systems, by the nature of the software they compromised —a password manager— it is also very likely that the malware had full access to customers’ password stores.

但是，虽然尚未过度验证攻击者从受感染的系统所需的攻击，但是，由于它们受到损害的软件的性质 - 密码管理器 - Malware也很有可能从客户的密码存储完全访问。

As Guerrero-Saade pointed out on Twitter, there are tools currently available that can decrypt the Passwordstate vaults and recover cleartext passwords.

作为Guerrero-Saade指出Twitter，有工具目前可用的工具可以解密PasswordState Vault并恢复ClearText密码。

As a result, the Australian firm has recommended that customers change all the passwords they stored inside compromised Passwordstate password managers as soon as possible.

因此，澳大利亚公司建议客户更改他们存储内部密码密码管理器中存储的所有密码。

Since this is a password manager is sold primarily in bulk to enterprises, to whom it is advertised as an on-premises system, changing passwords won’t involve just email and website accounts, but also passwords for internal gear such as firewalls, VPNs, switches, routers, network gateways, and others, which many employees would most likely have saved inside the app thinking it was a secure local storage system.

由于这是一个密码管理器，主要以批量销售给企业，它被宣传为内部部署系统，更改密码不会涉及电子邮件和网站账户，还不会涉及电子设备，如防火墙，VPN等密码，许多员工最有可能在应用程序内保存的交换机，路由器，网络网关和其他人认为这是一个安全的本地存储系统。

“This is a real annoying breach,”  William Thomas, a malware analyst at UK security firm  Cyjax, told  The Record. “Imagine having to change all your passwords for each device on the network, on a Friday.”

“这是一个真正的恼人的违法，”英国安全公司Cyjax的恶意软件分析师威廉·托马斯告诉记录。 “想象一下，在周五，必须为网络上的每个设备更改所有密码。”

Audacity 3.0.0
Audacity 3.0.0.

2021-03-18 05:49:15

We’re pleased to announce release of Audacity 3.0.0 which replaces all previous versions for Windows, macOS and Linux.

我们很高兴地宣布发布Audacity 3.0.0，它替换了Windows，MacOS和Linux的所有先前版本。

Audacity 3.0.0 is a major update on our previous Audacity 2.4.2. We’ve changed the format in which we save Audacity projects! Previously we saved projects as a sometimes large number of small files, with an ‘.aup’ file to coordinate the lot. This way of doing things is sometimes called ‘pile of files’ storage. The problem, which happened all too often, was that data files and .aup file parted ways. Users quite reasonably expected the .aup file to contain the entire project. Well, the new .aup3 file does contain the data as well. The technical detail is that we are using an open source database, SQLite3, to store everything in one .aup3 file. That all happens ‘behind the scenes’. SQLite3 is open source, and it is a delight to work with. Nevertheless, this was a huge change, and we decided it was too risky to include many other changes we wanted to make at the same time – so 3.0.0 is almost entirely about this big format change.

Audacity 3.0.0是我们以前的Audacity 2.4.2的重大更新。我们改变了我们节省了Audacity项目的格式！以前我们将项目保存为有时大量的小文件，具有“.aup”文件来协调批次。这种做法的方式有时被称为“堆文件”存储。这个问题频繁发生了，是数据文件和.aup文件分开了方法。用户非常合理地预期.AUP文件包含整个项目。嗯，新的.aup3文件也包含数据。技术细节是我们正在使用开源数据库SQLite3，将所有内容存储在一个.aup3文件中。这一切都发生在幕后'。 SQLite3是开源的，它很高兴与之合作。然而，这是一个巨大的变化，我们决定太冒险了，包括许多我们想要同时制作的其他改变 - 所以3.0.0几乎完全是关于这种大幅变化的变化。

Working with .aup3 projects editing audio should on most machines be a little faster than before, because there are fewer files being worked on. Finishing and closing a project at the end of working can be quite a lot slower, since there is more to do when a project is closed. We think the trade offs are worth it.

使用.aup3项目编辑音频应该在大多数机器上比以前更快，因为正在进行更少的文件。在工作结束时完成和关闭项目可能会更慢，因为项目关闭时有更多才能做到。我们认为权衡是值得的。

Importantly note that you can open your older .aup projects in Audacity 3.0.0 where they will be converted to the new .aup3 format.

重要的是要注意，您可以在Audacity 3.0.0中打开较旧的项目。它们将转换为新的.aup3格式。

We did have time to improve our ‘Noise Gate’ effect and add a new analyzer, ‘Label Sounds’, that can label sounds and silences. We also made a few small tweaks elsewhere. You can now import and export macros, and there are a couple of new commands for using the last used tool or last used analyzer that you can give shortcuts to.

我们确实有时间提高我们的“噪音门”效果，并添加一个新的分析仪，“标签声音”，可以标记声音和沉默。我们还在其他地方进行了一些小调整。您现在可以导入和导出宏，并使用您可以为快捷方式提供快捷方式的最后使用的工具或最后使用的分析器有几个新命令。

We also fixed over 160 bugs that had been accumulating over the years. This is quite a staggering amount of work. The majority of these bugs were minor problems, easily worked around. Some though were really juicy high priority bugs that would have mattered a lot to the people affected by them. We’re really glad to have these bugs fixed now.

我们还修复了多年来累积的160多个错误。这是一个非常惊人的工作量。大多数这些错误都是轻微的问题，很容易解决。有些人真的是真正多汁的高优先突发，对受他们影响的人来说很重要。我们现在真的很高兴解决这些错误。

We hope you enjoy using Audacity 3.0.0 as much as we enjoyed putting it together.

我们希望您喜欢使用Audacity 3.0.0，因为我们喜欢将其放在一起。

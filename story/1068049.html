在PC在我们的生活中发挥更核心作用的时候，Windows 11旨在让您更接近您所爱的东西。

Windows一直存在成为世界创新的阶段。这是全球企业的骨干，陷入扰乱的初创公司成为家喻户晓的名字。网络出生并在窗户上长大。这是我们许多人写过我们的第一封电子邮件的地方，播放了我们的第一个PC游戏并写了我们的第一行代码。 Windows是人们要创造的地方，要联系，学习和实现 - 一个超过十亿人的平台依靠。

设计许多人的责任是我们不轻视的人。过去18个月带来了令人难以置信的转变，我们如何使用PC;我们从将PC融入我们的生活，试图将整个生活融入PC。我们的设备不仅仅是我们去参加会议，课程和完成事物的地方，但我们在与朋友玩游戏的地方，狂欢观看我们最喜欢的节目，也许最有意义地相互联系。在这个过程中，我们发现自己重建办公室聊天，走廊喋喋不休，锻炼，快乐时光和度假庆典 - 数字化。

什么是如此强大的是我们看到和感受到的PC中的转变 - 从实用和功能的东西到个人和情感的东西。当我们建立下一代Windows时，这就是激励我们的激励。为您建立一个感觉熟悉的地方，您可以在那里创建，学习，播放和最重要的是，以所有新的方式连接。

今天，我很谦卑，很高兴向你介绍Windows 11，这是让你更接近你所爱的窗户。

我们简化了设计和用户体验，以赋予您的生产力并激励您的创造力。它是现代，新鲜，清洁和美丽的。从新的开始按钮和任务栏到每个声音，字体和图标，一切都是故意让您控制并带来平静和轻松的感觉。我们在中心开始，使得更容易快速找到所需的东西。开始利用云和Microsoft 365的权力向您展示您最近的文件，无论您在早期查看哪些平台或设备，即使它位于Android或IOS设备上也是如此。

Windows一直是关于帮助您通过提供多个Windows的灵活性以及并排捕捉应用程序的灵活性来帮助您的工作方式。新的Windows 11中，我们正在引入捕捉布局，捕捉组和桌面，为多任务提供更强大的方式，并保持您需要完成的内容。这些是新功能，旨在帮助您组织您的Windows并优化您的屏幕房地产，以便您可以看到您在视觉清洁的布局中所需的方式。您还可以为您的生活中的每个部分创建单独的桌面，并为您的喜好定制它们 - 想象有一个桌面工作，游戏或学校。

让你更接近你所爱的东西让你更接近你所爱的人。最后18个月推动了我们如何用数字创建有意义的联系的新行为。即使我们开始返回更多的亲自互动，我们也希望继续让人们在他们身边彼此保持靠近。我们不希望您的设备或平台成为障碍。

使用Windows 11，我们很高兴从CassBar中集成的Microsoft团队介绍聊天。现在，您可以通过所有个人联系人，任何地方，无论在Windows，Android或iOS上都能立即通过文本，聊天，语音或视频连接到所有个人联系人。如果您连接到另一端的人没有下载团队应用程序，则仍然可以通过双向SMS连接它们。

Windows 11还通过团队与朋友和家人联系，让您立即与朋友联系，允许您立即静音和取消静音，或者直接从任务栏开始呈现。

如果您是游戏玩家，Windows 11为您制作。游戏一直是窗户的根本。今天，世界上围绕世界游戏周围的数亿人，并通过游戏找到与亲人和朋友的快乐和联系。 Windows 11解锁系统硬件的全部潜力，将一些最新的游戏技术为您服务。像：DirectX 12终极，它可以使令人叹为观止，高帧速率下的沉浸式图形; DirectStorage用于更快的负载时间和更详细的游戏世界;和汽车HDR为一个更广泛，更生动的颜色范围，以获得真正的迷人视觉体验。我们对硬件兼容性承诺的任何内容都没有改变 - Windows 11支持您最喜欢的PC游戏配件和外围设备。通过Xbox游戏Pass for PC或Ultimate，游戏玩家可以访问100多种高质量的PC游戏，并在所有的时间内添加新游戏，并且无论他们在播放的情况下，找不到人们才能很容易。 PC或控制台。

Windows 11将您更接近您关心的新闻和信息，以便使用小部件 - 通过AI和Microsoft Edge的最佳浏览器性能提供了新的个性化Feed。即使我们在我们最专注和创造力的时候，我们仍然需要休息 - 一会儿与外界联系或给自己一个心理重置。今天，我们经常拿起我们的手机来检查新闻，天气或通知。现在，您可以直接从桌面打开类似的策练视图。当您打开个性化的饲料时，它就像一块玻璃板一样滑过屏幕，因此它不会扰乱您正在做的事情。对于创建者和发布者，小部件还在Windows内打开新的房地产来提供个性化内容。我们的愿望是为全球品牌和当地创作者创造一个充满活力的管道，以至于消费者和创造者都可以受益。

新的Microsoft Store是您为应用程序和内容的单个可信赖的地点，用于观看，创建，播放，工作和学习。它被重建为速度，并以全新的设计为美观而易于使用。我们不仅会为您带来更多的应用程序，我们还在制作所有内容 - 应用程序，游戏，节目，电影 - 更容易搜索和发现策划故事和集合。我们很兴奋很快欢迎Microsoft团队，Visual Studio，Disney +，Adobe Creative Cloud，Zoom和Canva等第一和第三方应用程序到Microsoft Store - 所有这些都提供令人难以置信的娱乐，激励和连接您。当您从商店下载应用程序时，您可以了解它已被测试为安全和家庭安全进行了测试。

我们还宣布我们首次将Android应用程序带到Windows。从今年晚些时候开始，人们将能够在Microsoft商店发现Android应用程序，并通过Amazon AppStore下载它们 - 想象一下，从Tiktok或使用Khan Academy Kids从您的PC中获取虚拟学习的视频。我们将在未来几个月内有更多的分享这一经验。我们期待使用他们的英特尔桥技术与亚马逊和英特尔的合作关系。

我们正在采取措施进一步开设Microsoft商店以解锁创造者和开发人员的更大的经济机会。我们正在启用开发人员和独立的软件供应商（ISV），以带来他们的应用程序，无论是作为Win32，渐进式Web应用程序（PWA）还是Universal Windows应用程序（UWP）或任何其他应用程序框架，都会创建机会与更多人联系并聘用。我们还宣布了我们的收入股份政策的进步变革，其中应用程序开发商现在可以将自己的商业带入我们的商店并保留100％的收入 - 微软一无所获。 App Developers仍然可以使用我们的商业，竞争性收入份额为85/15。我们相信创建一个更开放的生态系统最终使我们的客户受益 - 给他们安全，无摩擦访问他们想要的应用程序，游戏，电影，节目和网页内容。

Raspberry Pi 4 V3DV graphics driver achieves Vulkan 1.0 conformance
Raspberry Pi 4 V3DV图形驱动程序实现了Vulkan 1.0的一致性

2020-11-25 16:40:07

Just a couple of weeks ago, we reported on the status of Raspberry Pi 4 Vulkan driver & future plans based on a presentation made by Igalia at the Open Source Summit 2020 at the end of October.

就在几周前，我们根据Igalia在10月底举行的2020年开源峰会上的演讲，报告了Raspberry Pi 4 Vulkan驱动程序的状态和未来计划。

At the time, the V3DV Vulkan Mesa driver for Raspberry Pi 4 was merged into Mesa, passed over 100,000 tests in the Kronos Conformance Test Suite (CTS), and was said to implement the full Vulkan 1.0 API. So it should come as no surprise that Khronos has now declared the Raspberry Pi drivers to be conformant with Vulkan 1.0 specifications.

当时，用于Raspberry Pi 4的V3DV Vulkan Mesa驱动程序已合并到Mesa中，在Kronos一致性测试套件（CTS）中通过了100,000多个测试，并据说实现了完整的Vulkan 1.0 API。因此，Khronos现在宣布Raspberry Pi驱动程序符合Vulkan 1.0规范就不足为奇了。

This was tested in Raspberry Pi OS with Linux 5.4.51 using X11 display server at 1920×1080 resolution on Raspberry Pi 4. Vulkan 1.0 conformance means the V3DV Mesa driver has passed all tests from Khronos CTS and should be compatible with most applications using this version of the API. The drivers will work on all Broadcom BCM2711 platforms including Raspberry Pi Computer Module 4 and Raspberry Pi 400 keyboard computer.

这已在Raspberry Pi 4上使用X11显示服务器在Linux 5.4.51的Raspberry Pi OS上以1920×1080分辨率在Raspberry Pi 4上进行了测试。符合Vulkan 1.0的要求是V3DV Mesa驱动程序通过了Khronos CTS的所有测试，并且应该与使用此工具的大多数应用程序兼容。 API版本。该驱动程序可在所有Broadcom BCM2711平台上使用，包括Raspberry Pi计算机模块4和Raspberry Pi 400键盘计算机。

But as we’ve seen in our previous post there’s still more work to do, as Igalia plans to improve performance by making better use of Texture Filtering Units (TFUs), optimize the Windows System Integration (WSI), as well as start working on the Vulkan 1.1 API implementation.

但是，正如我们在前一篇文章中看到的那样，由于Igalia计划通过更好地利用纹理过滤单元（TFU），优化Windows系统集成（WSI）以及开始进行开发来提高性能，因此还有更多工作要做Vulkan 1.1 API实现。

The Raspberry Vulkan driver is already included on the latest Mesa 20.3 release 20.3, but it may take a bit more time before it’s included in Raspbian/Raspberry Pi OS repositories.

Raspberry Vulkan驱动程序已经包含在最新的Mesa 20.3版本20.3中，但是可能要花一些时间才能包含在Raspbian / Raspberry Pi OS存储库中。

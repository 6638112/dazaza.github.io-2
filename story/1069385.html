如何发现一个好的假身份证

2021-07-24 08:52:49

我最喜欢的事情之一是人们在工作过程中培养的非常具体、奇怪的能力。木匠或许能够通过观察来估计任何东西的长度，建筑师或许能够从记忆中快速勾勒出非常准确的图画，等等。这篇文章来自这些特定能力之一。我在波士顿的一个朋友是一家繁忙酒吧的保镖。从他的工作中，他非常擅长识别假身份证。这不是很多人想到的技能。甚至酒吧也不会经常考虑它。大多数酒吧只是对假身份证进行粗略的检查：弯曲身份证以确保它是用正确的卡片纸制成的，尝试剥开边缘，检查照片与人的对比，并可能根据人的地址对人进行测验。如果他们有能力，酒吧也可能会尝试扫描 ID，看看扫描是否匹配。所有这些都有助于发现不良的假身份证。它们对于发现好的假身份证没有用。好的假身份证将使用与真实身份证相同的卡片。他们的层压板将无法被剥离。他们的照片将是该人自己拍摄的照片。他们的数据将是该人自己输入的实际数据，ID 将是可扫描的。它将显示完全相同的 ID 数据。为了发现一个好的假身份证，你必须比这更聪明。感受您的实际驾驶执照的边缘。你会注意到它的圆角很光滑。这是因为您的 ID 是激光切割的。

假 ID 的边缘可能很粗糙，或者几乎没有螺纹。这是手工切割或机器切割 ID 的标志。这些都是假的。硬塑料上的层压很难正确完成。自从它只是胶合层压板的日子以来，假身份证就已经出现了，但这并不意味着层压板是完美的。在您的真实身份证上，除了轻微的光泽外，层压板几乎不引人注意。它在卡片的末端结束，但如果您仔细观察卡片的边缘，您可以看到两到三层：层压板、卡片纸，可能还有另一层层压板。在假身份证上，层压板通常不会很好地粘在上面。层压板下面有褶皱或污垢，尤其是在边缘周围。它还经常在边缘之外戳一点点，如果直接在边缘上闪光，你就可以看到。即使层压板正确粘贴，假 ID 制造商也很难获得与真实 ID 制造商相同类型的层压板。他们使用替代品，而这些替代品往往太过于反思。您也经常可以在假护照上看到这种极端的反射率。您的 ID 有几个部分带有凸起和缩进的字样。在我的马萨诸塞州身份证中，我右下角的出生日期是凸起的，照片下方的姓名首字母和出生年份是缩进的。这样做既困难又昂贵。更便宜的 ID 制造商可能会试图欺骗它，尤其是缩进的字母。他们只会跳过缩进，使字母与卡片纸的其余部分保持一致。您将能够用手指感受它。

彩色、凸起的字体尤其困难。假冒 ID 制造商很难将字体完全着色，从而在字体上留下斑点或斑点。全息图曾经是 ID 的高级安全功能。不幸的是，情况不再如此。大多数假身份证制造商对全息图都没有问题。然而，他们仍然会在全息图的质量上便宜。例如，在我的马萨诸塞州执照上，有一只鸟的全息图。它有一些伪造 ID 制造商很难复制的特征：一个小的镂空眼睛，上面写着“马萨诸塞州”的小字样，里面嵌入了我的姓名首字母和出生年份。很少有假身份证制造商能够直接复制这种质量的全息图。所以他们会试图作弊。他们可能只是直接在卡片纸上打印全息图，希望人们不会意识到它根本没有凸起。或者，他们可能会放弃切除眼睛。无论如何，如果您熟悉正确的全息图的外观，尤其是全息图升起并与卡本身分离的方式，那么当假冒 ID 制造商放弃时，您就可以毫无问题地发现它。假身份证制造商的业务是生产足够好的假身份证，以至于过度劳累的保镖或刚花完最后 100 美元的大学生都满意。这两个群体有什么共同点？他们真的检查身份证的正面，并浏览背面。

假身份证制造商知道这一点。所以他们通常会在背面的字母上便宜。与正面的字母不同，背面的字母可能错位、有斑点或连在一起。另一个非常常见的安全功能是激光打孔。如果您查看 ID 的背面，则更容易查看。你会注意到微小的小洞，可能是你所在州的形状。要制作干净、排列整齐的孔，您需要昂贵的激光器。正如您现在可能已经发现的那样，这不是假 ID 制造商倾向于拥有的东西。因此，他们尽最大努力使用机器冲压。这会导致排列不良的孔，可能看起来很凌乱或有点伤痕累累。你还记得你为驾照拍照的时候吗？他们让你站在一个空白的背景前。你脸上有强烈的直射光，然后他们拍了一张高质量的照片。这不是人们拍摄假身份证照片的方式。他们的朋友用他们的照相手机在空白墙前拍照。然后假身份证制造商将照片photoshop到身份证上。因此，在伪造的身份证照片中，可能会有 Photoshop 留下的伪影，例如头发边缘。强烈的直射光可能不会产生阴影。照片可能质量差或模糊。

假身份证比以往任何时候都好，这在很大程度上推动了 19 岁年轻人在酒吧喝酒的共生需求，以及美国城市对未成年人饮酒的罚款酒吧。这可能是有问题的，因为这也意味着非未成年饮酒犯罪分子也可以获得令人难以置信的高质量假身份证。但是，这些假身份证并不完美。即使您不熟悉个人身份证或国民护照的外观，您仍然可以发现伪造的。你只需要仔细观察。

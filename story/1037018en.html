MODNet: Is a Green Screen Necessary for Real-Time Human Matting?
MODNet：实时人为拍摄是否需要绿屏？

2020-11-28 23:24:35

This is the official project of our paper Is a Green Screen Really Necessary for Real-Time Human Matting?

这是我们论文的正式项目。实时人类抠图真的需要绿屏吗？

MODNet is a trimap-free model for human matting in real time (on a single GPU).

MODNet是一种无需Trimap的模型，可实时（在单个GPU上）进行人类抠像。

I have received some requests for accessing our code. I am sorry that we need some time to get everything ready since this repository is now supported by Zhanghan Ke alone. Our plans in the next few months are:

我已经收到一些访问我们代码的请求。很抱歉，我们需要一些时间来准备好一切，因为此存储库现在仅由Zhanghan Ke支持。我们未来几个月的计划是：

We first plan to publish an online image/video matting demo along with the pre-trained model in Dec. 2020.

我们首先计划在2020年12月发布在线图像/视频抠像演示以及预先训练的模型。

We then plan to release the code of supervised training and unsupervised SOC in Jan. 2021.

然后，我们计划在2021年1月发布监督培训和无监督SOC的代码。

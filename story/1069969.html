Element 是一款基于去中心化 Matrix 协议的消息应用，融资 3000 万美元

2021-07-27 15:15:33

Element 在 B 轮融资中获得了 3000 万美元。部分资金将用于扩展其技术，以继续迎合已经使用它的大型组织的用例。其中包括覆盖约 550 万公务员的法国政府； Dataport（正在将其纳入德国的教育和公共管理系统）；该公司表示，BWI 提供德国武装部队使用的通信系统——总共约有 10 次政府参与。 Element 的首席执行官兼开源非个人资料 Matrix 协议的技术联合创始人 Matthew Hodgson 也表示，其中一些将用于继续投资公司的点对点架构，以消除对服务器的任何需求。第三个投资领域将是建立公司的分散式语音和视频会议服务。 “这将是安全方面的变革，”他在接受采访时说。今天的 Element 可用作内部部署系统或基于云的平台。这家初创公司表示，大多数公共部门客户选择前者，而私营部门客户则选择后者。过去 12 个月，云收入增长了 300%，虽然该公司没有透露实际收入有多少，但这表明 Element 正在增长（另一个现在接受资金的原因，尽管作为联合创始人 Amandine Le Pape 指出，它不需要它）。这一轮包括来自 Protocol Labs（libp2p、IPFS 和 Filecoin 背后的开源研发实验室）和 Metaplanet（由 Skype 联合创始人 Jaan Tallinn 设立的基金）的投资。 Automattic 和 Notion——过去的投资者——也参与其中。没有披露估值。迄今为止，Element 已筹集了 4800 万美元。 Matrix 是 Element 所基于的非盈利协议，旨在通过创建一个地方来改变互联网的工作方式，在这个地方可以将多个孤立的通信环境集成并参与到一个有凝聚力的单一平台中，并且这些通信反过来可以由组织它的实体“拥有”。这里的想法是，通过将所有内容整合在一起，可以更轻松地从安全性和实际使用的角度管理这些对话。 Matrix 本身一直在增长，在过去 12 个月中使用率增长了 190%，现在已达到 75,000 次部署，拥有超过 3500 万“可寻址”用户。这些部署可以在小到 Raspberry Pi 或由政府运行的大型服务器上进行。但需要明确的是，“活跃”的数量远小于可寻址用户。虽然 Matrix，更不用说 Element，无法“看到”其他人如何使用 Matrix（这是政府喜欢的安全优势之一），但霍奇森指出，在其自己的服务器上有 120 万活跃用户。

Element - 最初被称为 Riot - 被设定为一种 Slack/Discord 竞争对手，它是 Matrix 之上的本地消息传递平台，在那里捕获已经使用该协议做其他事情的观众，为他们提供干净和安全的替代方案在只有少数平台控制着世界上大部分消息传递数据的时代。 “我们开发消息传递技术已经超过 15 年了，”Le Pape 说。 “让几个玩家[支配]并保留每个人的数据作为人质，这感觉很疯狂。这是为了打开任何人的交流。这就是我们正在努力解决的问题，数据主权。”在这方面，Element（和 Matrix）处于一个更大的技术趋势的核心，该领域的一些最聪明的人正在考虑这一切如何运作的默认值，已经决定了一个替代方案，一个不那么集中的随着数据变得更有价值，通信渠道变得越来越重要，这种方法必须成为更大等式的一部分。 “互联网通信协议已成为人类的基础。今天，大多数消息传递发生在变幻无常的专有集中围墙花园中，这些花园让我们受制于他们的短期业务前景，”Protocol Labs 首席执行官 Juan Benet 在一份声明中说。 “Matrix 是希望的灯塔：一个开放的网络，用于安全、分散的通信，以健全的互联网基础设施原则构建。 Element — 产品和公司 — 赋予组织控制权、所有权，并为他们的所有对话提供出色的用户体验。”虽然市场上还有许多其他加密消息平台正在被大规模采用——包括 Telegram、Signal，甚至在某种程度上 Facebook 的 WhatsApp——但 Element（和 Matrix 的）早期采用者一直是大型组织。然而，随着 Element 引擎 Matrix 也在其 Blue Sky 去中心化平台工作中与 Twitter 等公司进行早期合作，这可能会给 Element 带来除早期采用者之外的另一次用户增长。但正如云层有一线希望，晴朗的前景也会投下阴影……与 Telegram 和 Rocket.chat 等其他去中心化通信平台一样，端到端加密可以双向切割：它可以帮助防止通信被黑客入侵，但是它还可用于在计划恶意活动时逃避检测。 Matrix（以及默认情况下的 Element）一直在探索后门进入加密系统的替代方案，这是政府正在考虑解决此问题的一种途径，其论点是强制一个只会将不良行为者转移到另一个。最终，尽管存在这些疑虑和存在的问题，但这里的投资和 Element 的使用是对坚持去中心化方向的投票。

“当交流集中化时，它就会成为一个非常有吸引力的滥用目标；无论是通过宣传、监视、审查还是更糟的方式，”Metaplanet 的塔林说。 “消费者需要从监视资本主义中解脱出来，组织需要一种安全的中立沟通方式。 Matrix 是提供缺失的通信层的最先进平台。”

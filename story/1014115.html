今年的Xbox游戏展示会清楚地表明，微软对下一代Xbox游戏的真正关注点不是硬件，而是Xbox游戏通行证订阅

2020-07-25 20:29:05

跳到主要内容如果你昨天前往微软的Xbox游戏展示会，见证下一代游戏和“世界上最强大的游戏机”的重要性，你可能会有点失望。微软的活动展示了Xbox和Xbox Game Studios内容的未来，但它并没有说服我为什么应该购买Xbox Series X。相反，它明确表示，Xbox Series X只是你玩Xbox游戏的众多方式之一，微软真正的下一代重点是Xbox Game Pass。

几个月来，我一直在写关于Xbox Game Pass的重要性，以及微软利用订阅服务比传统游戏机接触到数百万Xbox玩家的战略。微软希望为视频游戏打造Netflix，初步迹象显示，它的赌注已经开始获得1000万订户的回报。一些开发者也报告了游戏销量的增长和更多的玩家，微软在XCloud的游戏通行证方面有一些宏伟的计划-特别是在即时玩游戏或演示的能力方面。

微软昨天展示了其15家工作室中的9家，推出了5款Xbox游戏工作室的新游戏和4款新的第三方游戏。它是Xbox Series X及其内容的路线图，但更重要的是，它是游戏通行证的预期路线图。展出的游戏种类繁多，尽管还不清楚我们什么时候能玩到大部分游戏。然而，微软在整个展示过程中做出了明确的承诺：每一款游戏都将在第一天进入Xbox Game Pass。

甚至连与谷歌体育场专业版(Stadia Pro)订阅捆绑在一起的游戏“命运2”(Destiny 2)也将在9月份推出Xbox Game Pass。尽管“命运2”的内容存在问题，但在Xbox One、PS4和PC上，它仍然是一个每天有100多万人玩的游戏。邦吉还对“命运2”制定了宏伟的计划，该计划将延续到2022年以后，虽然这里没有进行独家内容交易，但它肯定会在未来几年将Xbox打造成“命运”的平台。

邦吉的Xbox游戏通行证承诺是在微软首次展示Halo无限游戏后20分钟左右做出的。被描述为邦吉最初的Halo的“精神重启”：战斗的进化，微软已经选择在可预见的未来将Halo带向开放世界的方向。一个8分钟的游戏演示展示了酋长的新格斗钩，开放世界的光环，以及野兽的回归。

虽然看到首席大师回来与咕噜和野兽战斗会让铁杆的光环粉丝感到高兴，但这款游戏几乎没有展示为什么Xbox Series X很重要。许多人认为光晕无限将作为Xbox系列X的视觉或技术展示，特别是因为它计划与新游戏机一起发布。

“光晕无限”的游戏展示让粉丝们对游戏的玩性和视觉效果不屑一顾，整个艺术方向感觉过时了。其他粉丝只是想知道为什么开放世界很重要，以及343工业公司对光环无限在未来几年将如何适应的计划。但是，在离发射只有几个月的时候，仍然有很多这样的问题没有得到回答。

“”光环无限“延续了”光环4“和”光环5“中师傅酋长的传奇故事，这真的是”光晕“下一代游戏的开始，”“光环无限”工作室负责人克里斯·李(Chris Lee)在昨天的媒体问答环节中解释道。“我们认为这是一个平台，将随着时间的推移不断发展壮大，并将继续为其带来新的故事。您不需要有多个版本才能有一个完整的故事，包括开头、中间和结尾。我们正在考虑如何在未来几年做到这一点。“

目前还不清楚光环无限是否会成为你通常会与下一代游戏机联系在一起的那种系统销售商，但这可能甚至无关紧要。微软不希望你必须购买光环无限的副本，甚至升级到新的Xbox Series X游戏机才能玩它，因为它将在PC和Xbox One上都可以使用。微软只是想让你订阅Xbox游戏通行证--今年晚些时候，“光晕无限”将在Xbox游戏通行证的第一天推出，这是你这么做的一个重要原因。

微软的Xbox游戏展示让Xbox Series X感觉像是Xbox游戏通行证的另一个切入点，而不是一个必要的购买。现有的游戏机也可以访问Xbox Game Pass，比如微软将继续制造和销售的Xbox One S。还有数十亿台Windows10PC或数百万移动设备将通过XCloud流媒体访问Xbox游戏。Xbox Series X是玩Xbox游戏的最佳方式之一，但它肯定不是微软在游戏领域雄心勃勃的唯一方式或更广泛的重点。

微软预计还将推出第二款更便宜的下一代Xbox，最快可能在下个月发布。第二个控制台设计为更实惠的选项，考虑到了1080p和1440p显示器。微软可能会公布Xbox Series X和代号为“Lockhart”的第二款游戏机的价格，一旦它准备好正式确认它的存在。

第二代Xbox似乎越来越有可能成为微软对Xbox游戏通行证的大力推动，并成为一款基于订阅的Xbox。微软已经有了Xbox All Access订阅，提供带有Xbox Game Pass旗舰版的Xbox One S全数字版，其中包括Xbox Live Gold和Xbox Game Pass访问。第二个更便宜的下一代Xbox将完美地嵌入Xbox All Access，作为Xbox Game Pass Access的实惠订阅。

微软对Xbox和Xbox Series X的整体策略与索尼对PlayStation5的整体策略明显不同。索尼正在采取更传统的路线，推出下一代产品，专注于PS5的独家游戏。微软正在对内容进行长期投资，以营销Xbox Game Pass，并试图接触到比传统下一代游戏机发布更多的人。

微软在过去的几个月里一直在戏弄其Xbox Series X游戏机的技术细节，但昨天显然是Xbox游戏通行证内容的游戏展示窗口，而不仅仅是X系列游戏机。微软也淡化了Xbox Series X游戏机的重要性，承诺你不需要购买它就可以玩“我们在未来几年发布的Xbox游戏工作室的游戏”。虽然这一消息昨天变得有点模糊，但很明显，微软展示的一些游戏在未来几年内根本不会推出。一旦微软明年升级了服务器，Xbox One的拥有者很可能仍然可以通过XCloud流媒体播放它们。

Xbox Series X仍然是一款功能强大的游戏机，它将为下一代改进现有的和即将推出的游戏。但微软仍需要清楚地证明这一点。我们现在知道微软计划在未来几年推出Xbox Series X和Xbox Game Pass内容，但还需要更多来证明为什么“世界上最强大的游戏机”如此重要。

虽然下一站是非常重要的Xbox Series X价格，但可以肯定的是，Xbox游戏通行证订阅是微软下一代Xbox游戏的真正关注点。微软不再以游戏机销量来衡量其Xbox的成功，而是以未来有多少人通过订阅Xbox游戏通行证来玩Xbox游戏。

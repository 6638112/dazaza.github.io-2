AWS派出了一台在场所运行的机架式计算机。最近的re：Invent举办了许多关于前哨站变更的会议。在没有大张旗鼓的情况下发生的一个变化是新的更低的价格（注意：LOW-ER，而不是LOW）。去年发布后不久，我查看了Outposts的定价。与该分析一样，数字中还隐藏着更多的故事和奇怪之处：客户和销售反馈，爆破气球，长达数月的误会，以及（与任何大流行时代的故事一样）恢复为乏味。

AWS Outposts是驻留在您自己的设施（DC或Colo）中的云的扩展。这是一个42U机架，带有不同数量的EC2实例类型（因此，还有不同数量的CPU线程（vCPU），DRAM，GPU和SSD容量）。它不是私有云，因为它紧密绑定到AWS控制平面（以及特定的AWS区域和可用性区域）。相反，它实际上是与您的其他系统接近的一小部分AWS。

在去年的分析中，我们发现在开始添加CPU和DRAM等有用组件之前，Outpost机架的封面费用约为20万美元。那是机架，电源，开关，风扇等其他东西吗？车轮？如果这听起来并不疯狂，那就是。深。这些组件的边缘一定是天文数字。也许有些顾客注意到并反叛了。

所有型号的机架价格降低121,651.20美元，是造成所有价格变化的原因，但3除外：一种合理，一种烦人和一种愚蠢。

一年前最便宜的开发人员单位（实际上是两个型号，仅按其外部网络类型区分）有特别折扣。公布每个AWS输出模型均支持各种EC2实例类型。使用一些代数（或Excel Solver），我在Outposts中计算了每种实例类型的价格。大多数产品的价格与其云计算EC2类似产品（预留3年实例）的价格类似（±8％）。但是，去年，m5.12xlarge的价格降低了80％，但该实例类型仅出现在最便宜的开发单位中，从表面上看，这使它成为吸引人的模型，可以用它来踢轮胎。

不再！在2021年，m5.12xlarge不再是像差（请参见上文）。开发部门仍然比一年前便宜，但仅减少了5.6万美元（而不是12.1万美元），因为它实际上已经折价了6.5万美元。茉莉花赠与茉莉花被带走。这也意味着它不再是最便宜的模型-从来没有应该这样做，因为以前具有更少vCPU和DRAM的其他开发模型更昂贵。

最小的开发单元带有96个线程和192GB的DRAM。即使新的低价（低价）为11.8万美元，也似乎是……剥削性的。戴尔的同类服务器（单个）约为该价格的1/5。

请注意，在Outposts中出现的实例类型中，与3年保留实例价格进行比较时，m5.12xlarge是唯一看到价格大幅下降的实例类型。我对此感到非常惊讶。 AWS代表了不懈的资本主义与稳定的技术进步的交集。我们不是承诺云中的计算将越来越便宜吗？我的摩尔定律红利在哪里？与许多增量收益一样，这些收益不成比例地流向了无情的资本家。稍后会详细介绍，但这里有个展览A，说明为什么未来将不再是Jeff Bezos拥有和操作地球上每台计算机的地方。

OR-LR4DI6P（带有星号）是折扣最明显的Outpost模型。它支持少量实例类型，可容纳1152个vCPU和5TB DRAM。好消息是它的价格比其他任何产品都下降了4万美元。坏消息是它之前的定价过高了4万美元。去年，我把这个家伙称为异常。试图减少型号数量是否定价过高？对于一年前的新产品线来说，这似乎很奇怪。寻找愚蠢客户的水井是否有用呢？无论走到哪里，实验似乎都结束了。

另一个短暂的实验是已经停产的OR-M6GJ8FE。这是2 x g4dn.12xlarge和4 x m5.24xlarge的，这是唯一包含启用GPU的实例的开发单元。到目前为止，它也是定价最高的模型，比按实例类型的单价计算应该高出21％/ 12.5万美元。很难想像它不应该升级到OR-H7KSNNO的情况-多花2万个g4dn.12xlarge和3个m5.24xlarge可以多赚9万美元。它已或多或少地被OR-LMF5LJ1取代，后者只有2 x g4dn.12xlarge，价格为16.1万美元。

降价幅度最大的是OR-HUZEI16，具有8个m5.24xlarge和4个c5.24xlarge（1152个vCPU和3840GB DRAM）。这个人从$ 791,586.16下降了$ 200k到$ 590,216.36。

奇怪的是，单位数学中没有任何变化。一年前，它的价格正好等于其各部分之和。今年同样的故事。事实证明，改变的是它的组成部分。一年前，这4个c5.24xlarge实例是4 x r5.24xlarge； c5表示“将m5的内存减半”，r5表示“将m5的内存加倍”。去年，更高的价格适用于带有额外2304GB DRAM的单元。

回顾一年前的数据，我现在发现差异来源不一致。网站表示r5，而控制台（和API）表示c5。

某个地方肯定有转录错误；显然，定价团队中的量化人员正在查看网站（r5），而不是控制台（c5）。有人在过去的一年中弄清楚了。我想认为这是一个客户，他们一直在寻找缺少的2TB DRAM。

AWS尚未宣布其即将推出的2U Outpost模型的价格，但是据此我们可以做出一个很好的猜测（1U将使用Graviton处理器，因此我们没有从该模型推断的数据）。 128个vCPU，512 GB DRAM，8 TB SSD，以及一些用于精简机箱的额外设备，我希望它的价格约为7.5万美元。真是疯了。

也许AWS会让我们感到惊讶，但我没有屏息。前哨站不仅价格昂贵，而且价格高估-尽管下降幅度很大，而且相当可观。

再接再厉，但是我从戴尔网站上挑选了一些看起来很公平的零件。前哨基地中可能会有更昂贵的零件，我无法解释服务器中的所有其他东西，但是CPU / DRAM / GPU / SSD将主导其他所有事情。前哨基地的定价至少保持温和。

我了解客户为何需要前哨站。尽管这篇文章变得越来越松懈，但我还是AWS的忠实拥护者，并对他们的创新速度表示敬意。客户需要API驱动的基础架构，它提供的灵活性和开发速度。由EMC / Dell / VMware企业集团最简洁地代表的传统本地供应商未能将自己的方式发展为这种体验，这就是为什么超级扩展程序（包括Outposts）的硬件和软件看起来截然不同的原因。客户面临着字面上的困境：一方面，他们可以继续从Dell / HPE / etc购买价格适中，操作昂贵，使用效率低下的旧设备，或者他们可以为AWS Outposts之类的东西付出高昂的代价。前者是为了赚钱而又是为了赚钱。后者将越来越多的控制权交给了AWS，并将它们锁定在无法负担的未来。

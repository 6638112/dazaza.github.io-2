支持LLVM的Bazel构建系统

2020-10-29 10:37:51

This repo contains standalone Bazel BUILD configuration for part of the LLVM project that could be shared by dependent projectsusing the Bazel build system.

此回购包含部分LLVM项目的独立Bazel构建配置，这些配置可由使用Bazel构建系统的相关项目共享。

It is similar to the BUILD files for LLVMand MLIRin the TensorFlow project, but aims to besuitable for more general usage.

它类似于TensorFlow项目中LLVM和MLIR的构建文件，但旨在适合更通用的用途。

MLIR has Cuda support and the mlir-vulkan-runner disabled. LLVM does not yethave configuration detection and hardcodes values in the config.

MLIR支持CUDA，并且禁用了mlir-vulkan-runner。LLVM不会进行配置检测，并在配置中硬编码值。

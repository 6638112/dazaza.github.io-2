从泄露的源代码编译的Windows XP和Server 2003

2020-10-01 03:26:52

一位Windows开发人员上周从网上泄露的源代码中成功编译了Windows XP和Windows Server2003。

上周，在4chan论坛上，一组43 GB的微软源代码作为洪流发布。

该集合包括许多Microsoft操作系统的源代码，包括Windows XP SP1、Windows Server 2003、MS-DOS、Windows CE和Windows NT。

源代码发布后，一位名为NTDEV&39；的Windows开发人员从泄露的源代码成功编译了Windows XP。

但是，Windows XP源代码缺少关键文件，如Winlogon.exe，这使得无法安装操作系统。

因此，在进一步检查时，XP源代码可能不像我们之前认为的那样完整，因为它没有编译关键文件winlogon.exe。然而，泄漏中也有一个Server2003 sc，所以让我们看看这会走到什么程度。敬请关注！"；开发商在推特上写道。

在Windows SERVER 2003源代码方面，NTDEV取得了更大的成功，能够编译操作系统并将其安装到虚拟机中。

Bleepingcomputer不会提供指向泄露的源代码的链接，因为它是微软的知识产权。

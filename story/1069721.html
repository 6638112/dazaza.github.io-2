Haiku R1/beta3 发布

2021-07-26 05:06:30

诗歌在运动。 Haiku 项目及其开发人员和团队成员宣布，Haiku 操作系统于 2021 年 7 月 25 日发布了第三个 beta 版本，版本 R1/Beta3。版本 R1B3 延续了更频繁发布的趋势，为用户和开发人员提供了一个最新且稳定的平台工作。此版本结合了 Haiku 作为 BeOS 精神继承者的最佳历史和充满激情的社区的辛勤工作。它提供了几个使 Haiku 变得更好的新功能和性能改进。 Beta 3 包括系统和捆绑应用程序的新捷克语翻译。有了这个添加，俳句现在有 28 种不同的语言。 Haiku 项目感谢那些付出时间确保 Haiku 支持多种语言的人，以便操作系统可以在世界各地使用。由 Haiku 团队 WebPositive 开发的捆绑式 Web 浏览器的后端 WebKit 获得了多项重大改进。这为 WebPositive 的进一步改进以及改进的浏览和网站渲染体验提供了良好的基础，开发人员将继续关注下一个版本 Beta 4 和 asHaiku 接近其第一个初始版本 R1。更好的安装过程，对磁盘分区程序 (DriveSetupGUI) 进行了各种修复，使安装过程更轻松、更顺畅。各种软件的新端口和更新端口，包括弃用 Python 2 和使 Python 3 成为安装的默认版本 所有这些改进都已为社区和社区实施，以便最终用户更好地控制他们每天使用的设备.

当然，这一切都离不开世界各地捐助者的慷慨捐助，也离不开一个充满热情和勤奋的社区。俳句项目感谢那些付出时间或捐款的人。 Haiku 可以随时使用开发人员和志愿者，为任何有兴趣的人提供大量机会，让他们留下自己的印记，让计算世界变得更美好。 Haiku 是用于个人计算的免费开源操作系统 - 下载以 ISO 文件形式提供，该文件可以刻录到物理介质上，然后安装到物理硬件上，或在虚拟机中使用。请注意，此版本的新闻资料袋中包含屏幕截图、徽标和其他资源，以便在出版物中使用。 Haiku 是一个受 BeOS 启发并在 MIT 许可下许可的免费开源操作系统。 Haiku 专门针对个人计算，是一个快速、高效、简单易用、易学且功能强大的系统，适用于各个级别的计算机用户，可以从官方项目网站免费下载 ISO 文件。由热情友好的志愿者社区（统称为“Haiku 项目”）维护，Haiku 欢迎所有志愿者和贡献者，无论他们的技能水平或背景如何。有关更多信息，请访问 https://haiku-os.org Haiku Inc.一个 501(c)(3) 非营利组织（在美国注册），致力于支持俳句项目和俳句的发展。 Haiku Inc. 持有 Haiku 的商标注册，代表 Haiku 项目募集捐款并管理财务。过去，收集的捐款用于资助有偿开发合同和 Haiku Code Drive——一个类似于谷歌代码之夏的计划。欲了解更多信息，请访问 https://haiku-inc.org

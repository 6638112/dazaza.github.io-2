Valve 正在与 AMD 合作，使 Steam 平台支持 Windows 11

2021-08-10 01:30:52

Valve 的目标是让其 Steam Deck 掌上游戏 PC 为 Windows 11 做好准备。虽然我们几周前就知道 Steam Deck 可以运行 Windows，但目前尚不清楚 Valve 对它的支持程度，或者是否可以选择将启用可信平台模块 (TPM) 以在 Steam 平台上安装 Windows 11。现在，Valve 已经确认它一直专注于 Windows 支持。 “现在有一些工作着眼于 TPM，” Valve Steam Deck 设计师 Greg Coomer 在接受 PC Gamer 采访时说。 “到目前为止，我们已经非常关注 Windows 10，但我们还没有真正深入研究它。我们的期望是我们能够实现这一目标。” Valve 正在与 AMD 合作，以确保在 BIOS 级别支持 TPM，并且 Steam 平台已为 Windows 11 做好准备。“所以没有任何迹象表明 Windows 11 会出现任何问题，”解释说库默。对于今年晚些时候推出的 Steam 平台上安装 Windows 11 的能力来说，这听起来令人鼓舞。虽然手持设备将配备 SteamOS（Linux 的自定义版本），但 Valve 将支持 Windows 安装。那么你为什么要在 Steam 平台上使用 Windows？ Valve 仍在努力让带有反作弊功能的游戏在这款掌机上开箱即用，并且不能保证 Apex Legends、Destiny 2、PUBG、Fortnite 和 Gears 5 等游戏在没有 Windows 的情况下也能运行。 “我们正在与 BattlEye 和 Easy Anti-Cheat 合作，以便在 Proton 发布之前获得支持，”Valve 说。 Steam Deck 使用 Valve 的 Proton 软件，让很多官方不支持的 Windows 游戏在设备上运行，但反作弊一直是 Proton 近年来最头疼的问题。 Windows 支持在这里避免了明显的兼容性问题，但它会带来一个不适合 7 英寸屏幕的界面，以及许多未知数，直到我们看到操作系统在 Steam 平台上的运行情况。

Bash-LSP: A language server for Bash
BASH-LSP：BASH的语言服务器

2021-04-02 03:13:32

Bash language server implementation based on Tree Sitter and its grammar for Bashwith explainshell integration.

基于树立视图的Bash语言服务器实现及其语法为Bashwith ExplateShell集成。

If you encounter installation errors, ensure you have node version 8 or newer ( node --version).

如果遇到安装错误，请确保您有节点版本8或更新（节点--version）。

For Vim 8 or later install the plugin prabirshrestha/vim-lsp and add the following configuration to .vimrc:

对于VIM 8或更高版本安装插件PRABIRSHRESTHA / VIM-LSP并将以下配置添加到.vimrc：

if executable( 'bash-language-server ') au User lsp_setup call lsp#register_server({ \ 'name ': 'bash-language-server ', \ 'cmd ': {server_info ->[& shell, & shellcmdflag, 'bash-language-server start ']}, \ 'allowlist ': [ 'sh '], \ }) endif

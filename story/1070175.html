KeyCastr – 开源按键可视化工具

2021-07-29 21:24:38

KeyCastr 需要访问 macOS Accessibility API 才能接收您的按键事件并广播您感兴趣的按键。在较新版本的 macOS (10.15+) 上，系统偏好设置应用程序中的安全和隐私下有一个新的输入监控菜单，并且 KeyCastr 会在您第一次运行时自动出现在那里。只需解锁此菜单并选中 KeyCastr 旁边的框即可启用它。在旧版本的 macOS 上，或者如果由于某种原因应用程序没有出现在输入监控菜单下（或者如果您想预先启用它），那么您可以手动将其添加到辅助功能菜单中的应用程序列表中。要将 KeyCastr 添加到列表中，请单击 + 按钮并从文件系统中选择 KeyCastr。如果 KeyCastr 已经在列表中，则单击 - 按钮并再次添加 KeyCastr 以确保选择了正确的应用程序。如果您想显示的不仅仅是修饰键，请确保选中“显示所有按键”复选框。或者，取消选中此框以仅显示修饰键（例如 ⇧ ⌃ ⌥ ⌘）

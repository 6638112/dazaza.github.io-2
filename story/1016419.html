三星和微软扩大了合作伙伴关系，在Galaxy Note 20设备上增加了与Office和Windows 10的更深层次集成

2020-08-05 23:43:16

作为8月5日Galaxy Unpack虚拟活动的一部分，三星宣布计划将其新的Galaxy Note 20系列设备与微软的Windows 10和Office应用程序进行更多整合。这些整合建立在两家公司之间可以追溯到几年前的现有合作伙伴关系之上。三星官员今天表示，它将支持微软手机应用程序即将推出的更多功能。目前，三星使用现有的Windows链接集成功能，使用户能够直接从Windows10电脑访问手机信息、通知、照片和通话。某些三星手机已经允许用户从他们的PC上控制手机应用程序，Note 20系列设备也将获得这一功能。微软正在努力让手机上的应用程序屏幕镜像功能更加完整。我猜这将是想要在Windows上运行Android应用的用户实现这一目标的方式。

此外，根据今天的公告，今年晚些时候，用户将可以在他们的Windows10个人电脑上并行运行多个移动应用程序。(这些功能也将在新的Galaxy S7选项卡上运行，对于任何想要以这种方式使用它的人来说，它都可以作为Windows10的第二个屏幕。)。最新消息：微软有一篇博客文章解释了这一功能的工作原理，以及它将如何推动你的手机向前发展。帖子中写道：

当您启动应用程序时，它会在一个单独的窗口中打开，让您可以同时处理多项任务，从而省去了登录或设置的时间和麻烦。你还可以将手机的Android应用程序固定在Windows10任务栏或开始菜单上，以便快速方便地访问，让你在结束电话会议时查看社交信息源或点午餐等事情变得更加容易，而无需拿起手机。"；

官员们今天表示，三星和微软还将使Samsung Notes应用程序能够与Microsoft OneNote和Outlook同步，并提醒与Outlook、待办事项和团队同步。此同步功能将于今年晚些时候在OneNote for Web、OneNote for Windows 10和Outlook for Web中提供。为了使其正常工作，用户需要登录到相同的Microsoft帐户(注意：与OneNote的同步仅为单向/导出)。三星还表示，从9月15日开始，它将在Galaxy Note 20上支持XCloud游戏流媒体(现在称为云游戏)和Xbox Game Pass旗舰版。Xbox Game Pass旗舰版的测试版将于9月15日上市。从2020年8月21日开始，Galaxy Note20系列将由运营商提供，并通过Samsung.com、运营商和在线零售商解锁。

至少从2015年开始，微软就开始用自己的应用程序支持三星Android设备。今年，微软将推出其首款微软品牌的Android手机，即Surface Duo。随着时间的推移，这两家公司是否会改变微软-三星合作的性质，这将是一件有趣的事情。

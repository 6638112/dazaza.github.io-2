Flower – A Friendly Federated Learning Framework
花–友好的联合学习框架

2020-12-22 23:05:59

Flower ( flwr) is a framework for building federated learning systems. Thedesign of Flower is based on a few guiding principles:

Flower（flwr）是用于构建联合学习系统的框架。 Flower的设计基于以下指导原则：

Customizable: Federated learning systems vary wildly from one use case toanother. Flower allows for a wide range of different configurations dependingon the needs of each individual use case.

可自定义：联合学习系统从一个用例到另一个用例，差异很大。根据每个单独用例的需要，Flower允许使用各种不同的配置。

Extendable: Flower originated from a research project at the Univerity ofOxford, so it was build with AI research in mind. Many components can beextended and overridden to build new state-of-the-art systems.

可扩展：花起源于牛津大学的一个研究项目，因此它的构建考虑了AI研究。可以扩展和覆盖许多组件，以构建新的最新系统。

Framework-agnostic: Different machine learning frameworks have differentstrengths. Flower can be used with any machine learning framework, forexample, PyTorch, TensorFlow, or even raw NumPyfor users who enjoy computing gradients by hand.

与框架无关的：不同的机器学习框架具有不同的优势。 Flower可以与任何机器学习框架一起使用，例如PyTorch，TensorFlow，甚至是喜欢手工计算渐变的原始NumPy用户。

Understandable: Flower is written with maintainability in mind. Thecommunity is encouraged to both read and contribute to the codebase.

可以理解：在编写花时要牢记可维护性。鼓励社区阅读和贡献代码库。

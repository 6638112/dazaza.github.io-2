A fully public domain, highly portable first person shooter running on 32kb RAM
一个完全公共领域的，高度便携的第一人称射击游戏，运行在32kb RAM上

2020-11-24 12:05:01

THIS IS SPECIAL This game runs everywhere and adheres to great simplicity. It is much more efficient and portable than Doom and has completely no dependencies. Not even floating point is used, in case your computer doesn't have the HW unit. The game can fit into 200 KB (including assets!) and can run with just 32 KB RAM. No build system, library, internet connection or package manager is inherently required for compilation as the whole game is written in pure C language.

特别提示：该游戏无处不在，并且非常简单。它比Doom更加有效和便携，并且完全没有依赖关系。如果您的计算机没有硬件单元，则甚至不使用浮点数。该游戏可以容纳200 KB（包括资产！），并且仅需32 KB RAM即可运行。由于整个游戏都是用纯C语言编写的，因此编译本身并不需要构建系统，库，Internet连接或程序包管理器。

Windows Defender Bug用数千个文件填充Windows 10 Boot驱动器

2021-05-06 09:02:44

Windows Defender Bug会创建数千个小文件，这些文件浪费了Windows 10硬盘驱动器上的存储空间的千兆字节。

该错误始于Windows Defender Antivirus Engine 1.1.18100.5，并将导致C：\ ProgramData \ Microsoft \ Windows Defender \ Scans \ History \ Store文件夹以填充数千个具有似乎MD5哈希的名称。

从BleepingComputer看到的系统，所创建的文件的大小范围从600字节到稍微超过1KB。

虽然我们只看过大约1MB的文件，但其他Windows 10用户报告其系统已填充了数百种文件，其中在一个情况下，使用了30GB的存储空间。

在较小的SSD系统驱动器（C :)上，这可以是浪费不必要的文件的相当大量的存储空间。

据第一次报道这个问题的Deskmodder说，该错误现已在最新的Windows Defender引擎中修复，版本1.1.18100.6。

Windows 10用户可以通过转到Windows安全性检查其Windows Defender版本，单击屏幕左下方的设置Cog图标，然后选择约。

微软的 Windows 365 云 PC 服务每个用户每月的费用从 20 美元到 162 美元不等

2021-08-03 03:06:13

当微软上个月推出其 Windows 365 云 PC 桌面即服务产品时，官员们表示他们将在服务普遍可用的那天，即 8 月 2 日公布定价。正如承诺的那样，该公司已经公布了定价，并且它的范围从最低端 SKU 的每个用户每月 20 美元到最昂贵的每个用户每月 162 美元。 Windows 365 是 Microsoft 最新的远程桌面产品，它建立在 Azure 虚拟桌面之上并对其进行了补充。 Windows 365 将允许用户将他们的 Windows 10 或（一旦它在今年秋季晚些时候推出）、Windows 11 桌面、应用程序、工具、数据和设置带到他们的个人和工作设备，包括 PC、Mac、iPad、Linux 和 Android设备，通过本机远程桌面应用程序或 Web 浏览器。 7 月 15 日，微软官员（意外地？）透露了少数 Windows 365 SKU 之一的定价，每位用户每月 31 美元。今天是所有定价公开的日子。定价基于客户能够在其“云 PC”虚拟机中访问的虚拟内核、RAM 和存储的数量。 Windows 365 有两个版本：Windows 365 商业版和 Windows 365 企业版。 Windows 365 商业版 SKU 的上限为每个组织 300 个用户。每位用户每月 20 美元的商业价格适用于单个虚拟核心、2 GB RAM 和 64 GB 存储——并且需要 Windows 混合权益。 （混合权益是 Microsoft 的自带许可模式，它允许客户将现有（或新）许可证应用于产品成本。）如果没有混合权益折扣，则相同的 SKU 为每位用户每月 24 美元。在高端，具有 8 个虚拟核心、32 GB RAM 和 512 GB 存储空间的商业 SKU 每位用户每月花费 162 美元——或者使用 Windows 混合权益为每位用户每月花费 158 美元。 Windows 365 的企业 SKU 定价相似。单个虚拟核心、2 GB RAM 和 64 GV 存储空间的价格为每位用户每月 20 美元。在高端，8 个虚拟核心、32 GB 内存、512 GB 存储 SKU 的价格为每位用户每月 158 美元。 Windows 365/Cloud PC 将不仅支持 Microsoft 应用程序；该服务还将允许用户远程登录可在 Windows 10 或 11 上运行的任何应用程序。说。并且由于应用程序是虚拟运行的，因此无论他们使用的设备规格如何，用户都可以访问高计算应用程序，例如视频编辑软件或图形设计程序。为了使用 Windows 365 商业版，客户只需拥有 Microsoft 365订阅。要使用 Windows 365 Enterprise，用户需要满足以下要求： 更新：我看到很多人将 Windows 365 的价格与购买具有可比规格的新 PC 的成本进行比较，并嘲笑 Windows 365 的成本。实际上，正如微软合作伙伴 Nerdio 在这篇文章中所建议的那样，更恰当的比较可能是 Azure 虚拟桌面（用户为使用的 Azure 服务付费）和 Windows 365/云 PC。相比之下，Windows 365 可以为一些客户省钱。但是，是的，这些 Windows 365 价格相当令人震惊。需要记住的两件事：微软可能会为那些有兴趣批量购买的人提供这些估计零售价的大幅折扣。其次：虚拟化桌面绝对不适合所有人，尽管它们对某些特定用例有意义。更新 2：对 IT 专业人员的警告。从 8 月 14 日起，微软将提供 Windows 365 作为一种自助服务类型，这意味着用户可以选择订阅而无需集中批准——除非管理员使用 PowerShell 脚本和/或其他方式阻止订阅。 Microsoft 提供为期两个月的免费 Windows 365 Basic、Standard 或 Premium 试用版。免费试用期结束后，订阅会自动转换为付费订阅，除非用户采取行动阻止订阅。

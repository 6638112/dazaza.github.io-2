Otter Browser Aims To Bring Chromium To Decades-Old OS/2 Operating System
Otter Browser旨在将Chromium引入数十年前的OS/2操作系统

2022-02-17 18:40:53

Every computer needs a capable web browser, but that’s easier said than done on computers running something besides Windows, macOS, or Linux. There are still a few people using the OS/2 operating system, which was created by Microsoft and IBM and ended mainstream development in 2001, and is in desperate need of a functional web browser. With any luck, the decades-old operating system could have the same web engine as modern smartphones.

每台计算机都需要一个功能强大的网络浏览器，但在运行Windows、macOS或Linux以外的计算机上，这说起来容易做起来难。仍然有一些人在使用OS/2操作系统，它是由微软和IBM创建的，并在2001年结束了主流开发，现在迫切需要一个功能强大的web浏览器。如果运气好的话，这套已有数十年历史的操作系统可以拥有与现代智能手机相同的网络引擎。

Roderick Klein, president of the OS/2 Voice community, revealed in an announcement article that a public beta of the new Otter Browser will arrive “in the last week of February or the first week of March.” The new browser is based on the Chromium engine, the same one that powers Google Chrome, Microsoft Edge, Vivaldi, and most other browsers that aren’t Firefox or Safari. The interface is written in the cross-platform Qt framework.

OS/2语音社区主席罗德里克·克莱因（Roderick Klein）在一篇公告文章中透露，新水獭浏览器的公测版将“在2月的最后一周或3月的第一周”发布新浏览器基于Chrome引擎，该引擎支持Google Chrome、Microsoft Edge、Vivaldi和大多数其他非Firefox或Safari浏览器。该接口是在跨平台Qt框架中编写的。

OS/2 was the operating system developed jointly by IBM and Microsoft in the late 1980s and early 1990s, with the intended goal of replacing all DOS and Windows-based systems. However, Microsoft decided to focus on Windows after the immense popularity of Windows 3.0 and 3.1, leaving IBM to continue development on its own. IBM eventually stopped working on OS/2 in 2001, but two other companies licensed the operating system to continue where IBM left off — first eComStation, and more recently, ArcaOS.

OS/2是IBM和微软在20世纪80年代末和90年代初联合开发的操作系统，其目标是取代所有基于DOS和Windows的系统。然而，在Windows 3.0和3.1大受欢迎之后，微软决定将重点放在Windows上，让IBM自己继续开发。IBM最终在2001年停止了OS/2的开发，但另外两家公司授权该操作系统继续IBM停止的工作——第一家是eComStation，最近一家是ArcaOS。

BitWise Works GmbH and the Dutch OS/2 Voice foundation started work on Otter Browser in 2017, as it was becoming increasingly difficult to keep an updated version of Firefox available on OS/2 and ArcaOS. Firefox 49 ESR from 2016 is the latest version available, because that’s around the time Mozilla started rewriting significant parts of Firefox with Rust code, and there’s no Rust compiler for OS/2. Since then, the main focus has been porting Qt 5.0 to OS/2, which includes the QtWebEngine (based on Chromium). This effort also has the side effect of making more cross-platform ports possible in the future.

按位工作的GMH和荷兰OS/ 2语音基金会2017开始在Outter浏览器上工作，因为在OS/2和ARCAOS上保持更新的Firefox版本变得越来越困难。2016年推出的Firefox 49 ESR是最新版本，因为那时Mozilla开始用锈迹代码重写Firefox的重要部分，并且OS/2没有锈迹编译器。从那时起，主要关注点一直是将Qt 5.0移植到OS/2，其中包括QtWebEngine（基于Chromium）。这一努力还有一个副作用，那就是未来可能会有更多的跨平台端口。

Steven Levine, who helped with development, said in an announcement post, “the Qt5 and Otter browser ports have basically been a one man project, with help other bww bitwise works employees and from the community. What Dmitriy [the main developer] is doing is porting two large, complex projects that were never designed to run on a 32-bit platform and were never designed to run on OS/2. IMO, there are few developers that could have accomplished what he has done in the same timeframe. From what I can see, the browser is close to being usable on a daily basis. There are already a number of sites, that matter to me, that I can now access with Otter, rather than having to work on another platform.”

帮助开发的史蒂文·莱文在一篇公告中说，“Qt5和Otter浏览器端口基本上是一个人的项目，在bww bitwise works其他员工和社区的帮助下。什么是Dmitry[主要开发者]正在做的是移植两个大型、复杂的项目，它们从未被设计为在32位平台上运行，也从未被设计为在OS/2上运行。在我看来，几乎没有开发人员能够在同一时间内完成他所做的事情。据我所见，这款浏览器几乎可以每天使用。已经有很多对我来说很重要的网站，我现在可以通过Otter访问，而不必在另一个平台上工作。"

Even though OS/2 is not a modern operating system by any stretch of the imagination, the ArcaOS distribution/fork is still under development, primarily targeting legacy systems and retro enthusiasts. The latest update was v5.0.7 in December 2021, which included updated drivers and kernel fixes. The developers of Otter Browser are accepting donations on the OS/2 Voice website.

尽管OS/2在任何想象中都不是现代操作系统，但ArcaOS发行版/fork仍在开发中，主要针对遗留系统和复古爱好者。最新的更新是v5。0.7在2021年12月，其中包括更新的驱动程序和内核修复。Otter Browser的开发者正在OS/2语音网站上接受捐款。

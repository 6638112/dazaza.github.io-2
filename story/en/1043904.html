Gekko: OS software used by the Danish gov. for economic forecasting
Gekko：丹麦政府使用的OS软件。 用于经济预测

2021-01-11 01:03:22

Gekko Timeseries and Modeling Software is a free timeseries-oriented software package for timeseries data management, and for solving and analyzing large-scale timeseries-based models (e.g. economic models). It is recommended to use the Gekko 3.1.x version, cf. this overview page.

Gekko时间序列和建模软件是面向时间序列的免费软件包，用于时间序列数据管理以及用于解决和分析基于时间序列的大型模型（例如经济模型）。建议使用Gekko 3.1.x版本，请参阅。此概述页面。

Gekko interfaces to Excel, Python, R, GAMS, and file formats like .csv, .prn, .tsd, etc.

Gekko可与Excel，Python，R，GAMS和文件格式（例如.csv，.prn，.tsd等）交互。

Instead of Gekko, why not use for instance Python, or R, or GAMS, or MATLAB, or SAS, or Excel, or … ?

为什么不使用Gekko而不是Gekko，例如为什么不使用Python，R，GAMS，MATLAB，SAS，Excel或…？

Software packages are typically aimed at some purpose, and this is reflected in, among other things, the underlying data structures and the syntax of the command language.

软件包通常是针对某个目的的，除其他外，这反映在基础数据结构和命令语言的语法中。

Since Gekko is timeseries-oriented, the syntax for handling and analyzing timeseries and timeseries-based models is concise and natural. Gekko does not try to do everything, but tries to focus on its strong points, while simultaneously providing good interfaces to other software packages.

由于Gekko是面向时间序列的，因此处理和分析时间序列和基于时间序列的模型的语法简洁而自然。 Gekko不会尝试做所有事情，而是尝试着重于自己的长处，同时为其他软件包提供良好的界面。

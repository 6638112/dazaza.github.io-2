Firefox 在 3 年内流失了近 5000 万用户

2021-08-05 21:34:48

它一直是 Linux 用户和每个平台上注重隐私的用户的默认选择。然而，即使拥有作为最好的网络浏览器之一的所有优势，它在过去几年中也失去了控制力。老实说，我们甚至不需要统计数据就可以说，我们中的许多人已经切换到基于 Chromium 的浏览器或 Chromium 本身，而不是 Firefox 或 Google Chrome。然而，我看到了 u/nixcraft 的 Reddit 帖子，其中强调了自 2018 年以来 Firefox 用户群下降的更多细节。令人惊讶的是，这些信息的原始来源是 Firefox 的公共数据报告。根据官方统计，截至 2018 年底，报告的活跃（每月）用户数量约为 2.44 亿。而且，到 2021 年第二季度末，这一数字似乎已下降至 1.98 亿。

考虑到 2021 年是专注于隐私的工具的用户群大幅增长的一年，Mozilla 的 Firefox 正在不断下降。如果您从未使用过 Firefox 或已经离开了很长时间，Brave 和 Firefox 之间的比较突出表明，它仍然是迄今为止可靠的网络浏览器选择。 Microsoft Edge 作为 Windows 的默认网络浏览器（自然拥有巨大的市场份额） Google.com（最大的搜索引擎）建议用户安装 Google Chrome（这可能是一种反竞争行为） 除此之外，还有Firefox 可能做错的一些事情： 当然，没有网络浏览器是完美的，但这需要担心吗？嗯，我想，是的。 Mozilla 的 Firefox 是基于 Chromium 的浏览器唯一可行的竞争对手。如果 Firefox 消失，用户将无法选择不同的浏览器引擎。

我相信你会同意垄断是不好的。因此，我们需要一些东西来替代谷歌的 Chrome 引擎。事实上，大量网站通过牢记基于 chrome 的浏览器来优化用户体验。因此，最终，如果下降趋势继续下去，像我们这样的用户可能会被迫通过适应新的工作流程而切换到其他浏览器。即使我们通过争辩说它在技术上更好而忽略了谷歌 chrome 引擎在网络上的主导控制，Firefox 仍然是一种宝贵的东西。因为它提供了更多的自定义方式，并不断改进其与其他任何人不同的隐私实践。换句话说，仅仅因为所有的竞争对手都更喜欢使用 Chromium 作为基础或从事反竞争活动，我们将（强行）失去很多好东西。也许，您现在对 Firefox 感到沮丧并转向其他东西。那完全是你的选择。但是，当 Firefox 由于影响其衰落的所有因素而不再存在时，如果您没有其他选择，您会怎么想？

因此，为了保持平衡，我认为我们应该不断反对科技巨头的反竞争行为，并开始使用 Mozilla Firefox（以任何身份，甚至作为辅助浏览器）。虽然他们正忙于引入 VPN 服务、电子邮件中继和其他服务集成，但在用户体验改进方面并未取得成功。至少，我是这么认为的。多年来，我一直使用 Firefox 作为我的主要浏览器，但我最终偶尔会切换到其他浏览器，尤其是在每次重大 UI 大修之后。我很想知道您对此有何看法，以及似乎影响 Firefox 用户群下降的因素。您更喜欢使用什么作为您的主要网络浏览器？在下面的评论中让我知道这一切！如果您喜欢我们在 It's FOSS 所做的工作，请考虑捐款以支持我们的独立出版物。您的支持将帮助我们继续发布专注于桌面 Linux 和开源软件的内容。

Windows 11 现已发布首个 Beta 版

2021-07-30 04:24:50

如果您从 Verge 链接购买东西，Vox Media 可能会获得佣金。请参阅我们的道德声明。微软已经发布了 Windows 11 的第一个测试版，可供注册其 Windows Insider Program 的用户使用。直到今天，访问 Windows 11 意味着安装开发预览版，微软称其适用于“高技术用户”，因为它具有“粗糙的边缘”。根据微软的说法，测试版的波动性较小，构建由微软验证（尽管它仍然可能是你想要在测试机器或第二个分区上安装的东西）。当然，要安装测试版，您需要一台兼容的计算机。确定您的硬件是否适用于下一个版本的 Windows 是出了名的难以确定，但微软关于为 Insider 构建做准备的文章将人们引导到其系统要求页面。该公司表示将密切关注第 7 代英特尔和 AMD Zen 1 CPU 在测试期间的工作情况，因此这些系统可能会被允许运行测试版，但不能运行最终版本。 Beta 版本对于我们这些安装了开发预览版以尽快使用 Windows 11 的人来说也是个好消息，但实际上并不需要处于流血（阅读：buggy）边缘。如果您和我一样遇到这种情况，您可以通过转到“设置”>“Windows 更新”>“Windows 预览体验计划”，然后单击“选择您的预览体验计划设置”来切换到 Beta 频道。通常从 Dev 切换到 Beta 需要重新安装完整的操作系统，但根据 Windows Insider Twitter 帐户，它可以在“短时间内”完成。如果您不需要留在开发频道，最好尽快加入。我自己已经测试过了，切换到 Beta 频道只需要快速重启——这是一个很小的代价，希望在实际发布之前可以更顺利地航行。对于那些仍在运行 Windows 10，但又足够冒险加入 Beta 的用户，您可以在此处注册 Microsoft 的 Beta 计划。当然，虽然微软表示 Beta Channel 版本比 Dev Channel 版本更稳定，但它们仍然是 Beta。可能会有错误、崩溃和缺失的功能——微软甚至在其博客文章中列出了当前问题的完整列表，其中还表示开发频道中某些人可用的 Teams Chat 功能目前不适用于 beta 用户然而。但是，如果您一直渴望试用 Windows 11，那么它现在处于一个足够稳定的位置，微软愿意称其为早期采用者做好准备（对于那些生活在边缘的人，可以说）。

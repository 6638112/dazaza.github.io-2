德州仪器的新计算器采用 Python 编程语言

2021-07-31 10:34:51

CDT 2021 年 7 月 29 日下午 3:36 — 2021 年 7 月 30 日下午 4:39 更新 CDT 位于达拉斯的德州仪器 (TI) 的最新一代计算器正在通过添加编程语言 Python 进行现代更新。目标是通过美国高中和大学几乎都需要的设备来扩展学生探索科学、技术、工程和数学的能力。德州仪器 (TI) 成立于 1930 年，自设备开始普及以来，一直为学生提供创新的计算器。尽管该公司 140 亿美元的年收入大部分来自半导体，但其图形计算器仍然是其最受认可的消费产品。这种最新的 TI-84 型号的价格在 120 美元到 160 美元之间，具体取决于零售商，旨在适应现代世界日益增长的编程重要性。 TI 教育技术总裁 Peter Balyta 表示，随着 STEM 在日常生活中变得越来越重要，编码应该被视为下一代的必需品。他说编程语言就是——第二语言。 “我相信 STEM 技能是绝对的生存技能，我们的未来实际上取决于孩子们的 STEM 智能，”Balyta 说。

Balyta 说，Python 已成为大学中最受欢迎的编码语言之一，现在学生可以使用“他们在数学和科学课上已经使用的相同的专用、无干扰的工具”尽早学习它。 Python 因其广泛的用途而大受欢迎，它被认为是最容易学习的编程语言之一。 Python 被业余爱好者和专业人士使用，它是免费和开源的，任何有计算机访问权限的人都可以访问它。 Balyta 说，德州仪器 (TI) 结合 Python 为学生提供了有趣且富有创意的学习方式。他补充说，学生可以使用 Python 来做一些事情，比如创建小游戏，甚至为机器人汽车编程。 “编程是接口，这非常重要，但我们希望他们能够发明和创造事物，我们希望他们尽早完成，”Balyta 说。 Balyta 说，教育工作者需要通过从专注于重复的模式转变为培养创造潜力的模式，弥合“现实世界”对教师和学生的意义之间的差距。 “它需要有趣，我们需要帮助孩子们睁开眼睛，了解学习 STEM 的乐趣。例如，这与‘钻杀’相反，”巴利塔说。随着 Python 的加入，新型号的计算器还将具有耐用性、长效电池和全彩屏幕等熟悉的功能。该设备还被批准用于大学入学考试以及大学预修课程和国际文凭考试。

越来越多的学校已经开始将计算机编程纳入课堂。 2016 年，大学理事会推出了大学先修课程计算机科学原理，希望能吸引更多女性和有色人种参与编码。自 1988 年以来，该组织已经开设了另外两门计算机科学课程。对更多计算机科学课程的需求持续增长，但 Code.org 和计算机科学教师协会 2019 年的一份报告显示，只有 45% 的高中提供这些课程。 “我不知道他们要做什么，我真的迫不及待想看看他们这个学年做什么，”巴利塔说。

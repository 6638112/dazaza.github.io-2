例如，假设您用Hugo、Jekyll、Gatsby或Hexo制作了一个静态网站。或者，您可能在某个流行的框架中制作了一个JavaScript应用程序，这些框架包括ReactJS、ANGING、Vue.js、Svelte…。它可能是一个简单的副产品，或者仅仅是一个用于学习的应用程序，但它总是会遇到这样的问题：在哪里托管它，使用哪个提供商，以及哪个提供商免费提供大多数功能，这样你就可以向每个人展示你的作品。

嗯，现在是2020年，免费托管你的静态网站从来没有像现在这样容易。每一个都在某些方面有所不同，但本质上它们都提供相同的服务，所以我们创建了一个18个免费主机提供商的列表，您可以考虑在当前或下一个项目中使用这些提供商。

您是否在使用这些提供商中的一些？让我们在评论中知道你与他们的经历。

亚马逊AWS可能是互联网上最知名的主机提供商之一，即使你不选择AWS，你也有可能最终使用AWS，因为下面的许多提供商都是建立在它们之上的。因此，您可以考虑直接在AWS上托管您的网站。

AWS在使用的第一年是完全免费的，在此之后，您需要为您使用的内容付费。

注册后，新的AWS客户将获得5 GB的S3Standard存储级别的Amazon S3存储；20,000个GET请求；2,000个PUT、COPY、POST或LISTRequests；以及为期一年的每月15 GB数据传出。

Netlify以其无麻烦的服务和功能丰富的慷慨计划在开发人员社区中赢得了良好的声誉。这是一个提供分析、表单、无服务器功能和更多功能的一体化平台。

虽然Firebase拥有启动完整堆栈应用程序的所有功能，但它也是一个非常方便的主机提供商。与其Firebase CLI工具配合使用非常简单。

Heroku对于非商业应用是免费的。它没有为静态网站提供开箱即用的托管服务，但需要一些老套的技巧，请查看本教程，了解如何托管您的静态网站。

GitHub Pages是非商业性项目非常受欢迎的选择。对于每个GitHub账户和组织，您可以免费托管一个站点，并且可以不限数量的项目站点。

GitLab页面永远是免费的，没有限制，但网站需要在GitLab存储库中。

Vercel，以前被称为Zeit，非常关注终端用户的性能。这是一个专门为前端开发人员构建的平台。

Surge已经存在很长一段时间了，开发人员喜欢它，因为它非常简单，可以使用命令行工具Surge立即部署项目。它为无障碍的前端开发提供了丰富的功能。

Render是一个完全托管的云平台，您可以在其中托管静态站点、后端API、数据库、cron作业和所有其他应用程序。

Stormkit专门针对JavaScript应用程序，通过简化和自动化JavaScript应用程序的整个生命周期中的复杂流程，为现代应用程序提供了强大的基础设施。

Microsoft Azure Static Web Apps是微软推出的一项相对较新的服务，目前还没有定义价格，但预览版是免费的。

Hostman是全栈主机提供商，在AWS和DigitalOcean之上工作，以提供尽可能最佳的可靠性，使托管和管理您的应用程序变得轻松，并提供大量可用功能。对于免费计划的静态网站来说，这也是相当不错的。

嗯，Oracle云不需要太多的介绍，除了提供大规模的托管基础设施之外，它也是一个您应该考虑免费托管低流量项目的地方。

Deta是一项相对较新的云服务，用于构建FAST应用程序，提供慷慨的免费计划，但Deta Drive服务仍在开发中，它允许您上传、托管和提供图像和文件，因此请密切关注这家初创公司。

Fosshost仅对免费和开放源码软件社区可用。如果你有一个免费的开源项目，那么你应该考虑这个提供商。

Fly.io是需要在全球运行的应用程序的平台。它在用户附近运行您代码，并在您的应用程序最繁忙的城市进行计算。但是你也可以免费运行一个静态网站！Fly为您的网站每月提供10美元的信用额度。

特技飞行是专门为在杰基尔，雨果，赫克索，盖茨比…制作的静态网站而建的。等。通过在高度调整的、专门构建的CDN上提供惊人的快速性能。

每个新网站都有30天的试用期来测试所有特技飞行功能，之后自定义域名和SSL等功能的起步价为15美元/月。

如果上面列出的这些提供程序都不适合您，那么您可以考虑自托管，这肯定有它的优点。非常适合小型项目和学习，让您有机会掌握服务器的一些微调和优化。互联网上有很多教程，但你可能想从树莓派教程开始。

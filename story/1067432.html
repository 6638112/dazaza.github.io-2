攻击曲面分析仪是一个Microsoft开发的开源安全工具，分析了目标系统的攻击面，并在安装软件或系统错误配置期间引入的潜在安全漏洞的报告。

如果安装了.NET Core SDK，您可以使用Dotnet Tool Install -g Microsoft.cst.AttackSurfaceAnalyzer.cli安装攻击曲面分析仪。

苹果援引App Store指导方针，不让Stadia或XCloud进入iOS

2020-08-08 03:34:49

云游戏正越来越成为一种东西，一种可以让你在设备上玩AAA游戏的游戏，而不管硬件规格是什么。如果你的设备可以播放视频，那么它很可能就可以在谷歌体育场(Google Stadia)上玩“红死救赎”(Red Dead Reemption)，或者在微软(Microsoft)的XCloud上玩Halo(从技术上讲，XCloud现在被称为“云游戏(Beta)”，与Xbox Game Pass Ultimate一起使用)。不过，如果你的设备是iPhone或iPad，那你就不走运了。苹果公司表示，这些应用程序违反了其App Store政策，将不允许进入苹果的“围墙花园”。

创建App Store的初衷是为客户提供一个安全、值得信赖的地方来发现和下载应用程序，并为所有开发人员提供巨大的商机。在上架我们的商店之前，所有的应用程序都会根据一套相同的指导方针进行审查，这些指导方针旨在保护客户，并为开发人员提供公平和公平的竞争环境。

我们的客户享受数以百万计的开发者提供的优秀应用程序和游戏，游戏服务完全可以在App Store上推出，只要他们遵循适用于所有开发者的同一套指导方针，包括单独提交游戏供审查，并显示在图表和搜索中。除了App Store，开发者还可以选择通过App Store上的Safari和其他浏览器通过网络接触到所有iPhone和iPad用户。

苹果应用商店的宣传是，它有真人亲自审查每个应用的安全和质量，给用户一个单一的、值得信赖的地方来获取他们所有的应用。苹果希望单独批准这些游戏，并让用户通过App Store对它们进行单独评分。苹果引用的指导方针断然禁止在远程电脑上显示类似商店的界面，以及为基于云的应用程序显示瘦客户端，而Stadia和XCloud都与这两种界面相冲突。

审批过程还确保开发者遵守苹果的开发者政策，其中包括对iOS上发生的所有交易的独家权利-苹果希望任何支付都通过自己的服务进行，在那里它可以从收入中分得一杯羹。App Store确实有一些需要支付的费用，比如所有这些人工应用程序审批人的工资、开发人员支持和托管成本，但据估计，苹果公司2019年从App Store收入中削减的金额为150亿美元。苹果正在赚取巨额利润。

微软之前在iOS上对XCloud进行了测试版测试，听起来对苹果的指导方针相当不满。微软向The Verge发送了以下声明：

我们针对iOS的Project XCloud预览应用程序的测试期已过。不幸的是，我们没有办法通过苹果应用商店将Xbox Game Pass旗舰版云游戏的愿景带给iOS上的游戏玩家。苹果是唯一一个拒绝消费者使用云游戏和Xbox Game Pass等游戏订阅服务的通用平台。而且，它一贯以不同的方式对待游戏应用程序，对非游戏应用程序应用更宽松的规则，即使它们包括互动内容。Xbox游戏通行证目录中的所有游戏都由独立的行业评级机构(如ESRB和地区同等机构)对内容进行评级。我们致力于寻找一条道路，将Xbox Game Pass旗舰版的云游戏带到iOS平台上。我们认为，客户应该是游戏体验的核心，游戏玩家告诉我们，他们想要在任何地方玩、连接和分享，无论他们在哪里。我们同意。

微软声称，苹果对待游戏应用的态度是不同的，这是一个相当有力的论据，因为Xbox Game Pass作为一种随心所欲的游戏订阅服务，与Netflix或迪士尼+等视频流媒体服务并没有太大不同，这两家公司都可以在App Store上使用Xbox Game Pass。苹果对远程计算的厌恶可能被视为推动更一致、更高性能的原生应用程序(以及平台保护主义)，但在这些服务上推出的AAA级游戏要么不可能，要么作为原生iOS应用程序非常不切实际。例如，“红色死神救赎2”在个人电脑上占用150 GB的空间。

另一方面，谷歌尚未对Stadia的iOS禁令做出回应。谷歌没有微软那么争辩，因为Stadia是一家商店--游戏除了每月10美元的4K分辨率费用外，还需要花钱，所以它不是网飞公司(Netflix)的XCloud那样的游戏。谷歌肯定比微软更习惯于这种方式，谷歌和苹果的App Store可以追溯到早期，比如2009年谷歌语音(Google Voice)因为提供了另一种打电话方式而被屏蔽了一年。从技术上讲，Stadia有一款iOS应用程序，但它只适用于Stadia奇怪的设置过程。谷歌的应用程序描述特别指出，你不能直接在iOS设备上使用Stadia应用程序玩游戏，但你可以在其他设备上使用该应用程序管理体育场。

苹果的App Store政策最近经常出现在新闻中。像Basecamp和Eic Games这样的开发商批评了该公司从每一次App Store销售中抽成的做法。在欧盟，来自Spotify和Telegram等公司的投诉正在推动对苹果应用商店政策的反垄断调查。App Store今天的另一条新闻是Facebook的游戏应用-Facebook的Twitch.tv克隆-已经在iOS上删除了游戏部分。除了观看直播流，Android上的Facebook游戏还可以让你玩植物大战僵尸等HTML游戏，但苹果公司屏蔽了这款应用，因为它绕过了应用商店(App Store)。Facebook表示，它根据WWDC宣布的新的App Store上诉程序对这一决定提出上诉，但从未得到回应。

苹果的许多政策都可以被辩解为推动开发人员选择更一致、更用户友好的选项，或者帮助支付App Store基础设施和审查过程的成本。然而，很难想出一个以用户为中心的防御措施来禁止云游戏。苹果不提供任何类似的服务，而且这些游戏也不可能通过其他方式在iOS上运行。你可以说，云游戏将意味着从苹果游戏生态系统购买的游戏减少，但Netflix也可以这样说，让人们的眼球从苹果的视频内容中移开。

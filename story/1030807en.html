TwitGrid-用于阅读Twitter提要的简单的宽页布局

2020-10-24 07:26:35

TwitGrid is a simple, broadsheet layout for reading Twitter feeds that shows adense, instant overview of the latest posts from only the users you care about.It doesn't mix multiple feeds up into an infinite-scrolling jumble.

TwitGrid是一个简单的宽页布局，用于阅读Twitter订阅源，它只显示你关心的用户的Adense，即时概览最新帖子。它不会将多个订阅源混合在一个无限滚动的混乱中。

Page Up / Down will smooth scroll the whole viewport, snapping to the next set of feeds. Mouse Wheel Up / Down scrolls the individual user feeds. Home / End scrolls to the top / bottom, as expected.

向上/向下翻页将平滑滚动整个视口，捕捉到下一组提要。鼠标滚轮向上/向下滚动各个用户摘要。如预期的那样，主页/结尾滚动到顶部/底部。

Because most people have usable long-term memory, you'll know what you have readand haven't, somewhat in the vein of RSS (though less automatic).

因为大多数人都有可用的长期记忆力，所以你会知道你读到了什么，而不是像RSS那样(虽然没有那么自动)。

There will not be an algorithm intermediating what you want to see and what youactually see or surfacing things it thinks you might be interested in.

不会有一个算法将你想看的东西和你实际看到的东西联系起来，也不会让它认为你可能感兴趣的东西浮出水面。

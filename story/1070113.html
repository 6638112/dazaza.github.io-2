评论：Framework 的轻量模块化笔记本电脑是赢家

2021-07-28 01:11:21

当我们最初讨论初创公司 Framework 的雄心勃勃的计划时，该计划旨在打造一款完全可修复的模块化笔记本电脑，旨在与戴尔的 XPS 13 等行业冠军竞争，这似乎是一个很长的机会。该公司提议的 13 英寸笔记本电脑将成为其第一款产品，其中包括其他公司从未提供过的定制硬件功能——最值得注意的是，四个模块化托架取代了通常的硬接线笔记本电脑 I/O 端口系列。每个托架都有一个深凹的 USB-C 端口，可以接受一个模块，为用户提供外部 USB-A、USB-C、DisplayPort、HDMI 或 MicroSD 端口。设计和构建这样的功能是一个很大的风险——这是第一次这样做，同时计划在重量、宽度、厚度和价格方面与行业领先的紧凑型笔记本电脑（如戴尔的 XPS 13）在重量、宽度、厚度和价格方面进行平等竞争，这让我们大吃一惊。但是我们已经在一个审查部门度过了几天——Framework 已经满足了它的所有主要要求。它的新笔记本电脑并不完美，但它是兑现其承诺的可靠竞争对手。购买 Framework 笔记本电脑时，新客户可以选择三种预制配置中的一种，或者自己“DIY”并选择每个单独的部件——尽管 DIY 版本也需要在抵达时进行组装。我们关于 Framework 的最大问题围绕着物理设计及其承诺的可修复性，所以我们当然去了 DIY。我们的 DIY 构建基本上是为登月而设计的——它具有公司最快的 CPU 产品（英特尔 i7-1185G7）、32GiB 的快速 RAM、1TB SSD，甚至是英特尔 Wi-Fi 6E。检查几乎每个盒子（我们选择不购买 Windows 10 许可证或笔记本电脑可以处理的最大 64GiB RAM）导致价格为 1,900 美元——尽管我们将上面的系统列为 2,100 美元零售，因为我们使用 Windows 10大多数测试。如果您想以更低的总价获得包含 Windows 副本的相同版本，Professional trim 具有所有相同的硬件和价值 2,000 美元的 Windows 10 Pro 许可证，或者 Linux 用户可以执行我们所做的，完全跳过 Windows 许可证，并且从专业套餐的成本中节省 100 美元。您没有任何选择的一个组件是显示器——它采用 3:2 纵横比 2.5k 设计，有光泽且非触控，具有 400+ 尼特的亮度和 100% 的 sRGB 色域。我们测量了显示器的最大亮度为 418nits；我们无法科学地测试 sRGB 色域。虽然我们无法直接测量 sRGB 色域，但我们可以说，deadmau5 的迷幻色彩视频 Pomegranate 在 Framework 的显示器上显然比在 24 英寸 HP VH240a 显示器上更有活力，后者本身声称只有 72% 的 sRGB 色域。

DIY Framework 到货时经过了大量拆解——主板在大部分组装的笔记本电脑中，但 NVMe SSD、Wi-Fi 卡和系统 RAM 都放在全新的小盒子里，以及一些 Framework 的自定义 I/O模块。这意味着 DIY 者有一些工作要做——但开放框架以添加所有组件对于资深 PC 机械师来说是一种奇妙的体验。第一步是使用随附的驱动器从笔记本电脑底部卸下五颗 Torx T5 螺钉。所有五个螺丝都是固定的，这意味着即使在完全拧下后它们仍会固定在笔记本电脑上。一旦螺丝完全松开，下一步就是再次将笔记本电脑竖起并打开，就像正常使用一样。下一步只需将笔记本电脑的整个顶板（包括键盘和触摸板）直接提起。松开底部 Torx 螺丝后，唯一将顶板固定到位的是一堆小磁铁；没有塑料闩锁或拉环可以松开。接下来，将一根手指伸入将键盘和触摸板连接到主板的单条带状电缆上的环中，然后笔直向上提起 - 然后将顶板（键盘和鼠标仍然连接）放在一边以备后用。在主板完全暴露的情况下，两个 RAM DIMM、一个 M.2 NVMe SSD 和一个 M.2 Wi-Fi 卡的插槽都有醒目的标签，无障碍且易于填充。由于笔记本电脑的开放式设计，即使将小巧的天线引线连接到 AX210 Wi-Fi 卡也不像通常那样繁琐。我们经常在比这款旅行尺寸的笔记本电脑更难的大型中塔式机器上工作！在插入 RAM、SSD 和 Wi-Fi 之后，是时候再次关闭框架备份了——这个过程就像拆卸一样愉快。轻轻地将键盘/触摸板电缆的连接器直接向下按入其插座，然后将顶板放在主板上。轻轻晃动后，底盘中的磁铁会发出咔嗒声，将顶板牢固地拉到位，剩下的唯一事情就是重新拧紧底部的五颗 Torx T5 螺钉，再次使用随附的驱动器。如果您在分离组件时遇到困难，T5 驱动器的反向尖端有一个功能完善的塑料撬棒——但我们认为没有多少人会需要它。尽管顶板紧紧地固定在机箱上，但由于没有闩锁或卡舌，因此仅用一个缩略图就可以很容易地将其分开。现在笔记本电脑已经组装好了，是时候填充模块化 I/O 托架了。我们选择了一个 USB-A 模块、一个 HDMI 模块和两个 USB-C 模块。施加牢固、平稳的压力轻松插入四个模块中的三个——但我们与 HDMI 模块有轻微错位，需要稍微摆动才能解决。

我们使用 HDMI 模块的困难似乎是因为它和我们为其选择的托架处于笔记本电脑机械容差规范的极端两端——HDMI 模块在所有其他三个托架和所有其他三个模块中运行顺畅在我们为 HDMI 模块选择的托架中顺利运行。虽然我们将 HDMI 模块插入到一个相对不喜欢的托架中，但我们建议经验不足的用户在遇到相同问题时考虑尝试使用不同的托架 - 除非出现轻微错位，否则没有多大意义你绝对需要一个模块进入那个海湾而不是其他地方。当每个模块完全插入时，用户会听到并感受到令人满意的咔嗒声——这是一个保持杆的接合。要卸下模块，您需要按下笔记本电脑底部的释放按钮，并在模块背面和托架正面之间画一个缩略图。即使按下释放按钮，也需要适度的力才能移除模块。我们发现，对于设计为很少更换的部件而言，这种贴合度是理想的选择——安装的模块感觉不像是松散连接的配件，它们只是感觉像是普通笔记本电脑的正常部件。不幸的是，我们确实遇到了一个非机械问题——笔记本电脑应该附带一个 USB Windows 10 安装程序，框架的驱动程序已经插入其中。我们到达时没有安装程序，所以我们尝试使用我们自己的 - 但笔记本电脑出于某种原因拒绝将其识别为可启动设备。我们通过使用另一个已安装 Windows 10 的 1TB NVMe SSD，然后执行出厂重置来解决此问题。在出厂重置我们自己预装的 Windows 10 NVMe 后，我们只需要驱动程序 — Framework 的支持站点上还没有这些驱动程序。联系我们的框架联系人为我们提供了单个 800MiB 驱动程序包的直接下载链接，这解决了我们剩余的软件问题。安装了硬件、操作系统和驱动程序后，Framework 笔记本电脑是一个强大的竞争对手。我们不太喜欢它的 3:2 纵横比，这使得笔记本电脑明显比竞争对手的戴尔 XPS 13 更深——但我们知道许多读者喜欢近乎方形的显示器，如果你喜欢纵横比，还有很多喜欢关于显示器。与我们测试过的大多数笔记本电脑显示器相比，它锐利、明亮，色彩更鲜艳。

我们发现键盘和触摸板令人愉悦但不起眼。键盘最糟糕的功能是通常的压缩箭头键布局；除此之外，我们对它没有任何抱怨——它的触感很好，我们立即用它快速准确地进行了触摸输入。触摸板具有多指尖手势支持，以及像十年前一样左键单击和右键单击的物理棘爪；当设备处于重负载下时，它是准确的并保持凉爽。指纹读取功能并未内置于触摸板中，而是内置于键盘右上角的方形 LED 电源按钮中。在正常使用中，笔记本电脑既凉爽又安静——但在半小时的多线程 Cinebench 运行中大力推动它迫使它转动风扇。在几分钟的最大负载下，键盘会明显变热 - 但幸运的是，触摸板和周围环境（一个人可能会用手掌托住一些重量）仍然像笔记本电脑卸载时一样凉爽。我们没有发现按键上的热量让您感到非常不舒服——但如果您的工作量涉及将系统长时间推到绝对极限，那么在这种运行过程中明显过热的按键可能需要考虑。运行完成后，键盘温度在一两分钟内恢复正常。继续，我们很高兴地说模块化托架的使用比组装时更好——我们需要插入和拔出 USB-A 和 USB-C 设备很多，这样做并没有任何感觉与结构良好的“普通”笔记本电脑不同。

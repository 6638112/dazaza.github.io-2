Simple Password Management with GPG
使用GPG进行简单的密码管理

2020-11-30 04:47:10

If you want a password management system that is plain-text based, let me introduce to you my setup which utilizes GPG encrypted files to do that.

如果您想要一个基于纯文本的密码管理系统，那么让我介绍一下利用GPG加密文件进行设置的设置。

It uses GPG to encrypt markdown files containing my login credentials, and Syncthing to synchronize them to my Android phone to realize cross-device availability.

它使用GPG加密包含我的登录凭据的markdown文件，并使用Syncthing同步它们到我的Android手机，以实现跨设备可用性。

On my computer, I use Git to version control the folder so I have full history of my login credentials

在我的计算机上，我使用Git对文件夹进行版本控制，因此我拥有完整的登录凭据历史记录

I could just Syncthing the markdown files without encryption to my phone, but on the off chance that my phone is stolen / lost / hacked, I don’t want my passwords to go down with it, so it is safer to have it gpg encrypted.

我可以将降价文件同步而不加密到我的手机上，但是如果我的手机被盗/丢失/被黑客入侵，我不希望我的密码随它一起丢失，因此对gpg进行加密更安全。

I’m aware of pass and tried it before I came to this setup, but I want to put more info than just a password string in my files, so that’s why pass doesn’t fit my need.

我知道通过密码并在进入此设置之前尝试过它，但是我想在文件中放入更多信息，而不仅仅是密码字符串，因此这就是为什么不适合通过密码的原因。

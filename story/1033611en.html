Mostly to remove PHP as a dependency. There's a lot of people who work with JSON who don't have PHP installed.

主要是为了消除PHP的依赖关系。有很多使用JSON的人没有安装PHP。

jq is awesome, and a lot more powerful than gron, but with that power comescomplexity. gron aims to make it easier to use the tools you already know, like grep and sed.

JQ很棒，比GRON强大得多，但伴随着这种强大，复杂性也随之而来。Gron的目标是让你更容易使用你已经知道的工具，比如grep和sed。

gron's primary purpose is to make it easy to find the path to a value in a deeply nested JSON blobwhen you don't already know the structure; much of jq's power is unlocked only once you know that structure.

Gron的主要目的是在你还不知道其结构的情况下，让你很容易在一个嵌套得很深的JSON BLOB中找到通向某个值的路径；只有在你知道该结构之后，JQ的大部分功能才会被释放。

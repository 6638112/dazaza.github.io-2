Jump to navigation Jump to search Canon Cat is a task-dedicated desktop computer released by Canon Inc. in 1987 at the price of U.S. $1,495. On the surface, it was not unlike dedicated word processors popular in the late 1970s to early 1980s, but it was far more powerful, and incorporated many unique ideas for data manipulation.

Canon Cat is primarily the creation of Jef Raskin, the originator of the Macintosh project at Apple. After leaving the company in 1982, he began designing a new computer closer to his original vision of an inexpensive, utilitarian "people's computer"; BYTE in 1987 described the Cat as "a spiritual heir to the Macintosh". [1] It features a text user interface, not making use of any pointer ( mouse), icons, or graphics. All data are seen as a long "stream" of text broken into several pages. Instead of using a traditional command line interface or menu system, the Cat makes use of its special keyboard, with commands being activated by holding down a "Use Front" key and pressing another key. The Cat also uses special "Leap keys", which, when held down, allows the user to incrementally search for strings of characters.

佳能猫最初是由杰夫·拉斯金（Jef Raskin）创立的，后者是苹果公司Macintosh项目的创始人。 1982年离开公司后，他开始设计一种新计算机，使其更接近于最初的廉价，实用型计算机的构想。 1987年，BYTE将Cat描述为Macintosh的精神继承者。 [1]它具有文本用户界面，不使用任何指针（鼠标），图标或图形。所有数据都被视为很长的＆＃34; stream＆＃34;分为几页的文本。 Cat不使用传统的命令行界面或菜单系统，而是使用其特殊的键盘，通过按住＆＃34; Use Front＆＃34;来激活命令。键，然后按另一个键。 Cat还使用特殊的“飞跃键”，当按住该键时，用户可以逐步搜索字符串。

The machine's hardware consists of a 9-inch (229 mm) black-and-white monitor, a single 3½-inch 256 KB floppy disk drive, and an IBM Selectric–compatible keyboard. It uses a Motorola 68000 CPU (like the Macintosh, Lisa, Atari ST, and Amiga) running at 5 MHz, has 256 KB of RAM, and an internal 300/1200 bit/s modem. Setup and user preference data are stored in 8 KB of non-volatile (battery backed-up) RAM. The Cat's array of I/O interfaces encompasses one Centronics parallel port, one RS-232C serial port ( DB-25), and two RJ11 telephone jacks for the modem loop. The total weight of the system is 17 pounds (7.7 kg).

An extensive range of application software is built into 256 KB of ROM: standard office suite programs, communications, a 90,000 word spelling dictionary, and user programming toolchains for Forth and assembly language.

256 KB的ROM中内置了广泛的应用软件：标准的办公套件程序，通信，一个90,000个单词拼写词典以及用于Forth和汇编语言的用户编程工具链。

A text-only machine, the Cat includes graphics routines in ROM as well as connectors for a mouse or other pointing device that are never used. [2]

Cat是一台纯文本机器，包括ROM中的图形例程以及从未使用过的鼠标或其他定点设备的连接器。 [2]

There is a defunct software project called Archy, initiated by Raskin, to develop a similar yet even more capable system for current computing systems. The project was designed to eventually replace current software user interfaces.

由Raskin发起的一个名为Archy的已停产软件项目，旨在为当前的计算系统开发类似但功能更强大的系统。该项目旨在最终替代当前的软件用户界面。

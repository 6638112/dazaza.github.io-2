财务独立的特征

2020-12-06 16:03:56

尽管FI / RE运动相对较新，但在许多方面仍然模棱两可。这些隔壁的百万富翁是谁？他们之间共有哪些共同特征？幸运的是，根据对r / financialIndependence子reddit中近1,400名人员进行的调查所获得的数据，我们不仅可以进一步了解那些追求FI / RE的人，还可以了解已经实现FI / RE的人。特别感谢u / melonbalon首先进行调查！

作为此次调查的一部分，获得了如此多的数据，而以上视觉效果中所报告的数字仅触及了可以得出的宝贵见解。但是值得注意的是以下几项：

通勤时间很短。超过20％的受访者指出，上班时间不到10分钟。此外，近70％的乘客上下班时间不到30分钟。这些较小的通勤时间可能会减少运输成本，这可能是一个诱因。

拥有FI / REd的个人受过高等教育。不仅近85％的受访者拥有大学学位，而且还有7％的受访者拥有多个研究生学位。也许这些人有更多时间去退休后再上学。这种额外的教育也可能首先帮助了他们进行FI / RE。无论如何，与仅仅由于平均年龄较大而尚未接受FI / RE的受访者相比，期望接受更多的教育。

受访者同样偏爱城市和郊区环境。另外，在追求FI / RE和FI / RE的人群中，该比率保持恒定。我原本希望郊区居住更多，但也许减少的交通成本足以抵消城市环境中生活成本的增加所抵消。

Microsoft宣布推出Windows 11，使用简化的UI，新的Windows Store，“开始”菜单等

2021-06-24 23:18:54

微软正式确认下一个Windows的下一个Windows的名称：Windows 11.经过几个月的戏弄，第11个的提示和巨大的Windows 11泄漏，微软的新操作系统是官方的。 Windows 11的大焦点是简化Windows用户界面，新Windows Store以及性能和多任务处理的改进。

Windows 11包括新的“开始”菜单和更新的“开始”按钮，它们都以任务栏为中心。这个UI非常类似于我们在Windows 10x中看到的东西，该项目最初计划用于微软最终取消的双屏幕设备。 Windows 11中出现了大量进入Windows 10x的UI工作。

新的“开始”菜单会丢弃最初使用Windows 8引入的实时差块，并选择您在Chrome OS或Android中找到的更多典型启动器。有应用程序，最近的文档和单独的搜索界面。大部分居中的外观都是由麦克斯和Chrome OS的影响，并且Windows 11还包括我们在Android和iOS中看到的圆角。

Windows首席Panos Panay说：“该团队已经痴迷于每个细节。” Windows 11还将包括更新的黑暗和光线模式，这看起来比我们今天在Windows中看到的更好。

还有Microsoft正在调用捕捉布局的东西，允许您将应用程序快速捕捉到Windows 11支持的各种模式。此新版本的Windows 11还记得您的应用程序在哪里存储，感谢您称为Snap组布局的内容。

它看起来像支持多个监视器的有用方法，并确保应用程序始终在正确的屏幕上打开。如果您使用挂钩到监视器的笔记本电脑或具有多个显示器的传统桌面机器，则特别有用。

性能也是Windows 11的大焦点.Windows更新较小40％，而且在后台发生时，它们的效率更高。希望这意味着Windows 11在工作中间不会打扰你。

微软确认 Windows 11 中的任务栏不再支持拖放

2021-07-23 05:31:54

与旧的 Windows 10 相比，Windows 11 包含一些非常显着的视觉变化。 安装 Windows 11 预览版（目前在 Windows Insider 程序中可用）后，您会立即注意到一个新的居中任务栏、全新的“开始”菜单，以及其他视觉改进。像往常一样，新操作系统带来了新功能并删除了一些以前可用的功能。 Microsoft 已经发布了 Windows 11 中已删除和弃用的功能列表，但事实证明，还有一些更多未宣布且非常奇怪的更改。用户注意到 Windows 11 中的任务栏不再支持拖放。在 Windows 10 中，您可以通过将文件拖到任务栏上的应用程序来打开文件。此外，相同的逻辑适用于固定应用程序快捷方式。在 Windows 11 中似乎这些都不起作用。在 Windows 11 中任何拖放文件或任务栏快捷方式的尝试最终都会在光标附近出现一个停止图标。虽然有些人认为无法将对象拖放到 Windows 11 中的任务栏是一个错误，但微软确认这一更改是有意的。在反馈中心，一位微软代表表示 Windows 11 目前不支持拖放，但该公司“感谢用户的反馈”。在 Windows 11 中，任务栏失去了用户在 Windows 10 中喜欢的其他一些非常受欢迎的功能。例如，您不能再将任务栏移到顶部、右侧或左侧。 Windows Timeline 也不见了，但可以肯定地说，很少有用户会错过该功能。很难说微软为什么要在 Windows 11 中删除任务栏的拖放功能。如果这个有争议的更改让您以及其他成千上万的内部人员感到沮丧，请务必将您的反馈发送给 Microsoft。

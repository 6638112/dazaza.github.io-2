ISA8019 is a modern, open-source hardware design for an ISA-bus network card.It uses an RTL8019 chipset and is NE2000-compatible, which makes it easy to use with just about any operating system or software available.

ISA8019是用于ISA总线网卡的现代开放源代码硬件设计，它使用RTL8019芯片组并与NE2000兼容，因此可以轻松地与几乎所有可用的操作系统或软件一起使用。

Only the RJ45 jack, jumper headers and the DIP-28 socket for the Boot ROM have to be soldered.Jumpers and Boot ROM are optional, the card can be used without any jumpers configured.

仅需焊接RJ45插孔，跳线接头和Boot ROM的DIP-28插座即可。跳线和Boot ROM是可选的，无需配置任何跳线即可使用该卡。

Jumper mode (JP jumper set) This mode allows for manual configuration of any parameters (I/O address, IRQ, BROM address & size, etc.) manually via jumpers. Refer to the back of the PCB for details.

跳线模式（设置了JP跳线）此模式允许通过跳线手动配置任何参数（I / O地址，IRQ，BROM地址和大小等）。有关详细信息，请参见PCB的背面。

EEPROM mode (also called "Jumperless", no jumpers set) In this mode, all parameters can be configured via the RSET8019.EXE dos utility and will be stored in the 9364 I²C EEPROM.

EEPROM模式（也称为＆＃34; Jumperless＆＃34 ;，未设置跳线）在此模式下，所有参数都可以通过RSET8019.EXE dos实用程序进行配置，并将存储在9364I²CEEPROM中。

PNP mode (no jumpers set) For modern computers and operating systems like Windows 9x or Linux, this mode is ideal. Resource management will be done by the BIOS or OS.

PNP模式（未设置跳线）对于现代计算机和Windows 9x或Linux等操作系统，此模式是理想的。资源管理将由BIOS或OS完成。

RSET8019.EXE for DOS, can set EEPROM parameters, also includes diagnostic functions pg8019 for DOS, can initialize EEPROM, set MAC address, vendor/product IDs

用于DOS的RSET8019.EXE，可以设置EEPROM参数，还包括用于DOS的诊断功能pg8019，可以初始化EEPROM，设置MAC地址，供应商/产品ID

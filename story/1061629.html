Ubuntu认证系统在AMD R1000上运行，低至6W

2021-05-10 09:44:34

DFI揭开了一个关于其Ryzen R1000的GHF51 SBC周围的“EC90A-GH”工业迷你电脑。这些是支持6W R1102G和第一个工业系统提供认证的Ubuntu核心OTA更新的产品之一。 DFI宣布了一个无风扇，110 x 80 x 60毫米嵌入式系统，称为EC90A-GH，内置GHF51 SBC。 EC90A-GH和GHF51 SBC运行Ubuntu，Ubuntu核心，或在AMD的Ryzen嵌入式R1000上赢得10个IOT Enterprise，是DFI紧凑的“工业PI”系列嵌入式产品的一部分。它们也是第一个支持两种低功耗R1000部件的产品，TDPS低至6W。 EC90A-GH（左）和GHF51（点击图片放大）DFI注意到EC90A-GH和GHF51是第一个工业嵌入式计算机之一，以接收Ubuntu核心OTA更新的认证（见下文）。 DFI还宣布了一个“免费样品事件”，您可以在其中注册免费的EC90A-GH，GHF51或两台Apollo Lake基于Apolo Lake的DFI工业PI产品：PI-尺寸ALF51和PICO-ITX AL05P SBCS（见下文）。

EC90A-GH不仅仅是覆盆子PI大小的机箱版本，84 x 55mm ghf51。有一些新的端口和几种可选配置。与GHF51一样，EC90A-GH支持所有四个Ryzen嵌入式R1000型号。其中包括原始双核，2.6GHz / 3.5GHz R1606G和2.4GHz / 3.3GHz R1505G，两者都有12-25W TDPS（DFI是指12W）。

EC90A-GH和GHF51也是我们从简单的NUC的Red Oak（NP-FP5）和橡木（BP-FP5）MINI-PC上看的第一个产品，该PCS支持两个新的低TDP R1000型号之一。与SillyNuc系统一样，DFI计算机可以加载双核，1.5GHz / 2.8GHz R1305G，8-10W。它们也可以配有双核，1.2GHz / 2.6GHz R1102G，具有6W TDP。与GHF51不同，EC90A-GH缺乏对高端Quad-Core V1605B的支持。 EC90A-GH，前后（点击图片放大），如GHF51，EC90A-GH配有2GB至8GB焊接，3200MHz DDR4加16GB至64GB EMC。 GBE端口提供了一个Intel I211AT或I211IT控制器的选择。快速的USB 3.1 Gen 2类型C端口和2x Micro-HDMI 1.4端口，最多可达4096 x 2160 @ 24Hz。

全尺寸迷你PCIE插槽（PCIe Gen2，USB 2.0）可选地配备带有双天线的LTE或WiFi / BT模块。没有列出SIM卡插槽，但系统据说系统是“用户友好安装SIM卡设计”。

还有一个带有双端口的后面板选项，带有双2x RS-232端口，其互斥无线选项。第三个选项为您提供一个带有8位DIO和SMBus的组合端口，它看起来与无线选项兼容。

EC90A-GH提供了一个看门狗和TPM 2.0芯片，但没有提到SBC的RTC。该系统用可锁定的12VDC插孔供电。 700克，金属和铝制包装系统配有墙壁和VESA安装。

除了使用SBC提供的加热器外，EC90A-GH还具有散热器冷却结构。它具有比SBC更窄，0至45°C的范围，提供5至90％RH（非冷凝）的相对湿度耐受性。计算机增加了5G冲击和1.5G振动阻力，但仅在不运行时。

2月25日，Canonical宣布GHF51 SBC和即将到来的EC90A-GH以及DFI的第7届kaby Lake基于EC70A-Ku嵌入式计算机，并在Ubuntu IoT硬件认证合作伙伴计划下认证。 DFI是第一个加入该计划的工业系统供应商。基于Kaby Lake的EC70A-KU（左）和Remoguard的架构图与Ubuntu Core（点击图片放大），在DFI产品，Ubuntu核心，带有空中（OTA），App Store和事务回滚功能，与DFI自己的RemoGuard带外（OOB）远程管理解决方案集成。去年8月宣布，RemoGuard“可以在关闭之前及时预测设备状态，并且有效地激活OS自动备份和恢复尽管主机崩溃，”DFI说。

RemoGuard与Innodisk的Innoage 2.5英寸SSD集成，其中基于微软的专注于基于Solfir为主的Linux的Azure Sphere技术，用OOB软件送货。我们之前在Aetina的AN810-XNX载板上看到了作为NVIDIA的Jetson Xavier NX的M.2 SSD选项的Innoage。

使用Remoguard及其无创存储，设备运行状况自动收集到云端。用户可以通过可自定义的UX仪表板监控连接的设备。使用RemoGuard和Ubuntu Core的捕捉包装机制，SSD的OS分区“可以远程修复无限次数，”DFI说。 “可以积极地预测SSD生命周期以确定准确的更换时间，以避免耗时，现场维护，并大大减少应用程序服务的停机时间。”

DFI宣布为GHF51 SBC，EC90A-GH和另外两种工业PI产品进行免费样品计划。输入您的首选产品，以及您计划与之建造的项目的描述。如果你然后“提交我们的销售面试”，那么你可以赢得免费样品的机会。该计划于5月1日至7月31日。Alf51（左）和基于ALF51的EC90A-A1（点击图片放大）该程序中的另外两种工业PI产品是基于DFI的Apollo Lake Al05P Pico-ITX板（100 x 72mm ），我们在1月份涵盖，另一个Linux友好的Apollo湖SBC称为ALF51。 84 x 55mm Alf51船用2GB至4GB焊接LPDDR4-2400加16GB至64GB EMMC。

ALF51 SBC配备2倍GBE，2x USB 3.1 Gen1和Mini DP ++端口。内部功能包括迷你PCIe插槽和单串行，USB 2.0，DIO和SMBus标头。 12V端子块供电的SBC还具有看门狗，RTC，带硬币电池和TPM 2.0，以及工业温度选项。

ALF51还为EC90A-AL嵌入式MINI-PC供电。但是，这似乎不是样本程序的一部分。

基于R1000的EC90A-GH系统预计将于8月份以未公开的价格发货。可以在DFI的产品页面上找到更多信息。

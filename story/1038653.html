Xbox云游戏将于2021年春季登陆iOS和PC

2020-12-09 23:18:06

微软的云游戏服务最初称为Project xCloud，可让您在非Xbox设备上玩Xbox游戏。游戏在您附近的数据中心的服务器上运行。视频将流式传输到您的设备，并且您的交互将实时中继到服务器。

Xbox云游戏不是单独的订阅。订阅Xbox Game Pass Ultimate（每月14.99美元）的人可以访问云游戏作为其订阅的一部分。该计划还包括访问游戏库，EA Play和Xbox Live Gold。

对于新设备，您很快就可以通过PC在Xbox云游戏上启动游戏。该服务将在Xbox应用程序中并使用网络浏览器提供。

如果您是Xbox Game Pass Ultimate订阅者，可以将游戏下载到PC上，但对于计算机中没有强大GPU的人来说，云游戏将特别有用。看看Microsoft是否将其服务限制为Windows计算机上运行的Web浏览器将会很有趣。我敢肯定，很多人也希望从Mac上访问该服务。

对于iOS，由于限制性的App Store规则，Microsoft将通过Web浏览器专门启动云游戏。英伟达已经为iOS Web浏览器发布了Beta版GeForce Now。我从iPhone和iPad上尝试了Nvidia的服务，并且网络浏览器的解决方法非常有效。

如果您有Android手机或平板电脑，则可以从Xbox Game Pass应用程序访问Xbox云游戏。您的体验会因互联网连接和Wi-Fi网络的质量而异。

您还需要位于数据中心附近，以最大程度地减少延迟。因此，Xbox云游戏只能在少数几个国家/地区使用-大多数是美国，加拿大，韩国和欧洲部分地区。

微软在其博客文章中表示，该公司将开放更多的数据中心并扩展到新市场。 您可以期待很快在澳大利亚，巴西，日本和墨西哥出现Xbox云游戏。

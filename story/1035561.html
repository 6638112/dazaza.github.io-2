Windows转动35

2020-11-21 12:59:39

跳到主要内容PC革命本周开始于35年前。微软于1985年11月20日发布了其第一版Windows，以继承MS-DOS。这是一个巨大的里程碑，为我们今天使用的现代Windows版本铺平了道路。尽管Windows 10看起来不像Windows 1.0，但它仍然具有许多原始的基础知识，例如滚动条，下拉菜单，图标，对话框以及记事本和MS Paint等应用程序。

Windows 1.0还为鼠标设置了舞台。如果您使用的是MS-DOS，则只能键入命令，但是在Windows 1.0中，您会拾起鼠标并通过指向和单击来移动窗口。除了原始的Macintosh，鼠标还彻底改变了消费者与计算机交互的方式。当时，许多人抱怨Windows 1.0过于关注鼠标交互而不是键盘命令。微软的第一个版本的Windows可能并没有广受欢迎，但它引发了苹果，IBM和微软之间的一场向大众提供计算的斗争。

早在1985年，Windows 1.0需要两个软盘，256 KB内存和一个图形卡。如果要运行多个程序，则需要一台具有硬盘和512 KB内存的PC。您将无法在现代计算机上运行只有256 KB内存的任何内容，但是这些基本规格仅仅是开始。尽管苹果当时在生产鼠标驱动的GUI方面处于领先地位，但它仍然专注于硬件和软件的组合。微软已经为IBM PC创建了低成本PC DOS操作系统，并已牢固地定位为软件公司。

借助Windows 1.0，Microsoft迈出了重要的一步，专注于应用程序和核心软件。 IBM坚持PC架构的基础已有几年了，但是Microsoft使竞争对手和软件开发人员易于创建应用程序，从而确保Windows相对开放并且易于重新配置和调整。 PC制造商蜂拥至Windows，并且该操作系统吸引了重要软件公司的支持。这种为硬件合作伙伴出售自己的计算机提供软件的方法为Microsoft创建了一个巨大的平台。这个平台可让您升级到所有Windows版本，如经典的YouTube片段所示。

Windows已经在个人计算领域占据了35年的主导地位，并且Mac和PC的广告系列几乎没有改变这种趋势，但是它们确实很有趣。微软继续调整Windows，并在跨设备，在企业中以及现在迁移到云中为其创建新用途。直到现在，随着现代智能手机和平板电脑的普及，Windows面临着最严峻的挑战。微软可能尚未度过移动风暴，但只有通过重新树立起真正的软件公司的根基，它才能做到。在2055年，我们不太可能再以同样的方式庆祝Windows的35周年，所以让我们回顾一下微软的操作系统自卑微的诞生以来所发生的变化。

一切从哪里开始：Windows 1.0引入了GUI，鼠标支持和重要的应用程序。比尔·盖茨（Bill Gates）在Mac上从事软件开发工作多年后，领导了操作系统的开发。 Windows 1.0是Microsoft的第一个图形PC操作系统，在MS-DOS之上具有16位外壳。

Windows 2.0继续使用VGA图形以及Word和Excel的早期版本进行16位计算。它允许应用程序相互重叠，并且桌面图标使Windows在1987年12月发布2.0时更易于使用。微软六个月后继续发布Windows 2.1，这是Windows 2000的第一个版本。 Windows需要硬盘驱动器。

Windows 3.0继承了MS-DOS之上的GUI的传统，但它包括具有更好的UI的新程序和文件管理器。 Windows 3.1更新也带来了《扫雷》，这款益智游戏充满了隐藏的地雷。

Windows NT 3.5是NT的第二个版本，它确实标志着Microsoft借助重要的安全性和文件共享功能推动了业务计算。它还包括对TCP / IP的支持，TCP / IP是我们今天都用来访问Internet的网络通信协议。

Windows 95是Windows现代化时代开始的地方。它是Windows最重要的更新之一。 Microsoft移至32位体系结构，并引入了“开始”菜单。应用程序的新纪元出现了，Internet Explorer引入了Windows 95的更新。

Windows 98通过改善硬件支持和性能而基于Windows 95的成功。微软在启动时还专注于Web，并捆绑了诸如Active Desktop，Outlook Express，Frontpage Express，Microsoft Chat和NetMeeting等应用程序和功能。

Windows ME专注于多媒体和家庭用户，但是它不稳定且容易出错。 Windows Movie Maker首次出现在ME中，同时还有Windows Media Player和Internet Explorer的改进版本。

Windows 2000是为企业内的客户端和服务器计算机设计的。它基于Windows NT，设计为具有新文件保护，DLL缓存和硬件即插即用的安全性。

Windows XP确实结合了Microsoft在家庭和商业方面的努力。 Windows XP专为企业内的客户端和服务器计算机而设计。它基于Windows NT，设计为具有新文件保护，DLL缓存和硬件即插即用的安全性。

Windows Vista像我一样受到好评。 Vista引入了新的Aero UI并改进了安全功能，而Microsoft花费了大约六年的时间来开发Windows Vista，并且仅在新硬件上运行良好。用户帐户控制受到严重批评，Windows Vista仍然是Windows版本糟糕周期的一部分。

Windows 7于2009年问世，以清理Vista的混乱局面。微软在调整和改进用户界面以及减少用户帐户控制的烦恼方面做得很好。 Windows 7现在是Windows最受欢迎的版本之一。

Windows 8是对熟悉的Windows界面的重新设计。 Microsoft删除了“开始”菜单，并将其替换为全屏“开始”屏幕。新的“ Metro风格”应用程序旨在替代老化的桌面应用程序，Microsoft真正专注于触摸屏和平板电脑。对于大多数台式机用户来说，这太过激烈了，微软不得不重新考虑Windows的未来。

回到开始：Windows 10返回熟悉的“开始”菜单，并引入了一些新功能，例如Cortana，Microsoft Edge和流式传输到PC的Xbox One。它是为混合型笔记本​​电脑和平板电脑而设计的，因此经过深思熟虑，并且Microsoft已改用Windows即服务模型，以使其在将来定期更新。

Windows 10在过去五年中并没有发生太大变化。 Microsoft一直在调整操作系统的各个部分以完善它。更多的系统设置已从传统的“控制面板”移至新的“设置”应用程序，并且“开始”菜单现在看起来更加简洁。我们仍在等待Windows 10X（最初是为双屏设备设计的）所带来的，但是微软也一直在改进Windows 10的系统图标。2021年可能会给Windows 10带来更大的视觉效果。

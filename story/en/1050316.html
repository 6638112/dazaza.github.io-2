You can close the tag window anytime using ^T again. To quit, use ^Q.For more details on the possible commands and keybindings, read the files vicmd.w and exec.c. Clone the repository git://c9x.me/ed.git for the latestchanges. You can also browse the repository. Compile with GNU make, you will also need cweb (as provided bymost LaTeX distributions) because one file is written in literateprogramming. A minimalpackage is available on Knuth's webpage, note that it is a tarbomb!

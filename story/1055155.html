五年的琐事

2021-03-30 04:58:28

这一切都开始了一封电子邮件。回到2015年，我曾在我的单人公司，Atomic Squirrel担任顾问，我希望聘请Devops工程师与我一起使用一些项目。一个9月早上，我醒来在我的收件箱中找到以下消息：

我看到你正在寻找一个Devops顾问，这几乎是我做的。

这是来自名叫Josh Padnick的人，他通过几年前发布的一些博客帖子找到了我关于一些开源的Web框架。在挖掘乔希的背景之后，我发现他有自己的单人咨询公司，称为凤凰德国人，并在我们所做的事情上有很多重叠。我决定有一个聊天并通过电子邮件向josh回馈，并在未来的描述中描述：

首先，对原子松鼠的一个非常快速的介绍。我与公司的目标是为整个技术堆栈做些什么亚马逊＆amp;其他云提供商为数据中心做了。也就是说，这些天几乎没有启动运行自己的数据中心或机架自己的服务器 - 他们只是使用AWS或类似的服务器。我认为其余的技术堆栈有类似的机会。不需要初创公司将几个月粘在一起的rails + mysql + github + travis + ansible + docker + aws +新遗物+ koggly ...（更多项目）...我认为可以创建预包装，结束 - 结束，战斗测试的技术堆栈可以显着减少推出某些东西的时间和您最终启动后的技术债务金额。

Josh和我最终谈了，然后通过咨询彼此的公司来共同努力，最后，通过在一个名为Gruntwork的单一公司加入武力。本月，犯罪作业转弯5岁，所以我花了时间暂停，挖掘乔希的旧电子邮件，并反思我们从那时起我们走了多远：

我们在电子邮件中创建了“预打包，端到端，战斗测试的技术堆栈”：请参阅基础架构作为代码库和参考体系结构。

在这个博客文章中，我将分享5个课程，帮助我们到了这里：

媒体喜欢讲述孜孜不倦地工作的英雄企业家的故事，蔑视所有赔率，并完全成功，因为他们的辉煌决策，祭祀和砂砾。但这不是我们的故事。要诚实，那不是任何人的故事。

建造一家公司有点像烹饪：你需要正确的成分，在合适的时间内在正确的比例。良好的决策，牺牲和辛勤工作是3个必要的成分，而乔希和我陷入了每一部分，但你会发现在一家好公司的每一个食谱中，还有另一个成分绝对必要，但经常出现无情：少许运气。

这是一个简单的思想实验，展示了运气的作用：以你最喜欢的，最有才华，最崇拜的企业家，并改变关于他们的生活的重要事物 - 例如，将它们送回100年，或将它们移到另一个国家，具有完全不同的文化或经济 - 并询问他们是否可以建立成功的公司。例如，Bill Gates可以建立一个世纪前的微软吗？或者他可以在苏联建造它吗？

你经常会发现答案是否定的。现在这是关键点：这些是企业家无法控制的因素。谈到这些因素时，他们只是在正确的时间幸运地在正确的地方。比尔盖茨很幸运，足以在世界的一部分中长大，相对安全，稳定，富裕，支持的创业;幸运能够获得优秀的教育，包括广泛访问计算机（在20世纪60年代不寻常）;足够幸运能从IBM的许多战略误导中受益;并且幸运地在数字革命的尖牙（因为他所说的那样）。拿走任何一个成分，也许我们都不会听说比尔盖茨。

现在，让我在这一点上额外清楚：独自运气是不够的。比尔盖茨唯一的成功，因为他很幸运。但就像一个人一样运气不够，也不是艰苦的工作或良好的决策。虽然每个人都魅力后一种物品，但前者的注意力不足。所以即使Josh和我无处可去的比尔盖茨的成功水平，我们仍然非常幸运能够成为我们所在的地方，我们希望明确承认只有一些令人惊叹的运气允许我们来到这里：

电子邮件和博客。乔希的电子邮件是纯粹的运气。我们彼此不认识，这是纯粹的运气，他碰巧正在使用戏剧框架，并遇到几年前发表的一些博客帖子。这也是纯粹的运气，我希望在原子松鼠上建造团队，乔希碰巧当时有一些可用性。情况下的任何微小的变化，也许Josh从未发送过电子邮件，而且犯罪作业永远不会存在。

云，devops，iac。我们在Gruntwork解决的问题是多年来一直在困扰我们的问题。但在云之前，在Devops运动之前，在代码（IAC）工具出现的基础设施之前，我们无法建立一个现成的基础架构库。如此多的商业成功归结为时间：例如，电脑鼠标在GUI之前发明的经典案例。

家庭。乔什和我很幸运能够有精彩的，支持的家庭，这些家庭已经支持了我们的每一步。

团队。乔希和我自己没有建造犯罪作业。我们通过艰苦的工作，灵感和牺牲了一个超过20人的努力，我们很幸运能找到这个世界。

谈到我们的惊人团队，这是一个有趣的事实，关于在Gruntwork工作的20多个咕噜声：他们出生的地方代表了10多个不同的国家（包括美国，巴西，保加利亚，拉脱维亚，乌克兰，尼日利亚，澳大利亚，塞尔维亚，日本和芬兰）和他们居住的地方有点不同的8个国家（包括美国，英国，德国，加拿大，尼日利亚，爱尔兰，芬兰和澳大利亚）。在Gruntwork中，许多咕噜声选择分享他们的个人用户指南 - 有点像一个自述或使用说明书，了解如何与您最佳工作 - 我发现它发现了人们认为所有不同方式思考，工作，生活方式和相互影响。

自1日起，Gruntwork一直是100％全球分布的团队，所以我们很幸运能够先经历第一手的想法（通常归因于Leila Janah），才能均匀地分布在世界各地，但机会不是。与Covid-19强迫如此许多其他公司也分布式分发，我认为世界其他地方将变得越来越意识到这一事实。

当然，作为一个分布式公司，由于被迫由于全球大流行和检疫而被迫分发。尽管如此，一旦你有一种偏远的工作 - 当你的招聘不仅限于一些任意办公室位置的10英里半径时，就可以与伟大的人一起工作，无论在哪里工作从你想要的地方，当你想要的时候，没有花时间通勤 - 这很难回去。分布式工作有很多缺点，特别是如果你被迫进入它，而不是在它周围设计的公司，所以无疑，许多企业将恢复到他们在Covid-19之前的方式。但我认为很多，遥控工作的权衡将是净获胜，所以大量公司将留下分发，更重要的是，我怀疑数百万人有味道的分布式工作，未来开始的大多数新业务将分发给第一天。

我们应该在未来十年内为一个更广泛的全球劳动力准备。我们应该为真正的全球化做好准备：不是外包，其中一些大公司拥有中央办公室，正在进行初级工作和离岸团队进行外围工作，但完全分布的偏远第一家公司，没有中央办公室，而且所有的工作由全球的团队完成。在大多数情况下，我认为这对世界来说将是一个非常好的东西，因为机会终于蔓延到一些特权国家的边界​​之外。

也就是说，在过去的5年里完成了分布式公司的事情，事实是，大多数政府都没有为这个勇敢的新世界做好准备。大多数国家的法律法规 - 特别是劳动法和税法 - 没有继续支持支持这一新模式所需的步伐。许多这些法律是基于现在感到过时的假设，即使是古迹，甚至是古迹，也是每个公司都必须拥有物理办公室的想法，或者产品是物理物品，在一个位置建造和销售。

因此，招聘国际 - 有时候，甚至招聘单个国家，例如跨多个美国的国家 - 令人困惑，复杂，耗时，昂贵，特别是对于小型创业公司。所有选项都是蹩脚的：

承包商：您可以聘请人们作为承包商，这很便宜，而且在大多数国家，拥有承包商为一家公司进行全职工作是非法的。

全职就业：您可以直接雇用人们，但在大多数国家，劳动力和税法要求您在该国有一个实际办公室。这涉及大量的开销，即建立子公司，提交文书，在多个国家的申请税，遵守当地劳动法等。对于小型，引导启动，这条路线往往是不可行的。

共同就业：另一种选择是利用一项合作计划，员工通过专业雇主组织（PEO）提供技术聘用，这是一个在每个国家的实际办公室的大公司，但您签署合同他们工作的所有知识产权都属于您的公司。 PEO处理所有当地的劳动法，工资单，税收等。在Exchange中，您通常会向每个员工的薪资支付PEO标记：例如，如果员工每年赚取100亿美元，您可能会支付20％的标记，或每年约20,000万美元，因此雇用的总费用人每年为120亿美元。这意味着每个雇用都更昂贵，而且因为它避免了在世界各地建立子公司的相当大的前面和正在进行的开销，这通常是小型，引导的初创公司的最佳选择。这是我们在Gruntwork遵循的路线，在美国和Remote.com和首都Ges中使用Justworks作为我们的Peos。

这些都是不完美的，痛苦的选择。但是哪里有痛苦，有机会。可能不会有一个“下一个硅谷” - 所有人才必须在一个地点池的想法将在未来时代过时 - 但是任何国家都可以使其法律能够使遥远的工作理智，价格实惠将有一个巨大的在未来的经济中的腿。

大多数初创公司失败了。虽然确切的百分比因如何定义“启动”和“失败”而异，但典型的故障率约为75％。

在成功的几个初创公司中，需要十年。如果你很幸运能够成功作为一个创业，它将平均需要五到八年来达到成功的出口，例如收购或IPO，平均是向上培训。此外，作为创始人，你很少在出口后立即离开，而是持续至少2-3年，使总额约为10年。

大多数人听说过物品（1），但相对少的是意识到项目（2）。再次，媒体扭曲了这里的东西，有意思地讲述“一夜之间成功”的故事。但是，这不是我们的故事，这并不是任何人的故事。

现实是，如果你想建立一个成功的公司，你必须愿意至少投入你生命的十年。如果你现在在20多岁时，你会在你的30多岁时努力工作;如果您已经在30多岁，您将在您的公司上工作，直到您在40多岁;那就是如果你很幸运，就足以成为少数成功之一！

建造公司需要很长时间。我可以告诉你，在花了5年的犯罪作业工作后，它仍然感觉我们刚刚开始，而且还有更多的要做（更稍后更多）。事实证明，这类时间轴不是初创公司的独特：良好的软件至少需要10年，大的新想法可能需要更长。

这一切都意味着你需要将一个启动作为马拉松，而不是冲刺：

思考长期。在尝试挑选启动概念时，避免追逐时尚，时尚和富含富有的方案。将其视为10年的投资，只有选择一个想法，如果你可以真正看到自己在下一个醒着的时间里度过了下一年你的生活。

自我节奏。不要尝试执行80或100小时。忽略喧嚣的色情片。几十年来，没有办法保留那种节奏。好消息是你不必

谈到休假，我们尝试遵循GruntWork的核心原则之一是：

换句话说，我们计划在生活中的工作，而不是我们的生活。工作很重要，我们对我们的使命来充满热情，使其更容易构建软件，但我们也有其他激情。我们喜欢与家人和朋友共度时光;去旅行;运动;阅读;去徒步旅行;等等。

工作基本上是无限的;或者，根据Parkinson的法律，工作将扩展以填补您分配的时间。为了防止工作接管你生命中的每个方面，你必须先计划生活，只填写剩下的时间。

例如，如果您想在Gruntwork休假，则没有批准过程;相反，您提前将其添加到日历中，在我们的#OO​​O（Out Office）Slack频道中宣布它，然后我们计划周围的工作。事实上，在Gruntwork，我们有一个最低的假期政策，我们要求所有员工每年至少花4周（不包括假期，以及我们2周的冬季休息关闭）。为了让每个人都在轨道上，我们甚至建立了一个内部工具，可以从每个人的日历中读取休假时间，并显示您是否满足最小值：

在休假期间，不仅仅是在休假期间，才能在休假期间，而且在一天的基础上是很重要的。作为一个创业公司联合创始人，我可以告诉您，完全可以考虑每天24小时的初创公司。总有更多的工作要做，总是有更多的问题来解决，总是新的事情要担心。如果你让它，一个初创公司可以完全接管你的生活，填补每个醒着的时刻，也是很少的睡觉时刻。这不是一件好事。在你的生活中的一切成本上取得了成本，这不是我想要制作的折磨，而对于这一事件，我不确定与你的创业公司的24/7迷恋让你有任何更容易成功。

有一点有助于在您工作的地方和您居住的地方产生物理分离。我们提供所有犯罪作业员工，可用于提高生产力和健康的健康预算，包括支付合作空间，因此您可以在您生活的地方完全独立的地方进行工作，并支付健身房会员的工作，所以你可以保持健康。

在Covid-19期间，大多数人必须因锁定而留在家里，使分离很艰难。如果你很幸运，你家里可以用作办公室的独立房间（再次有那么运气！），这可能是一个巨大的帮助，因为你可以在一天结束时“留下”工作，关上你身后的门，更容易专注于其他东西。当然，不是每个人都有那种特权，而在家里的孩子们因Covid-19而在家里，从生活中分开工作变得更加艰难。因此，在2020年，我们将个人预算翻了一番，并冲出更多用例：例如，您可以使用它来支付保姆，保姆，儿童保育，Peloton会员，家庭健身房，家庭办公设备等。

不幸的是，Covid-19不是2020年唯一的问题。这对每个人来说都是一个艰难的一年。有时喜欢这些，在工作前提醒每个人都比以往任何时候都更重要：

在亚马逊，杰夫贝罗斯着名的，“这总是第1天。” 无论公司如何获得，无论他们有多少员工，无论他们周围多久，他们都会想到它的第1天，以保持创新，资源丰富和敏捷。 我们无处可去亚马逊的规模附近，我们仍然是一个微小的初创公司，但“它总是第1天”对我们有意义。 在某种程度上，它感觉自从Josh在5年前发送了这封电子邮件以来已经很长一段时间了。 但是，在其他方面，感觉就像我们刚刚开始 - 就像我们几乎没有划伤了可能的事情。 我们要建造的更多信息： 有很多事情要做 - 这是一件很好的事情。 我们很高兴能够在这些物品上工作。 我们很高兴让它更容易为每个人构建软件。 我们很兴奋，即使在5年的工作之后，它仍然是第1天。

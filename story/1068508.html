github copilot：您的AI对程序员

2021-06-29 23:00:42

今天，我们正在推出Github Copilot的技术预览，这是一个新的AI对程序员，可帮助您编写更好的代码。 github copilot从正在进行的代码中绘制上下文，暗示整行或整个函数。它可以帮助您快速发现解决问题，编写测试和探索新API的替代方法，而无需速度测量在互联网上寻找答案。当您键入时，它会符合您编写代码的方式 - 帮助您更快地完成工作。

GitHub Copilot与Openai合作开发，由Openai创建的一个新的AI系统供电。 Openai Codex具有广泛的了解人们如何使用代码，并且在代码生成中的GPT-3具有明显的能力，部分原因是，因为它培训了包括更大集中的公共源代码的数据集。 GitHub Copilot使用广泛的框架和语言，但此技术预览对于Python，JavaScript，Cyperscript，Ruby和Go尤其适用。

在GitHub CopIlot页面上了解更多信息。技术预览的斑点数量有限，所以今天注册！

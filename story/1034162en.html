Structslop：GO结构对齐分析器

2020-11-10 06:55:00

Package structslop defines an Analyzer that checks struct can be re-arranged fields to get optimal struct size.

Package structslop定义了一个分析器，它检查struct可以被重新排列的字段，以获得最佳的struct大小。

You can run structslop either on a Go package or Go files, the same way asother Go tools work.

您可以在GO包或GO文件上运行structslop，这与其他GO工具的工作方式相同。

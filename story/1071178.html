德州仪器全新 TI-84 Plus CE Python 图形计算器

2021-08-02 22:10:42

达拉斯，2021 年 7 月 27 日/美通社/ -- 最值得信赖、最受推荐的图形计算器品牌德州仪器 (TI) 宣布推出下一代 TI-84 Plus 系列——TI-84 Plus CE Python 图形计算器。 TI-84 Plus CE Python 图形计算器可及时用于返校，可帮助学生探索、可视化和更好地理解数学和科学，同时还向他们介绍 Python，Python 是世界上发展最快、最受欢迎的编程语言之一。 “在德州仪器，我们关心确保下一代学习技能以在未来的劳动力中取得成功，”TI 教育技术总裁 Peter Balyta 说。 “Python 是一种易于上手的编程语言，对初学者和高级程序员都有好处，在应用程序开发和人工智能等领域被许多世界领先的技术公司广泛使用。”新的 TI-84 Plus CE Python 图形计算器包括熟悉的功能，如全彩屏幕、轻巧的设计和一次充电可使用长达一个月的可充电电池。与笔记本电脑和平板电脑不同，TI-84 Plus CE Python 图形计算器没有 Wi-Fi、蓝牙或相机等任何干扰，让孩子们专注于学习。家长们也会欣赏到，这款计算器的设计可以满足年复一年的课堂需求，并且它将让中学数学和科学课程的学生一直到大学。 “通过将 Python 添加到许多学生已经熟悉并在课堂上使用的计算器，我们让所有学生都更容易和平易近人地使用编程，无需教师保留单独的计算机实验室来教授这些重要技能，”Balyta 说。 “使用 Python，学生不仅在学习编程；他们还在学习关于批判性和创造性思考以解决现实世界问题的终生课程。”重要的是，TI-84 Plus CE Python 图形计算器已获准用于高风险考试，包括 PSAT/NMSQT*、SAT* 和 ACT® 大学入学考试以及允许或需要图形的 Advanced Placement* 和 IB® 考试计算器。学生可以通过选择彩虹般大胆的数学和科学主题颜色来表达自己的个性，例如无限虹膜、完全蓝绿色或正珊瑚色。 SAT* 和 AP* 是美国大学入学考试委员会的注册商标 PSAT/NMSQT* 是美国大学委员会和国家优异奖学金公司的注册商标、耐用且无干扰的工具，将课堂与现实世界联系起来，帮助教师和学生以交互方式探索数学和科学。 TI 的产品和服务经过了公认的第三方研究的严格测试，结果表明图形计算器的有效使用可以提高学生的数学技能和他们对数学的态度。如需更多信息，请访问education.ti.com。

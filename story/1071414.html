为金融机构提供身份验证 API 的 SentiLink 筹集了 7000 万美元 B 轮融资，由 Craft Ventures 领投，a16z 和其他公司参与

2021-08-05 21:21:42

Felicis Ventures、Andreessen Horowitz (a16z) 和 NYCA 也参与了融资，使公司自 2017 年成立以来的总融资额达到 8500 万美元。该公司拒绝透露这笔资金的估值是多少，只是说它是 2019 年由 a16z 领投的 1500 万美元 A 系列的“非常高的倍数”。 Naftali Harris 和 Max Blumenfeld 在获得 Affirm 数据科学家的经验后于 2017 年年中创立了 SentiLink，在那里他们建立了公司的风险职能。在分期贷款提供商任职期间，他们曾遇到一起“特殊”欺诈案件，其中 12 个身份使用相同的姓名和出生日期但 12 个不同的社会安全号码申请贷款。这对夫妇惊讶地发现，尽管不是真人，但所有 12 个身份都有真实的信用报告，信用评分超过 750。那时他们才意识到身份验证的问题有多大，做得有多差，并在Affirm CEO和创始人Max Levchin的加持和投资下创立了SentiLink。位于旧金山的 SentiLink 旨在通过实时 API 帮助银行、贷方和金融机构在应用程序点检测欺诈。具体来说，这些 API 会检测新帐户应用程序的虚假和被盗身份。例如，银行在发行新信用卡之前，会将申请信息发送给 SentiLink。 SentiLink 的模型和技术评估信用卡申请的各种欺诈风险，包括身份盗用和合成欺诈。如果 SentiLink 检测到该申请为高风险，银行将要求提供更多信息或拒绝该申请。据哈里斯称，SentiLink 与 100 多家金融机构合作，其中包括美国前 10 大银行中的 3 家。迄今为止，SentiLink 已验证了数亿份申请，其收入比去年增长了“5 倍”。 Craft Ventures 联合创始人兼普通合伙人大卫·萨克斯 (David Sacks) 表示，SentiLink 的增长轨迹是他见过的“最快的”之一。 “他们对从初创公司到美国主要银行的公司的吸引力令人印象深刻，”他补充道。 “这一切都源于团队对欺诈和身份的深刻理解。通过与 Naftali 和 Max 交谈，我了解到了我什至认为不可能发生的欺诈攻击。”

哈里斯表示，该公司距离 B 轮融资还有几个月的时间才能实现盈利，但认为这个机会“太大而不能缓慢增长”。因此，它选择推迟盈利，以便可以更加“积极”地扩张。由于该领域的公司众多，很难区分是什么让一个公司与其他公司相比与众不同。对于 Harris 来说，SentiLink 方法的最大区别在于它在多大程度上强调“对我们模型中的欺诈和身份的深入理解”。 “我们有一个欺诈调查团队，每天手动审查申请以寻找欺诈，我们在欺诈模型和技术中使用他们的见解和发现，”他告诉 TechCrunch。 “这种深刻的理解对我们来说非常重要，以至于每个星期五整个公司都会花一个小时来审查欺诈案件。” Harris 补充说，SentiLink 专注于“深入”了解欺诈和身份，然后使用技术将这些见解生产化。这些发现包括电话/姓名匹配数据的恶化和揭露“同名”欺诈。 “这种深刻的理解非常重要，以至于 SentiLink 聘请了一个风险分析师团队，他们的全职工作是调查新型欺诈行为并发现欺诈者在做什么，”该公司表示。与许多其他初创公司一样，SentiLink 在 COVID-19 大流行期间看到了业务增长。

“ 各种政府援助计划充斥着欺诈行为。这对整个金融服务产生了连锁效应，成功窃取政府资金的欺诈者试图将其洗入金融系统，”哈里斯说。 “因此，我们一直很忙，尤其是在支票和储蓄账户方面，迄今为止欺诈相对较少。”这家初创公司计划利用其新资本来构建其产品套件并进行一些招聘。如今，它拥有 25 名员工，其中 5 人接受了邀请，预计今年年底员工人数将达到 45-50 人。 “身份验证有很多方面和方法，因此我们计划将我们的产品套件显着扩展到我们开始使用的评分 API 之外，”哈里斯告诉 TechCrunch。 “其中一部分将包括继续对我们的产品团队进行大量投资，其中一部分将涉及与其他公司的合作，还可能包括收购。”例如，该公司计划在 9 月底推出 KYC（“了解您的客户”）解决方案。 Craft 的负责人 Mike Marg 表示，Harris 和 Blumenfeld 在 Affirm 的经历“清楚地表明他们是这方面的专家。” “当创始人对一个外人不了解的大问题有一个赢得的秘密或洞察力时，我们喜欢它，而 SentiLink 就是一个完美的例子，”他说。 “他们的快速增长只能证明他们为客户解决的问题是紧迫而痛苦的。”最近筹集资金的身份验证领域的其他公司包括 Persona 和 Socure。

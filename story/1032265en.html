CTRL-Z, CTRL-Y (or S,F) to undo/redo CTRL-C, CTRL-V to copy and paste between doodles W,R to switch between doodles (or use the menu buttons) TAB to toggle menu Mouse wheel (or e,d) to change brush size RMB to pick up a colour RMB in menu colour palette to select secondary colour (used for patterns) LMB+RMB in menu colour palette to set the background colour

Ctrl-Z、CTRL-Y(或S、F)撤消/重做CTRL-C、CTRL-V在涂鸦W、R之间复制和粘贴的步骤切换涂鸦(或使用菜单按钮)Tab键切换菜单鼠标滚轮(或e、d)更改画笔大小人民币的步骤在菜单调色板中拾取颜色人民币的步骤选择菜单调色板中的次色(用于图案)LMB+人民币设置背景颜色。

Wobblepaint saves data to itself. To start a new wobble cart, type LOAD #WOBBLEPAINT from inside PICO-8 and then save it as something. The data storage is reasonably efficient so you can get around 20~100 doodles to a cart depending on complexity.

Wobblepaint将数据保存到自身。要启动新的摆动购物车，请在Pico-8中键入load#WOBBLEPAINT，然后将其另存为某个内容。数据存储的效率相当高，因此根据复杂性的不同，您可以将大约20到100个涂鸦放到一个购物车中。

To save a gif to desktop, use the gif button to record a second of looping wobble. If you want to record multiple doodles (e.g. for an animation or story), press tab to hide menu, CTRL-8 to start a gif, W,R to flip through the doodles, and then CTRL-9 to save the gif.

要将gif保存到桌面，请使用gif按钮记录一秒钟的循环摆动。如果要录制多个涂鸦(例如动画或故事)，请按Tab键隐藏菜单，按Ctrl-8开始gif，按W、R浏览涂鸦，然后按Ctrl-9保存gif。

Turn off the devkit input in the options menu ("turn off mouse") and use a gamepad:

关闭Options(选项)菜单中的devkit输入("；关闭鼠标"；)，并使用游戏手柄：

LRUD to move the cursor [X] to paint [O] + L/R to undo/redo [O] + U/D to adjust brush size In the menu, [X] and [O] behave the same as LMB,RMB

LRUD移动光标[X]以绘制[O]+L/R以撤消/重做[O]+U/D以调整菜单中的画笔大小，[X]和[O]的行为与LMB、人民币相同。

CTRL-C copies doodles in a text format that can be pasted into code (or bbs posts)

Ctrl-C以文本格式复制涂鸦，可以粘贴到代码(或BBS帖子)中。

Or alternatively, copy the binary data straight out of the spritesheet and use load_library (tab 2) to load all of the doodles into a table.

或者，将二进制数据直接从精灵表中复制出来，并使用LOAD_LIBRARY(选项卡2)将所有涂鸦加载到一个表中。

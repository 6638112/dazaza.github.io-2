How to force yourself to leave the computer using the leave command
如何使用请假命令强迫自己离开计算机

2020-11-21 23:19:14

This post is part of my Today I learned series in which I share all my learnings regarding web development.

这篇文章是我今天学习的系列文章的一部分，在该系列文章中，我分享了有关Web开发的所有知识。

While browsing a collection of shell one-liners, I found out about a command that I haven't seen before: leave.

在浏览一系列shell单行代码时，我发现了一个我之前从未见过的命令：离开。

The command is straightforward. Run it and define an absolute or relative time, and your terminal will remind you to leave it alone and go at a certain time.

该命令很简单。运行它并定义一个绝对或相对时间，您的终端会提醒您不要理会它，并在某个时间去。

# set the alarm for 2:10pm$ leave 1410Alarm set for Fri Nov 20 14:10:00 CET 2020. (pid 3121)# set the alarm in two minutes$ leave +2# set the alarm in two minutes interactively$ leaveWhen do you have to leave? +2Alarm set for Fri Nov 20 09:05:21 CET 2020. (pid 50717)

＃将闹钟设置为2:10 pm$休假1410将闹钟设置为2020年11月20日星期五12:10:00 CET 2020（pid 3121）＃在两分钟内设置闹钟$休假+ 2＃在两分钟内交互式设置闹钟你必须离开？ 2020年11月20日星期五09:05:21 CET设置为+2报警。（pid 50717）

When the time arrives, the command running in the background sends shell notifications and prints new lines telling you to go. And let me tell you the best part: it doesn't do that once; it does that repeatedly every minute after the set alarm time. It will try hard to make you leave!

时间到了，在后台运行的命令会发送shell通知并打印新行，告诉您执行此操作。让我告诉你最好的部分：它一次也做不到；它会在设置的闹钟时间后每分钟重复执行一次。它将努力使您离开！

You can only stop the running leave process from annoying you by logging out of your shell session (or manually stopping the process). 😆

您只能通过退出Shell会话（或手动停止该过程）来阻止正在运行的请假过程烦人。 😆

I love that! Maybe I should define leave 1900 in my .zshrc to force me to stop doing nerdy things late at night.

我喜欢那个！也许我应该在.zshrc中定义请假1900，以迫使我在深夜停止做书呆子的事情。

Edited: For the adventurous folks – Frederic Hemberger pointed out that you could also just shut down the system at a specific time using sudo shutdown -h 12:00 "Feierabend!". But be careful with this command; it could lead to data loss. 😆

编辑：对于喜欢冒险的人– Frederic Hemberger指出，您也可以使用sudo shutdown -h 12:00“ Feierabend！”在特定时间关闭系统。但是请谨慎使用此命令；它可能会导致数据丢失。 😆

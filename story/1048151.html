您应该选择哪种CentOS替代发行版？

2021-02-18 07:20:27

2014年1月，CentOS宣布正式加入Red Hat，同时在新的CentOS理事会的领导下独立于Red Hat Enterprise Linux（RHEL）。

但是，在2020年12月，红帽要求CentOS终止CentOS 8的开发。他们还宣布将支持转移到CentOS替代滚动发行Linux发行版，介于Fedora的上游开发和RHEL的下游开发之间。

CentOS曾经是并且仍然是非常常用的可用于生产环境的Linux服务器发行版。随着Red Hat的发展，CentOS已从一个非常稳定，受社区支持且与RHEL兼容的发行版转变为RHEL的早期滚动发行发行版，即CentOS Stream。

因此，包括我在内的许多用户都在寻找CentOS的替代品。 CentOS替代品应该提供长期支持，并且同等重要的是，它应该非常稳定。

为了响应我们所知的CentOS终止，CentOS创始人Gregory Kurtzer创建了CentOS原始任务的继任者，然后不久，CloudLinux也宣布了新的CentOS克隆。此外，还有其他选择。

因此，让我们看一下可行的CentOS替代方案，并在2021年12月31日截止日期之前开始我们的研究，讨论和决策。

Rocky Linux是CentOS项目的创始人领导的CentOS的分支。它是一个社区企业操作系统，旨在与RHEL 100％兼容。 Rocky Linux的目标是像CentOS以前那样充当下游构建，在上游供应商添加发行版本之后（而不是之前）构建发行版本。初始版本的当前ETA是2021年3月31日。

优点：在CentOS项目的创始人的带领下，迄今为止获得最多的关注和社区支持。

缺点：ETA将于2021年3月31日首次发布，这个新发行版本会继续流行并持续十年吗？

AlmaLinux是CloudLinux创建的最新RHEL克隆。该发行版是另一个免费的操作系统，由CloudLinux每年赞助100万美元，后者已承诺“通过更新AlmaLinux支持将来的RHEL版本。”预计发布日期为2021年第一季度。

优点：由Cloudlinux领导，其发行版被共享主机提供商广泛使用。流行的cPanel＆amp; WHM支持在CentOS，RHEL和CloudLinux OS上进行安装。因此，如果您在CentOS上运行cPanel，您可能会对此感兴趣。 cPanel最近发表了这样的声明：“我们已经承诺通过CloudLinux支持RHEL fork。”

缺点：不是自由软件的先驱。希望他们能在其首页上兑现承诺-“始终免费，始终开源”。 –并且不会分解为一堆付费插件。

Oracle的Linux发行版是免费提供的，他们已经创建了一个转换脚本，可以将您的CentOS Linux服务器切换到Oracle Linux。 Oracle Linux声称与CentOS服务器上的现有应用程序100％兼容。此Linux发行版还使用Oracle生产的Unbreakable Enterprise Kernel（UEK）密切关注RHEL发行周期。

优点：来自Oracle，稳定。具有定制的，经过严格测试的Linux内核。与Oracle的硬件和软件产品（包括大多数数据库应用程序）紧密集成。

缺点：来自Oracle。业内许多人仍然认为Oracle因诱饵和转换策略而臭名昭著。 （例如，Oracle处理Java。

Springdale Linux是另一个基于Red Hat Enterprise Linux的Linux发行版。该分布由高级研究学院和普林斯顿大学开发。这个项目在CentOS或其他项目可用之前就已经开始了。最新版本是8.3，于2020年11月20日发布。

ClearOS Linux基于CentOS和Red Hat Enterprise Linux。设计用于中小型企业。这是一个免费的Linux发行版，社区可以免费使用。

Amazon Linux 2（AL2）也是基于CentOS的RHEL克隆。向Amazon EC2用户免费提供了Amazon Linux 2 AMI。因此，如果您在AWS上拥有CentOS服务器，那么值得一试的是Amazon Linux 2。 AL2为KVM，Microsoft Hyper-V，Oracle VM VirtualBox和VMware ESXi虚拟化平台提供长期支持和多个虚拟机映像。

CentOS 8的正式替代品是CentOS Stream。阅读：从CentOS Linux转换为CentOS Stream。

假设您担心担心长期支持。然后可能是时候掏出你的钱包了。红帽企业版Linux（缩写为RHEL）是红帽为商业市场开发的Linux发行版。红帽企业版Linux 8（代号：Ootpa）基于Fedora 28，上游Linux内核4.18，GCC 8.2，Glibc 2.28，systemd 239，GNOME 3.28以及向Wayland的转换。 RHEL 8已于2019年5月7日正式发布。

优点：Linux Enterprise发行版的鲜奶油。以上所有克隆的来源。

缺点：不是免费的，也不便宜的，这是从CentOS之下拿出地毯的同一家公司获得的。

Debian –超过20年的经验，拥有超过60,000个软件包。 Ubuntu LTS服务器基于Debian SID。 OpenSUSE Leap –专注于创建可用的开源工具。 SUSE Linux Enterprise –提供与Windows和其他平台的互操作性。 Ubuntu Server –特定的服务器版本，非常受商业支持。另请参阅-选择最佳Linux服务器发行版。

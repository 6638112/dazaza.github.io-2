“Facebook取了我的开源项目的名字。”

2020-11-16 03:02:04

Hi, I'm Armin Sebastian, the maintainer of Label Actions, a GitHub app that Facebook has been using for the React Native repository.

大家好，我是Armin Sebastian，Label Actions的维护者，这是一款GitHub应用程序，Facebook一直在使用它作为Reaction Native存储库。

It has come to my attention that Héctor Ramos, one of your collegues at Facebook, has reimplemented the app with almost the same features and configuration syntax using GitHub actions, and shared it on GitHub using the same project name.

我注意到，你在Facebook的一位同事赫克托·拉莫斯(Héctor Ramos)使用GitHub的操作，用几乎相同的功能和配置语法重新实现了这款应用，并使用相同的项目名称在GitHub上分享了它。

The action has been published on the GitHub Marketplace using the same name as my open source project, preventing me from publishing my project as a GitHub action using the project name.

该操作已使用与我的开源项目相同的名称发布在GitHub Marketplace上，阻止我使用项目名称将我的项目发布为GitHub操作。

Despite your bot taking heavy inspiration from my project, and despite taking the name of my project, you haven't credited me or the original project in any way, possibly not complying with the MIT license.

尽管你的机器人从我的项目中获得了很大的灵感，尽管你用了我的项目的名字，但你没有以任何方式归功于我或最初的项目，可能没有遵守麻省理工学院的许可证。

The GitHub app is part of a set of open source projects that are funded by contributors. By hijacking the name of my project, and taking it's name on GitHub Marketplace, you are making it harder for me to receive donations for my open source work, funds that help me pay rent and buy groceries.

GitHub应用程序是由贡献者资助的一系列开源项目的一部分。通过劫持我的项目名称，并在GitHub Marketplace上使用它的名称，你让我更难为我的开源工作获得捐款，这些资金可以帮助我支付租金和购买食品杂货。

In the last month this is the second time that Facebook has caused me grief that borders on illegality. Recently I have signed up for Wit.ai by creating a Facebook account. The account was automatically blocked after a couple of days for no reason, and I have been asked to provide additional personal data, after which the account has remained banned.

上个月，这是Facebook第二次给我带来近乎违法的悲痛。最近，我注册了一个Facebook账号，注册了Wit.ai。该账户在几天后被无缘无故地自动屏蔽，我被要求提供额外的个人数据，之后该账户仍处于被屏蔽状态。

Facebook has a photo of my face that they have requested during review, my legal name and date of birth, my verified email address and phone number, and I cannot exercise any control over this data, only download it, which is obviously illegal under GDPR.

Facebook上有他们在审查期间要求的我的照片，我的法定姓名和出生日期，我的认证电子邮件地址和电话号码，我不能对这些数据进行任何控制，只能下载它，这在GDPR下显然是非法的。

I have reached out to the Wit.ai team at Facebook, but so far my request for help has been ignored.

我已经联系了Facebook的Wit.ai团队，但到目前为止，我的帮助请求一直被忽视。

Please find a different name for the action that has been published on GitHub Marketplace, and change the name of the repository.

请为已在GitHub Marketplace上发布的操作找到其他名称，并更改存储库的名称。

Please also reach out to your collegues at Wit.ai, or the people who can override automated blocks at Facebook, so that I receive due process for my Facebook account, and avoid having to file a GDPR complaint regarding the loss of control over my personal data.

也请联系你在Wit.ai的同事，或者可以覆盖Facebook自动屏蔽的人，这样我的Facebook账户就可以得到适当的程序，而不必就失去对我的个人数据的控制向GDPR投诉。

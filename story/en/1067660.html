Jump to navigation Jump to search Wikidata is a free, collaborative, multilingual, secondary database, collecting structured data to provide support for Wikipedia, Wikimedia Commons, the other wikis of the Wikimedia movement, and to anyone in the world.

跳转到导航跳转以搜索Wikidata是一款免费，协作，多语言，辅助数据库，收集结构化数据，为维基百科，维基梅德米亚，其他维基运动的Wikis提供支持，以及世界上任何人。

Free. The data in Wikidata is published under the Creative Commons Public Domain Dedication 1.0, allowing the reuse of the data in many different scenarios. You can copy, modify, distribute and perform the data, even for commercial purposes, without asking for permission.

自由。 Wikidata中的数据在Creative Commons公共领域Dementication 1.0下发布，允许在许多不同场景中重用数据。即使是商业目的，您也可以复制，修改，分发和执行数据，而不要求许可。

Collaborative. Data is entered and maintained by Wikidata editors, who decide on the rules of content creation and management. Automated bots also enter data into Wikidata.

合作。由Wikidata编辑器输入和维护数据，他决定内容创建和管理规则。自动机器人还将数据输入Wikidata。

Multilingual. Editing, consuming, browsing, and reusing the data is fully multilingual. Data entered in any language is immediately available in all other languages. Editing in any language is possible and encouraged.

多种语言。编辑，消费，浏览和重用数据是完全多语言的。以任何语言输入的数据立即以所有其他语言提供。以任何语言编辑是可能的，并鼓励。

A secondary database. Wikidata records not just statements, but also their sources, and connections to other databases. This reflects the diversity of knowledge available and supports the notion of verifiability.

辅助数据库。 Wikidata不仅记录了陈述，还记录了他们的来源和与其他数据库的连接。这反映了可用知识的多样性，并支持可验证的概念。

Collecting structured data. Imposing a high degree of structured organization allows for easy reuse of data by Wikimedia projects and third parties, and enables computers to process and “understand” it.

收集结构化数据。强加高度的结构化组织允许维基媒体项目和第三方轻松重用数据，并使计算机能够处理和“了解”它。

Support for Wikimedia wikis. Wikidata assists Wikipedia with more easily maintainable information boxes and links to other languages, thus reducing editing workload while improving quality. Updates in one language are made available to all other languages.

支持Wikimedia Wikis。 Wikidata有助于维基百科，具有更轻松的可维护信息框和链接到其他语言，从而减少编辑工作负载，同时提高质量。所有其他语言都可以使用一种语言更新。

Anyone in the world. Anyone can use Wikidata for any number of different ways by using its application programming interface.

世界上任何人。任何人都可以使用其应用程序编程接口使用Wikidata以任何数量的不同方式使用Wikidata。

Wikidata is a central storage repository that can be accessed by others, such as the wikis maintained by the Wikimedia Foundation. Content loaded dynamically from Wikidata does not need to be maintained in each individual wiki project. For example, statistics, dates, locations and other common data can be centralized in Wikidata.

Wikidata是一个中央存储库，可以由其他人访问，例如由Wikimedia Foundation维护的Wiki。每个Wiki项目不需要维护从Wikidata动态加载的内容。例如，统计信息，日期，位置和其他常用数据可以集中在Wikidata中。

The Wikidata repository consists mainly of items, each one having a label, a description and any number of aliases. Items are uniquely identified by a Q followed by a number, such as Douglas Adams .

Wikidata存储库主要由项目组成，每个项目都具有标签，描述和任意数量的别名。项目是唯一标识的，后跟一个数字，如道格拉斯亚当斯。

Statements describe detailed characteristics of an Item and consist of a property and a value. Properties in Wikidata have a P followed by a number, such as with educated at .

语句描述了项目的详细特征，并由属性和值组成。 Wikidata中的属性具有P，然后是一个数字，例如受过教育的。

For a person, you can add a property to specify where they were educated, by specifying a value for a school. For buildings, you can assign geographic coordinates properties by specifying longitude and latitude values. Properties can also link to external databases. A property that links an item to an external database, such as an authority control database used by libraries and archives, is called an identifier. Special Sitelinks connect an item to corresponding content on client wikis, such as Wikipedia, Wikibooks or Wikiquote.

对于一个人来说，您可以通过指定学校的价值来添加一个属性来指定他们受过教育的地方。对于建筑物，您可以通过指定经度和纬度值来分配地理坐标属性。属性也可以链接到外部数据库。将项目链接到外部数据库（例如库和存档使用的权限控制数据库）的属性称为标识符。特殊的Sitelinks将物品连接到客户端Wiki上的相应内容，例如维基百科，Wikibook或Wikipote。

All this information can be displayed in any language, even if the data originated in a different language. When accessing these values, client wikis will show the most up-to-date data.

即使数据源自不同的语言，也可以以任何语言显示所有这些信息。访问这些值时，客户端Wiki将显示最新的数据。

There are a number of ways to access Wikidata using built-in tools, external tools, or programming interfaces.

使用内置工具，外部工具或编程接口访问Wikidata的许多方法。

Wikidata Query and Reasonator are some of the popular tools to search for and examine Wikidata items. The tools page has an extensive list of interesting projects to explore.

Wikidata查询和推理是搜索和检查Wikidata项目的一些流行工具。 “工具”页面具有广泛的有趣项目列表探索。

Client wikis can access data for their pages using a Lua Scribunto interface. You can retrieve all data independently using the Wikidata API.

客户端Wiki可以使用Lua Scribunto接口访问其页面的数据。您可以使用Wikidata API独立检索所有数据。

The Wikidata tours designed for new users are the best place to learn more about Wikidata.

为新用户设计的Wikidata Tours是了解有关Wikidata更多信息的最佳地点。

Go ahead and start editing. Editing is the best way to learn about the structure and concepts of Wikidata. If you would like to gain understanding of Wikidata's concepts upfront, you may want to have a look at the help pages. If you have questions, please feel free to drop them in the project chat or contact the development team.

继续编辑并开始编辑。编辑是了解Wikidata的结构和概念的最佳方式。如果您想了解Wikidata＆＃39;概念前期，您可能希望查看帮助页面。如果您有疑问，请随时将其放入项目聊天或联系开发团队。

Wikidata is an ongoing project that is under active development. More data types as well as extensions will be available in the future. You can find more information about Wikidata and its ongoing development on the Wikidata page on Meta. Subscribe to the Wikidata mailing list to receive up-to-date information about the development and to participate in discussions about the future of the project.

Wikidata是一个正在进行的积极发展的持续项目。将来将提供更多数据类型以及扩展。您可以在META上找到有关Wikidata及其在Wikidata页面上的持续开发的更多信息。订阅Wikidata邮件列表，以接收有关开发的最新信息，并参与项目未来的讨论。

The Pac-Man projects were developed for UC Berkeley's introductory artificial intelligence course, CS 188. They apply an array of AI techniques to playing Pac-Man. However, these projects don't focus on building AI for video games. Instead, they teach foundational AI concepts, such as informed state-space search, probabilistic inference, and reinforcement learning. These concepts underly real-world application areas such as natural language processing, computer vision, and robotics.

PAC-MAN项目是为UC Berkeley＆＃39; S介绍性人工智能课程，CS 188。他们应用了一系列AI技术来玩Pac-Man。然而，这些项目不专注于为视频游戏构建AI。相反，他们教授基础AI概念，例如知情的状态空间搜索，概率推断和加强学习。这些概念依据自然语言处理，计算机视觉和机器人等现实世界应用领域。

We designed these projects with three goals in mind. The projects allow students to visualize the results of the techniques they implement. They also contain code examples and clear directions, but do not force students to wade through undue amounts of scaffolding. Finally, Pac-Man provides a challenging problem environment that demands creative solutions; real-world AI problems are challenging, and Pac-Man is too.

我们设计了这些项目，以三大目标为设计。该项目允许学生可视化他们实施的技术的结果。它们还包含代码示例和清晰的方向，但不要强制学生通过不必要的脚手架削弱。最后，Pac-Man提供了需要创造性解决方案的具有挑战性的问题环境;现实世界的AI问题是挑战性，PAC-MAN也是如此。

In our course, these projects have boosted enrollment, teaching reviews, and student engagement. The projects have been field-tested, refined, and debugged over multiple semesters at Berkeley. We are now happy to release them to other universities for educational use.

在我们的课程中，这些项目促进了入学，教学评论和学生参与。该项目已经经过现场测试，精致和调试伯克利的多个学期。我们现在很乐意将它们释放到其他教育用途的其他大学。

Instructor's Guide for Projects Using the Projects on the edX Platform Contact Information UNIX/Python Tutorial This short UNIX/Python tutorial introduces students to the Python programming language and the UNIX environment.

教练＆＃39;使用EDX平台的项目指南，使用EDX平台联系信息UNIX / Python教程此简短的UNIX / Python教程将学生推出到了 Python编程语言和UNIX环境。

Search Students implement depth-first, breadth-first, uniform cost, and A* search algorithms. These algorithms are used to solve navigation and traveling salesman problems in the Pacman world.

搜索学生利用深度第一，广度，统一的成本和*搜索算法。 这些算法用于解决导航和旅行推销员问题 Pacman World。

Multi-Agent Search Classic Pacman is modeled as both an adversarial and a stochastic search problem. Students implement multiagent minimax and expectimax algorithms, as well as designing evaluation functions.

多代理搜索经典Pacman被建模为普通话和随机搜索问题。 学生实现多元素Minimax和Ippectimax算法，以及 设计评估功能。

Reinforcement Learning Students implement model-based and model-free reinforcement learning algorithms, applied to the AIMA textbook's Gridworld, Pacman, and a simulated crawling robot.

加强学习学生实现基于模型和无模型的加强学习算法， 适用于Aima教科书＆＃39; S Gridworld，Pacman和模拟爬行机器人。

Ghostbusters Probabilistic inference in a hidden Markov model tracks the movement of hidden ghosts in the Pacman world. Students implement exact inference using the forward algorithm and approximate inference via particle filters.

隐藏马尔可夫模型中的幽灵票是概率推断追踪隐藏的运动 Pacman世界的鬼魂。学生使用前进实施精确推断 算法和粒子滤波器的近似推断。

Classification Students implement standard machine learning classification algorithms using Naive Bayes, Perceptron, and MIRA models to classify digits. Students extend this by implementing a behavioral cloning Pacman agent.

分类学生使用标准机器学习分类算法使用 天真的贝叶斯，Perceptron和Mira模型来分类数字。学生通过 实施行为克隆Pacmman代理。

Contest: Pacman Capture the Flag Students create strategies for a team of two agents to play a multi-player capture-the-flag variant of Pacman.

比赛：Pacman捕获旗帜学生为两名代理团队创造战略，以发挥多人 捕获 - Pacman的旗帜变体。

The Pac-Man projects are written in pure Python 2.7 and do not depend on any packages external to a standard Python distribution.

PAC-MAN项目以纯Python 2.7编写，不依赖于标准Python分布外部的任何包。

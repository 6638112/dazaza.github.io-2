Facebook 宣布 Passthrough API，它允许 Oculus Quest 2 开发人员使用通过传感器收集的真实世界图像来创建混合现实应用程序

2021-07-24 08:09:19

Facebook 宣布了一个新的 API，可以让开发人员将来自 Quest 2 传感器的视频整合到他们的游戏和应用程序中，从而创造一种混合现实体验。借助 Passthrough API，开发人员不仅能够将头显传感器的黑白​​图像混合到他们的体验中，而且还能够自定义它最终对玩家的看法、应用效果和甚至让真实世界出现在特定的表面上。 Facebook 的公告包括 API 如何通过允许集成现实键盘和办公桌来提高软件生产力的示例，以及游戏如何包含您的真实家具以供敌人躲在后面。也很容易想象将您的真实环境与虚拟世界混合在一起的其他有趣体验 - 您在自己的房子里解决门户式的谜题，或者能够看到您的墙壁被虚拟油漆覆盖？当被问及与第一代 Quest 的兼容性时，Facebook 告诉 UploadVR，该 API 目前仅适用于 Quest 2。在公告中，Facebook 表示 API 的图像处理是在设备上完成的，使用它的应用程序将无法查看或存储来自 Quest 2 传感器的图像。 Facebook 正在将 API 的实验版本添加到其即将推出的开发人员工具中，它将首先提供给使用 Unity 构建软件的程序员。 Facebook 承诺该 API 将在未来获得对其他开发平台的支持。 Quest、Quest 2 和 Rift S 耳机已经有一个 Passthrough 技术版本，可以让用户快速了解现实世界中发生的事情。 Facebook 还允许你将 Passthrough 设置为你的虚拟环境，为你提供一个真实环境的版本（尽管是黑白的），你可以在其中导航 Oculus UI。它甚至在关于其无限办公室功能的视频中展示了这种能力。长期以来，将您的真实环境集成到虚拟世界的能力一直是带有集成摄像头的耳机的更令人兴奋的承诺之一。微软甚至在将其 VR 平台命名为 Windows Mixed Reality 时引用了这个想法（无论实际交付了多少）。然而，Oculus 的 API 的消息并不意味着您今天就可以拿起耳机并拥有混合现实体验。 Facebook 在其公告中表示，它希望让开发人员在“今年晚些时候”向玩家提供使用 Passthrough 的软件。尽管如此，这个 API 对开发人员的可用性还是令人兴奋的，因为它应该允许在 Oculus 平台上获得新的体验类型。

Fun with INTERACTIVE Unix 3.0 – Fun with virtualization
交互式Unix 3.0的乐趣–虚拟化的乐趣

2020-11-30 00:09:43

I’ve been hunting for a complete set of INTERACTIVE UNIX System for quite some time. While I had the “basic set” of it, the real stuff Visix Looking Glass graphical environment was nowhere to be found. Recently I got my hands on a box containing a massive set of 5.25″ floppy disks. Without further delay this is how the famous 386 UNIX GUI looks like: The os has a long and convoluted history. It started as PC/IX for the IBM XT created by Interactive Systems Corporation. Later updated to 386/ix and renamed as INTERACTIVE UNIX System V/386. The company was acquired by Kodak and the OS was later sold to Sun Microsystems.

我一直在寻找一套完整的INTERACTIVE UNIX System。当我拥有“基本设置”时，找不到真正的Visix Looking Glass图形环境。最近，我把手放在装有大量5.25英寸软盘的盒子上。不用再拖延，这就是著名的386 UNIX GUI的样子：os有着悠久而复杂的历史。它最初是由Interactive Systems Corporation创建的IBM XT的PC / IX。后来更新为386 / ix，并重命名为INTERACTIVE UNIX System V / 386。该公司被柯达（Kodak）收购，其操作系统后来卖给了Sun Microsystems。

Sun has reportedly used the OS to help port SunOS/Solaris to x86 platform. However they also sold and supported it as a stand alone product. The system was widely used as part of Reuters Terminal and other embedded applications. It briefly survived Oracle acquisition making the os Oracle INTERACTIVE UNIX? This article specifically covers version 3.0 released by ISC around 1991. The installation is pretty straightforward except for swapping 50 disk images and later configuration of NIC, TCP/IP and VGA/X11. The floppy images are available here. I have spent quite a lot of time to get TCP/IP working and VGA at half decent resolution. Fortunately thanks to 86Box you can enjoy a fully working OS in a “high-res” (1024×768) mode, 256 colors and working TCP/IP. You can even pretend to browse the web using early version of Mosaic and WRP:

据报道，Sun使用该操作系统来帮助将SunOS / Solaris移植到x86平台。但是，他们也作为独立产品销售和支持它。该系统被广泛用作路透社终端和其他嵌入式应用程序的一部分。它短暂地在Oracle收购中幸存下来，从而使os Oracle INTERACTIVE UNIX？本文专门介绍了ISC在1991年左右发布的3.0版。安装过程非常简单，除了交换50个磁盘映像以及以后配置NIC，TCP / IP和VGA / X11之外。软盘映像位于此处。我花了很多时间使TCP / IP和VGA的分辨率达到一半。幸运的是，得益于86Box，您可以在“高分辨率”（1024×768）模式，256色和有效的TCP / IP下享受可以正常运行的操作系统。您甚至可以假装使用早期版本的Mosaic和WRP浏览网络：

On a text console side, the OS has virtual consoles switchable via SYSRQ + F key. Console is on F8.

在文本控制台方面，操作系统具有可通过SYSRQ + F键切换的虚拟控制台。控制台在F8上。

One should probably appreciate that PC had such a cool Unix version before Linux was even born. Unfortunately this stuff was all prohibitively expensive and mortals could not afford it to run on their 386s.

人们可能应该赞赏PC甚至在Linux诞生之前就拥有如此出色的Unix版本。不幸的是，这些东西实在是太贵了，凡人无法承受运行在386上的东西。

You can download 86Box version here. Make sure to look at readme for some last minute updates. Especially around configuring TCP/IP and Looking Glass licensing. There also is a VirtualBox OVA, however it only works in 800×600 and no networking/tcpip. Additional software can be downloaded from funet ftp.

您可以在此处下载86Box版本。确保查看自述文件以获取最新的一些更新。特别是在配置TCP / IP和Looking Glass许可方面。还有一个VirtualBox OVA，但是它只能在800×600的分辨率下工作，并且不能联网。可以从funet ftp下载其他软件。

Also you may be interested in more in-depth look at the VP/ix DOS hypervisor included with the os.

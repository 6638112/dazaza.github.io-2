Or are you just looking for file format information? As with all living websites, this one is perpetually under construction, so a lot of the focus has been on editors wishing to participate; but if you're coming here as an end-user looking for information on some files, disks, tapes, or other data you've stumbled upon, there may be information of use. You can try navigating the hierarchy of data types at File Formats (which, despite the name, includes a bunch of other things like physical media formats). Or, if you found a file and don't know what it is, but at least it's got a file extension (those letters at the end of the filename after a dot), you might try browsing the file formats by extension category. Good luck!

还是您只是在寻找文件格式信息？与所有正在运行的网站一样，这个网站一直在建设中，因此很多焦点都集中在希望参与的编辑上。但是，如果您以最终用户的身份来这里寻找有关您偶然发现的某些文件，磁盘，磁带或其他数据的信息，则可能会有使用信息。您可以尝试在“文件格式”处导航数据类型的层次结构（尽管有名称，但还包括诸如物理媒体格式之类的许多其他内容）。或者，如果找到文件但不知道它是什么，但至少具有文件扩展名（点名后的文件名末尾的那些字母），则可以尝试按扩展名类别浏览文件格式。祝好运！

Quiver: A modern commutative diagram editor
Quiver：现代的交换图编辑器

2020-11-26 03:56:41

quiver is a modern, graphical editor for commutative and pasting diagrams, capable ofrendering high-quality diagrams for screen viewing, and exporting to LaTeX via tikz-cd.

quiver是用于交换图和粘贴图的现代图形编辑器，能够渲染高质量的图以用于屏幕查看，并可以通过tikz-cd导出到LaTeX。

Creating and modifying diagrams with quiver is orders of magnitude faster than writing theequivalent LaTeX by hand and, with a little experience, competes with pen-and-paper.

用颤振器创建和修改图要比手工编写等效的LaTeX快几个数量级，并且由于经验不足，可以与笔和纸竞争。

quiver features an efficient, intuitive interface for creating complex commutative diagrams andpasting diagrams. It's easy to draw diagrams involving pullbacks and pushouts,

quiver具有高效，直观的界面，可用于创建复杂的交换图和粘贴图。绘制涉及回撤和推出的图很容易，

Object placement is based on a flexible grid that resizes according to the size of the labels.

对象放置基于可根据标签大小调整大小的灵活网格。

quiver is intended to look good for screenshots, as well as to export LaTeX that looks as closeas possible to the original diagram.

quiver的目的是使屏幕截图看起来更好，并导出看起来与原始图尽可能近的LaTeX。

Diagrams may be created and modified using either the mouse, by clicking and dragging, or using the keyboard, with a complete set of keyboard shortcuts for performing any action.

可以使用鼠标，单击和拖动，或者使用键盘以及用于执行任何操作的完整的键盘快捷键集来创建和修改图。

When you export diagrams to LaTeX, quiver will embed a link to the diagram, which will allow youto return to it later if you decide it needs to be modified, or to share it with others.

当您将图表导出到LaTeX时，颤振器将嵌入到该图表的链接，如果您决定需要对其进行修改或与他人共享，则它会允许您稍后返回到该图表。

Support for custom macro definitions: simply paste a URL corresponding to the file containing your \newcommands.

支持自定义宏定义：只需粘贴与包含\ newcommands的文件相对应的URL。

Make sure you have installed yarn and have a version ofmake that supports .ONESHELL (e.g. GNU Make 3.82).

确保已安装yarn并具有支持.ONESHELL的makemake版本（例如GNU Make 3.82）。

If you have any problems building quiver, open an issue detailing the problem and I'll try tohelp.

如果您在构建箭袋时遇到任何问题，请打开一个问题详细说明的问题，我将尽力提供帮助。
